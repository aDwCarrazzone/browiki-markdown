# Tempestade de Flechas

|  |  |
| --- | --- |
| **Tempestade de Flechas Tempestade de Flechas** | |
| Tempestade de Flechas | |
| **Arrow Storm / Tormenta de Flechas** | |
| **ID** | RA\_ARROWSTORM (2233) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 4) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,3 + 2 seg. |
| **Recarga** | 3,2 segundos |
| **Alvo** | Oponente |
| **Alcance** | ([Olhos de Águia](Olhos de Águia.md "Olhos de Águia") + 10) células |
| **Área** | 5x5 a 9x9 células |
| **Munição** | [Over Arrow](Munições.md "Munições") 5 Flechas |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - [Disparo Certeiro](Disparo Certeiro.md "Disparo Certeiro") [Disparo Certeiro](Disparo Certeiro.md "Disparo Certeiro") Nv. 5 | |

Atira flechas no alvo, causando dano físico a distância nos oponentes ao redor.

O dano aumenta se o usuário estiver em estado de [![Disparo Selvagem](images/6/6b/Disparo_Selvagem.png)](Disparo Selvagem.md "Disparo Selvagem") [Disparo Selvagem](Disparo Selvagem.md "Disparo Selvagem").

| 2 | 560% | 700% | 28 |
| 3 | 740% | 950% | 32 |
| 4 | 920% | 1.200% | 36 |
| 5 | 1.100% | 1.450% | 40 |
| 6 | 1.280% | 1.700% | 7x7 células | 44 |

| 8 | 1.640% | 2.200% | 52 |

| 10 | 2.000% | 2.700% | 9x9 células | 60 |

### Fórmula

`Dano = [Dano base × (Nv. de base ÷ 100)]%`
