# Concentrar

|  |  |
| --- | --- |
| **Concentrar Concentrar** | |
| Concentrar | |
| **Improve Concentration / Mejorar Concentración** | |
| **ID** | AC\_CONCENTRATION (45) |
| **Tipo** | Suporte |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 5) |
| **Duração** | 40 + (Nv. da hab. × 20) segundos |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Ícone** | I Concentração |
| **Pré-requisitos** | |
| - [Olhos de Águia](Olhos de Águia.md "Olhos de Águia") [Olhos de Águia](Olhos de Águia.md "Olhos de Águia") Nv. 1 - [Olhos de Coruja](Olhos de Coruja.md "Olhos de Coruja") [Olhos de Coruja](Olhos de Coruja.md "Olhos de Coruja") Nv. 3 | |

Aumenta a [DES](DES.md "DES") e a [AGI](AGI.md "AGI").

Revela oponentes em [esconderijo](Esconderijo.md "Esconderijo") ao redor do personagem.
