# Explosão Espiritual

|  |  |
| --- | --- |
| **Explosão Espiritual Explosão Espiritual** | |
| Explosão Espiritual | |
| **Rampage Blast / Ráfaga de Explosiones** | |
| **ID** | SR\_RAMPAGEBLASTER (2332) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 100 |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 10 segundos |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Requisito** | 3 Esferas espirituais e em [Fúria Interior](Fúria Interior.md "Fúria Interior") |
| **Pré-requisitos** | |
| - [Impacto Sísmico](Impacto Sísmico.md "Impacto Sísmico") [Impacto Sísmico](Impacto Sísmico.md "Impacto Sísmico") Nv. 2 | |

Só pode ser usada se estiver em [Fúria Interior](Fúria Interior.md "Fúria Interior").

Causa dano físico à distância seu ao redor.

Oponentes [*impactados*](Impacto Sísmico.md "Impacto Sísmico") recebem dano extra.

Custa 3 Esferas, mas em estado de [Fervor Espiritual](Punho Labareda.md#Fervor_Espiritual "Punho Labareda"), elas não são consumidas.

| Nv. | [ATQ](ATQ.md "ATQ") | |
| --- | --- | --- |
| Normal | Impactado |
| 1 | 350% | 550% |
| 2 | 700% | 1.100% |
| 3 | 1.050% | 1.650% |
| 4 | 1.400% | 2.200% |
| 5 | 1.750% | 2.750% |

### Fórmula

`Normal = {[Dano base + (Nv. de Fúria × 200)] × (Nv. de base ÷ 100)}%`

`Impactado = {[Dano base + (Nv. de Fúria × 300)] × (Nv. de base ÷ 100)}%`
