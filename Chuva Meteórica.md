# Chuva Meteórica

|  |  |
| --- | --- |
| **NPC Skill Icon Chuva Meteórica** | |
| Chuva Meteórica | |
| **ID:** | NPC\_RAINOFMETEOR (769) |
| **Níveis:** | 10 |
| **Exclusiva:** | Sim |
| **Propriedade:** | [Fogo](Monstros.md#Propriedade "Monstros") |

Invoca meteoros para atingir oponentes na área.

Um único meteoro pode infligir de 3 a 7 golpes.

Causa dano mágico de propriedade [Fogo](Fogo.md "Fogo") equivalente a 500% do [ATQM](ATQM.md "ATQM").

| Nv. | Meteoros | Golpes | Área |
| --- | --- | --- | --- |
| 1 | 1 Meteoro | 3 | 3x3 células |
| 2 | 4 |
| 3 | 5 |
| 4 | 6 |
| 5 | 7 |
| 6 | 8 Meteoros |
| 7 | 5x5 células |
| 8 | 7x7 células |
| 9 | 9x9 células |
| 10 | 11x11 células |

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Chuva Meteórica](http://www.divine-pride.net/database/skill/769)*
- O padrão da habilidade é determinada pela [instância](Instância.md "Instância").
  - Padrão em **×** = Atinge oponentes na diagonal.
  - Padrão em **+** = Atinge oponentes em linha reta.
