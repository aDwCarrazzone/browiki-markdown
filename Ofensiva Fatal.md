# Ofensiva Fatal

|  |  |
| --- | --- |
| **Ofensiva Fatal Ofensiva Fatal** | |
| Ofensiva Fatal | |
| **Fatal Menace / Amenaza Mortal** | |
| **ID** | SC\_FATALMENACE (2284) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 6 + (Nv. da habilidade × 4) |
| **Pós-conjuração** | 0,5 segundos |
| **Alvo** | Oponente |
| **Alcance** | 2 células |
| **Área** | 3x3 a 5x5 células |
| **Pré-requisitos** | |
| - [Rapto](Rapto.md "Rapto") [Rapto](Rapto.md "Rapto") Nv. 5 | |

Ameaça um alvo, afetando oponentes ao redor dele.

Causa dano físico corpo a corpo.

Aumentar o nível da habilidade reduz a penalidade da [Precisão](Precisão.md "Precisão") do ataque.

Se usada com adagas, o dano é dobrado.

O dano aumenta se [Ameaça](Ofensiva Abissal.md "Ofensiva Abissal") estiver ativo.

| --- | --- | --- | --- | --- | --- |
| Normal | [Ameaça](Ofensiva Abissal.md "Ofensiva Abissal") |
| 1 | 120% | 150% | -30 | 3x3 células | 10 |
| 2 | 240% | 300% | -25 | 14 |
| 3 | 360% | 450% | -20 | 18 |
| 4 | 480% | 600% | -15 | 22 |
| 5 | 600% | 750% | -10 | 26 |
| 6 | 720% | 900% | 0 | 5x5 células | 30 |
| 7 | 840% | 1.050% | +5 | 34 |
| 8 | 960% | 1.200% | +10 | 38 |
| 9 | 1.080% | 1.350% | +15 | 42 |
| 10 | 1.200% | 1.500% | +20 | 46 |

### Fórmula

Normal:  
`Dano = {[Dano base + (AGI × 2)] × (Nv. de base ÷ 100)}%`

Ameaça:  
`Dano = {[Dano base + (AGI × 3)] × (Nv. de base ÷ 100)}%`
