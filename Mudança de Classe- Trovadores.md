# Mudança de Classe: Trovadores

| JQ ico Mudança de Classe: Trovadores | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 99 |
| **Nv. de classe** | 70 |
| **Tipo** | Classe |
| **Classe** | [Menestrel](Menestrel.md "Menestrel") |
| **Caça** | 33 [Ursinhos](http://www.divine-pride.net/database/monster/1622/) |
| **[Zeny](Zeny.md "Zeny")** | 5.000 |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 2795.png 1 [Anel de Maçã Verde](http://www.divine-pride.net/database/item/2795/)  5751.png 1 [Inspiração do Artista](http://www.divine-pride.net/database/item/5751/) |
| **Recompensa** | Muda a classe para [Trovador](Trovador.md "Trovador") |

1. Encontre um Bardo (alberta 196,133)

   :   ![MdCTr01](images/d/df/MdCTr01.png)
2. Fale com a Mulher Bondosa (prontera 141,97)

   :   ![MdCTr02](images/8/8b/MdCTr02.png)
3. Localize o Homem de Óculos (prontera 140,331)

   :   ![MdCTr03](images/9/99/MdCTr03.png)
4. Converse com a Mulher (prontera 146,218)

   :   ![MdCTr04](images/3/37/MdCTr04.png)
5. Retorne ao Bardo em [Alberta](Alberta.md "Alberta") (alberta 196,133)
6. No [aeroplano](Aeroplano.md "Aeroplano") doméstico, fale com Karian (airplane 222,66). Ele teleportará você para [Hugel](Hugel.md "Hugel").

   :   ![MdCTr0BC](images/7/73/MdCTr0BC.png)
7. Fale com Karian (hu\_in01 267,5)

   :   ![MdCTr05a](images/6/64/MdCTr05a.png)
8. Converse mais de uma vez com a Pessoa Surda (hu\_in01 361,103) e fique selecionando: *Continuar escutando*.

   :   ![MdCTr06a](images/d/d0/MdCTr06a.png)
9. Volte para o hotel em [Hugel](Hugel.md "Hugel") para falar com Karian (hu\_in01 267,5)
   - *Dica:* para se recuperar do HP perdido conversando com a Pessoa Surda, ao entrar novamente no Hotel, converse com a [NPC](NPC.md "NPC") Estalagem e pague 5.000z para Descansar, você recuperará seu [HP](HP.md "HP") e [SP](SP.md "SP") e já estará na frente do Karian novamente.
10. Entre na [Torre de Thanatos](Torre de Thanatos.md "Torre de Thanatos") pagando a entrada com o Gerente da Torre (hu\_fild01 140,157)

    :   ![GerentedaTorre](images/b/b9/GerentedaTorre.png)
11. Aproxime-se do portal para abrir um diálogo e receber 1 [Recibo de Loja](http://www.divine-pride.net/database/item/6271/).

    :   ![MdCTr07a](images/e/ed/MdCTr07a.png)
12. Fale com o Sr. Click (yuno 143,170)

    :   ![MdCTr05](images/6/65/MdCTr05.png)

    - Sua tarefa é eliminar 33 [Ursinhos](http://www.divine-pride.net/database/monster/1622/).
    - Você receberá 1 [Foto de Mello Dias](http://www.divine-pride.net/database/item/6270/).
13. Caminhe em volta da Garotinha (lighthalzen 162,126)

    :   ![MdCTr06](images/8/83/MdCTr06.png)
14. Entre na Caverna de Gelo e feche os diálogos (ra\_fild01 233,329)

    :   ![MdCTr08](images/7/7a/MdCTr08.png)

    - Seu personagem será teleportado para [Rachel](Rachel.md "Rachel").
    - Se tiver alguém nas coordenadas, é provável que o teleporte não ocorra.
15. Converse com a Mulher Velha (ra\_in01 358,130)

    :   ![MdCTr07](images/5/52/MdCTr07.png)
16. Leia a Placa de Aviso (ve\_fild07 131,132)

    :   ![MdCTr01avi](images/9/97/MdCTr01avi.png)

    - Selecione:
      - Pegue o caiaque
      - Jogar a água para fora
    - Seu personagem será teleportado para [Comodo](Comodo.md "Comodo").
17. Procure pelo Mestre Kayak (comodo 192,119)

    :   ![MdCTr09](images/d/db/MdCTr09.png)
18. Fale com a Churrasqueira (comodo 159,314)

    :   ![MdCTr010](images/2/27/MdCTr010.png)
19. Agora você deve se aproximar das coordenadas abaixo para ativar um diálogo e seguir Karian.
    1. Escadarias da Guilda dos Mercenários (moc\_fild16 205,231)

       :   ![MdCTr011](images/3/37/MdCTr011.png)
    2. Entrada da Torre do Relógio (aldebaran 142,128)

       :   ![MdCTr012](images/1/15/MdCTr012.png)
    3. Sul de [Prontera](Prontera.md "Prontera") (prontera 157,39)

       :   ![MdCTr013](images/9/97/MdCTr013.png)
20. Karian aparecerá para teleportar você para uma sala. Lá dentro, fale com o [NPC](NPC.md "NPC") para mudar de classe e receber 1 [Anel de Maçã Verde](http://www.divine-pride.net/database/item/2795/) e 1 [Inspiração do Artista](http://www.divine-pride.net/database/item/5751/).

    :   ![MdCTr014](images/b/bb/MdCTr014.png)

- Ao chegar no nível 100, pegue sua *[Caixa Inicial](Conquista.md "Conquista")* para ajudar na sua nova jornada.
