# Explosão Rúnica

|  |  |
| --- | --- |
| **Explosão Rúnica Explosão Rúnica** | |
| Explosão Rúnica | |
| **Storm Blast / Ráfaga de Tormentas** | |
| **ID** | RK\_STORMBLAST (2017) |
| **Tipo** | Ofensiva |
| **Níveis** | 1 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 2 seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 1 segundo |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Item** | 12732.png 1 [Runa Wyrd](http://www.divine-pride.net/database/item/12732/) |
| **Pré-requisitos** | |
| - [Perícia em Runas](Perícia em Runas.md "Perícia em Runas") [Perícia em Runas](Perícia em Runas.md "Perícia em Runas") Nv. 3 | |

Cria um grande estrondo, causando dano físico corpo a corpo nos oponentes ao redor.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

### Fórmula

`Dano = {[Nv. de Perícia em Runas + (FOR ÷ 8)] × 140} × (Nv. de base ÷ 100)`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- A conjuração não pode ser interrompida.
