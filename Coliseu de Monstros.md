# Coliseu de Monstros

| NQ ico Coliseu de Monstros | |
| --- | --- |
| Enfrente o monstro que desejar ou veja outros heróis desafiando inimigos no famoso Coliseu! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| Premiação | |
| **Recompensa** | Enfrenta monstros |

1. Procure pelo Homem de Ferro (gonryun 180,118)

   :   ![Coliseu de Monstros01](images/6/67/Coliseu_de_Monstros01.png)

   - Seu primeiro teste será carregar algumas [pedras](http://www.divine-pride.net/database/item/7049/) que ele dará.
   - Se você falhar, tente ficar com [peso](Peso.md "Peso") zero para carregar as pedras.
   - Em caso de sucesso, você poderá entrar no Coliseu de Monstros na próxima vez que falar com ele.
2. No salão de batalha, existem 2 [NPCs](NPC.md "NPC"):
   - Para entrar como Desafiante, entre na sala de chat;

   :   ![Coliseu de Monstros02](images/2/2d/Coliseu_de_Monstros02.png)

   - Para entrar como Espectador, fale com a Guia de Batalha e selecione a opção: *Quero entrar.*

   :   ![Coliseu de Monstros03](images/c/c2/Coliseu_de_Monstros03.png)

   - Como espectador, você terá uma melhor visão da batalha, mas não poderá interferir nela. Algumas [habilidades](Habilidades.md "Habilidades") não alcançam o desafiante, nem mesmo disparos de longas distâncias.
   - Para sair, basta falar com o [NPC](NPC.md "NPC") na área dos espectadores, que manda o usuário novamente para a pequena sala.
3. Como desafiante, você deve falar com o Conjurador, ele mostrará uma lista de grupos com 6 monstros, sendo que você só poderá escolher apenas 1 monstro por desafio.

   :   ![Coliseu de Monstros04](images/d/d3/Coliseu_de_Monstros04.png)

   - O monstro conjurado não fornece [experiência](Experiência.md "Experiência") e [itens](Itens.md "Itens").
   - Ao entrar na arena, você terá 5 minutos para invocar e derrotar o monstro.
