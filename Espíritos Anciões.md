# Espíritos Anciões

|  |  |
| --- | --- |
| **Espíritos Anciões Espíritos Anciões** | |
| Espíritos Anciões | |
| **Soul Strike / Golpe de Alma** | |
| **ID** | MG\_SOULSTRIKE (13) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 18 a 42 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,1 + 0,4 seg. |
| **Pós-conjuração** | 1,4 segundos |
| **Alvo** | Oponente |
| **Alcance** | 9 células |
| **Propriedade** | [Fantasma](Monstros.md#Propriedade "Monstros") |
| **Pré-requisitos** | |
| - [Ataque Espiritual](Ataque Espiritual.md "Ataque Espiritual") [Ataque Espiritual](Ataque Espiritual.md "Ataque Espiritual") Nv. 4 | |

Invoca de 1 a 5 espíritos ancestrais para causar 100% do [ATQM](ATQM.md "ATQM") de propriedade [Fantasma](Fantasma.md "Fantasma").

Em monstros da propriedade [Maldito](Maldito.md "Maldito") causa dano mágico extra.

| Nv. | Golpes | Dano extra | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 1 Espírito | 5% | 18 |
| 2 | 10% | 14 |
| 3 | 2 Espíritos | 15% | 24 |
| 4 | 20% | 20 |
| 5 | 3 Espíritos | 25% | 30 |
| 6 | 30% | 26 |
| 7 | 4 Espíritos | 35% | 36 |
| 8 | 40% | 32 |
| 9 | 5 Espíritos | 45% | 42 |
| 10 | 50% | 38 |
