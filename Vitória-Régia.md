# Vitória-Régia

| NQ ico Vitória-Régia | |
| --- | --- |
| A lenda de uma flor nos confins das cataratas de Brasilis parece estar relacionada a uma menina apaixonada pela Lua! | |
| Requisitos mínimos | |
| **Nv. de base** | 50 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 513.png 10 [Bananas](http://www.divine-pride.net/database/item/513/)  11515.png 5 [Cocos](http://www.divine-pride.net/database/item/11515/) |
| **Quests anteriores** | [Loira do Banheiro](Loira do Banheiro.md "Loira do Banheiro") |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 50.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 10.000 |
| **[Itens](Itens.md "Itens")** | 5302.png 1 [Chapéu de Vitória-Régia](http://www.divine-pride.net/database/item/5302/) |

1. Fale com a Botânica Carmen (brasilis 203,285)

   :   ![Vitória-Régia01](images/7/7d/Vitória-Régia01.png)
2. Converse com a Anciã Jurema (bra\_in01 142,26)

   :   ![Vitória-Régia02](images/0/05/Vitória-Régia02.png)
3. Localize uma Menina (brasilis 271,145)

   :   ![Vitória-Régia03](images/8/84/Vitória-Régia03.png)

   - Selecione:
     - *Qual é o problema?*
     - *Eu só queria ajudar*
     - *Ajudar Jacira*.
4. Com 10 [Bananas](http://www.divine-pride.net/database/item/513/) no inventário, interrogue o Ermitão (bra\_dun02 67,204)

   :   ![Vitória-Régia04](images/8/8a/Vitória-Régia04.png)
5. Tente colher a flor de Vitória-Régia ao lado do [NPC](NPC.md "NPC"), mas o Ermitão te impedirá.

   :   ![Vitória-Régia05](images/9/99/Vitória-Régia05.png)
6. Continue conversando com o Ermitão até ele citar o suco de uma fruta.
7. Reencontre a Menina.
8. Entregue os 5 [Cocos](http://www.divine-pride.net/database/item/11515/) para o Ermitão.
9. Clique na Vitória Régia para receber a [Flor de Vitória-Régia](http://www.divine-pride.net/database/item/7553/).
10. Conte a história para a Menina, que agradece com um Chapéu de Vitória-Régia encantado com uma das seguintes [cartas](Cartas.md "Cartas"):
    - [Carta Gato de Folha](http://www.divine-pride.net/database/item/4195/) → [Chapéu de Vitória-Régia da Pedra Azul](http://www.divine-pride.net/database/item/5302/)
    - [Carta Dríade](http://www.divine-pride.net/database/item/4177/) → [Chapéu de Vitória-Régia da Pedra Verde](http://www.divine-pride.net/database/item/5302/)
    - [Carta Leib Olmai](http://www.divine-pride.net/database/item/4188/) → [Chapéu de Vitória-Régia da Pedra Vermelha](http://www.divine-pride.net/database/item/5302/)
11. Volte a falar com a Botânica Carmen para receber seu prêmio em [experiência](Experiência.md "Experiência").
