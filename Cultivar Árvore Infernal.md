## A Árvore Infernal

### Atributos

| Atributo | Fórmula + Nv. de [Botânica](Botânica.md "Botânica") |
| --- | --- |
| [HP](HP.md "HP") | 170% do [HP](HP.md "HP") do usuário |
| [ATQ](ATQ.md "ATQ") | 180% do [ATQ](ATQ.md "ATQ") do usuário |
| [ATQM](ATQM.md "ATQM") | 200% do [ATQ](ATQ.md "ATQ") do usuário |

| [Esquiva](Esquiva.md "Esquiva") | 50% da [Esquiva](Esquiva.md "Esquiva") do usuário |
| [Precisão](Precisão.md "Precisão") | Igual ao do usuário |

### Habilidades

|  |
| --- |
| HellTree |
| NPC Skill Icon **Grito do Inferno** |
| - Causa 16.250% de dano mágico de propriedade [Sombrio](Sombrio.md "Sombrio") ao redor de 13x13 células da planta. - Dano aumenta em 1% para cada nível de base do usuário. - Tem chance de infligir [Tristeza](Tristeza.md "Tristeza") e [Azar](Azar.md "Azar").   - `Chance = [(SAB do Cientista ÷ 10)]%`   - Se o [Cientista](Cientista.md "Cientista") possuir 0 de [SAB](SAB.md "SAB"), a chance é de 0%.   - Se o [Cientista](Cientista.md "Cientista") possuir 100 de [SAB](SAB.md "SAB"), a chance é de 10%. |
| NPC Skill Icon **Cheiro de Enxofre** |
| - Conjurada a cada 40 segundos. - Aplica o estado de [*Aroma Infernal*] no Cientista: - I HellDusty **Aroma Infernal**:   - Dano físico à distância +20%.   - Dano físico contra a raça [Planta](Planta.md "Planta") e [Amorfo](Amorfo.md "Amorfo") +50%.   - Não pode ser removido por [Gênese](Gênese.md "Gênese") ou [Desencantar](Desencantar.md "Desencantar").   - Dura 30 segundos. |

## Notas

- Apenas 1 Árvore Infernal pode estar ativa por vez.
