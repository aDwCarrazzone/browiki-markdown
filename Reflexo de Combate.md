# Reflexo de Combate

|  |  |
| --- | --- |

| **Weapon Blocking / Bloqueo de Arma** | |
| **ID** | GC\_WEAPONBLOCKING (2028) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **Pós-conjuração** | 2 segundos |
| **Duração** | 180 segundos |
| **Alvo** | Usuário |
| **Ícone** | I Reflexo de Combate |
| **Pré-requisitos** | |
| - [Perícia com Mão Esquerda](Perícia com Mão Esquerda.md "Perícia com Mão Esquerda") [Perícia com Mão Esquerda](Perícia com Mão Esquerda.md "Perícia com Mão Esquerda") Nv. 5 | |

Reage à um ataque rapidamente.

Tem chance de bloquear danos físicos corpo a corpo recebidos.

Ao ativar, concede o estado de *Reflexo* por 10 segundos.

Nesse estado, pode usar [Estilhaçar Arma](Estilhaçar Arma.md "Estilhaçar Arma") ou [Retaliação](Retaliação.md "Retaliação").

Enquanto estiver ativa, drena 3 de [SP](SP.md "SP") a cada 5 segundos.

| Nv. | Chance | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 12% | 40 |
| 2 | 14% | 36 |
| 3 | 16% | 32 |
| 4 | 18% | 28 |
| 5 | 20% | 24 |

## Notas

- Se conjurada novamente, a habilidade é cancelada.
- É ativada mesmo se você se esquivar do ataque.
- Ativa com danos físicos corpo a corpo de habilidades.
- Estado de *Reflexo* não pode ser removido por [Desencantar](Desencantar.md "Desencantar") ou [Gênese](Gênese.md "Gênese").
- Se o estado for ativado novamente, a duração é reiniciada.
