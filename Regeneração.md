# Regeneração

|  |  |
| --- | --- |
| **Regeneração Regeneração** | |
| Regeneração | |
| **Regeneration** | |
| **ID** | GD\_REGENERATION (10011) |
| **Tipo** | Suporte |
| **Níveis** | 3 |
| **Recarga** | 3 minutos |
| **Duração** | 390 segundos |
| **Alvo** | Líder do [clã](Clã.md "Clã") |
| **Área** | 31x31 células |
| **Pré-requisitos** | |
| [Expandir Clã](Expandir Clã.md "Expandir Clã") [Expandir Clã](Expandir Clã.md "Expandir Clã") Nv. 5 [Comando de Batalha](Comando de Batalha.md "Comando de Batalha") [Comando de Batalha](Comando de Batalha.md "Comando de Batalha") Nv. 1 | |

Acelera a velocidade de regeneração natural de [HP](HP.md "HP") e [SP](SP.md "SP") de todos os membros do [clã](Clã.md "Clã").

| Nv. | Efeito |
| --- | --- |
| 1 | Regeneração natural de [HP](HP.md "HP") +200% |
| 2 | Regeneração natural de [HP](HP.md "HP") e [SP](SP.md "SP") +200% |
| 3 | Regeneração natural de [HP](HP.md "HP") e [SP](SP.md "SP") +300% |

## Notas

- Funciona apenas dentro dos castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
