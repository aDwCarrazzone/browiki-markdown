# Instrutores do Éden

As missões dadas pelos Instrutores do Éden são voltadas para iniciantes e consistem em lições básicas do jogo.

## Instrutora

| GQ ico Instrutores do Éden | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 1 |
| **Tipo** | Tutorial |
| **[Itens](Itens.md "Itens")** | Nenhum |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 10.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 10.000 |
| **[Itens](Itens.md "Itens")** | 5 [Ervas Azuis](http://www.divine-pride.net/database/item/510/), 5 [Ervas Verdes](http://www.divine-pride.net/database/item/511/), 5 [Poções Brancas](http://www.divine-pride.net/database/item/504/), 1 [Semente de Yggdrasil](http://www.divine-pride.net/database/item/608/) |

1. Aceite o desafio da Instrutora, no segundo andar do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").

   :   ![MestredoEden1](images/9/92/MestredoEden1.png)
2. Responda às perguntas dos [NPCs](NPC.md "NPC") próximos a ela na seguinte ordem:

   :   ![MestredoEden2](images/1/1e/MestredoEden2.png)

   1. Mestre de Batalha
   2. Mestre de Grupo
   3. Mestre de Guerra
   4. Mestre de Instância
   5. Mestre de Mapeamento
3. Retorne à Instrutora para receber sua recompensa.

## Instrutor

| GQ ico Instrutores do Éden | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 15 |
| **Tipo** | Tutorial |
| **[Itens](Itens.md "Itens")** | 1 [Catálogo de Vendas Prata](http://www.divine-pride.net/database/item/12580/), 10 [Jellopys](http://www.divine-pride.net/database/item/909/), 1 Pergaminho de [assistente](Assistente.md "Assistente") |
| **[Zeny](Zeny.md "Zeny")** | 10.200 |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 5.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 2.500 |

1. Aceite o desafio do Instrutor, no segundo andar do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").

   :   ![InstrutorEden1](images/6/61/InstrutorEden1.png)
2. Compre um pergaminho de [assistente](Assistente.md "Assistente"), a guilda mais próxima está em (prontera 41,337)
3. Use o pergaminho e retorne ao Instrutor, que pedirá que você confira sua janela de missões, sem seguida, digite:

   :   `Jellopy`
4. Compre um [Catálogo de Vendas Prata](http://www.divine-pride.net/database/item/12580/) com o o [Catalogador Mágico](Catálogo de Vendas.md "Catálogo de Vendas").
5. Retorne ao Instrutor, que pedirá que você use o item.
   - Dica: basta jogar o catálogo no chão, clicar no [NPC](NPC.md "NPC") e pegar novamente.
6. Traga 10 [Jellopys](http://www.divine-pride.net/database/item/909/) ao instrutor para receber 3 [Capotes do Tutorial](http://www.divine-pride.net/database/item/15033/).
7. Pela última vez, fale com o Instrutor para receber sua recompensa. Os Capotes serão removidos de seu inventário.
