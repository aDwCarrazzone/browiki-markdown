# Luz da Criação

|  |  |
| --- | --- |

| **ID** | LG\_RAYOFGENESIS (2321) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 40 + (Nv. da habilidade × 5) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,5 + 1 + (Nv. × 1) seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 2 segundos |
| **Alvo** | Usuário |
| **Área** | 11x11 células |
| **Propriedade** | [Sagrado](Monstros.md#Propriedade "Monstros") |
| **Efeito negativo** | [Cegueira](Cegueira.md "Cegueira") |
| **Pré-requisitos** | |
| - [Crux Magnum](Crux Magnum.md "Crux Magnum") [Crux Magnum](Crux Magnum.md "Crux Magnum") Nv. 5 | |

Concentra a luz do Criador nos pés para pisar nos alvos ao seu redor.

Causa dano mágico de propriedade [Sagrado](Sagrado.md "Sagrado").

Se [Consagração](Consagração.md "Consagração") estiver ativo, a propriedade altera para [Neutro](Neutro.md "Neutro").

Em [mortos-vivos](Mortos-vivos.md "Mortos-vivos") e [demônios](Demônio.md "Demônio"), haverá 50% de chance de infligir [cegueira](Cegueira.md "Cegueira").

`Dano = [(Dano base + INT × 3) × (Nv. de base ÷ 100)]%`
