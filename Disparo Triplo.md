# Disparo Triplo

|  |  |
| --- | --- |
| **Disparo Triplo Disparo Triplo** | |
| Disparo Triplo | |
| **Triangle Shot / Tiro Triangular** | |
| **ID** | SC\_TRIANGLESHOT (2288) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 2) |
| **Pós-conjuração** | 0,35 segundos |
| **Recarga** | 0,2 segundos |
| **Alvo** | Oponente |
| **Alcance** | 7 a 11 células |
| **Munição** | [Over Arrow](Munições.md "Munições") 3 Flechas |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") Nv. 7 | |

Atira 3 flechas para desenhar um triângulo.

Causa dano físico a distância em um alvo único.

| Nv. | [ATQ](ATQ.md "ATQ") | Alcance | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 230% | 7 células | 22 |
| 2 | 460% | 24 |
| 3 | 690% | 26 |
| 4 | 920% | 9 células | 28 |
| 5 | 1.150% | 30 |
| 6 | 1.380% | 32 |
| 7 | 1.610% | 34 |
| 8 | 1.840% | 11 célula | 36 |
| 9 | 2.070% | 38 |
| 10 | 2.300% | 40 |

### Fórmula

`Dano = {[Dano base + (AGI × 3)] × (Nv. de base ÷ 100)}%`
