# Escamas Rochosas

|  |  |
| --- | --- |

| **ID** | RK\_STONEHARDSKIN (2015) |
| **Tipo** | Ativa |
| **Níveis** | 1 |
| **[HP](HP.md "HP")** | 20% |
| **[Conjuração](Conjuração.md "Conjuração")** | 2 + 0 seg. |
| **Duração** | 900 segundos |
| **Alvo** | Usuário |
| **Item** | 12733.png 1 [Runa Hagalaz](http://www.divine-pride.net/database/item/12733/) |
| **Ícone** | I Escamas Rochosas |
| **Pré-requisitos** | |
| - [Perícia em Runas](Perícia em Runas.md "Perícia em Runas") [Perícia em Runas](Perícia em Runas.md "Perícia em Runas") Nv. 4 | |

Enrijece a própria pele, fornecendo um bônus de [DEF](DEF.md "DEF") e [DEFM](DEFM.md "DEFM") por 15 minutos.

Personagens que causarem dano no usuário terão 30% de chance da arma ser quebrada.

Se o oponente for um [monstro](Monstro.md "Monstro"), reduz 25% do [ATQ](ATQ.md "ATQ") por 10 segundos.

### Fórmula

`DEF e DEFM = [(Nível de Classe × Perícia em Runas) ÷ 4]`

## Notas

- A redução do [ATQ](ATQ.md "ATQ") não funciona em [monstros](Monstros.md "Monstros") Chefes.
