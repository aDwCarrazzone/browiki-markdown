# Ocultação

|  |  |
| --- | --- |
| **Ocultação Ocultação** | |
| Ocultação | |
| **Cloaking Exceed / Senda Secreta** | |
| **ID** | GC\_CLOAKINGEXCEED (2033) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 45 |
| **Pós-conjuração** | 2 segundos |
| **Alvo** | Usuário |
| **Ícone** | I Ocultação |
| **Pré-requisitos** | |
| - [Furtividade](Furtividade.md "Furtividade") [Furtividade](Furtividade.md "Furtividade") Nv. 3 | |

Entra em um estado de [esconderijo](Esconderijo.md "Esconderijo").

Impede a detecção até de [insetos](Inseto.md "Inseto") e [demônios](Demônio.md "Demônio").

Enquanto ativa, drena [SP](SP.md "SP") por segundo e permite receber até 3 ataques sem cancelar o efeito.

| Nv. | Movimento | Ataques | Dreno de [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 100% | 1 | 9 por segundo |
| 2 | 110% | 8 por segundo |
| 3 | 120% | 2 | 7 por segundo |
| 4 | 130% | 6 por segundo |
| 5 | 140% | 3 | 5 por segundo |

## Notas

- É cancelada se receber o limite de ataques permitidos pela habilidade, se o personagem atacar um alvo ou utilizar qualquer outra habilidade.
- Durante o efeito, é possível usar [Furtividade](Furtividade.md "Furtividade"), reiniciando a contagem dos ataques permitidos.
- Personagens escondidos podem ser detectados pela [Clarividência](Clarividência.md "Clarividência").
