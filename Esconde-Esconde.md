# Esconde-Esconde

| NQ ico Esconde-Esconde | |
| --- | --- |
| Tente acalmar uma criança que está desesperada por ter se perdido do pai. | |
| Requisitos mínimos | |
| **Nv. de base** | 50 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 12051.png 1 [Caranguejo no Vapor](http://www.divine-pride.net/database/item/12051/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 5.000 |
| **[Itens](Itens.md "Itens")** | 12065.png 3 [Saladas Verdes](http://www.divine-pride.net/database/item/12065/) |

1. Encontre Torpy (hugel 107,67)

   :   ![Hugel01](images/f/ff/Hugel01.png)
2. Localize a Mãe do Torpy (hu\_fild06 190,367)

   :   ![Hugel02](images/c/c8/Hugel02.png)
3. Retorne a Torpy.
4. Entregue as [Garras de Caranguejo no Vapor](http://www.divine-pride.net/database/item/12051/) ao Barril Suspeito (hugel 100,176)

   :   ![Hugel03](images/a/ab/Hugel03.png)
5. O pai da criança surgirá, fale com ele.

   :   ![Hugel04](images/7/7f/Hugel04.png)
6. Volte a Torpy.
7. Converse com a Mãe do Torpy para receber sua premiação.
8. Fale com Torpy.
9. Clique no Barril Suspeito para encerrar.
