1. Receba a [Carta do Irmão](http://www.divine-pride.net/database/item/6546/) de Wuhari (eclage 282,255)

   :   ![Preocupações de uma Fada1](images/d/db/Preocupações_de_uma_Fada1.png)
2. Entre na prisão através do o Guarda de Eclage (ecl\_in02 98,32)

   :   ![Preocupações de uma Fada2](images/7/74/Preocupações_de_uma_Fada2.png)
3. Insista em conversar com Wuhuru (ecl\_in02 149,117)

   :   ![Preocupações de uma Fada3](images/2/2b/Preocupações_de_uma_Fada3.png)
4. Retorne a Wuhari.
5. Fale com Masione (eclage 200,272)

   :   ![Preocupações de uma Fada4](images/2/23/Preocupações_de_uma_Fada4.png)
6. Encontre Goatie (eclage 163,228)

   :   ![Preocupações de uma Fada5](images/f/f1/Preocupações_de_uma_Fada5.png)
7. Pegue o [Livro Manchado de Pesquisas](http://www.divine-pride.net/database/item/6547/) com Svery (eclage 170,195)

   :   ![Preocupações de uma Fada6](images/0/0e/Preocupações_de_uma_Fada6.png)
8. Entregue o [Livro Manchado de Pesquisas](http://www.divine-pride.net/database/item/6547/) para Wuhari e aguarde 5 minutos para falar com ele novamente.
9. Retorne a Svery.
10. Volte a Goatie.
11. Localize Pompe (eclage 126,151)

    :   ![Preocupações de uma Fada7](images/3/31/Preocupações_de_uma_Fada7.png)
12. Procure Giranni (ecl\_fild01 70,285)

    :   ![Preocupações de uma Fada8](images/8/81/Preocupações_de_uma_Fada8.png)
13. Examine o Cadáver de Esplêndido (ecl\_fild01 269,165)

    :   ![Preocupações de uma Fada9](images/4/46/Preocupações_de_uma_Fada9.png)

    - Você receberá um [Pedaço de Asa dos Esplêndidos](http://www.divine-pride.net/database/item/6548/).
14. Giranni agradece você com 1 [Velha Caixa Roxa](http://www.divine-pride.net/database/item/0000/) e 100.000 de [experiência](Experiência.md "Experiência").
15. Volte a Wuhuru na prisão.
16. Retorne a Wuhari.
17. Pegue o [Pedaço de Asa dos Esplêndidos](http://www.divine-pride.net/database/item/6548/) com Giranni.
18. Consiga o [Cabelo Seco](http://www.divine-pride.net/database/item/6545/) no Cadáver de Esplêndido.
19. Entregue os itens a Wuhari para receber 5 [Moedas Esplendor](http://www.divine-pride.net/database/item/6081/) e 300.000 de [experiência](Experiência.md "Experiência").
