# Barreira de Gelo

|  |  |
| --- | --- |

| **ID** | WZ\_ICEWALL (87) |
| **Tipo** | Ativa |

| **[SP](SP.md "SP")** | 20 |

| **Alvo** | Chão |

| **Área** | 1x5 células |
| **Pré-requisitos** | |
| - [Petrificar](Petrificar.md "Petrificar") [Petrificar](Petrificar.md "Petrificar") Nv. 1 - [Rajada Congelante](Rajada Congelante.md "Rajada Congelante") [Rajada Congelante](Rajada Congelante.md "Rajada Congelante") Nv. 1 | |

Conjura ilimitadamente uma parede com gelo cristalino, impedindo a passagem.

A barreira é destruída se a duração terminar ou se o [HP](HP.md "HP") chegar a zero.

| Nv. | HP | Duração |
| --- | --- | --- |
| 1 | 400 | 8 segundos |
| 2 | 600 | 12 segundos |
| 3 | 800 | 16 segundos |
| 4 | 1.000 | 20 segundos |
| 5 | 1.200 | 24 segundos |
| 6 | 1.400 | 28 segundos |
| 7 | 1.600 | 32 segundos |
| 8 | 1.800 | 36 segundos |
| 9 | 2.000 | 40 segundos |
| 10 | 2.200 | 44 segundos |

## Notas

- A cada segundo ativada, a barreira perde 50 de [HP](HP.md "HP").
- Não pode ser usada em castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium"), [Tumba da Honra](Tumba da Honra.md "Tumba da Honra") e algumas [Instâncias](Instâncias.md "Instâncias").
- As células da barreira tornam-se inválidas para movimentação, permitindo que [monstros](Monstros.md "Monstros") fiquem presos entre várias paredes.
- É possível mirar magias de alvo único na barreira pressionando a tecla SHIFT.
- Usar a habilidade para atrapalhar outros personagens é passível de punição.
- A barreira pode sofrer dano de ataques básicos e de magias em área.
