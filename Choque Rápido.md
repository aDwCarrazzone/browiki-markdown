# Choque Rápido

|  |  |
| --- | --- |

| **ID** | PA\_SHIELDCHAIN (480) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 25 + (Nv. da habilidade × 3) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,2 + 0,8 seg. |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | 7 a 11 células |
| **Propriedade** | [Neutro](Monstros.md#Propriedade "Monstros") |
| **Equipamento** | Escudo |
| **Pré-requisitos** | |
| - [Escudo Bumerangue](Escudo Bumerangue.md "Escudo Bumerangue") [Escudo Bumerangue](Escudo Bumerangue.md "Escudo Bumerangue") Nv. 5 | |

Rapidamente atira o escudo no oponente para causar dano físico a distância.

Pode golpear o alvo até 5 vezes, dependendo da sua [Precisão](Precisão.md "Precisão").

No momento do ataque, garante [Precisão](Precisão.md "Precisão") +20.

Se [Escudo Reforçado](Arremessar Escudo.md "Arremessar Escudo") estiver ativo, há um bônus no dano causado.

| Nv. | [ATQ](ATQ.md "ATQ") | | Alcance | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| Normal | [Bônus](Arremessar Escudo.md "Arremessar Escudo") |
| 1 | 500% | 700% | 7 células | 28 |
| 2 | 700% | 950% | 31 |
| 3 | 900% | 1.200% | 9 células | 34 |
| 4 | 1.100% | 1.450% | 37 |
| 5 | 1.300% | 1.700% | 11 células | 40 |

### Fórmula

Normal:  
`ATQ = {[Dano base + Peso + (Refino × 4)] × (Nv. de base ÷ 100)}`

Escudo Reforçado:  
`ATQ = {[(Nv. da habil. × (250 + (Nv. de Perícia com Escudo × 15) + 400)] + Peso + (Refino × 4)} × (Nv. de base ÷ 100 ]`

- *Peso* e *Refino* se referem ao Peso e Refino do escudo.

## Notas

- É afetada por [Investigar](Investigar.md "Investigar").
- A [propriedade](Propriedade.md "Propriedade") do ataque não pode ser alterada pela arma equipada.
- Apesar da animação, todos os golpes estão conectados em um só, sendo duplicado por [Lex Aeterna](Lex Aeterna.md "Lex Aeterna").
