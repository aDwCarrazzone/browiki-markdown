# Remoção Total

|  |  |
| --- | --- |
| **Remoção Total Remoção Total** | |
| Remoção Total | |
| **Full Divestment / Despojo Total** | |
| **ID** | ST\_FULLSTRIP (476) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 2) |
| **Pós-conjuração** | 1 segundo |
| **Duração** | 60 + (Nv. da hab. × 15) segundos |
| **Alvo** | Oponente |
| **Pré-requisitos** | |
| - [Remover Arma](Remover Arma.md "Remover Arma") [Remover Arma](Remover Arma.md "Remover Arma") Nv. 5 | |

Prepara-se para fragilizar o alvo.

Tem chance de remover a armadura, a arma, o equipamento para cabeça e o escudo do oponente, impedindo-o de equipá-los temporariamente.

Em [monstros](Monstros.md "Monstros"), reduz o [ATQ](ATQ.md "ATQ") em 10% e a [DEF](DEF.md "DEF") em 30%.

| Nv. | Remoção | | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| Chance base | Duração base |
| 1 | 7% | 75 segundos | 17 |
| 2 | 9% | 90 segundos | 19 |
| 3 | 11% | 105 segundos | 21 |
| 4 | 13% | 120 segundos | 23 |
| 5 | 15% | 135 segundos | 25 |

### Fórmula

`Chance = {Chance base + [(DES do usuário - DES do alvo) ÷5]}%`  
`Duração = {Duração base + [(DES do usuário - DES do alvo) ÷5]}%`

## Notas

- A chance de sucesso não pode ser menor que 5% e a duração não pode ser menor que 2 segundos.
- Não exibe animação em caso de falha.
- Não funciona contra [Mecânicos](Mecânicos.md "Mecânicos") pilotando um [MECHA](Licença de Pilotagem.md "Licença de Pilotagem").
- Funciona em [monstros](Monstros.md "Monstros") do tipo chefe e em alvos sofrendo de [congelamento](Congelamento.md "Congelamento") e [petrificação](Petrificação.md "Petrificação").
- O efeito de remoção pode ser prevenido por [Proteção Química Total](Proteção Química Total.md "Proteção Química Total") e cancelado por [Canto de Batalha](Canto de Batalha.md "Canto de Batalha") ou desconectar e conectar no jogo novamente.
