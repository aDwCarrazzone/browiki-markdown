# Noivo de Euslan

| NQ ico Noivo de Euslan | |
| --- | --- |
| Uma mulher doente está preocupada com seu noivo que saiu em busca de um remédio para ela. Será que ele é capaz de arriscar a própria vida para conseguir uma cura? | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 7511.png 5 [Runas da Escuridão](http://www.divine-pride.net/database/item/7511/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 90.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 60.000 |

1. Converse mais de uma vez com Euslan (yuno 111,156)

   :   ![Euslan1](images/b/b3/Euslan1.png)
2. Pergunte sobre o noivo de Euslan para Hoi (airplane 236,63)

   :   ![SdA01](images/e/e1/SdA01.png)
3. Encontre Kaci (airplane 33,69)

   :   ![SdA02](images/1/1e/SdA02.png)
4. Insista na conversa com o Capitão (airplane 236,163)

   :   ![SdA04](images/9/94/SdA04.png)
5. Retorne a Euslan.
6. Entre na residência em [Hugel](Hugel.md "Hugel"). Um diálogo surgirá, diga que deseja falar com o Gerente para teleportar para uma nova sala. Lá, fale com Eukran (que\_bingo 54,17)

   :   ![Euslan2](images/d/d4/Euslan2.png)
7. Caminhe pelo Templo de Odin até aparecer um Jovem (odin\_tem01 135,125)

   :   ![Euslan3](images/e/e3/Euslan3.png)
8. Aproxime-se do Barqueiro (odin\_tem01 108,146)

   :   ![Barquero1](images/5/5d/Barquero1.png)
9. Após ouvir o diálogo do Barqueiro, retorne ao Jovem e entregue 5 [Runas da Escuridão](http://www.divine-pride.net/database/item/7511/).
10. Volte para Euslan para ganhar sua recompensa.
