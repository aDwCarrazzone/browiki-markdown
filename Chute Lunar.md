# Chute Lunar

|  |  |
| --- | --- |
| **Chute Lunar Chute Lunar** | |
| Chute Lunar | |
| **Full Moon Kick / Patada de Luna Llena** | |
| **ID** | SJ\_FULLMOONKICK (2576) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | [25 + (Nv. da habilidade × 5)] |
| **Recarga** | 1 segundo |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Efeito negativo** | [Cegueira](Cegueira.md "Cegueira") |
| **Postura** | [Lunar](Postura Lunar.md "Postura Lunar") |
| **Pré-requisitos** | |
| - [Eclipse Lunar](Eclipse Lunar.md "Eclipse Lunar") [Eclipse Lunar](Eclipse Lunar.md "Eclipse Lunar") Nv. 7 | |

Só pode ser usada no estado de [Penumbra](Eclipse Lunar.md "Eclipse Lunar").

Cancela o status de [Penumbra](Eclipse Lunar.md "Eclipse Lunar").

Causa dano físico corpo a corpo nos oponentes na área de 7x7 células ao seu redor.

Tem chance de infligir [Cegueira](Cegueira.md "Cegueira") nos alvos.

| Nv. | [ATQ](ATQ.md "ATQ") | [Cegueira](Cegueira.md "Cegueira") | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 1.200% | 20% | 30 |
| 2 | 1.300% | 25% | 35 |
| 3 | 1.400% | 30% | 40 |
| 4 | 1.500% | 35% | 45 |
| 5 | 1.600% | 40% | 50 |
| 6 | 1.700% | 45% | 55 |
| 7 | 1.800% | 50% | 60 |
| 8 | 1.900% | 55% | 65 |
| 9 | 2.000% | 60% | 70 |
| 10 | 2.100% | 65% | 75 |

### Fórmula

`ATQ = [1.100 + (Nv. da habilidade × 100)] × (Nv. de base ÷ 100)]%`
