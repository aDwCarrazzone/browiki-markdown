# Lope e Euridi

| NQ ico Lope e Euridi | |
| --- | --- |
| O desaparecimento de uma pessoa amada pode acarretar em uma busca perigosa pelo labirinto enevoado. | |
| Requisitos mínimos | |
| **Nv. de base** | 100 |
| **Tipo** | Normal |

| Premiação | |

1. Ofereça ajuda a Euridi (mora 117,66)

   :   ![Lopeuride1](images/e/e1/Lopeuride1.png)
2. Fale com Pitt (mora 65,145).

   :   ![Lopeuride2](images/6/6c/Lopeuride2.png)
3. Retorne a Euridi.
4. Colete 30 [Pistas sobre Lope](http://www.divine-pride.net/database/item/6383/) e mostre-as a Pitt, mas não entregue a ele.
5. Volte a Euridi.
6. Insista a conversa com Pitt até conseguir um [Anel de Lope](http://www.divine-pride.net/database/item/6384/)
7. Converse com Euridi novamente.
8. Fale com a Amiga de Euridi (mora 115,68)

   :   ![Lopeuride3](images/7/72/Lopeuride3.png)
9. Aproxime-se de Lope (bif\_fild01 131,338)

   :   ![Lopeuride4](images/2/2b/Lopeuride4.png)
10. Diga a Euridi que Lope está morto.
11. Fale com a Amiga de Euridi para receber 1.000.000 de [experiência](Experiência.md "Experiência") de base, 2.000.000 de [experiência](Experiência.md "Experiência") de classe e 10 [Moedas de Mora](http://www.divine-pride.net/database/item/6380/).
12. Entregue 30 [Pistas sobre Lope](http://www.divine-pride.net/database/item/6383/) para a Regente dos Cavaleiros (mora 114,163)

    :   ![AmazonaVizinha](images/7/74/AmazonaVizinha.png)

    - Você receberá 1.000.000 de [experiência](Experiência.md "Experiência") de base, 1.000.000 de [experiência](Experiência.md "Experiência") de classe e 2 [Moedas de Mora](http://www.divine-pride.net/database/item/6380/).
