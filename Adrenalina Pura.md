# Adrenalina Pura

|  |  |
| --- | --- |
| **Adrenalina Pura Adrenalina Pura** | |
| Adrenalina Pura | |
| **Adrenaline Rush / Ráfaga de Adrenalina** | |
| **ID** | BS\_ADRENALINE (111) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 17 + (Nv. da habilidade × 3) |
| **Duração** | (Nv. da habilidade × 30) segundos |
| **Alvo** | Usuário |
| **Área** | 37x37 células |
| **Arma** | Maça ou Machado |
| **Ícone** | I Adrenalina Pura |
| **Pré-requisitos** | |
| - [Martelo de Thor](Martelo de Thor.md "Martelo de Thor") [Martelo de Thor](Martelo de Thor.md "Martelo de Thor") Nv. 2 | |

Só pode ser usada ao equipar Machados ou Maças.

Garante [velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") +10% e aumenta a [Precisão](Precisão.md "Precisão") de todos do [grupo](Grupo.md "Grupo") ao redor.

Usuários que são [Ferreiros](Ferreiros.md "Ferreiros") e evoluções também receberão [Std](Std.md "Std") +7.

Usuários que não são [Ferreiros](Ferreiros.md "Ferreiros") receberão [Std](Std.md "Std") +5.

| Nv. | [Precisão](Precisão.md "Precisão") | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | +8 | 30 segundos | 20 |
| 2 | +11 | 60 segundos | 23 |
| 3 | +14 | 90 segundos | 26 |
| 4 | +17 | 120 segundos | 29 |
| 5 | +20 | 150 segundos | 32 |

## Notas

- A duração pode ser aumentada por [Punho Firme](Punho Firme.md "Punho Firme").
- É cancelada ao trocar de arma se não for Maça ou Machado.
- Pode ser cancelada por [Pântano dos Mortos](Pântano dos Mortos.md "Pântano dos Mortos"), [Diminuir Agilidade](Diminuir Agilidade.md "Diminuir Agilidade") e [Não me Abandones](Não me Abandones.md "Não me Abandones").
- O bônus de [velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") é similar aos efeitos de equipamentos.
- Para que outros membros do grupo recebam o efeito, eles devem estar equipados de uma Maça ou Machado.
