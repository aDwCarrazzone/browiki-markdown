# Sifão de Alma

|  |  |
| --- | --- |
| **Sifão de Alma Sifão de Alma** | |
| Sifão de Alma | |
| **Soul Siphon / Succionar Alma** | |
| **ID** | PF\_SOULBURN (375) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 70 + (Nv. da habilidade ×10) |
| **Recarga** | 10 a 15 segundos |
| **Alvo** | Oponente |
| **Alcance** | 10 células |
| **Pré-requisitos** | |
| - [Desencantar](Desencantar.md "Desencantar") [Desencantar](Desencantar.md "Desencantar") Nv. 3 - [Espelho Mágico](Espelho Mágico.md "Espelho Mágico") [Espelho Mágico](Espelho Mágico.md "Espelho Mágico") Nv. 3 - [Cancelar Magia](Cancelar Magia.md "Cancelar Magia") [Cancelar Magia](Cancelar Magia.md "Cancelar Magia") Nv. 5 | |

Aquece a alma do alvo, criando uma chance de drenar todo o [SP](SP.md "SP").

No nível 5, o calor aumenta, causando dano mágico de acordo com o [SP](SP.md "SP") drenado.

Em caso de falha, o efeito será aplicado no usuário.

| Nv. | Chance | Dano | Recarga | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| 1 | 40% | Nenhum | 10 segundos | 80 |
| 2 | 50% | 90 |
| 3 | 60% | 100 |
| 4 | 70% | 110 |
| 5 | 200% | 15 segundos | 120 |

## Notas

- Só pode ser usada contra personagens em áreas de [PvP](PvP.md "PvP").
- O dano pode ser reduzido pela [DEFM](DEFM.md "DEFM").
