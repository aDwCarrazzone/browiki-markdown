Cria um terremoto violento na área selecionada.

A cada intervalo de tempo, um pilar de pedra é erguido para causar dano mágico.

Cada Pilar de Pedra ocupa uma área de 7x7 células dentro do terreno que você criou.

Os pilares são erguidos de maneira aleatória dentro da área.

Quanto maior o nível, mais pilares poderão ser erguidos.

| Nv. | Pilares de Pedra | | | Área | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| [ATQM](ATQM.md "ATQM") | Núm. | Intervalo |
| 1 | 120% | 4 | 1,50 segundos | 7x7 células | 80 |
| 2 | 240% | 8 | 0,75 segundos | 90 |
| 3 | 360% | 12 | 0,50 segundos | 9x9 células | 100 |
| 4 | 480% | 16 | 0,37 segundos | 110 |
| 5 | 600% | 20 | 0,30 segundos | 120 |

### Fórmula

`ATQM por pilar = {[Dano base + (5 × FEI)] × (Nv. base ÷ 100)}%`

### Potencialização

- Essa habilidade pode atingir seu clímax a cada nível de [![Potencializar Magia](images/4/49/Potencializar_Magia.png)](Potencializar Magia.md "Potencializar Magia") [Potencializar Magia](Potencializar Magia.md "Potencializar Magia").
- Seu solo mágico muda de cor conforme o nível usado.

|  |  |
| --- | --- |
| Clímax | Nível 1 |
| - Dobra o número de Pilares erguidos. - Dano de Pilares de Pedra -50%.   - Esse valor é cumulativo ou anulado por [equipamentos](Equipamentos.md "Equipamentos") de efeitos semelhantes. | |
| Clímax | Nível 2 |
| - Aumenta o tamanho do Pilar de Pedra de 7x7 para 9x9 células. - Não aumenta a área do terreno. | |
| Clímax | Nível 3 |
| - Dano de Pilares de Pedra +100%.   - Esse valor é cumulativo com [equipamentos](Equipamentos.md "Equipamentos") de efeitos semelhantes. | |
| Clímax | Nível 4 |
| - Não causa mais dano. Inflige o estado de [*Empalamento*] em quem estiver na área. - **Ícone:** I Empalamento - **Duração:** 30 segundos - **Efeitos no oponente:**   - Resistência à propriedade [Terra](Terra.md "Terra") -100%. | |
| Clímax | Nível 5 |
| - Independente do nível de Pilares de Pedra, altera a área do terreno para 7x7 células. | |

## Notas

- A duração da habilidade acaba quando todos os pilares foram erguidos.
- O estado de *Empalamento* não pode ser removido por [Desencantar](Desencantar.md "Desencantar") ou [Gênese](Gênese.md "Gênese").
- Não causa dano, nem inflige efeito em oponentes dentro do [Proteger Terreno](Proteger Terreno.md "Proteger Terreno").
- [Pólen](Florescer.md "Florescer") e *Empalamento* podem estar atuantes simultaneamente.
