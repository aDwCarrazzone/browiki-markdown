1. Ajude Muff (comodo 224,187)

   :   ![Muff01](images/a/a8/Muff01.png)
2. Cave o Pedaço de Terra até encontrar uma carteira (cmd\_fild01 55,160)

   :   ![Muff2](images/c/cd/Muff2.png)
3. Reencontre Muff, em [Comodo](Comodo.md "Comodo").
4. Dirija-se a Dorian (izlude\_in 59,94)

   :   ![Muff3 new](images/2/2b/Muff3_new.png)

   - Digite:
     - `Teoria das Ondas Tricentrífugas`
   - Entregue os itens pedidos.
5. Clique na Máquina ao lado de Dorian.

   :   ![NPMaquina nw](images/b/bc/NPMaquina_nw.png)

   - Primeiro troque o Motor.
   - Depois, aperte o parafuso várias vezes lendo os diálogos até o som mudar para de "*krrr*" para "*clique*".
   - Agora selecione "*Apertar botão*" e siga a sequência abaixo.
   - Repita o processo várias vezes até consertar a máquina.
     - Se vibrar *levemente*, aperte o botão *Vermelho*.
     - Se vibrar *mais forte*, aperte o botão *Azul*.
     - Se as vibrações forem *mais estáveis*, aperte o botão *Amarelo*.
     - Se vibrar *bem forte*, aperte o botão *Verde*.
6. Ao consertar a máquina, retorne a Dorian para receber uma [Lista de Débitos](http://www.divine-pride.net/database/item/7722/).
7. Entregue a [Lista de Débitos](http://www.divine-pride.net/database/item/7722/) para Muff, que agradecerá com uma recompensa.
