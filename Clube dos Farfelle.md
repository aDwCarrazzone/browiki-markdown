# Clube dos Farfelle

| NQ ico Clube dos Farfelle | |
| --- | --- |
| Estrangeiros que chegam em [Lasagna](Lasagna.md "Lasagna") podem ajudar os Farfelle a entender os motivos do navio Conchiolina não ter embarcado! | |
| Requisitos mínimos | |
| **Nv. de base** | 1 |
| **Tipo** | Normal |
| **Caça** | 20 [Basiliscos General](http://www.divine-pride.net/database/monster/3504/) ou 20 [Basiliscos Guarda](http://www.divine-pride.net/database/monster/3502/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | **Nv. 1 a 139**: 1.000 **Nv. 140 ou +**: 1.200.000 |
| **[Itens](Itens.md "Itens")** | 22984.png 5 [Leites de Kaluna](http://www.divine-pride.net/database/item/22984/) |
| **Recompensa** | Libera as missões [Maldição do Barqueiro](Maldição do Barqueiro.md "Maldição do Barqueiro"), [Mundo de Sabores](Mundo de Sabores.md "Mundo de Sabores") e [História Perdida](História Perdida.md "História Perdida") |

1. *Pergunte sobre os Farfelle* ao Guia (lasagna 173,161)

   :   ![GuiaFarfelle](images/2/22/GuiaFarfelle.png)
2. Procure pelo Chefe Peperoccino (lasagna 235,198)

   :   ![Peperoccino](images/b/bd/Peperoccino.png)
3. Ao lado do Chefe, fale com Vigilante Olio (lasagna 232,198)

   :   ![Olio](images/9/92/Olio.png)

   - Você receberá 400 de [EXP](EXP.md "EXP") de base, 500 de [EXP](EXP.md "EXP") de classe e 1 [Sandálias Ecológicas](http://www.divine-pride.net/database/item/22076/).
4. Fale com o Chefe Peperoccino novamente, ele vai te enviar para um quarto secreto!
5. No quarto, converse com o Chefe Peperoccino e depois com o Basilisco.

   :   ![Basilisco](images/9/90/Basilisco.png)

   - Após a conversa, o Chefe te retira do local para trazer materiais.
6. Saia de [Lasagna](Lasagna.md "Lasagna") e vá até a Floresta Ravioli, clique na Raiz Grossa (lasa\_fild02 341,250)

   :   ![RaizesGrossas](images/7/72/RaizesGrossas.png)

   - Você precisa clicar 5 vezes para conseguir 5 [Raízes Grossas](http://www.divine-pride.net/database/item/25044/).
7. Com as [Raízes Grossas](http://www.divine-pride.net/database/item/25044/) no inventário, retorne ao Chefe Peperoccino (lasagna 235,198)

   :   ![Peperoccino](images/b/bd/Peperoccino.png)
8. Dentro do quarto secreto, entregue os itens para Jindou fazer uma [Flauta de Raízes](http://www.divine-pride.net/database/item/25043/).

   :   ![Jindou](images/d/d5/Jindou.png)

   - Agora, converse com o Basilisco, mas preste atenção: você **não** deve ameaçá-lo.
   - Sempre selecione as opções de *Largar a flauta e contar uma piada*.
   - Você deve repetir isso umas 5 vezes.
   - Quando acabar, haverá duas opções de arma para você escolher à vontade:
     - ![1684.png](http://static.divine-pride.net/images/items/item/1684.png) [Cauda de Gato Resiliente](http://www.divine-pride.net/database/item/1684/)
     - ![1683.png](http://static.divine-pride.net/images/items/item/1683.png) [Cauda de Gato Arcana Simples](http://www.divine-pride.net/database/item/1683/)
   - Você também receberá 400 de [EXP](EXP.md "EXP") de base e 500 de [EXP](EXP.md "EXP") de classe.
9. Em [Lasagna](Lasagna.md "Lasagna") novamente, fale com o Chefe Peperoccino (lasagna 235,198)

   :   ![Peperoccino](images/b/bd/Peperoccino.png)
10. Procure por Tali (lasagna 77,263)

    :   ![DoramTali](images/f/fe/DoramTali.png)

    - Após a conversa, seu objetivo é coletar 2 [Tecidos Peculiares](http://www.divine-pride.net/database/item/25045/).
    - Estes itens são encontrados ao caçar [Sapos de Rodda Doces](http://www.divine-pride.net/database/monster/3499/).
    - Os itens vão diretamente para o seu inventário.
    - Entregue os [Tecidos Peculiares](http://www.divine-pride.net/database/item/25045/) para Tali.
11. Volte a falar com o Chefe Peperoccino (lasagna 235,198)

    :   ![Peperoccino](images/b/bd/Peperoccino.png)

    - Você receberá 700 de [EXP](EXP.md "EXP") base, 500 de [EXP](EXP.md "EXP") classe, 1 [Camiseta Ecológica](http://www.divine-pride.net/database/item/15151/) e 1 [Passe de Embarque](http://www.divine-pride.net/database/item/25046/).
12. A missão começa falando com Miauxuruca (lasagna 205,326)

    :   ![Miauxuruca](images/3/39/Miauxuruca.png)

    - Você recebe 1 [Kaluna](http://www.divine-pride.net/database/item/25047/).
13. Dentro do navio, procure pelo Comissário.

    :   ![Conchissario](images/f/f6/Conchissario.png)
14. Entre no portal que fica entre os dois lances de escada, fale com o Capitão Gamberi.

    :   ![Gamberi](images/8/83/Gamberi.png)
15. Em seguida, você deve encontrar o arquivista. Ele está após o NPC Comissário, no portal a direita.

    :   ![Arquivisty](images/4/43/Arquivisty.png)
16. Saia pelo portal, siga a direita, e entre no último portal no meio para falar com o Sous-chef.

    :   ![Sous-Chef](images/4/47/Sous-Chef.png)
17. Retorne ao Comissário, e depois retorne para falar novamente com o Sous-chef.
18. Após o diálogo, retorne ao Comissário.
19. Reporte a situação ao Stello.

    :   ![Stello](images/8/83/Stello.png)
20. Procure pelo Pannacotta (lasagna 298,84)

    :   ![Pannacotta](images/d/db/Pannacotta.png)
21. No [Ninho do Dragão](http://www.divine-pride.net/database/map/lasa_dun01/), encontre Pannacotta (lasa\_dun01 153,103)

    :   ![Cottapanna](images/0/00/Cottapanna.png)

    - Se seu nível for abaixo do 140, Pannacotta teleportará seu personagem para uma réplica do 3º andar do [Ninho do Dragão](http://www.divine-pride.net/database/map/lasa_dun03/).
    - Se o seu nível for maior que 140, você deve seguir para o último andar da caverna.
22. Dirija-se a Tartufi, no centro do mapa (lasa\_dun03 162,157)

    :   ![Tartufi](images/b/bc/Tartufi.png)
23. Procure por Imugi Gong (lasa\_dun03 161,289)

    :   ![Imugigong](images/9/9a/Imugigong.png)

    - Se você não for nível 140 ou mais, pode pular essa etapa.
24. Volte a falar com Tartufi.
    - Seu objetivo é eliminar 20 [Basiliscos General](http://www.divine-pride.net/database/monster/3504/).
    - Se seu nível for 139 ou menor, terá de eliminar 20 [Basiliscos Guarda](http://www.divine-pride.net/database/monster/3502/).
25. Assim que completar a missão, volte ao Conchiolina e fale com Stello.

    :   ![Stello](images/8/83/Stello.png)

    - Você receberá 1.200.000 de [EXP](EXP.md "EXP") de base e 5 [Leites de Kaluna](http://www.divine-pride.net/database/item/22984/).
    - Se seu nível for menor que 140, você recebe 1.000 de [EXP](EXP.md "EXP") de base e 5 [Leites de Kaluna](http://www.divine-pride.net/database/item/22984/).
    - Seu personagem será teleportado para [Izlude](Izlude.md "Izlude").
