# Destruidor de Almas

|  |  |
| --- | --- |
| **Destruidor de Almas Destruidor de Almas** | |

| **Tipo** | Ofensiva |
| **Níveis** | 10 |

| **[Conjuração](Conjuração.md "Conjuração")** | 0,25 + 0,25 seg. |

| **Recarga** | 0,25 segundos |
| **Alvo** | Oponente |

| **Pré-requisitos** | |
| - [Envenenar](Envenenar.md "Envenenar") [Envenenar](Envenenar.md "Envenenar") Nv. 5 - [Furtividade](Furtividade.md "Furtividade") [Furtividade](Furtividade.md "Furtividade") Nv. 3 - [Ataque Duplo](Ataque Duplo.md "Ataque Duplo") [Ataque Duplo](Ataque Duplo.md "Ataque Duplo") Nv. 5 - [Envenenar Arma](Envenenar Arma.md "Envenenar Arma") [Envenenar Arma](Envenenar Arma.md "Envenenar Arma") Nv. 6 | |

Atinge a alma do alvo com toda sua força, causando dano físico a distância.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual à metade do [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") |
| --- | --- |
| 1 | 150% |
| 2 | 300% |
| 3 | 450% |
| 4 | 600% |
| 5 | 750% |
| 6 | 900% |
| 7 | 1.050% |
| 8 | 1.200% |
| 9 | 1.350% |
| 10 | 1.500% |

### Fórmula

`ATQ = {[(Dano base + INT + FOR) × (Nv. de base ÷ 100)}%`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- É afetada por [Investigar](Investigar.md "Investigar") e por armas que ignoram a [DEF](DEF.md "DEF").
- Pode ser bloqueada por [Escudo Sagrado](Escudo Sagrado.md "Escudo Sagrado") e por [Troca de Pele](Troca de Pele.md "Troca de Pele").
- [Perícia com Mão Esquerda](Perícia com Mão Esquerda.md "Perícia com Mão Esquerda") e [Perícia com Mão Direita](Perícia com Mão Direita.md "Perícia com Mão Direita") não influenciam no dano.
