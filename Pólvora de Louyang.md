# Pólvora de Louyang

| NQ ico Pólvora de Louyang | |
| --- | --- |
| Ajude uma pessoa misteriosa a encontrar pessoas dispostas a causar uma verdadeira revolução na cidade! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| **[Grupo](Grupo.md "Grupo")** | 8 Pessoas, incluindo um personagem com [carrinho](Usar Carrinho.md "Usar Carrinho") |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 668.png 8 [Envelopes Vermelhos](http://www.divine-pride.net/database/item/668/) |

1. Crie um [grupo](Grupo.md "Grupo") e fale com a Pessoa Escondida (lou\_in02 77,37)

   :   ![Pólvora de Louyang01](images/f/f2/Pólvora_de_Louyang01.png)
2. A próxima etapa requer adicionar 8 pessoas em seu grupo, para isso dar certo, faça o seguinte:
   - Convide uma pessoa para seu [grupo](Grupo.md "Grupo");
   - O novo membro do [grupo](Grupo.md "Grupo") deve conversar com o [NPC](NPC.md "NPC");
   - Repita esse procedimento até que 8 pessoas estejam no seu [grupo](Grupo.md "Grupo").
     - Não adicione as 8 pessoas de uma vez.
3. Ao realizar o procedimento corretamente, você receberá 8 [Montes de Lama](http://www.divine-pride.net/database/item/7004/), 8 [Madeiras Queimadas](http://www.divine-pride.net/database/item/7068/) e 8 [Pedras Pome](http://www.divine-pride.net/database/item/7096/).
   - Coloque os itens no [carrinho](Usar Carrinho.md "Usar Carrinho") de algum membro do seu [grupo](Grupo.md "Grupo").
   - Nenhum membro do [grupo](Grupo.md "Grupo") deverá ter esses itens em seu inventário, do contrário a missão estará comprometida.
4. Dirija-se ao Palácio e procure pelo Produtor de Pólvora (lou\_in01 43,147)

   :   ![Pólvora de Louyang02](images/6/63/Pólvora_de_Louyang02.png)
5. O dono do [carrinho](Usar Carrinho.md "Usar Carrinho") deverá entregar 1 [Monte de Lama](http://www.divine-pride.net/database/item/7004/), 1 [Madeira Queimada](http://www.divine-pride.net/database/item/7068/) e 1 [Pedra Pome](http://www.divine-pride.net/database/item/7096/) para cada membro de seu [grupo](Grupo.md "Grupo").
6. Cada membro do [grupo](Grupo.md "Grupo") deve falar com o Produtor de Pólvora para fabricar uma [Pólvora](http://www.divine-pride.net/database/item/7204/).
   - Novamente, guarde todas as [Pólvoras](http://www.divine-pride.net/database/item/7204/) no [carrinho](Usar Carrinho.md "Usar Carrinho") e saiam do Palácio.
7. Retornem ao lugar da Pessoa Escondida, o líder do [grupo](Grupo.md "Grupo") deve entregar as 8 [Pólvoras](http://www.divine-pride.net/database/item/7204/) para o [NPC](NPC.md "NPC").
8. Como recompensa, o líder receberá 8 [Envelopes Vermelhos](http://www.divine-pride.net/database/item/668/), que devem ser distribuídos para os membros do [grupo](Grupo.md "Grupo").
   - Cada [Envelope Vermelho](http://www.divine-pride.net/database/item/668/) oferece uma quantia aleatória de [zeny](Zeny.md "Zeny"), entre 1.000 a 10.000.
