# Amigo da Natureza

|  |  |
| --- | --- |
| **Amigo da Natureza Amigo da Natureza** | |
| SemImagem | |
| **Nature's Friend / Amigo de la Naturaleza** | |
| **ID** | WH\_NATUREFRIENDLY (5325) |
| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Pré-requisitos** | |
| - Nenhum | |

Aprimora seu treinamento com aves de rapina.

Ao realizar ataques básicos, haverá uma chance de autoconjurar [Mergulho Aéreo](Mergulho Aéreo.md "Mergulho Aéreo").

A cada 3 pontos de [CON](CON.md "CON"), a chance aumenta em 1%.

Quanto maior o nível da habilidade, mais dano [Mergulho Aéreo](Mergulho Aéreo.md "Mergulho Aéreo") causa.

| Nv. | Fator | [Mergulho Aéreo](Mergulho Aéreo.md "Mergulho Aéreo") [Mergulho Aéreo](Mergulho Aéreo.md "Mergulho Aéreo") |
| --- | --- | --- |
| 1 | 0,2 | +10% |
| 2 | 0,4 | +20% |
| 3 | 0,6 | +30% |
| 4 | 0,8 | +40% |
| 5 | 1,0 | +50% |

### Fórmula

`Chance = [(Fator) × (CON ÷ 5)]%`

## Notas

- A chance de autoconjuração funciona somente se o usuário estiver equipado com um arco.
