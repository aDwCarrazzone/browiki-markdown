# Mudança de Classe: Kagerou e Oboro

| JQ ico Mudança de Classe: Kagerou e Oboro | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 99 |
| **Nv. de classe** | 70 |
| **Tipo** | Classe |
| **Classe** | [Ninja](Ninja.md "Ninja") |
| **[Itens](Itens.md "Itens")** | Ver tabela |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 13075.png 1 [Kurenai](http://www.divine-pride.net/database/item/13075/) ou  13311.png 1 [Sombra Huuma](http://www.divine-pride.net/database/item/13311/) |
| **Recompensa** | Mudança de Classe para [Kagerou](Kagerou.md "Kagerou") ou [Oboro](Oboro.md "Oboro") |

1. Dirija-se para a Guilda dos Ninjas (que\_ng 29,152)

   :   ![Qninja3](images/4/40/Qninja3.png)
2. Na casa, atravesse a parede falsa.

   :   ![Qninja4](images/1/12/Qninja4.png)

   - Se der certo, você ouvirá uma conversa.
   - Há momentos em que seu personagem sempre será teleportado para longe do local, isso acontece por conta de outros jogadores terem ativado a missão.
   - Quando isso ocorrer, você deve aguardar o [NPC](NPC.md "NPC") *sombra* sumir ou tentar contornar usando [Corte das Sombras](Corte das Sombras.md "Corte das Sombras") para pular uma célula mais distante dele.
3. Toque no desenho da Parede com Pintura (que\_ng 21,76) para teleportar.

   :   ![Qkageboro1](images/7/74/Qkageboro1.png)
4. Fale com Kage Gion.

   :   ![Qkageboro3](images/8/83/Qkageboro3.png)

   - Responda:
     - Eu sou um Ninja
     - Preciso dos seus ensinamentos
   - Seu personagem será mandado para fora da guilda, basta retornar e tocar na pintura de novo.
5. Fale com Kage Gion novamente para ouvir todas as explicações dos 4 testes.
   - A ordem dos testes pode ser feita à sua escolha.
   - Ao selecionar um teste, seu personagem será enviado para uma sala.

   :   :   **Teste de Conhecimento**
6. Responda as 10 perguntas de Cougar, devendo acertar 9, no mínimo, para passar.

   :   ![Qkageboro2](images/2/27/Qkageboro2.png)

   - As perguntas são muitas e aleatórias, recomendamos que você utilize *CTRL + F* para encontrá-las abaixo.
   - Caso alguma pergunta não esteja de acordo com a que está no jogo, pedimos que você corrija nesse guia.

:   :   :   **Teste de Sobrevivência**
:   7. Esse teste funciona como um tabuleiro de jogos, onde cada casa possui um [NPC](NPC.md "NPC") chamado *Visor*.

    :   :   ![Qkageboro4](images/2/25/Qkageboro4.png)
:   8. Aproxime-se do primeiro visor para iniciar o teste.

    :   - Você começará com 100 *P*ontos de *V*ida (PV).
        - Para andar no tabuleiro, os visores rolarão os dados, se tirar 3, por exemplo, você deve andar 3 casas.
        - Cada vez que rolar o dado, você perde 1 PV.
        - Cada *Visor* possui um efeito listado no quadro abaixo. Se tiver em branco, significa que não há efeitos.
        - Você passa de fase se chegar na casa 40.
        - Se seus PV acabarem antes de chegar na Casa 40, você deverá reiniciar o teste.

:   :   :   **Teste de Perícia**
:   9. Fale com Red Leopard Joe para aprender a forjar.

    :   :   ![Qkageboro6](images/8/8c/Qkageboro6.png)
:   10. Clique em um dos Forjadores. Escolha as opções de minerais para derreter.

    :   :   ![Qkageboro7](images/5/51/Qkageboro7.png)

        - Não há necessidade de ter todos os itens listados abaixo, apenas uma boa quantidade de um deles ou um pouco de cada. A quantidade pode variar:
          - ![1002.png](http://static.divine-pride.net/images/items/item/1002.png) 5 [Minérios de Ferro](http://www.divine-pride.net/database/item/1002/)
          - ![998.png](http://static.divine-pride.net/images/items/item/998.png) 10 [Ferros](http://www.divine-pride.net/database/item/998/)
          - ![999.png](http://static.divine-pride.net/images/items/item/999.png) 7 [Aços](http://www.divine-pride.net/database/item/999/)
          - ![756.png](http://static.divine-pride.net/images/items/item/756.png) 7 [Minérios de Oridecon](http://www.divine-pride.net/database/item/756/)
          - ![757.png](http://static.divine-pride.net/images/items/item/757.png) 7 [Minério de Elunium](http://www.divine-pride.net/database/item/757/)
          - ![1010.png](http://static.divine-pride.net/images/items/item/1010.png) 20 [Fracon](http://www.divine-pride.net/database/item/1010/)
        - A quantidade de Minérios gasto é aleatória.
:   11. Fique derretendo o mineral até o diálogo avisar que você precisa esfriar o metal em um molde, selecione Adaga ou Shuriken para continuar.

    :   - Selecione alternadamente as opções:
          - Polir a arma
          - Temperar a arma
        - Faça isso em torno de 5 vezes cada , pois há uma chance do metal quebrar se não estiver bem polido e temperado.
        - Se deu tudo certo, você receberá 1 [Adaga Ninja](http://www.divine-pride.net/database/item/13074/) ou 1 [Huuma Ninja](http://www.divine-pride.net/database/item/13312/).
        - Se estiver sem materiais, clique na Cômoda para voltar para [Amatsu](Amatsu.md "Amatsu").

        :   ![Qkageboro8](images/9/99/Qkageboro8.png)
:   12. Fale com Red Leopard Joe para a próxima etapa.
:   13. Equipe a arma forjada e clique em um dos Refinadores.

    :   :   ![Qkageboro8a](images/7/75/Qkageboro8a.png)

        - Seu objetivo é refinar a arma criada até o +7.
        - Não há necessidade de trazer minérios de [refinamento](Refinamento.md "Refinamento") e em casos de falha, o refino reduz em 1.
:   14. Mostre a arma +7 para Red Leopart Joe aprovar seu teste e enviar você de volta a Kage Gion.
:   15. Kage Gion dará o último teste: encontrar e eliminar o *Segredo **de** Família* numa pequena sala.

    :   :   ![Qkageboro10](images/6/61/Qkageboro10.png)

        - No lugar, existem vários [monstros](Monstros.md "Monstros") com nomes parecidos, se eliminar o errado, o teste reinicia.
        - Cuidado para não confundir com o [monstro](Monstro.md "Monstro") Segredo **da** Família.
:   16. Em caso de sucesso, volte a falar com Kage Gion para mudar de classe e receber a arma escolhida no teste de perícia.

    :   - Se optou por Adaga, você receberá um [Kurenai](http://www.divine-pride.net/database/item/13075/) com o seu nome.
        - Se optou por Shuriken, você receberá um [Sombra Huuma](http://www.divine-pride.net/database/item/13311/) com o seu nome.

- Ao chegar no nível 100, pegue sua *[Caixa Inicial](Conquista.md "Conquista")* para ajudar na sua nova jornada.
