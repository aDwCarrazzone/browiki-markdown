# Refletir Veneno

|  |  |
| --- | --- |
| **Refletir Veneno Refletir Veneno** | |
| Refletir Veneno | |
| **Poison React / Reacción Venenosa** | |
| **ID** | AS\_POISONREACT (139) |
| **Tipo** | Ativa |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 25 a 60 |
| **Duração** | 20 a 60 segundos |
| **Alvo** | Usuário |
| **Efeito negativo** | [Envenenamento](Envenenamento.md "Envenenamento") |
| **Ícone** | I Refletir Veneno |
| **Pré-requisitos** | |
| - [Envenenar Arma](Envenenar Arma.md "Envenenar Arma") [Envenenar Arma](Envenenar Arma.md "Envenenar Arma") Nv. 3 | |

Entra em um estado de reação.

Ao receber danos físicos da propriedade [Veneno](Veneno.md "Veneno"), tem chance de refletir o dano.

Em ataques físicos de outras propriedades, o usuário vai autoconjurar [Envenenar](Envenenar.md "Envenenar") em nível 5.

A habilidade termina quando a duração acabar ou atingir o limite máximo de autoconjurações.

| Nv. | Dano refletido | Limite | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| 1 | 130% | 1 | 20 segundos | 25 |
| 2 | 160% | 25 segundos | 30 |
| 3 | 190% | 2 | 30 segundos | 35 |
| 4 | 220% | 35 segundos | 40 |
| 5 | 250% | 3 | 40 segundos | 45 |
| 6 | 280% | 45 segundos | 50 |
| 7 | 310% | 4 | 50 segundos | 55 |
| 8 | 340% | 55 segundos | 60 |
| 9 | 370% | 5 | 60 segundos | 45 |
| 10 | 400% |

## Notas

- A chance de reflexão e autoconjuração é influenciada pelo nível aprendido de [Ataque Duplo](Ataque Duplo.md "Ataque Duplo").
