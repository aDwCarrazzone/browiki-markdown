# Missão Regional de Hugel

## Problema no Lago

| NQ ico Missão Regional de Hugel | |
| --- | --- |
| Se você não presta atenção no seu trabalho, acidentes podem acontecer.  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 120 |
| **Tipo** | Normal |
| **Caça** | 30 [Ferus Escarlates](http://www.divine-pride.net/database/monster/1714/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | Nenhum |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | Nenhum |

1. Fale com Nahadan (hugel 69,121)

   :   ![HugelRegional01](images/2/23/HugelRegional01.png)
2. Elimine 30 [Ferus Escarlates](http://www.divine-pride.net/database/monster/1714/), mas não retorne à [NPC](NPC.md "NPC") em [Hugel](Hugel.md "Hugel").
3. No [Lago do Abismo](http://www.divine-pride.net/database/map/hu_fild05/), fale com Nahadan (hu\_fild05 185,36)

   :   ![HugelRegional02](images/4/47/HugelRegional02.png)
4. Retorne a Nahadan em [Hugel](Hugel.md "Hugel") (hugel 69,121)
   - Agora você pode iniciar as missões diárias. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
   - Se esta for a sua primeira vez fazendo as diárias com a [NPC](NPC.md "NPC"), você receberá o dobro de [EXP](EXP.md "EXP").

| --- | --- |
| [Hugel](Hugel.md "Hugel") é o lar de pessoas misteriosas e com passados desconhecidos. Conheça um pouco da história desse velho pescador.  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 40 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 966.png [Carnes de Ostra](http://www.divine-pride.net/database/item/966/) |
| **Caça** | 10 [Molusco](http://www.divine-pride.net/database/monster/1074/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 8.400 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 8.380 |
| **[Itens](Itens.md "Itens")** | 12053.png 1 [Sopa de Ostra](http://www.divine-pride.net/database/item/12053/) |

1. Fale com o Pescador Raimundo (hugel 100,149)

   :   ![HugelRegional03](images/9/95/HugelRegional03.png)
2. Elimine 10 [Moluscos](http://www.divine-pride.net/database/monster/1074/) e traga 5 [Carnes de Ostra](http://www.divine-pride.net/database/item/966/).
   - Você receberá 8.400 de [EXP](EXP.md "EXP") base, 8.380 de [EXP](EXP.md "EXP") de classe e 1 [Sopa de Ostra](http://www.divine-pride.net/database/item/12053/).
   - Se esta for a sua primeira vez, você receberá o dobro de [EXP](EXP.md "EXP").
   - Você pode iniciar as missões diárias. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
