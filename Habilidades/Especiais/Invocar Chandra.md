Desperta o espírito da montanha, Chandra.

Concede bônus especiais para seu mestre.

Manter o elemental vivo drena [SP](../../Redirecionamentos/SP.md "SP") a cada 10 segundos.

Ao ser invocado, o elemental terá os seguintes [atributos](../../Sistema%20e%20Mec%C3%A2nicas/Atributos.md "Atributos"):

| Atributo | Fórmula |
| --- | --- |
| [HP](../../Redirecionamentos/HP.md "HP") | [(HP do usu. ÷ 3) + (INT × 10) + (Nv. de classe × 20)] × [(Nv. de Invocar + 2) ÷ 3] |
| [SP](../../Redirecionamentos/SP.md "SP") | (SP do usuário ÷ 4) |
| [ATQ](../../Redirecionamentos/ATQ.md "ATQ") (mín.) | [SP do usuário x (Nv. de Invocar ÷ 18)] + (1 ~ 100) |
| [ATQ](../../Redirecionamentos/ATQ.md "ATQ") (máx.) | [SP do usuário x (Nv. de Invocar ÷ 18)] |
| [ATQM](../../Redirecionamentos/ATQM.md "ATQM") | {Nv. de Invocar × [(INT ÷ 2) + (DES ÷ 4)} |
| [Velocidade de ataque](../../Redirecionamentos/Velocidade%20de%20Ataque.md "Velocidade de ataque") | {150 + [(DES ÷ 10) + (Nv. de Invocar × 3)]} |
| [DEF](../../Redirecionamentos/DEF.md "DEF") | {DEF do usuário + [Nv. de base ÷ (5 - Nv. de Invocar )]} |
| [DEFM](../../Redirecionamentos/DEFM.md "DEFM") | {DEFM do usuário + [INT ÷ (5 - Nv. de Invocar )]} |
| [Esquiva](../../Redirecionamentos/Esquiva.md "Esquiva") | {Esquiva do usuário + [Nv. de base ÷ (5 - Nv. de Invocar )]} |
| [Precisão](Precisão (Atributo).md "Precisão (Atributo)") | (Precisão do usuário + Nv. de base) |

## Notas

- Apenas 1 elemental pode estar ativo por vez.
- Os elementais não funcionam como os [homunculus](../../Hom%C3%BAnculos/Homunculus.md "Homunculus"), não havendo necessidade de programar uma [AI](../../Redirecionamentos/AI.md "AI") ou alimentá-los.
- As habilidades dos elementais são conjuradas de acordo com o modo definido por [Domínio Elemental](Dom%C3%ADnio%20Elemental.md "Domínio Elemental").
