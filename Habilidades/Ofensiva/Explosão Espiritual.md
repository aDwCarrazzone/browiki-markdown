| **Rampage Blast / Ráfaga de Explosiones** | |
| **ID** | SR\_RAMPAGEBLASTER (2332) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](../../Redirecionamentos/SP.md "SP")** | 100 |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 10 segundos |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Requisito** | 3 Esferas espirituais e em [Fúria Interior](../Suporte/F%C3%BAria%20Interior.md "Fúria Interior") |
| **Pré-requisitos** | |
| - [![Impacto Sísmico](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/07/Impacto_S%C3%ADsmico.png)](Impacto%20S%C3%ADsmico.md "Impacto Sísmico") [Impacto Sísmico](Impacto%20S%C3%ADsmico.md "Impacto Sísmico") Nv. 2 | |

Só pode ser usada se estiver em [Fúria Interior](../Suporte/F%C3%BAria%20Interior.md "Fúria Interior").

Causa dano físico à distância seu ao redor.

Oponentes [*impactados*](Impacto%20S%C3%ADsmico.md "Impacto Sísmico") recebem dano extra.

Custa 3 Esferas, mas em estado de [Fervor Espiritual](Punho%20Labareda.md#Fervor_Espiritual "Punho Labareda"), elas não são consumidas.

| Nv. | [ATQ](../../Redirecionamentos/ATQ.md "ATQ") | |
| --- | --- | --- |
| Normal | Impactado |
| 1 | 350% | 550% |
| 2 | 700% | 1.100% |
| 3 | 1.050% | 1.650% |
| 4 | 1.400% | 2.200% |
| 5 | 1.750% | 2.750% |

### Fórmula

`Normal = {[Dano base + (Nv. de Fúria × 200)] × (Nv. de base ÷ 100)}%`

`Impactado = {[Dano base + (Nv. de Fúria × 300)] × (Nv. de base ÷ 100)}%`
