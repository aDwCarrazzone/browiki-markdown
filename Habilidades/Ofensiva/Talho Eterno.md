Só pode ser usada em estado de [Reflexo](../Especiais/Reflexo%20de%20Combate.md "Reflexo de Combate").

Faz um corte no alvo, causando dano físico corpo a corpo.

Se usada novamente dentro de 3 segundos, o número de cortes acumula.

Você pode acumular até 5 cortes em um ataque.

Se [Carrasco Sombrio](../Suporte/Carrasco%20Sombrio.md "Carrasco Sombrio") estiver ativo, o dano é aumentado.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é a metade do [CRIT](../../Redirecionamentos/CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](../../Redirecionamentos/ATQ.md "ATQ") por corte | |
| --- | --- | --- |
| Normal | [Carrasco Sombrio](../Suporte/Carrasco%20Sombrio.md "Carrasco Sombrio") |
| 1 | 350% | 500% |
| 2 | 700% | 1.000% |
| 3 | 1.050% | 1.500% |
| 4 | 1.400% | 2.000% |
| 5 | 1.750% | 2.500% |

### Fórmula

Normal:  
`ATQ = [Dano base + (POD x 5)] x (Nv. de base ÷ 100)]%`

Com Carrasco Sombrio:  
`ATQ = [Dano base + (POD x 7)] x (Nv. de base ÷ 100)]%`
