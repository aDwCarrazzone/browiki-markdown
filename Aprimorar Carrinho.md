# Aprimorar Carrinho

|  |  |
| --- | --- |
| **Aprimorar Carrinho Aprimorar Carrinho** | |
| SemImagem | |
| **Cart Remodeling / Remodelación de Carreta** | |

| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Pré-requisitos** | |
| - Nenhum | |

Com conhecimento avançado, melhora seu carrinho.

Aumenta a capacidade de carga do carrinho.

Aumenta a [precisão](Precisão.md "Precisão") ao atacar com habilidades de carrinho.

| Nv. | Carga | [Precisão](Precisão.md "Precisão") |
| --- | --- | --- |
| 1 | +500 | +4 |
| 2 | +1.000 | +8 |
| 3 | +1.500 | +12 |
| 4 | +2.000 | +15 |
| 5 | +2.500 | +20 |
