# Suprimento de Armas

| NQ ico Suprimento de Armas | |
| --- | --- |
| Proteja suprimentos misteriosos da Corporação Rekenber que estão sendo atacados por pessoas estranhas no Aeroplano! | |
| Requisitos mínimos | |
| **Nv. de base** | 70 |
| **Tipo** | Normal |
| **Caça** | [Brutamontes](http://www.divine-pride.net/database/monster/1592/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 140.000 |
| **[Itens](Itens.md "Itens")** | 504.png 1 [Poção Branca](http://www.divine-pride.net/database/item/504/) |

1. Encontre o Velho (lighthalzen 141,162)

   :   ![SuprArm01](images/7/77/SuprArm01.png)
2. Aceite a proposta do Jovem Kazien (lhz\_in01 174,258)

   :   ![SuprArm02](images/b/be/SuprArm02.png)
3. Fale com um dos Guardas do Laboratório (einbroch 55,52)

   :   ![SprArm05](images/f/f2/SprArm05.png)

   - Selecione:
     - *Eu... eu volto mais tarde*
4. Reporte o acontecido à Kazien (lhz\_in01 174,258)

   :   ![SuprArm02](images/b/be/SuprArm02.png)
5. Procure Lyozien no aeroplano com destino de Izlude, Juno e Rachel (airplane\_01 96,48)

   :   ![SuprArm021](images/3/38/SuprArm021.png)
6. Entregue a encomenda ao Velhaco (izlude 186,57)

   :   ![Entregas5 new](images/a/a3/Entregas5_new.png)
7. Retorne a Lyozien (airplane\_01 96,48)

   :   ![SuprArm021](images/3/38/SuprArm021.png)
8. Receba [experiência](Experiência.md "Experiência") de Kazien.

   :   ![SuprArm02](images/b/be/SuprArm02.png)
9. Volte a Lyozien, que será atacado por 2 [Brutamontes](http://www.divine-pride.net/database/monster/1592/), mate-os para o [NPC](NPC.md "NPC") reaparecer.
10. Converse com Lyozien.

    :   ![SuprArm021](images/3/38/SuprArm021.png)
11. Reentregue as encomendas ao Velhaco, em [Izlude](Izlude.md "Izlude").

    :   ![Entregas5 new](images/a/a3/Entregas5_new.png)
12. Receba [experiência](Experiência.md "Experiência") de Kazien.

    :   ![SuprArm02](images/b/be/SuprArm02.png)
13. Retorne a Lyozien, que sofrerá um novo ataque.

    :   ![SuprArm021](images/3/38/SuprArm021.png)
14. Após derrotar os 3 [Brutamontes](http://www.divine-pride.net/database/monster/1592/), caminhe ao redor das coordenadas (airplane 99,47)

    :   ![SuprArm022](images/b/be/SuprArm022.png)
15. Quando Lyozien reaparecer, fale com ele e receba uma [Poção Branca](http://www.divine-pride.net/database/item/504/).

    :   ![SuprArm021](images/3/38/SuprArm021.png)
16. Fale com o Velhaco pela última vez.

    :   ![Entregas5 new](images/a/a3/Entregas5_new.png)
17. Receba [experiência](Experiência.md "Experiência") de Kazien.

    :   ![SuprArm02](images/b/be/SuprArm02.png)
18. Finalize a missão caminhando ao redor das coordenadas (lhz\_in01 189,248)

    :   ![SuprArm03](images/f/fd/SuprArm03.png)
