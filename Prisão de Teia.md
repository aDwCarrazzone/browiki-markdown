# Prisão de Teia

|  |  |
| --- | --- |

| **Fiber Lock / Bloqueo Arácnido** | |
| **ID** | PF\_SPIDERWEB (405) |
| **Tipo** | Ativa |
| **Níveis** | 1 |
| **[SP](SP.md "SP")** | 30 |
| **Duração** | 8 segundos |
| **Alvo** | Oponente |
| **Alcance** | 10 células |
| **Item** | 1025.png [Teia de Aranha](http://www.divine-pride.net/database/item/1025/) |
| **Pré-requisitos** | |
| - [Dragonologia](Dragonologia.md "Dragonologia") [Dragonologia](Dragonologia.md "Dragonologia") Nv. 4 | |

Imobiliza o alvo com uma teia de aranha, fornecendo [esquiva](Esquiva.md "Esquiva") -50.

Ataques físicos ou mágicos de [fogo](Fogo.md "Fogo") causarão 250% de dano adicional, mas libertarão o alvo da prisão.

## Notas

- Apenas 3 teias podem estar ativas simultaneamente.
- Não funciona em monstros [mortos-vivos](Mortos-vivos.md "Mortos-vivos").
- Dura 4 segundos em áreas de [PvP](PvP.md "PvP").
- Não pode ser usada na [Tumba da Honra](Tumba da Honra.md "Tumba da Honra").
