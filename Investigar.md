# Investigar

|  |  |
| --- | --- |
| **Investigar Investigar** | |
| SemImagem | |
| **Set Investigate** | |
| **ID** | - (-) |
| **Tipo** | Passiva |
| **Níveis** | 1 |
| **Pré-requisitos** | |
| Nenhum | |

Anula toda a [DEF](DEF.md "DEF") do alvo e aumenta o seu [ATQ](ATQ.md "ATQ") equivalente a metade da [DEF](DEF.md "DEF") anulada do oponente.

### Fórmula

`Dano = [(ATQ de atributos × 2) + (ATQ de equipamentos) + (DEF de equip. do alvo ÷ 2)]%`

## Notas

- O efeito de *Investigar* não é cumulativo. Se uma carta com o efeito é equipada em uma arma que também dá o efeito, valerá como se fosse apenas 1 efeito.
- Itens que ignoram a [DEF](DEF.md "DEF") terão prioridade.
- Investigar **não** afeta as seguintes habilidades:
  - [Sacrifício do Mártir](Sacrifício do Mártir.md "Sacrifício do Mártir")
  - [Faca Envenenada](Faca Envenenada.md "Faca Envenenada")
  - [Canhão de Prótons](Canhão de Prótons.md "Canhão de Prótons")
  - [Impacto Psíquico](Impacto Psíquico.md "Impacto Psíquico")
  - [Canhão](Canhão.md "Canhão")
  - [Sopro do Dragão](Sopro do Dragão.md "Sopro do Dragão")
  - [Bafo do Dragão](Bafo do Dragão.md "Bafo do Dragão")
