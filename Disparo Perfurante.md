# Disparo Perfurante

|  |  |
| --- | --- |
| **Disparo Perfurante Disparo Perfurante** | |
| Disparo Perfurante | |
| **Cannon Spear / Lanza Cañón** | |
| **ID** | LG\_CANNONSPEAR (2307) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 25 + (Nv. da habilidade × 5) |
| **Recarga** | 1,7 segundos |
| **Alvo** | Oponente |
| **Alcance** | 7 células |
| **Área** | 3x3 a 7x7 células |
| **Arma** | Lança |
| **Pré-requisitos** | |
| - [Estocada Precisa](Estocada Precisa.md "Estocada Precisa") [Estocada Precisa](Estocada Precisa.md "Estocada Precisa") Nv. 1 | |

Ergue a lança para causar dano físico a distância no alvo e em oponentes ao redor.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

Se [Cicatriz da Justiça](Lança da Justiça.md "Lança da Justiça") estiver ativo, o dano é maior.

| Nv. | [ATQ](ATQ.md "ATQ") | | Área | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| Normal | [Cicatriz](Lança da Justiça.md "Lança da Justiça") |
| 1 | 120% | 200% | 3x3 células | 30 |
| 2 | 240% | 400% | 35 |
| 3 | 360% | 600% | 5x5 células | 40 |
| 4 | 480% | 800% | 45 |
| 5 | 600% | 1.000% | 7x7 células | 50 |

### Fórmula

`Dano = {[Dano base + (FOR × Nv. da hab.)] × (Nv. de base ÷ 100)}%`
