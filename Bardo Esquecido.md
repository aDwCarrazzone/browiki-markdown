# Bardo Esquecido

| NQ ico Bardo Esquecido | |
| --- | --- |
| Um bardo andarilho não se lembra de um verso de uma das suas músicas. Caberá a você ajudá-lo a lembrar como era a canção original! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 611.png 1 [Lupa](http://www.divine-pride.net/database/item/611/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | Ver tabela |

1. Fale com o Bardo (geffen 132,38)

   :   ![O Bardo de Geffen01](images/7/71/O_Bardo_de_Geffen01.png)

   - Peça para ele tocar uma canção divertida. Eventualmente, ele pode esquecer do 8º trecho da canção, logo, você deve se oferecer para ajudá-lo.
2. Converse com o Bardo (payon 180,172)

   :   ![O Bardo de Geffen02](images/5/5b/O_Bardo_de_Geffen02.png)

   - Digite o texto abaixo, parágrafo por parágrafo, e fale com ele novamente.
     - `At One, I Fall in Love`
     - `Minty Errende`
3. Volte ao Bardo em [Geffen](Geffen.md "Geffen") e diga que Kino Kitty mudou a canção.
4. Fale com o Bardo (morocc 134,110)

   :   ![O Bardo de Geffen03](images/8/83/O_Bardo_de_Geffen03.png)

   - Selecione:
     - Desculpe, interrompê-lo
     - Estou procurando Kino Gato
5. Volte ao Bardo de [Geffen](Geffen.md "Geffen").
6. Mostre sua mão esquerda para o Velho (yuno 264,87)

   :   ![O Bardo de Geffen04](images/9/93/O_Bardo_de_Geffen04.png)
7. Leia o Livro Velho até encontrar a canção.

   :   ![O Bardo de Geffen05](images/e/ec/O_Bardo_de_Geffen05.png)
8. Volte ao Bardo de [Morroc](Morroc.md "Morroc").
   - Selecione:
     - Tá aqui
     - Não, não me importo.
9. Retorne ao Bardo de [Geffen](Geffen.md "Geffen") para receber sua recompensa em [experiência](Experiência.md "Experiência").
