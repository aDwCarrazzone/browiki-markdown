# Invasão ao Aeroplano

| SC ico Invasão ao Aeroplano | |
| --- | --- |
| O funcionamento de um aeroplano sempre foi alvo da curiosidade de muitas pessoas. Nem todas querem apenas "saber", existem outras que querem "roubar" esse conhecimento para si. Sobreviva ao ataque contra o Aeroplano e tente acalmar o Capitão, que enlouqueceu. | |
| Requisitos mínimos | |
| **Nv. de base** | 125 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| Skilldelayinfo **Retorno** | Reseta 04h da manhã |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa **apenas** |
| Premiação | |

| **[EXP base](Experiência de Base.md "Experiência de Base")** | Ver tabela |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | Ver tabela |

1. No [Espaço Dimensional](Espaço Dimensional.md "Espaço Dimensional"), acesse a [instância](Instância.md "Instância") pelo Salta-Dimensões (dali02 142,81)

   :   ![InvasaoAeroplano01](images/2/28/InvasaoAeroplano01.png)
2. Converse com Iris próxima à cabine do capitão.

   :   ![InvasaoAeroplano02](images/f/f5/InvasaoAeroplano02.png)

   - Um ataque será anunciado, não adianta atacar os [monstros](Monstros.md "Monstros") aqui, eles são apenas [NPCs](NPCs.md "NPCs")
   - Use o portal para acessar o interior do aeroplano.
   - Seu objetivo será eliminar os [monstros](Monstros.md "Monstros") que surgem.
     - [Beholder Invasor](http://www.divine-pride.net/database/monster/3177/), [Giro-Goblin Invasor](http://www.divine-pride.net/database/monster/3175/), [Gremlin Invasor](http://www.divine-pride.net/database/monster/3176/), [Wyvern Azul](http://www.divine-pride.net/database/monster/3179/) e [Wyvern Dourado](http://www.divine-pride.net/database/monster/3178/).
   - Para os [Arcebispos](Arcebispos.md "Arcebispos") e evoluções, uma versão mais fraca dos [monstros](Monstros.md "Monstros") aparecerão.
     - [Beholder Invasor](http://www.divine-pride.net/database/monster/3184/), [Giro-Goblin Invasor](http://www.divine-pride.net/database/monster/3182/), [Gremlin Invasor](http://www.divine-pride.net/database/monster/3183/), [Wyvern Azul](http://www.divine-pride.net/database/monster/3186/) e [Wyvern Dourado](http://www.divine-pride.net/database/monster/3185/).
3. Elimine todos os monstros e desça as escadas.
4. No andar inferior, derrote todos os monstros e fale com Iris em seguida.

   :   ![InvasaoAeroplano03](images/2/29/InvasaoAeroplano03.png)
5. Após falar com Iris, suba as escadas e elimine todos os monstros.
6. Suba as escadas e fale com Fenrir para iniciar a batalha.
   - Você enfrentará um [Wyvern Gigante](http://www.divine-pride.net/database/monster/3180/).
   - Para os [Arcebispos](Arcebispos.md "Arcebispos") e evoluções, um [Wyvern Gigante](http://www.divine-pride.net/database/monster/3187/) diferente aparece.
7. Converse com Iris até o Capitão Ferlock surtar.
   - Enfrente o [Capitão Ferlock](http://www.divine-pride.net/database/monster/3181/).
   - Para os [Arcebispos](Arcebispos.md "Arcebispos") e evoluções, um [Capitão Ferlock](http://www.divine-pride.net/database/monster/3188/) mais fraco aparece.
8. Ao fim da luta, fale com Chaos para receber sua [EXP](EXP.md "EXP") e sair pelo portal na ponte.

   :   ![InvasaoAeroplano04](images/f/ff/InvasaoAeroplano04.png)
