# Episódios

[Ragnarök Online](Ragnarök Online.md "Ragnarök Online") é um jogo que está sempre recebendo novas atualizações, chamadas de "episódios" que trazem novidades como [mapas](Mapas.md "Mapas"), [itens](Itens.md "Itens"), [monstros](Monstros.md "Monstros"), etc.

Por ser desenvolvido pela *Gravity*, os episódios lançados no servidor Coreano são referência para saber as próximas atualizações.

Na América Latina, o jogo foi lançado em 28 de Maio de 2025 no episódio *14.2 Eclage*.
