# Purificação do Santuário

| SC ico Purificação do Santuário | |
| --- | --- |
| A lenda de que o último rei de Glastheim sucumbiu à loucura por gostar de magia das trevas pode ser falsa... acompanhe Oscar, o Dragão-do-Tempo, nessa viagem ao passado e descubra a verdade por trás dos fatos. | |
| Requisitos mínimos | |
| **Nv. de base** | 170 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| Skilldelayinfo **Retorno** | Reseta 4 da manhã após 3 dias |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| **Quests anteriores** | [Edda de Glastheim](Edda de Glastheim.md "Edda de Glastheim") |
| Premiação | |
| **Chefe** | Nu tein |
| **[Itens](Itens.md "Itens")** | 1000405.png x [xx](http://www.divine-pride.net/database/item/1000405/) |

1. No primeiro andar, você deve abrir caminho derrotando os Guardas de Rachel.
2. Portais verdes aparecerão quando você tiver eliminado todos os guardas do trajeto.
3. Siga o caminho indicado em amarelo para chegar ao templo.
4. Para o segundo andar, encontre o Guarda na estrela laranja. Ele informará um código usado para trancar o templo.
5. Siga o caminho em vermelho e elimine todos os Heart Hunters.
6. Nos locais marcados em verde, você encontrará um NPC ao lado da porta onde deve inserir o código para trancá-la.
7. Após concluir, fale com o NPC do lado de fora da instância para receber dois conjuntos de recompensas:

- Primeira recompensa:
  - 4 Fragmentos de Ametista
  - Experiência Base: 12.681.245
  - Experiência de Classe: 3.000.000
- Segunda recompensa:
  - 4 Fragmentos de Ametista
  - Experiência Base: 9.222.714
  - Experiência de Classe: 3.000.000
