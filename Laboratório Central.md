# Laboratório Central

| SC ico Laboratório Central | |
| --- | --- |
| Fazer parte de um experimento perigoso pode trazer consequências muito graves quando o computador principal cria vida própria e controla os próprios comandos. | |
| Requisitos mínimos | |
| **Nv. de base** | 140 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| Skilldelayinfo **Retorno** | Reseta 4h da manhã |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| **Quests anteriores** | [Fantasmagórica](Fantasmagórica.md "Fantasmagórica") |
| Premiação | |
| **Chefe** | Aleatório |

1. Fale com o Pesquisador (verus01 149,155)

   :   ![PesquiCiviliza](images/3/37/PesquiCiviliza.png)
2. Entre na fila da instância clicando no Transportador.

   :   ![VerusTranspo](images/7/74/VerusTranspo.png)
3. Fale com o Cientista para iniciar a história.

   :   ![Centralab1](images/5/5a/Centralab1.png)
4. Pegue o **código do dia** com o Pesquisador Sênior.

   :   ![Centralab2](images/2/21/Centralab2.png)
5. Atrás do Cientista, haverão 8 Interruptores onde o líder deverá ligar ou desligá-los.

   :   ![Centralab3](images/e/ea/Centralab3.png)

   - Seu objetivo é ligar os interruptores certos para combinar com o **código do dia**.
   - O **código do dia** é aleatório para cada ida na [instância](Instância.md "Instância").
   - Cada interruptor ligado adiciona um número no código do dia, os números são:
     - Interruptor 1: 128
     - Interruptor 2: 64
     - Interruptor 3: 32
     - Interruptor 4: 16
     - Interruptor 5: 8
     - Interruptor 6: 4
     - Interruptor 7: 2
     - Interruptor 8: 1
   - Por exemplo, se o código do dia for **80**, basta ligar os interruptores 2 e 4, pois `64 + 16 = 80`
6. Com o código correto, clique no Interruptor Geral.

   :   ![Centralab4](images/2/27/Centralab4.png)
7. Dirija-se para o centro do Laboratório e se aproxime dos cientistas para iniciar a história.

   :   ![Centralab5](images/b/bd/Centralab5.png)

   - Durante a progressão da história, haverão 3 chances de sair um [monstro](Monstro.md "Monstro") aleatório.
   - Ao eliminar o terceiro monstro, a [instância](Instância.md "Instância") termina e você poderá sair pelo portal do lado oposto ao qual você entrou.
   - A seguir, a lista de monstros que podem ser invocados aleatoriamente:
