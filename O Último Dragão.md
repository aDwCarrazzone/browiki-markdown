# O Último Dragão

|  |  |
| --- | --- |
| **O Último Dragão O Último Dragão** | |
| O Último Dragão | |
| **Raging Thrust / Remate Furioso** | |
| **ID** | MO\_COMBOFINISH (273) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | (Nv. da habilidade + 2) |
| **Pós-conjuração** | 0,3 segundos |
| **Alvo** | Oponente |
| **Alcance** | 1 célula |
| **Consome** | 1 Esfera espiritual |
| **Pré-requisitos** | |
| - [Combo Quádruplo](Combo Quádruplo.md "Combo Quádruplo") [Combo Quádruplo](Combo Quádruplo.md "Combo Quádruplo") Nv. 3 | |

Só pode ser usada após 0,3 segundos depois de ativar [Combo Quádruplo](Combo Quádruplo.md "Combo Quádruplo").

Consome 1 Esfera Espirtual para causar dano físico corpo a corpo no oponente.

Se estiver em [Fúria Interior](Fúria Interior.md "Fúria Interior") e com 1 Esfera Espiritual sobrando, pode usar [Punho Supremo de Asura](Punho Supremo de Asura.md "Punho Supremo de Asura").

| --- | --- | --- |
| 1 | 600% | 3 |
| 2 | 750% | 4 |
| 3 | 900% | 5 |
| 4 | 1.050% | 6 |
| 5 | 1.200% | 7 |

### Fórmula

`ATQ = {[Dano base] + (FOR × 5)}%`
