# Mudança de Classe: Cavaleiros

| --- | --- |
| Requisitos mínimos | |

| **Tipo** | Classe |

1. Inicie sua jornada se alistando com o Chefe dos Cavaleiros. (prt\_in 88,101)

   :   ![Mudança de Classe- Cavaleiro01](images/1/17/Mudança_de_Classe-_Cavaleiro01.png)
2. Traga os itens pedidos pelo Cavaleiro Dedicado (prt\_in 75,107). Se seu nível de classe for 50, não há necessidade de entrega.

   :   ![Mudança de Classe- Cavaleiro02](images/7/73/Mudança_de_Classe-_Cavaleiro02.png)

   - Preste atenção ao diálogo, pois existem duas possibilidades:
     - Possibilidade 1:
       - ![1040.png](http://static.divine-pride.net/images/items/item/1040.png) 5 [Bigode de Anão Ancião](http://www.divine-pride.net/database/item/1040/)
       - ![7006.png](http://static.divine-pride.net/images/items/item/7006.png) 5 [Asa de Morcego Vermelha](http://www.divine-pride.net/database/item/7006/)
       - ![931.png](http://static.divine-pride.net/images/items/item/931.png) 5 [Amuleto dos Orcs](http://www.divine-pride.net/database/item/931/)
       - ![1057.png](http://static.divine-pride.net/images/items/item/1057.png) 5 [Pó de Traça](http://www.divine-pride.net/database/item/1057/)
       - ![903.png](http://static.divine-pride.net/images/items/item/903.png) 5 [Língua de Réptil](http://www.divine-pride.net/database/item/903/)
       - ![1028.png](http://static.divine-pride.net/images/items/item/1028.png) 5 [Crina](http://www.divine-pride.net/database/item/1028/)
     - Possibilidade 2:
       - ![1042.png](http://static.divine-pride.net/images/items/item/1042.png) 5 [Pata de Inseto](http://www.divine-pride.net/database/item/1042/)
       - ![950.png](http://static.divine-pride.net/images/items/item/950.png) 5 [Coração de Sereia](http://www.divine-pride.net/database/item/950/)
       - ![1032.png](http://static.divine-pride.net/images/items/item/1032.png) 5 [Flor de Planta Carnívora](http://www.divine-pride.net/database/item/1032/)
       - ![966.png](http://static.divine-pride.net/images/items/item/966.png) 5 [Carne de Ostra](http://www.divine-pride.net/database/item/966/)
       - ![7031.png](http://static.divine-pride.net/images/items/item/7031.png) 5 [Frigideira Velha](http://www.divine-pride.net/database/item/7031/)
       - ![946.png](http://static.divine-pride.net/images/items/item/946.png) 5 [Casco de Caramujo](http://www.divine-pride.net/database/item/946/)
3. Responda corretamente as 8 perguntas do Cavaleiro Honroso. (prt\_in 71,91)

   :   ![Mudança de Classe- Cavaleiro03](images/d/d4/Mudança_de_Classe-_Cavaleiro03.png)

:   :   | Perguntas | Respostas |
        | --- | --- |
        | Qual das seguintes armas não é afetada pela habilidade Rapidez com Duas Mãos? | Espada Flamberge |
        | Qual das habilidades a seguir não é necessária para aprender o Impacto de Tyr? | Provocar Nv. 10 |
        | Quais habilidades não são necessárias para aprender Brandir Lança? | Lança-bumerangue Nv. 3 |
        | Qual dos atributos a seguir é capaz de ferir um Pesadelo, um inimigo com o atributo Fantasma? | Zéfiro |
        | Qual é a porcentagem da velocidade de ataque normal que você terá depois de aprender a habilidade Perícia em Montaria Nível 3? | 80 % da velocidade de ataque normal |
        | O que você deve fazer ao encontrar um Aprendiz pedindo ajuda na cidade? | Indicar ao Aprendiz uma boa área de caça |
        | Agora, como você deve agir se fizer parte de um grupo? | Proteger a todos à frente da batalha. |
        | Qual é o valor mais importante que um Cavaleiro deve ter? | Honra |

:   4. Inicie o teste prático com o Cavaleiro das Lutas (prt\_in 79,94). Seu objetivo é derrotar todos os monstros em 3 minutos.

    :   ![Mudança de Classe- Cavaleiro04](images/2/28/Mudança_de_Classe-_Cavaleiro04.png)

    - São 3 salas de teste:
      - Sala Terra: 2 [Andres](http://www.divine-pride.net/database/monster/1114/), 2 [Pieres](http://www.divine-pride.net/database/monster/1114/), 2 [Deniros](http://www.divine-pride.net/database/monster/1114/), 2 [Argos](http://www.divine-pride.net/database/monster/1114/)
      - Sala Fogo: 2 [Frilldoras](http://www.divine-pride.net/database/monster/1114/), 4 [Drainliares](http://www.divine-pride.net/database/monster/1114/)
      - Sala Humanoides: 5 [Goblins](http://www.divine-pride.net/database/monster/1280/), 2 [Goblins a Vapor](http://www.divine-pride.net/database/monster/1280/)
:   5. Confirme o sucesso do seu teste com o Cavaleiro das Lutas (prt\_in 79,94)
:   6. Dirija-se até a Cavaleira Educada (prt\_in 69,107) para responder 10 questões, devendo acertar pelo menos 8. Nesse teste, existem **2 opções** de respostas certas.

    :   ![Mudança de Classe- Cavaleiro07](images/c/c0/Mudança_de_Classe-_Cavaleiro07.png)

:   :   | Perguntas | Respostas (1 ou 2) |
        | --- | --- |
        | Como você demonstraria suas intenções? | Abriria uma sala de bate-papo e aguardaria |
        | Procuraria pessoas que estivessem procurando Cavaleiros |
        | Você chega até o Nível 4 das Pirâmides com o seu grupo. O que você deve fazer? | Avaliar a situação e planejar os nossos próximos passos. |
        | Liderar o grupo em um avanço cuidadoso. |
        | O que você deve fazer? | Impedir que os monstros cheguem até o grupo. |
        | Defender os integrantes do grupo enquanto eles recuam |
        | A pessoa pede educadamente por ajuda. O que você deve fazer? | Pedir que o Sacerdote do grupo a ajude. |
        | Mas vocês encontraram um item raro durante a batalha. O que você deve fazer? | Dar o item para o integrante do grupo que realmente o merece. |
        | Decidir qual integrante do grupo deve ficar com o item. |
        | O que você vai fazer para vender o seu item? | Abrir uma sala de bate-papo e aguardar. |
        | Perguntar se há alguém interessado no meu item. |
        | Enquanto aguarda, alguém se aproxima e implora por itens e um pouco de zenys. O que você faz? | Dou dicas de um bom lugar para caçar |
        | Mas acaba encontrando alguém que está perdido. O que você deve fazer? | Dizer à pessoa como chegar à saída |
        | Levar a pessoa até a saída |
        | De repente, um Sacerdote cruza o seu caminho. Como você pediria ao Sacerdote que ele cure você? | Por favor, seria possível me curar? |
        | No caminho, você encontra um item raro no meio da rua. O que você deve fazer? | Procurar nas proximidades pelo dono do item |
        | Ignorar o item e continuar o meu caminho. |

:   7. Inicie um teste de paciência com o Cavaleiro Devotado (prt\_in 70,99)

    :   ![Mudança de Classe- Cavaleiro08](images/2/20/Mudança_de_Classe-_Cavaleiro08.png)

    1. Seu objetivo é permanecer em uma sala por 4 minutos **sem** atacar os monstros.
    2. Passado o tempo, confirme o resultado com o Cavaleiro Devotado (prt\_in 70,99)
:   8. Responda um questionário com o Cavaleiro Modesto (prt\_in 87,92).

    :   ![Mudança de Classe- Cavaleiro011](images/5/59/Mudança_de_Classe-_Cavaleiro011.png)

:   :   | Perguntas | Respostas |
        | --- | --- |
        | Primeiro... Por que você decidiu se tornar um Cavaleiro? | Para me tornar forte... |
        | Por que você acha que é tão bom ganhar força como um Cavaleiro? | Eu poderei proteger outras pessoas. |
        | Se você se tornasse um Cavaleiro neste instante, qual a primeira coisa que faria? | Há pessoas que esperam por mim. |
        | Quem está esperando você? | Meus amigos. |

:   9. Retorne ao Chefe dos Cavaleiros (prt\_in 88,101) para mudar sua classe e receber 7 [Poções do Despertar](http://www.divine-pride.net/database/item/656/).

    :   ![Mudança de Classe- Cavaleiro01](images/1/17/Mudança_de_Classe-_Cavaleiro01.png)
