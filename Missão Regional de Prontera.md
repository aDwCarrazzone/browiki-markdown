# Missão Regional de Prontera

## Cavalaria de Prontera

| NQ ico Missão Regional de Prontera | |
| --- | --- |
| Para proteger a cidade da expansão de outras raças agressivas, a Cavalaria de [Prontera](Prontera.md "Prontera") envia tropas para monitorar a Vila dos Orcs e a Vila Goblin.  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 40 |
| **Tipo** | Normal |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 4.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 3.000 |

1. Aceite a missão do Quadro de Caça (prontera 59,339)

   :   ![PronteraRegional01](images/8/8c/PronteraRegional01.png)
2. Em seguida, fale com a Escrivã ao lado. Ela deverá entregar 1 [Suprimentos da Cavalaria](http://www.divine-pride.net/database/item/25077/).
   - Se você perder o item, ela cobrará 200 [zeny](Zeny.md "Zeny") para lhe dar um novo.
3. Dirija-se à Vila dos Orcs e entregue os Suprimentos para o Líder da Tropa (gef\_fild10 78,340)

   :   ![PronteraRegional02a](images/e/e0/PronteraRegional02a.png)
4. Assim que entregar os Suprimentos, você já poderá iniciar as missões diárias de caçada dos Orcs, se possuir nível 50 ou maior. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
5. Na Vila Goblin e entregue os Suprimentos para o Vice-Líder da Tropa (gef\_fild11 128,351)

   :   ![PronteraRegional03](images/e/e0/PronteraRegional03.png)
6. Você já poderá iniciar as missões diárias de caçada dos Goblins. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
7. Essa missão de entrega de Suprimentos não está no [Grupo do Éden](Grupo do Éden.md "Grupo do Éden") e é resetada às 4 horas da manhã.
   - Você receberá 4.000 de [EXP](EXP.md "EXP") de base e 3.000 de [EXP](EXP.md "EXP") de classe.
   - Se esta for sua primeira vez, a [EXP](EXP.md "EXP") será dobrada.
8. Enquanto as missões de caçada estiverem ativas, você pode falar com alguns membros da Cavalaria nas Vilas para receber uma cura.
   - Vila dos Orcs (gef\_fild10 215,194)
   - Vila dos Goblins (gef\_fild11 177,227)
   - Caverna dos Orcs (orcsdun01 161,135)

## Catedral de Prontera

| NQ ico Missão Regional de Prontera | |
| --- | --- |
| Para provar sua fé, a Catedral envia missionários exorcistas para [Glastheim](Glastheim.md "Glastheim"), a antiga capital do reino. Entretanto, a falta de respostas de uma expedição tem deixado um membro da igreja preocupado com a situação.  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 70 |
| **Tipo** | Normal |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 5.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 3.000 |

1. Aceite ajudar o Arcebispo Laerte (prontera 235,311). Fale com ele novamente para se teleportar para [Glastheim](Glastheim.md "Glastheim").

   :   ![PronteraRegional04](images/9/93/PronteraRegional04.png)
2. Fale com Cláudio (gef\_fild10 78,340)

   :   ![PronteraRegional05](images/4/40/PronteraRegional05.png)

   - Você receberá 5.000 de [EXP](EXP.md "EXP") de base e 3.000 de [EXP](EXP.md "EXP") de classe.
3. Assim que terminar a conversa, você já poderá iniciar as missões diárias. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
4. Enquanto as missões de caçada estiverem ativas, você pode falar com alguns membros da Catedral para receber uma cura e abençoar sua arma com propriedade [Sagrado](Sagrado.md "Sagrado"). Basta levar 1 [Água Benta](http://www.divine-pride.net/database/item/909/).
   - Abadia (gl\_church 175,72)
   - Cemitério (gl\_chyard 144,119)
