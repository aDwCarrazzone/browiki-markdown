# Virar Tatami

|  |  |
| --- | --- |
| **Virar Tatami Virar Tatami** | |
| Virar Tatami | |
| **Flip Tatami / Voltear Tatami** | |
| **ID** | NJ\_TATAMIGAESHI (527) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 15 |
| **Pós-conjuração** | 3 segundos |
| **Duração** | 3 segundos |
| **Alvo** | Usuário |
| **Alcance** | até 3 células |
| **Empurra** | 5 células |
| **Área** | 3x3 células |
| **Pré-requisitos** | |
| Nenhum | |

Usa o tatami causar dano em alvos próximos e empurrá-los 5 células para trás.

No momento do uso, um [Escudo Sagrado](Escudo Sagrado.md "Escudo Sagrado") é ativado, bloqueando danos físicos a distância nos próximos 3 segundos.

| Nv. | [ATQ](ATQ.md "ATQ") | Alcance |
| --- | --- | --- |
| 1 | 110% | 1 célula |
| 2 | 120% | 2 células |
| 3 | 130% |
| 4 | 140% | 3 células |
| 5 | 150% |

## Notas

- Não atinge oponentes na mesma célula do personagem.
