# Intimidar

|  |  |
| --- | --- |
| **Intimidar Intimidar** | |
| Intimidar | |
| **Power of Flock / Gato Imitador** | |
| **ID** | SU\_POWEROFFLOCK (5045) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | 1 + (Nv. da habilidade - 5) seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 100 segundos |
| **Alvo** | Usuário |
| **Área** | 7x7 a 29x29 |
| **Efeito negativo** | [Medo](Medo.md "Medo") e [Congelamento](Congelamento.md "Congelamento") |
| **Pré-requisitos** | |
| [Riscar Fósforo](Riscar Fósforo.md "Riscar Fósforo") [Riscar Fósforo](Riscar Fósforo.md "Riscar Fósforo") Nv. 5 | |

Intimida os oponentes ao convocar sua gangue, tendo a chance de infligir [Medo](Medo.md "Medo") e [Congelamento](Congelamento.md "Congelamento") na área.

| Nv. | Área | Conjuração variável |
| --- | --- | --- |
| 1 | 7x7 células | 4 segundos |
| 2 | 9x9 células | 3 segundos |
| 3 | 11x11 células | 2 segundos |
| 4 | 13x13 células | 1 segundo |
| 5 | 29x29 células | 0 segundos |

### Fórmula

- Chance de inflição = `[20 + (DES do usuário ÷ 8) + (SOR do usuário ÷ 5)]%`

## Notas

- O oponente pode sofrer os dois efeitos ao mesmo tempo.
