# Rumores Sérios

| EP ico Rumores Sérios | |
| --- | --- |
| Há rumores de que os experimentos com humanos, que pensávamos ter parado, prosseguem. O que está havendo? É preciso verificar isso. | |
| Requisitos mínimos | |
| **Nv. de base** | 90 - 95 |
| **Tipo** | Episódio |
| **Caça** | [MVPs](MVP.md "MVP") da Instância |
| **Quests anteriores** | [A Empresa](A Empresa.md "A Empresa") |
| Premiação | |
| **[Itens](Itens.md "Itens")** | Ver tabela |

1. Converse com Giacomo Girolamo (lighthalzen 303,303)

   :   ![Investigando os Rumores01](images/6/6e/Investigando_os_Rumores01.png)
2. Ao entrar no portal para o quarto andar do [Laboratório de Somatologia](http://www.divine-pride.net/database/map/lhz_dun04/), um diálogo surgirá (lhz\_dun03 239,78)

   :   ![Investigando os Rumores03](images/d/d8/Investigando_os_Rumores03.png)
3. Ative o diálogo descendo as escadas. Prossiga pelo caminho até encontrar Wolfchev.

   :   ![Investigando os Rumores04](images/4/44/Investigando_os_Rumores04.png)

   - Wolfchev fugirá de você algumas vezes, será necessário falar com ele sempre que o encontrar.
   - Na última fuga, algumas [Cobaias Humanas](http://www.divine-pride.net/database/monster/2244/) atacarão.
   - Após eliminá-las, continue insistindo o diálogo com Wolfchev até ele acordar.
4. Siga ao norte do mapa para ouvir uma mensagem e voltar ao [Laboratório de Somatologia](http://www.divine-pride.net/database/map/lhz_dun04/).
5. Reserve a [instância](Instância.md "Instância") com o Pesquisador (lhz\_dun04 151,276)

   :   ![Investigando os Rumores07](images/6/6e/Investigando_os_Rumores07.png)
6. Acesse o [Laboratório de Wolfchev](Laboratório de Wolfchev.md "Laboratório de Wolfchev") pela Entrada do Laboratório.

   :   ![Investigando os Rumores08](images/b/bc/Investigando_os_Rumores08.png)

   - Sua missão é eliminar os seguintes [MVPs](MVP.md "MVP"):
     - [Algoz Eremes](http://www.divine-pride.net/database/monster/1647/)
     - [Mestre-Ferreiro Howard](http://www.divine-pride.net/database/monster/1648/)
     - [Lorde Seyren](http://www.divine-pride.net/database/monster/1646/)
     - [Menestrel Alphoccio](http://www.divine-pride.net/database/monster/2240/)
     - [Criador Flamel](http://www.divine-pride.net/database/monster/2236/)
     - [Paladino Randel](http://www.divine-pride.net/database/monster/2235/)
     - [Mestre Chen](http://www.divine-pride.net/database/monster/2238/)
     - [Sumo Sacerdotisa Margaretha](http://www.divine-pride.net/database/monster/1649/)
     - [Atiradora de Elite Cecil](http://www.divine-pride.net/database/monster/1650/)
     - [Arquimaga Kathryne](http://www.divine-pride.net/database/monster/1651/)
     - [Cigana Trentini](http://www.divine-pride.net/database/monster/2241/)
     - [Professora Celia](http://www.divine-pride.net/database/monster/2237/)
     - [Desordeira Gertie](http://www.divine-pride.net/database/monster/2239/)
7. Após ter completar a missão, fale com Wolfchev no final da [instância](Instância.md "Instância") para receber sua recompensa:
   - A premiação abaixo é dada aleatoriamente:
     - ![6471.png](http://static.divine-pride.net/images/items/item/6471.png) 10 a 20 [Frio dos Mortos](http://www.divine-pride.net/database/item/6471/) ou
     - ![6470.png](http://static.divine-pride.net/images/items/item/6470.png) 10 a 20 [Sede de Sangue](http://www.divine-pride.net/database/item/6470/) ou
     - ![6469.png](http://static.divine-pride.net/images/items/item/6469.png) 30 a 60 [Vontade do Guerreiro](http://www.divine-pride.net/database/item/6469/) ou
     - 1 [Arma ou Equipamento Biológico](Lighthalzen.md#Fabricação "Lighthalzen") aleatório
8. Você pode falar com o [NPC](NPC.md "NPC") novamente para repetir a missão de caçada e receber a recompensa por ter eliminado todos os [MVPs](MVP.md "MVP").
