# Mudança de Classe: Mercadores

| JQ ico Mudança de Classe: Mercadores | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Nv. de classe** | 1 |
| **Tipo** | Classe |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 1381.png 1 [Machado de Batalha](http://www.divine-pride.net/database/item/1381/) |
| **Recompensa** | Muda a classe para [Mercador](Mercador.md "Mercador") |

1. No 2º andar da [Academia Criatura](Academia Criatura.md "Academia Criatura"), fale com Salim Hamid (iz\_ac02 50,169)

   :   ![Aca Merchant01](images/7/7e/Aca_Merchant01.png)

   - Se você tiver alcançado o nível de classe 10, fale com Salim. Selecione "*Sim*" para seguir a [Alberta](Alberta.md "Alberta").
   - Caso contrário, matricule-se no curso dele. Ele pede que você junte 100 [zenys](Zenys.md "Zenys").
   - Ele não tomará seus [zenys](Zenys.md "Zenys"), mas dará 1 [Manual do Mercador](http://www.divine-pride.net/database/item/2823/).
   - Converse com Salim Hamid até pedir que você fale com o Mordomo Aidinn.
2. Converse com o Mordomo Aidinn.

   :   ![Aca Merchant02](images/2/29/Aca_Merchant02.png)

   - Após a explicação da classe, você recebe 1 [Caixa Velha de Equipamento](http://www.divine-pride.net/database/item/12847/).
3. Volte a falar com Salim Hamid até ele pedir que você fale com a Máquina Quebrada. (Ignore a máquina)
   - Você não precisa clicar na máquina, basta responder à ele: "*Garleta 27 z e Zargônio 334 z*"
   - Após a explicação, seu curso está finalizado.
   - Até aqui, todos esses passos são opcionais para mudar de classe, mas é o jeito mais fácil de chegar em [Alberta](Alberta.md "Alberta").
   - Você pode optar por mudar de classe diretamente na Academia, mas é recomendável mudar de classe na Guilda.
4. Na Guilda dos Mercadores e converse com o Mercador da Guilda (alberta\_in 56,43)

   :   ![QMercador1](images/e/e4/QMercador1.png)

   - Responda "*Sim*" para mudar de classe.
   - Mudando de Classe na Guilda, você recebe:
     - ![1381.png](http://static.divine-pride.net/images/items/item/1381.png) 1 [Machado de Batalha](http://www.divine-pride.net/database/item/1381/)
     - Aprende [![Grito de Guerra](images/0/05/Grito_de_Guerra.png)](Grito de Guerra.md "Grito de Guerra") [Grito de Guerra](Grito de Guerra.md "Grito de Guerra")
     - Aprende [![Personalizar Carrinho](images/8/82/Personalizar_Carrinho.png)](Personalizar Carrinho.md "Personalizar Carrinho") [Personalizar Carrinho](Personalizar Carrinho.md "Personalizar Carrinho")
     - Aprende [![Cavalo-de-Pau](images/b/b6/Cavalo-de-Pau.png)](Cavalo-de-Pau.md "Cavalo-de-Pau") [Cavalo-de-Pau](Cavalo-de-Pau.md "Cavalo-de-Pau")
5. Após a mudança, fale com o Mercador novamente para voltar a Izlude, siga para a [Academia Criatura](Academia Criatura.md "Academia Criatura") e fale com Sprakki (iz\_ac01 102,44)

   :   ![TutorialNewbie010a](images/f/fd/TutorialNewbie010a.png)

   - Lenore, do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden") aparecerá e lhe dará:
     - 1.000 de [EXP](EXP.md "EXP") de base e 200 de classe.
     - ![5583.png](http://static.divine-pride.net/images/items/item/5583.png) [Chapéu do Éden I](http://www.divine-pride.net/database/item/5583/)
     - ![2560.png](http://static.divine-pride.net/images/items/item/2560.png) [Capa do Éden I](http://www.divine-pride.net/database/item/2560/)
     - ![15009.png](http://static.divine-pride.net/images/items/item/15009.png) [Uniforme do Éden I](http://www.divine-pride.net/database/item/15009/)
     - ![2456.png](http://static.divine-pride.net/images/items/item/2456.png) [Botas do Éden I](http://www.divine-pride.net/database/item/2456/)
6. Fale com Sprakki novamente se deseja receber a missão [Explorando o Mundo](Explorando o Mundo.md "Explorando o Mundo").

## Treinamento

1. Diga que sabe de tudo para Salim Hamid (alberta\_in 75,48)

   :   ![TU Merchant02](images/6/65/TU_Merchant02.png)

   - Responda corretamente as perguntas sobre atributos para ganhar 10 [Vale-Armazém](http://www.divine-pride.net/database/item/7059/).
   - Fale com Hamid novamente, que explica sobre habilidades, você recebe 15 [Vale-Armazém](http://www.divine-pride.net/database/item/7059/).
2. Converse com o Mordomo (alberta\_in 69,51)

   :   ![TU Merchant03](images/9/9c/TU_Merchant03.png)

   - Apenas para [mercadores](Mercadores.md "Mercadores") entre os níveis de base 10 a 21.
   - As missões podem ser refeitas assim que completadas.
   - Quando completar uma missão, selecione "*Receber as recompensas*".
   - A partir do nível 21, você passará a receber 30% menos de [EXP](EXP.md "EXP").

            | --- | --- |
            | Caçar 20 [Rabo de Verme](http://www.divine-pride.net/database/monster/1024/) | - 1.755 de [EXP](EXP.md "EXP") base - 1.305 de [EXP](EXP.md "EXP") classe - 2 [Vale-Armazém](http://www.divine-pride.net/database/item/7059/) |
            | Caçar 20 [Jiboia](http://www.divine-pride.net/database/monster/1025/) | - 1.755 de [EXP](EXP.md "EXP") base - 1.305 de [EXP](EXP.md "EXP") classe - 2 [Vale-Armazém](http://www.divine-pride.net/database/item/7059/) |
            | Caçar 20 [Esporo](http://www.divine-pride.net/database/monster/1014/) | - 1.755 de [EXP](EXP.md "EXP") base - 1.305 de [EXP](EXP.md "EXP") classe - 2 [Vale-Armazém](http://www.divine-pride.net/database/item/7059/) |

:   :   3. Você pode se livrar dos seus equipamentos de [aprendiz](Aprendiz.md "Aprendiz") trocando na Máquina de Vendas.

        :   ![CriaturaAcademy011](images/a/a2/CriaturaAcademy011.png)

      - 1 [Bilhete para Passeio de Peco](http://www.divine-pride.net/database/item/7310/)
