# Silentium

|  |  |
| --- | --- |
| **Silentium Silentium** | |

| **Silentium** | |
| **ID** | AB\_SILENTIUM (2057) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 60 + (Nv. da habilidade × 4) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 4 seg. |
| **Recarga** | 15 segundos |
| **Alvo** | Usuário |
| **Alcance** | 31x31 células |
| **Área** | 9x9 a 17x17 células |
| **Efeito negativo** | [Silêncio](Silêncio.md "Silêncio") |
| **Pré-requisitos** | |
| - [Gênese](Gênese.md "Gênese") [Gênese](Gênese.md "Gênese") Nv. 1 | |

Recita uma oração em latim.

Concede a chance de infligir [Silêncio](Silêncio.md "Silêncio") em oponentes ao seu redor.

| Nv. | Área | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 9x9 | 20 segundos | 64 |
| 2 | 11x11 | 30 segundos | 68 |
| 3 | 13x13 | 40 segundos | 72 |
| 4 | 15x15 | 50 segundos | 76 |
| 5 | 17x17 | 60 segundos | 80 |

## Notas

- As chances de sucesso são baseadas no [nível de base](Nível de base.md "Nível de base"), [VIT](VIT.md "VIT") e [SOR](SOR.md "SOR") do alvo.
