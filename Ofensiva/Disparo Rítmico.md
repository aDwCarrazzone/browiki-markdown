| **Rhythm Shooting / Disparo Rítmico** | |
| **ID** | TR\_RHYTHMSHOOTING (5355) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](../Sem%20categoria/SP.md "SP")** | 60 + (Nv. da habilidade × 6) |
| **Regen. de [AP](../Sem%20categoria/AP.md "AP")** | **1 a 2** |
| **[Conjuração](../Sem%20categoria/Conjura%C3%A7%C3%A3o.md "Conjuração")** | 0 + 2 seg. |
| **Pós-conjuração** | 0,15 segundos |
| **Alvo** | Oponente |
| **Alcance** | 9 + [Olhos de Águia](../Passiva/Olhos%20de%20%C3%81guia.md "Olhos de Águia") células |
| **Munição** | [Over Arrow](../Sem%20categoria/Muni%C3%A7%C3%B5es.md "Munições") 1 flecha |
| **Arma** | Arco ou Instrumento musical ou Chicote |
| **Pré-requisitos** | |
| - Nenhum | |

Dispara uma flecha em ritmo de festa.

A flecha atinge 3 vezes o oponente, causando dano físico a distância.

Se o alvo estiver sofrendo de [Desarranjo Musical](Arranjo%20Musical.md "Arranjo Musical"), há um bônus no dano causado.

O dano é afetado pelo nível de [Presença de Palco](../Passiva/Presen%C3%A7a%20de%20Palco.md "Presença de Palco").

| Nv. | [ATQ](../Sem%20categoria/ATQ.md "ATQ") x 3 | | Regen. [AP](../Sem%20categoria/AP.md "AP") | [SP](../Sem%20categoria/SP.md "SP") |
| --- | --- | --- | --- | --- |
| Normal | [Bônus](Arranjo%20Musical.md "Arranjo Musical") ♬ |
| 1 | 120% | 156% | 1 | 66 |
| 2 | 240% | 312% | 72 |
| 3 | 360% | 468% | 78 |
| 4 | 480% | 624% | 2 | 84 |
| 5 | 600% | 780% | 90 |

### Fórmula

`Dano = {[Dano + (CON × 2) × Nv. de Presença de Palco] × (Nv base ÷ 100)}%`
