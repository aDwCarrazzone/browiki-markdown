Cria uma intensa chuva no chão por 4,5 segundos.

A cada 0,3 segundos, inflige dano mágico de propriedade [Sagrado](../Sem%20categoria/Sagrado.md "Sagrado") na área.

Se [Escudo Divino](../Suporte/Escudo%20Divino.md "Escudo Divino") estiver ativo, há um bônus no dano causado.

### Fórmula

Normal:  
`ATQM = {[Dano base + (Nv. de Maestria da Guarda x 5) + (5 x FEI)] x (Nv. de base ÷ 100)}%`

Escudo Divino:  
`ATQM = {[(Dano base + (Nv. de Maestria da Guarda x 10) + (5 x FEI)] x (Nv. de base ÷ 100)}%`
