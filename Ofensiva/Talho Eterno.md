Só pode ser usada em estado de [Reflexo](../Ativa/Reflexo%20de%20Combate.md "Reflexo de Combate").

Faz um corte no alvo, causando dano físico corpo a corpo.

Se usada novamente dentro de 3 segundos, o número de cortes acumula.

Você pode acumular até 5 cortes em um ataque.

Se [Carrasco Sombrio](../Suporte/Carrasco%20Sombrio.md "Carrasco Sombrio") estiver ativo, o dano é aumentado.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é a metade do [CRIT](../Sem%20categoria/CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](../Sem%20categoria/ATQ.md "ATQ") por corte | |
| --- | --- | --- |
| Normal | [Carrasco Sombrio](../Suporte/Carrasco%20Sombrio.md "Carrasco Sombrio") |
| 1 | 350% | 500% |
| 2 | 700% | 1.000% |
| 3 | 1.050% | 1.500% |
| 4 | 1.400% | 2.000% |
| 5 | 1.750% | 2.500% |

### Fórmula

Normal:  
`ATQ = [Dano base + (POD x 5)] x (Nv. de base ÷ 100)]%`

Com Carrasco Sombrio:  
`ATQ = [Dano base + (POD x 7)] x (Nv. de base ÷ 100)]%`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- Se o usuário for alvo de [Gênese](../Ativa/G%C3%AAnese.md "Gênese") ou [Desencantar](../Ativa/Desencantar.md "Desencantar") no meio da contagem de cortes, ela será resetada para 1.
