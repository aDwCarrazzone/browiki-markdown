# Caverna de Mors

| SC ico Caverna de Mors | |
| --- | --- |
| Enfrente um dos seguidores de Morroc nessa pequena caverna de invocação de demônios. | |
| Requisitos mínimos | |
| **Nv. de base** | 160 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| Skilldelayinfo **Retorno** | Reseta 4h da manhã |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| **Quests anteriores** | [Ilha Bios](Ilha Bios.md "Ilha Bios") |
| Premiação | |
| **Chefe** | Monster NecromanteMorroc [Necromante de Morroc](http://www.divine-pride.net/database/monster/3000/) |
| **[Itens](Itens.md "Itens")** | 6684.png 1 [Marca do Herói](http://www.divine-pride.net/database/item/6684/) |

1. Fale com o Rastreador Sênior (moro\_cav 61,69)

   :   ![Mors01](images/0/04/Mors01.png)
2. Reserve a [instância](Instância.md "Instância") clicando na Rosa Vermelha.

   :   ![Mors02](images/e/ed/Mors02.png)
3. Derrote todos os [Carniçais Profanos](http://www.divine-pride.net/database/monster/3001/) e fale com o Ceifador Ankou no final do corredor.

   :   ![Mors03](images/d/dc/Mors03.png)
4. Após uma breve conversa com [Morroc](http://www.divine-pride.net/database/monster/2998/), derrote-o.

   :   ![Mors04](images/5/5f/Mors04.png)

   - Se um membro do [grupo](Grupo.md "Grupo") entrar na [instância](Instância.md "Instância") o Ceifador Ankou teleportar todos, ele não será enviado para a sala de Morroc.
   - Entretanto, se ele estiver aguardando no mapa após Morroc ser eliminado, o membro será teleportado para a sala do passo a seguir.
5. Seu [grupo](Grupo.md "Grupo") será dividido em dois times. Todos ficarão imóveis enquanto hordas de [monstros](Monstros.md "Monstros") aparecerão.
   - Derrote todos os [monstros](Monstros.md "Monstros") dentro do limite de 3 minutos.
   - Se um dos times não completar a tarefa, todo o [grupo](Grupo.md "Grupo") será mandado para [Prontera](Prontera.md "Prontera").
   - Monstros invocados: [Múmia Profana](http://www.divine-pride.net/database/monster/3002/), [Arqueiro Profano](http://www.divine-pride.net/database/monster/3003/), [Aparição Profana](http://www.divine-pride.net/database/monster/3004/) e [Verit Profano](http://www.divine-pride.net/database/monster/3005/).
   - Se conseguirem eliminar os monstros, o [grupo](Grupo.md "Grupo") se reunirá novamente na próxima sala, que consiste em continuar eliminando os monstros.
   - Haverá um monstro com o nome do líder do [grupo](Grupo.md "Grupo") no centro do mapa, mas não adianta atacá-lo.
6. Enfrente o [Necromante](http://www.divine-pride.net/database/monster/2999/) e, em seguida, elimine o [Necromante de Morroc](http://www.divine-pride.net/database/monster/3000/).

   :   ![Mors06](images/3/30/Mors06.png)
7. Para sair da [instância](Instância.md "Instância") clique no Teleportador.

   :   ![Mors07](images/9/9b/Mors07.png)
8. Fale com o Rastreador Sênior para receber a [Marca do Herói](http://www.divine-pride.net/database/item/6684/).
   - Você pode usar as [Marca do Herói](http://www.divine-pride.net/database/item/6684/) para adquirir um [Anel Heroico](http://www.divine-pride.net/database/item/2981/) e encantá-lo.
   - Confira a lista de itens do [Prêmio do Herói](Ilha Bios.md#Prêmio_do_Herói "Ilha Bios").

### Encantamento

1. Na [Caverna de Fogo](http://www.divine-pride.net/database/map/moro_cav/), dentro do [Círculo de Fogo](http://www.divine-pride.net/database/map/moro_vol/), você pode trocar 30 [Marcas do Herói](http://www.divine-pride.net/database/item/6684/) por 1 [Anel Heroico](http://www.divine-pride.net/database/item/2981/) com a Negociante Especial (moro\_cav 32,73)

   :   ![Bios05](images/5/57/Bios05.png)
2. O [Anel Heroico](http://www.divine-pride.net/database/item/2981/) pode ser encantado gratuitamente pelo Encantador Byrnes (moro\_cav 34,65)

   :   ![Bios06](images/1/15/Bios06.png)

   - Resetar todos os encantos custará 3 [Marcas do Herói](http://www.divine-pride.net/database/item/6684/).
