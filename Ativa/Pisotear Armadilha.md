# Pisotear Armadilha

|  |  |
| --- | --- |
| **Pisotear Armadilha Pisotear Armadilha** | |
| Pisotear Armadilha | |
| **Trample / Pisotear** | |
| **ID** | LG\_TRAMPLE (2309) |
| **Tipo** | Ativa |
| **Níveis** | 3 |
| **[SP](../Sem%20categoria/SP.md "SP")** | 15 + (Nv. da habilidade × 15) |
| **Recarga** | 1 segundo |
| **Alvo** | Usuário |
| **Área** | 5x5 células |
| **Pré-requisitos** | |
| - Nenhum | |

Pisa para criar um forte impacto no chão.

Há chance de destruir armadilhas ao seu redor do usuário.

Liberta você e aliados presos, além de desativar armadilhas próximas.

Ao destruir armadilhas ofensivas, haverá uma chance de bloquear o dano.

| Nv. | Chance | | [SP](../Sem%20categoria/SP.md "SP") |
| --- | --- | --- | --- |
| Destruir | Bloquear |
| 1 | 50% | 5% | 30 |
| 2 | 75% | 10% | 45 |
| 3 | 100% | 15% | 60 |

### Fórmula

`Chance de bloquear = {Chance + [(DES + AGI) ÷ 4]}%`

## Notas

- Também remove [Hera Venenosa](../Ofensiva/Hera%20Venenosa.md "Hera Venenosa").
