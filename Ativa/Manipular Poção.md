Cria fármacos a partir de ingredientes.

A quantidade produzida varia de acordo com o nível aprendido.

Cada uso consome 1 [Béquer](http://www.divine-pride.net/database/item/1000275/).

| Nv. | Máx. de poções | Fator de Produção |
| --- | --- | --- |
| 1 | Até 11 itens | 600 |
| 2 | Até 12 itens | 500 |
| 3 | Até 13 itens | 400 |
| 4 | Até 14 itens | 300 |
| 5 | Até 15 itens | 200 |

### Fórmula

**Chance de sucesso:**  
`Comparação entre Fórmula A e Fórmula B`

**Fórmula A:**  
`(INT ÷ 3) + (CRV) + (SOR ÷ 3 ) + (Nv. base ÷ 5 ) +  
(Aleatório entre 50 ~ 200) + (Nv. base - 100 ) + (Nv. de Pesquisa de Poções × 5)`

**Fórmula B:**  
`Taxa do item (tabela abaixo) + Fator de Produção (tabela acima)`

Comparação:

- Se "*A*" - "*B*" for 550 pontos → Cria o número de poções = Máx. de poções
- Se "*A*" - "*B*" for 450 ~ 549 pontos → Cria o número de poções = [Máx. de poções -3 poções]
- Se "*A*" - "*B*" for 200 ~ 449 pontos → Cria o número de poções = [Máx. de poções -5 poções]
- Se "*A*" - "*B*" for 1 ~ 199 pontos → Cria o número de poções = [Máx. de poções -6 poções]
- Se "*A*" - "*B*" for negativo → Cria o número de poções = [Máx. de poções -7 poções]

## Notas

- O Béquer é consumido independente da escolha do fármaco.
- Os ingredientes devem ocupar o mesmo espaço no inventário.
  - Isso significa que "1x Frasco de Ácido" e "1x Frasco de Ácido *com Nome*" não contam com 2 unidades do mesmo ingrediente.
