| **Strip Accessory / Descartar Accesorio** | |
| **ID** | SC\_STRIPACCESSARY (2298) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](../Sem%20categoria/SP.md "SP")** | 12 + (Nv. da habilidade × 3) |
| **[Conjuração](../Sem%20categoria/Conjura%C3%A7%C3%A3o.md "Conjuração")** | 0 + 1 seg. |
| **Pós-conjuração** | 1 segundo |
| **Duração** | [50 + (Nv. da hab. × 10)] segundos |
| **Alvo** | Oponente |
| **Alcance** | 3 células |
| **Ícone** | I Remover Acessório |
| **Pré-requisitos** | |
| - [Remover Arma](Remover%20Arma.md "Remover Arma") [Remover Arma](Remover%20Arma.md "Remover Arma") Nv. 1 | |

Tenta remover o acessório do oponente, impedindo-o de equipá-lo temporariamente.

Em monstros, reduz a [INT](../Sem%20categoria/INT.md "INT"), [SOR](../Sem%20categoria/SOR.md "SOR") e [DES](../Sem%20categoria/DES.md "DES") em 20%.

| Nv. | Chance | Duração | [SP](../Sem%20categoria/SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 14% | 60 segundos | 15 |
| 2 | 16% | 70 segundos | 18 |
| 3 | 18% | 80 segundos | 21 |
| 4 | 20% | 90 segundos | 24 |
| 5 | 22% | 100 segundos | 27 |

## Notas

- Funciona em alvos imobilizados por [Pintar Armadilha](Pintar%20Armadilha.md "Pintar Armadilha").
- Não funciona contra [Mecânicos](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Mec%C3%A2nicos.md "Mecânicos") pilotando um [MECHA](../Passiva/Licen%C3%A7a%20de%20Pilotagem.md "Licença de Pilotagem").
- Após aprender a habilidade, [Remoção Total](Remo%C3%A7%C3%A3o%20Total.md "Remoção Total") não inclui os acessórios.
- Não há métodos para evitar a remoção de um acessório.
