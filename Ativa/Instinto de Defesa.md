# Instinto de Defesa

|  |  |
| --- | --- |
| **Instinto de Defesa Instinto de Defesa** | |
| Instinto de Defesa | |
| **Counter Instinct / Contrainstinto** | |
| **ID** | ST\_REJECTSWORD (390) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](../Sem%20categoria/SP.md "SP")** | 5 + (Nv. da habilidade ×5) |
| **Duração** | Até 3 ataques |
| **Alvo** | Usuário |
| **Ícone** | I Instinto de Defesa |
| **Pré-requisitos** | |
| - Nenhum | |

Apura seus instintos.

Concede uma chance de reduzir em 50% o dano de oponentes que atacarem com adagas ou espadas.

A outra metade do dano será refletida de volta ao atacante.

| Nv. | Chance | [SP](../Sem%20categoria/SP.md "SP") |
| --- | --- | --- |
| 1 | 15% | 10 |
| 2 | 30% | 15 |
| 3 | 45% | 20 |
| 4 | 60% | 25 |
| 5 | 75% | 30 |

## Notas

- Não possui tempo de duração, o efeito permanece no personagem até que 3 ataques sejam refletidos.
- Em [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros"), reflete 50% do dano, sem restrição de arma.
- Pode ser usada para refletir o dano de [Ataque Mortal](../Ofensiva/Ataque%20Mortal.md "Ataque Mortal") dos [Ninjas](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Ninjas.md "Ninjas") que usam adaga.
