| **Counter Attack / Contrataque** | |
| **ID** | KN\_AUTOCOUNTER (61) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](../Sem%20categoria/SP.md "SP")** | 3 |
| **Duração** | (Nv. da hab. × 0,4) segundos |
| **Alvo** | Usuário |
| **Pré-requisitos** | |
| - [Perícia com Espada de Duas Mãos](../Passiva/Per%C3%ADcia%20com%20Espada%20de%20Duas%20M%C3%A3os.md "Perícia com Espada de Duas Mãos") [Perícia com Espada de Duas Mãos](../Passiva/Per%C3%ADcia%20com%20Espada%20de%20Duas%20M%C3%A3os.md "Perícia com Espada de Duas Mãos") Nv. 1 | |

Fica imóvel por um curto tempo.

Durante esse período, se o usuário for atacado com danos físicos corpo a corpo, o dano será revidado.

O dano revidado será sempre na forma de ataque crítico.

| Nv. | Duração |
| --- | --- |
| 1 | 0,4 segundos |
| 2 | 0,8 segundos |
| 3 | 1,2 segundos |
| 4 | 1,6 segundos |
| 5 | 2,0 segundos |

## Notas

- Não funciona contra [habilidades](../Habilidades%20especiais/Habilidades.md "Habilidades") ou danos físicos a distância.
- Oponentes em uma certa distância poderão não ser atingidos se a arma não possuir o alcance necessário.
- Funciona em [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros") do tipo chefe.
- Pode ser ativado mesmo se o oponente tenha errado o usuário.
