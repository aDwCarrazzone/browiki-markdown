# Punho Foguete

|  |  |
| --- | --- |
| **Punho Foguete Punho Foguete** | |
| Punho Foguete | |
| **Knuckle Boost / Propulsión de Puño de Acero** | |
| **ID** | NC\_BOOSTKNUCKLE (2256) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | Nv. da habilidade × 5 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 0,2 seg. |
| **Alvo** | Oponente |
| **Alcance** | 11 células |
| **Pré-requisitos** | |
| - [Licença de Pilotagem](Licença de Pilotagem.md "Licença de Pilotagem") [Licença de Pilotagem](Licença de Pilotagem.md "Licença de Pilotagem") Nv. 1 | |

Ejeta o punho do M.E.C.H.A, atingindo um alvo 1 vez.

Causa dano físico a distância.

| Nv. | Dano base | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 340% | 5 |
| 2 | 580% | 10 |
| 3 | 820% | 15 |
| 4 | 1.060% | 20 |
| 5 | 1.300% | 25 |

### Fórmula

`Dano = [(Dano base + DES) × (Nv. de Base ÷ 100)]%`
