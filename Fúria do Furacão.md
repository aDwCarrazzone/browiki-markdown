# Fúria do Furacão

|  |  |
| --- | --- |
| **Fúria do Furacão Fúria do Furacão** | |
| Fúria do Furacão | |
| **Axe Tornado / Tornado de Hacha** | |
| **ID** | NC\_AXETORNADO (2280) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 45 |
| **Pós-conjuração** | 0,5 segundos |
| **Recarga** | [4,5 - (Nv. da hab. × 0,5)] segundos |
| **Alvo** | Usuário |
| **Área** | 5x5 a 7x7 células |
| **Arma** | Machado |
| **Pré-requisitos** | |
| - [Maestria com Machados](Maestria com Machados.md "Maestria com Machados") [Maestria com Machados](Maestria com Machados.md "Maestria com Machados") Nv. 1 | |

Rodopia o machado para criar um furacão.

Causa dano físico corpo a corpo nos oponentes ao seu redor.

Se o estado de [Esmagador](Machado Esmagador.md "Machado Esmagador") estiver ativo, causa dano maior.

| Nv. | [ATQ](ATQ.md "ATQ") | | Área | Recarga |
| --- | --- | --- | --- | --- |
| Normal | [Esmagador](Machado Esmagador.md "Machado Esmagador") |
| 1 | 380% | 480% | 5x5 células | 4,0 segundos |
| 2 | 560% | 730% | 3,5 segundos |
| 3 | 740% | 980% | 7x7 células | 3,0 segundos |
| 4 | 920% | 1.230% | 2,5 segundos |
| 5 | 1.100% | 1.480% | 2,0 segundos |

### Fórmula

Normal:  
`Dano = {[(Dano base + VIT) × (Nv. de base ÷ 100)}%`

Normal:  
`Dano = {[Dano base + (VIT × 2)] × (Nv. de base ÷ 100)}%`
