# Invocar Escravos

|  |  |
| --- | --- |
| **NPC Skill Icon Invocar Escravos** | |
| Invocar Escravos | |
| **ID:** | NPC\_CALLSLAVE (352) |
| **Níveis:** | 1 |
| **Exclusiva:** | Sim |

Invoca [monstros](Monstros.md "Monstros") que seguirão seu mestre.

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Invocar Escravos](http://www.divine-pride.net/database/skill/352)*
- Escravos ficarão imóveis até que seu mestre se locomova e só atacarão se receberem ataques ou se o mestre for atacado.
- Serão teleportados junto com o mestre.
- Morrem junto com o mestre e não derrubam [itens](Itens.md "Itens") nem dão [experiência](Experiência.md "Experiência").
