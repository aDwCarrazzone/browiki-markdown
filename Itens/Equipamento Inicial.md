# Equipamento Inicial

Para facilitar sua rota de [equipamentos](../Redirecionamentos/Equipamentos.md "Equipamentos") após o nível 99, os Equipamentos Iniciais podem ser uma mão na roda para quem está iniciando.

Para adquirir os Equipamentos Iniciais:

1. Completar a [Conquista](../Redirecionamentos/Conquista.md "Conquista") de alcançar o nível de base 100.
2. Abrir a ![23583.png](https://static.divine-pride.net/images/items/item/23583.png) [Caixa de Equipamento Inicial](http://www.divine-pride.net/database/item/23583/).
3. Para conseguir a Arma Inicial, leve o ![25427.png](https://static.divine-pride.net/images/items/item/25427.png) [Cupom Inicial](http://www.divine-pride.net/database/item/25427/) para o Gerente de Armas (prontera 157,191).

   :   ![RebeginerW](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/7f/RebeginerW.png)
