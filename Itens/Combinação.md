# Combinação

![CombinadorJanela](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/2f/CombinadorJanela.png)

Combinação é a interface que um jogador interage a partir de um item combinador.

Para abrir uma janela de combinação, basta dar um clique duplo em um item que possui essa interação.

O item combinador só será consumido se o usuário selecionar o botão de **Criar** na janela.

Combinadores podem ter várias funções, que variam de acordo com o item:

- Adicionar [Bônus Aleatórios](../Sem%20categoria/B%C3%B4nus%20Aleat%C3%B3rios.md "Bônus Aleatórios");
- [Refinar](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Refinamento.md "Refinamento") equipamentos específicos;
- Criar novos [itens](Itens.md "Itens") a partir de materiais;
- Converter armas refinadas em pergaminhos de refino.
