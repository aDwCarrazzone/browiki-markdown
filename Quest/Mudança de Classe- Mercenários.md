# Mudança de Classe: Mercenários

| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Nv. de classe** | 40 |
| **Tipo** | Classe |
| **Classe** | [Gatuno](../Sem%20categoria/Gatuno.md "Gatuno") |

| **Recompensa** | Muda a classe para [Mercenário](../Sem%20categoria/Mercen%C3%A1rio.md "Mercenário") |

1. Dirija-se para a Guilda dos Mercenários e fale com o Mercenário (in\_moc\_16 19,33)

   :   ![MdCsin01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/85/MdCsin01.png)
2. Seu teste começa ao se aproximar do Funcionário da Guilda, que se moverá para um outro ponto. Ande até ele para se inscrever.

   :   ![MdCsin02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/63/MdCsin02.png)
3. Na nova sala, siga em frente até um diálogo começar a fazer perguntas. Você deve acertar 9, no mínimo, para passar.

   :   ![MdCsin03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/cd/MdCsin03.png)

:   :   |  |  |
        | --- | --- |
        | Possibilidade I | |
        | Quantos bônus de Esquiva você tem ao maximizar 'Perícia em Esquiva'? | 30 |
        | Qual desses monstros ignora você estar escondido furtivo? | Andre |
        | Qual dessas combinações de armas é impossível? | Katar + Main Gauche |
        | Para qual cidade você vai para se tornar uma Gatuno? | Morroc |
        | Qual dessas cartas não dá bônus à Agilidade? | Carta Sussurro |
        | Mercenários são conhecidos por qual grande vantagem? | Qualquer resposta |
        | No nível 50 de classe, qual o bônus natural em Agi? | 10 |
        | Qual desses itens não pode ser usado por um Mercenário? | Elmo |
        | Quando um Aprendiz vai se tornar Gatuno, que tipo de cogumelo ele procura? | Cogumelo Borrachento |
        | Qual dessas cartas não é muito útil a um Mercenário? | Carta Salgueiro Ancião |
        | Possibilidade II | |
        | Qual dessas habilidades não é pré-requisito para 'Tocaia'? | Perícia com a Mão Direita Nvl 2 |
        | Ao usar a habilidade 'Envenenar Arma', sua arma se torna de qual elemento? | Veneno |
        | Qual o efeito de Nível 4 de 'Perícia com a Mão Direita' no dano com a mão direita, ao usar duas armas? | 90% do dano normal |
        | Qual item é utilizado na habilidade 'Névoa Tóxica'? | Gema Vermelha |
        | Que habilidade você pode aprender depois de saber 'Envenenar Arma' Nvl 5? | Névoa Tóxica |
        | Qual dessas habilidades permite que você ande sem ser visto? | Furtividade |
        | Das 3 opções, qual não é uma condição para que a 'Bomba Tóxica' funcione? | Ter uma Gema Vermelha |
        | Com uma arma que tenha a Carta Vadon (20% a mais de dano em monstro de Fogo), que monstro recebe dano extra?. | Salgueiro Ancião |
        | Quanta SP é consumida quando utiliza-se Ataque Duplo com uma Adaga? | 0, por ser uma habilidade passiva. |
        | Qual dessas armas armas causa mais dano no Túnel Submarino? | Main Gauche de Vento |
        | Possibilidade III | |
        | Que monstro você deve matar para conseguir uma Katar com slots extras? | Lobo do Deserto |
        | E para conseguir uma Jur com slots extras? | Martin |
        | Qual classe é capaz de produzir Armas Elementais? | Ferreiros |
        | Qual dessas armas não é uma Katar? | Gladius |
        | De que elemento são os monstros do Túnel Submarino? | Água |
        | Qual desses monstros não pode ser um Bichinho de Estimação? | Sapo de Rodda |
        | Com uma Adaga de Fogo, que monstro recebe dano extra?. | Goblin com Machado |
        | Qual dessas Katares não possui um elemento? | Infiltrador |
        | Todos estes monstros são do mesmo elemento, exceto um. Qual? | Ghostring |
        | Qual desses monstros não é do elemento Maldito? | Esporo (ou *Champignon*) |

:   4. Após o questionário, siga em frente até ouvir uma conversa. Para iniciar o teste, basta clicar no chat e aguardar sua vez.

    :   ![MdCsin04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/29/MdCsin04.png)

        - Seu objetivo será eliminar 6 [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros") de nome *Vítima* em 3 minutos.
        - Os [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros") são 3 Porings, 1 Lunático, 1 Poporing e 1 Drops
        - O teste falha se você:
          - Cair em um buraco.
          - Atacar um monstro que não tenha o nome *Vítima*.
:   5. Em caso de sucesso, a próxima etapa é chegar perto de Dante (in\_moc\_16 89,98) sem matar os [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros") que surgem no caminho. O ideal é abusar da sua [esquiva](../Sem%20categoria/Esquiva.md "Esquiva") e usar [esconderijo](../Ativa/Esconderijo.md "Esconderijo") quando puder.

    :   ![MdCsin07](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/ca/MdCsin07.png)
:   6. A última etapa consiste em aproximar-se do Líder da Guilda (in\_moc\_16 149,80)

    :   ![MdCsin06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/b8/MdCsin06.png)

        - Aproxime-se do Líder e responda às perguntas, suas escolhas não influenciam na missão. Você receberá 1 [Colar do Oblívio](http://www.divine-pride.net/database/item/1008/).
        - Se seu [nível de classe](../Sem%20categoria/N%C3%ADvel%20de%20classe.md "Nível de classe") estiver entre 40 a 48 quando começou a missão, você receberá 1 arma aleatória.
          - [Main Gauche](http://www.divine-pride.net/database/item/1207/) [3]
          - [Jur](http://www.divine-pride.net/database/item/1250/) [2]
          - [Stiletto](http://www.divine-pride.net/database/item/1216/) [2]
          - [Faca](http://www.divine-pride.net/database/item/1201/) [3]
          - [Katar](http://www.divine-pride.net/database/item/1252/) [1]
        - Se seu [nível de classe](../Sem%20categoria/N%C3%ADvel%20de%20classe.md "Nível de classe") for 49 ou 50 quando começou a missão, você escolherá a arma.
          - [Jur](http://www.divine-pride.net/database/item/1251/) [3]
          - [Katar](http://www.divine-pride.net/database/item/1253/) [2]
          - [Main Gauche](http://www.divine-pride.net/database/item/1208/) [4]
          - [Gladius](http://www.divine-pride.net/database/item/1220/) [3]
        - Se seu personagem for desconectado do servidor ou você cancelar a conversa com o Líder, será necessário caminhar ao redor dele novamente para continuar o diálogo.
:   7. Entregue o [Colar do Oblívio](http://www.divine-pride.net/database/item/1008/) ao Mercenário (in\_moc\_16 19,33) para mudar de classe.

    :   ![MdCsin01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/85/MdCsin01.png)
