# Montanha do Grito

| NQ ico Montanha do Grito | |
| --- | --- |
| Furte a mais famosa Sopa do restaurante da cidade para ter o direito de subir na montanha mais alta de Louyang! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| Premiação | |
| **Recompensa** | Permite enviar um [broadcast](../Sem%20categoria/Broadcast.md "Broadcast") |

1. Encontre Guyeonbo (lou\_fild01 195,177)

   :   ![A Montanha do Grito01](../images/a/ad/A_Montanha_do_Grito01.png)
2. No restaurante da cidade, clique na Panela (lou\_in02 49,185)

   :   ![A Montanha do Grito02](../images/b/be/A_Montanha_do_Grito02.png)

   - Tente sair de lá sem que outros empregados notem você.
   - Se falhar, repita o procedimento até conseguir sair com a panela.
   - Tente ficar na entrada do restaurante e girar a câmera do jogo para clicar na panela, sem se aproximar dela.
3. Retorne ao Guyeonbo e entregue a sopa.
   - Há uma possibilidade de não ser a sopa correta, então você deve tentar roubá-la novamente.
   - Se for a sopa correta, você terá a possibilidade de subir na montanha.
   - Lá de cima, fale com Guyeonbo, que deixará você falar qualquer coisa. Seu texto será visualizado por todos do mapa.
