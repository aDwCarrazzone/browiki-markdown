# Reunião Regular dos Gatos

| NQ ico Reunião Regular dos Gatos | |
| --- | --- |
| Enquanto navegavam pelos mares para sua terra natal, esses felinos atracaram em uma ilha com o formato de uma pata de gato! Agora eles precisam se reunir para determinar o que fazer daqui pra frente. | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Normal |
| **[Itens](../Itens/Itens.md "Itens")** | 12624.png 30 [Geleias Deliciosas](http://www.divine-pride.net/database/item/12624/) |
| Premiação | |
| **[Itens](../Itens/Itens.md "Itens")** | 6422.png 400 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/)  12636.png 20 [Peixes Enlatados](http://www.divine-pride.net/database/item/12636/) |
| **Recompensa** | Libera a missão [Aventuras em Malangdo](../Quests%20repet%C3%ADveis/Aventuras%20em%20Malangdo.md "Aventuras em Malangdo") |

1. Fale com Dolangmal (prontera 114,77)

   :   ![RRdG01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/36/RRdG01.png)

   - Selecione:
     - *Malangdo... Vá... Vá...*
     - *Lembrar das Precauções*
2. Procure por Tomas (mal\_in02 22,62)

   :   ![RRdG02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/7f/RRdG02.png)
3. Converse com o Presidente do Conselho (mal\_in02 182,61)

   :   ![RRdG03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/ef/RRdG03.png)
4. Enquanto a reunião vai acontecendo, você deve seguir clicando nos gatos na seguinte ordem:
   1. Scottfold
   2. Koon
   3. Devore
   4. Mali
   5. Brie
5. Fale sobre Rican com Brie (mal\_in02 181,55)

   :   ![RRdG04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1a/RRdG04.png)
6. Encontre Rican (mal\_in01 114,169)

   :   ![RRdG05](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/ba/RRdG05.png)
7. Colete 30 [Frutos de Yggdrasil Frescos](http://www.divine-pride.net/database/item/6435/) dos Montinhos de Areia Suspeitos.

   :   ![RRdG06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/85/RRdG06.png)

   - O Montinho de Areia sempre desaparecerá após ser clicado. Ele reaparece após 30 segundos.
   - Não é recomendado esperar um montinho reaparecer, este é o método mais demorado para conseguir os Frutos.
   - As chances de conseguir um [Fruto de Yggdrasil Fresco](http://www.divine-pride.net/database/item/6435/) é razoavelmente baixa.
   - Se você clicar no Montinho de Areia e apertar a tecla [enter] rapidamente e diversas vezes seguidas, conseguirá pegar [Frutos de Yggdrasil Frescos](http://www.divine-pride.net/database/item/6435/) sem que o Montinho de Areia desapareça.
8. Coloque os [Frutos de Yggdrasil Frescos](http://www.divine-pride.net/database/item/6435/) na Máquina de Secar (mal\_in01 116,168)

   :   ![RRdG07](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/82/RRdG07.png)

   - Você receberá 20 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
9. Volte a Brie (mal\_in02 181,55)
10. Ajude a Geologista Mali (mal\_in02 179,57)

    :   ![RRdG08](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/05/RRdG08.png)
11. Pegue o [Kit de Reparação](http://www.divine-pride.net/database/item/6434/) com Bob (mal\_in02 140,94)

    :   ![RRdG09](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/03/RRdG09.png)
12. Seu objetivo agora é *tampar* ou *grampear* as Rachaduras espalhadas na ilha.

    :   ![RRdG010](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/a/af/RRdG010.png)

    - Ao interagir com a Rachadura, há uma chance de cair no [Recife de Corais Estrelados](http://www.divine-pride.net/database/map/mal_dun01/).
    - Você deve continuar reparando as Rachaduras até surgir uma mensagem pedindo que você reporte a Bob.
13. Volte ao Bob (mal\_in02 140,94)
14. Retorne a Geologista Mali (mal\_in02 179,57)
    - Você recebe 20 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
15. Fale com o Linguista Devore (mal\_in02 174,64)

    :   ![RRdG011](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1e/RRdG011.png)
16. Encontre Bolacha e fale com Doce Estelar (mal\_dun01 136,122)

    :   ![RRdG012](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/5c/RRdG012.png)
17. Volte ao Linguista Devore (mal\_in02 174,64)
18. Converse com Bolacha selecionando todas as opções de diálogo.
19. Colete 30 [Geleias Deliciosas](http://www.divine-pride.net/database/item/12624/) e entregue-os para Bolacha (mal\_dun01 136,122)
20. Consiga 40 [Baldes](http://www.divine-pride.net/database/item/6431/) e 1 [Apagador](http://www.divine-pride.net/database/item/6433/) com o Faxineiro (mal\_in02 102,94)

    :   ![RRdG013](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3d/RRdG013.png)
21. Encontre a *Água* (malangdo 240,184)

    :   ![RRdG014](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3e/RRdG014.png)

    - Selecione:
      - *Pegar Água*
    - Você receberá um [Balde Cheio](http://www.divine-pride.net/database/item/6432/).
    - Repita esse procedimento até conseguir uma boa quantidade de Baldes vazios.
    - Os baldes cheios possuem 300 de peso, mas esse procedimento evita você fazer longas viagens entre vários lugares.
22. Retorne a Bolacha (mal\_dun01 136,122)
23. Escove as 4 peles da enorme estrela-do-mar onde Doce Estelar se encontra.

    :   ![RRdG015](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/bb/RRdG015.png)

    - Observações:
      - A ordem é:
        - Pele Seca (mal\_dun01 143,137)
        - Pele Espinhosa (mal\_dun01 131,150)
        - Pele Rachada (mal\_dun01 116,144)
        - Pele Dura (mal\_dun01 120,129)
      - Você deve repetir o procedimento de escovar até uma mensagem avisar que a pele está macia.
      - Algumas vezes será pedido que derrame água nas peles, mas então o diálogo fechará sem que você tenha feito algo. Isso acontece porque você não deixou alguns [Baldes](http://www.divine-pride.net/database/item/6431/) vazios.
      - Se isso acontecer, procure o Faxineiro novamente e pegue novos [Baldes](http://www.divine-pride.net/database/item/6431/), mas não ponha água dentro deles.
24. Finalizando o trabalho, fale com Bolacha
    - Você recebe 20 [Peixes Enlatados de Malangdo](http://www.divine-pride.net/database/item/12636/) e 100 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
25. Fale com o Linguista Devore (mal\_in02 174,64)
    - Você recebe 30 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
26. Se encontre com Sr. Tomas (mal\_in02 22,62)
    - Você recebe 30 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
27. Converse com Leleco (malangdo 137,200)

    :   ![RRdG016](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/f8/RRdG016.png)

    - Selecione:
      - *Almirante Tomas*
      - *Ele é gentil*
    - Você receberá 200 [Moedas Classe E](http://www.divine-pride.net/database/item/6422/).
