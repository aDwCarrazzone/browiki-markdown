# Tribo dos Jaty

| NQ ico Tribo dos Jaty | |
| --- | --- |
| Conheça a história de uma tribo tradicional da ilha e de uma ave do paraíso. | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Normal |
| **[Itens](../Itens/Itens.md "Itens")** | 6405.png 15 [Penas de Cendrawasih](http://www.divine-pride.net/database/item/6405/) |
| Premiação | |
| **[Itens](../Itens/Itens.md "Itens")** | 18520.png 1 [Coroa de Jaty](http://www.divine-pride.net/database/item/18520/)  11533.png 5 [Satés](http://www.divine-pride.net/database/item/11533/)  6406.png 1 [Pena Reluzente](http://www.divine-pride.net/database/item/6406/) |
| **Recompensa** | Acesso ao [Vulcão Krakatau](http://www.divine-pride.net/database/map/dew_dun01/) |

1. Entre na residência, assim que se aproximar, uma janela de conversa é ativada (dew\_fild01 60,273)

   :   ![A Tribo dos Jaty00](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/25/A_Tribo_dos_Jaty00.png)
2. Lá dentro, converse com Kasyapa, O Sábio (dew\_in01 22,48)

   :   ![A Tribo dos Jaty01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/e7/A_Tribo_dos_Jaty01.png)
3. Na sequência, fale com Failko, chefe da tribo.

   :   ![A Tribo dos Jaty02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/4/40/A_Tribo_dos_Jaty02.png)
4. Ainda no local, ative outra janela de conversa andando perto da fogueira (dew\_in01 15,34)
5. Retorne ao chefe Failko, escolhendo:
   - *Qual o problema?*
   - *Ok, vou ajudar*
6. Converse com Kasyapa, O Sábio.
7. Entregue 15 [Penas de Cendrawasih](http://www.divine-pride.net/database/item/6405/) a Failko.
8. Fale com Kasyapa para receber 1 [Pena Reluzente de Cendrawasih](http://www.divine-pride.net/database/item/6406/).
9. Aceite ajudar Failko a entregar Satés.
10. Entregue 2 [Satés](http://www.divine-pride.net/database/item/11533/) para o Guerreiro da Tribo Jaty (dew\_fild01 78,288)

    :   ![A Tribo dos Jaty03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/8f/A_Tribo_dos_Jaty03.png)
11. Retorne a Failko.
12. Entregue 2 [Satés](http://www.divine-pride.net/database/item/11533/) para o Intendente da Tribo (dew\_fild01 185,300)

    :   ![A Tribo dos Jaty04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3b/A_Tribo_dos_Jaty04.png)
13. Retorne a Failko.
14. Entregue 2 [Satés](http://www.divine-pride.net/database/item/11533/) para o Guardião do Vulcão (dewata 235,56)

    :   ![A Tribo dos Jaty05](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/7c/A_Tribo_dos_Jaty05.png)
15. Retorne a Failko.
16. Entregue 2 [Satés](http://www.divine-pride.net/database/item/11533/) para o Supervisor do Templo (dewata 78,192)

    :   ![A Tribo dos Jaty06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/c2/A_Tribo_dos_Jaty06.png)
17. Retorne a Failko para receber 5 [Satés](http://www.divine-pride.net/database/item/11533/) e uma [Coroa de Jaty](http://www.divine-pride.net/database/item/18520/).
    - Agora você tem acesso ao [Vulcão Krakatau](http://www.divine-pride.net/database/map/dew_dun01/) através do Guardião do Vulcão.
