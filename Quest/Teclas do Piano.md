# Teclas do Piano

| Sair do mundo dos mortos pode ser uma tarefa difícil para quem está vivo! Encontre as teclas do piano para conseguir voltar ao mundo dos humanos. | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| Premiação | |
| **Recompensa** | Teleporta para [Umbala](../Lugares%20em%20Ragnar%C3%B6k/Umbala.md "Umbala") |

1. Encontre Alreg (niflheim 224,243)

   :   ![Piano01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/55/Piano01.png)

   - Você receberá a [1a Tecla](http://www.divine-pride.net/database/item/7184/) ao custo de perder 60% do [HP](../Sem%20categoria/HP.md "HP") atual.
2. Entre na Taverna e fale com Kuzkahina (nif\_in 31,20)

   :   ![Piano02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1c/Piano02.png)

   - Você receberá a [2a Tecla](http://www.divine-pride.net/database/item/7184/).
3. Dentro de uma casa, converse com Crayu (nif\_in 104,81)

   :   ![Sinal26](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3b/Sinal26.png)

   - Assim que um diálogo vermelho aparecer, ele pedirá que você escreva uma das frases abaixo e te dará a [3a Tecla](http://www.divine-pride.net/database/item/7184/):
     - *Quando o Sol se põe na montanha do Oeste* = digite: `Pôr do sol na montanha a oeste`
     - *Onde aponta um raio envolvente* = digite: `Onde o crepúsculo roxo cai`
     - *Uma linda melodia será a chave* = digite: `Cercado por uma bela melodia`
     - *Para derrotar os Senhor dos Mortos* = digite: `Você se torna a chave que ignora seu mestre`
4. Caminhe ao redor de outro Túmulo (niflheim 169,71)

   :   ![Piano04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/ca/Piano04.png)

   - Selecione sim para receber a [4a Tecla](http://www.divine-pride.net/database/item/7184/).
5. Caminhe ao redor do Túmulo (niflheim 208,103)

   :   ![Piano05](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/10/Piano05.png)

   - Nesse passo você receberá a [5a Tecla](http://www.divine-pride.net/database/item/7184/).
   - Para receber a [6a Tecla](http://www.divine-pride.net/database/item/7184/), você deve repetir todo o processo, ou seja, falar com Alreg novamente e executar todos os pontos até o 5º.
6. Siga para o Castelo da Bruxa e aproxime-se do piano (niflheim 253,191)

   :   ![Piano06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/f6/Piano06.png)

   - O piano fica nas coordenadas (nif\_in 115,180)
7. Após o primeiro diálogo no piano, aproxime-se novamente para se teleportar até o quarto da Bruxa Kirkena.

   :   ![Piano06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/f6/Piano06.png)
8. Converse com a Bruxa (nif\_in 188,168)

   :   ![Piano08](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/f0/Piano08.png)

   - A [NPC](../Sem%20categoria/NPC.md "NPC") te mandará para [Umbala](../Lugares%20em%20Ragnar%C3%B6k/Umbala.md "Umbala").
   - Essa missão serve para que você saia de [Niflheim](../Lugares%20em%20Ragnar%C3%B6k/Niflheim.md "Niflheim") sem precisar morrer ou usar [Asa de Borboleta](http://www.divine-pride.net/database/item/602/).
   - Agora, sempre que você quiser voltar para [Umbala](../Lugares%20em%20Ragnar%C3%B6k/Umbala.md "Umbala"), basta aproximar-se do piano para ser levado para a sala da bruxa e ter com ela.
