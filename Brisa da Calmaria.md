# Brisa da Calmaria

|  |  |
| --- | --- |
| **Brisa da Calmaria Brisa da Calmaria** | |
| Brisa da Calmaria | |
| **Silent Breeze** | |
| **ID** | MH\_SILENT\_BREEZE (8026) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 36 + (Nv. da habilidade ×9) |
| **[Conjuração](Conjuração.md "Conjuração")** | 1,2 - (Nv. × 0,2) + 0,8 + (Nv. × 0,2) seg. |
| **Alvo** | Personagens e [monstros](Monstros.md "Monstros") |
| **Alcance** | 5~9 células |
| **Efeito negativo** | [Silêncio](Silêncio.md "Silêncio") |
| **Pré-requisitos** | |
| [Eira](Eira.md "Eira") nível 137 | |

Remove alguns [efeitos negativos](Efeitos negativos.md "Efeitos negativos") e regenera o [HP](HP.md "HP") do alvo, ao custo de deixá-lo em [silêncio](Silêncio.md "Silêncio") temporariamente.

| Nv. | Alcance | Duração do [Silêncio](Silêncio.md "Silêncio") | [Conjuração](Conjuração.md "Conjuração") | | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| Fixa | Variável |
| 1 | 5 células | 9 segundos | 1 segundo | 1 segundo | 45 |
| 2 | 12 segundos | 0,8 segundos | 1,2 segundos | 54 |
| 3 | 7 células | 15 segundos | 0,6 segundos | 1,4 segundos | 63 |
| 4 | 18 segundos | 0,4 segundos | 1,6 segundos | 72 |
| 5 | 9 células | 21 segundos | 0,2 segundos | 1,8 segundos | 81 |

### Efeitos removidos

- [Alucinação](Alucinação.md "Alucinação")
- [Caos](Caos.md "Caos")
- [Sono](Sono.md "Sono")
- [Sono Profundo](Sono Profundo.md "Sono Profundo")
- [Harmonizar](Harmonizar.md "Harmonizar")
- [Grito da Mandrágora](Grito da Mandrágora.md "Grito da Mandrágora")
- [Sedução](Sedução.md "Sedução")

## Notas

- A quantidade de [HP](HP.md "HP") recuperada é baseada no [ATQM](ATQM.md "ATQM") e o quão cheia a barra de fome da [Eira](Eira.md "Eira") está.
- Para usar no personagem, segure a tecla SHIFT.
- O [silêncio](Silêncio.md "Silêncio") não será aplicado se o alvo se for um [monstro](Monstro.md "Monstro") Chefe ou se possuir imunidade ao efeito.
