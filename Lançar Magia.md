# Lançar Magia

|  |  |
| --- | --- |
| **Lançar Magia Lançar Magia** | |
| Lançar Magia | |
| **Release / Liberación** | |
| **ID** | WL\_RELEASE (2230) |
| **Tipo** | Ofensiva |
| **Níveis** | 2 |
| **[SP](SP.md "SP")** | 10 |
| **Alvo** | Oponente |
| **Alcance** | 11 células |
| **Pré-requisitos** | |
| - Nenhum | |

Permite lançar uma Esfera Elemental.

No nível 1, libera a magia memorizada por [Memorizar](Memorizar.md "Memorizar").

No nível 2, lança todas as Esferas invocadas, causando dano mágico no alvo por esfera.

| Nv. | Efeito |
| --- | --- |
| 1 | Lança 1 magia memorizada |
| 2 | Lança todas as esferas |

### Fórmula

`ATQM por esfera = [300 × (Nv. de base ÷ 100)]%`

## Notas

- No nível 1, liberar uma magia de área, como [Nevasca](Nevasca.md "Nevasca") e [Cometa](Cometa.md "Cometa"), fará com que o alvo seja o centro da área da habilidade.
- No nível 2, a [propriedade](Propriedade.md "Propriedade") do dano mágico é determinada pelas esferas invocadas.
