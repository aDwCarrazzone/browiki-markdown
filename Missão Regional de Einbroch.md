# Missão Regional de Einbroch

| NQ ico Missão Regional de Einbroch | |
| --- | --- |
| Para conhecer a história de [Einbroch](Einbroch.md "Einbroch") e [Einbech](Einbech.md "Einbech"), basta perguntar a qualquer trabalhador das cidades. Apesar de estarem sempre ocupados e longe da família, eles sempre terão um tempo para conversar com você.  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 80 |
| **Tipo** | Normal |
| **Caça** | 20 [Metaling](http://www.divine-pride.net/database/monster/1613/) 20 [Holden](http://www.divine-pride.net/database/monster/1628/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 80.920 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 78.220 |

1. Fale com Enrico Schumacher (yuno\_fild08 172,194)

   :   ![EinbrochRegional01](images/f/f9/EinbrochRegional01.png)
2. Encontre Fred Schumacher (ein\_fild06 163,173)

   :   ![EinbrochRegional02](images/d/d6/EinbrochRegional02.png)

   - Você receberá 30.000 de [EXP](EXP.md "EXP") de base e de classe.
   - Você já poderá iniciar as missões diárias de caçada. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
3. Após eliminar 20 [Metaling](http://www.divine-pride.net/database/monster/1613/) e 20 [Holden](http://www.divine-pride.net/database/monster/1628/), fale com Fred novamente e selecione "*Apenas conversar*" para teleportar a [Einbech](Einbech.md "Einbech").
   - Você receberá 40.920 de [EXP](EXP.md "EXP") de base e 38.220 de classe.
4. Converse com o Guarda Aiden (einbech 156,200)

   :   ![EinbrochRegional03](images/8/87/EinbrochRegional03.png)

   - Durante a conversa, um residente deve aparecer, continue conversando com o Guarda.
   - Se seu nível for 85 ou mais, você já poderá iniciar as missões diárias de caçada. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
   - Fale novamente com o Guarda e selecione "*Sobre Einbroch*".
5. Procure pelo Guarda Aisen (einbroch 220,254)

   :   ![EinbrochRegional04](images/c/c0/EinbrochRegional04.png)

   - Você já poderá iniciar as missões diárias de caçada. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
   - Fale novamente com o Guarda e selecione "*Apenas conversar*" e "*Sobre Lighthalzen*".
6. Aisen pede que você encontre o Guarda Carlos (airport 133,42)

   :   ![EinbrochRegional05](images/8/81/EinbrochRegional05.png)

   - Você receberá 10.000 de [EXP](EXP.md "EXP") de base e de classe.
