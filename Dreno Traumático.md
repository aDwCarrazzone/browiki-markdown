# Dreno Traumático

|  |  |
| --- | --- |
| **Dreno Traumático Dreno Traumático** | |
| Dreno Traumático | |
| **Continual Break Combo** | |
| **ID** | MH\_CBC (8037) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 50) |
| **Recarga** | 0,3 segundos |
| **Alvo** | Oponente |
| **Alcance** | 1 células |
| **Consome:** | 2 Esferas |
| **Pré-requisitos** | |
| [Eleanor](Eleanor.md "Eleanor") nível 112 | |

Só pode ser usada 2 segundos após [Quebra-Costelas](Quebra-Costelas.md "Quebra-Costelas"), no modo [Garra](Estilo de Luta.md "Estilo de Luta").

Causa dano fixo que aumenta conforme o nível de base do [Homunculus](Homunculus.md "Homunculus").

Após o uso, é possível conjurar [Estado Traumático](Estado Traumático.md "Estado Traumático") em seguida.

| Nv. | Dano fixo | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 4.000 | 10 |
| 2 | 8.000 | 20 |
| 3 | 12.000 | 30 |
| 4 | 16.000 | 40 |
| 5 | 20.000 | 50 |

## Notas

- Funciona em [monstros](Monstros.md "Monstros") do tipo Chefe.
- O dano fixo ignora a [DEF](DEF.md "DEF"), mas não afeta Plantas, Cogumelos e monstros com [Invencibilidade](Invencibilidade.md "Invencibilidade").
