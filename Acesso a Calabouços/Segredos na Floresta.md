# Segredos na Floresta

| NQ ico Segredos na Floresta | |
| --- | --- |
| Quando uma lendária criatura é avistada por uma criança conhecida por ser mentirosa, você precisará explorar as profundezas da floresta para descobrir se a história é real. | |
| Requisitos mínimos | |
| **Nv. de base** | 100 |
| **Tipo** | Normal |
| **[Itens](../Itens/Itens.md "Itens")** | 506.png 1 [Poção Verde](http://www.divine-pride.net/database/item/506/) |
| **Quests anteriores** | [Vila Cautelosa](../Quests%20de%20experi%C3%AAncia/Vila%20Cautelosa.md "Vila Cautelosa") |
| Premiação | |
| **[EXP base](../Sem%20categoria/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 1.000.000 |
| **[EXP classe](../Sem%20categoria/Experi%C3%AAncia%20de%20Classe.md "Experiência de Classe")** | 600.000 |
| **[Itens](../Itens/Itens.md "Itens")** | 6497.png 5 [Amuletos Menores](http://www.divine-pride.net/database/item/6497/) |
| **Recompensa** | Acesso a [Caverna de Buwaya](../Quest/Caverna%20de%20Buwaya.md "Caverna de Buwaya") |

1. Localize o Guarda (malaya 305,281)

   :   ![Malaya01](../images/f/f9/Malaya01.png)
2. Converse com 1 dos 3 Aldeões (malaya 278,288)

   :   ![Malaya02](../images/1/11/Malaya02.png)
3. Cutuque o ombro de Totoy (ma\_in01 47,101)

   :   ![Malaya03](../images/d/d6/Malaya03.png)
4. Clique na gaveta (ma\_in01 44,104)

   :   ![Malaya04](../images/9/97/Malaya04.png)
5. Entregue a [Poção Verde](http://www.divine-pride.net/database/item/506/) a Totoy e continue conversando, você pode selecionar qualquer opção.
6. Procure pelo Mestre de Caça (malaya 290,152)

   :   ![Malaya05](../images/a/a5/Malaya05.png)
7. Colete 10 [Amostras Coletadas](http://www.divine-pride.net/database/item/6519/) dos Rastros Desconhecidos (ma\_fild02 155,235)

   :   ![Malaya06](../images/e/e2/Malaya06.png)

   - Existem 7 Rastros Desconhecidos e reaparecem após 30 segundos.
8. Fale com o Mestre de Caça até surgir um monte de areia ao lado.
9. Clique no monte de areia até descobrir que o monstro visto por Totoy é um Buwaya.

   :   ![Segredos na Floresta06](../images/0/00/Segredos_na_Floresta06.png)
10. Fale com o Mestre de Caça novamente
11. Converse com o Chefe da Guarda (malaya 290,340)

    :   ![Malaya07](../images/4/4c/Malaya07.png)
12. Aproxime-se das coordenadas (ma\_fild02 308,243)

    :   ![Segredos na Floresta08](../images/b/b6/Segredos_na_Floresta08.png)

    - Quando o cachorro aparecer, clique nele.
13. Converse com Totoy até receber 5 [Amuletos Menores](http://www.divine-pride.net/database/item/6497/).
14. Retorne ao Chefe da Guarda para receber sua recompensa em [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência").
    - Fale mais uma vez com ele para liberar o acesso da instância.
15. Se seu [nível de base](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/N%C3%ADvel.md "Nível") for 130, fale novamente com o Chefe da Guarda para receber a missão "*Livre-se de Buwaya*".
    - Ao derrotar o [Buwaya](http://www.divine-pride.net/database/monster/2319/), reporte ao Chefe da Guarda para receber 3 [Amuletos Menores](http://www.divine-pride.net/database/item/6497/).
    - É possível refazer este passo a cada 24 horas.
