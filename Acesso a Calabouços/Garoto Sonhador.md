# Garoto Sonhador

| --- | --- |
| Uma simples permissão para entrar na capital dos Esplêndidos pode gerar um pedido de ajuda atrás do outro! Tente buscar os materiais necessários para fazer uma grande festa de aniversário! | |
| Requisitos mínimos | |
| **Nv. de base** | 99 |
| **Tipo** | Normal |
| **[Itens](../Itens/Itens.md "Itens")** | ![6561.png](https://static.divine-pride.net/images/items/item/6561.png) 128 [Bolas de Poeira](http://www.divine-pride.net/database/item/6561/) ![6558.png](https://static.divine-pride.net/images/items/item/6558.png) 1 [Pilha de Avelãs](http://www.divine-pride.net/database/item/6558/) |
| Premiação | |
| **[Itens](../Itens/Itens.md "Itens")** | ![19543.png](https://static.divine-pride.net/images/items/item/19543.png) 1 [Capuz do Lobo Oliver](http://www.divine-pride.net/database/item/19543/) |
| **Recompensa** | Acesso a [Eclage](../Lugares%20em%20Ragnar%C3%B6k/Eclage.md "Eclage") |

1. Preencha um cadastro com o Guarda da Cidade (ecl\_fild01 100,323)

   :   ![Garoto Sonhador001](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/9/91/Garoto_Sonhador001.png)
2. Fale mais de uma vez com o Oficial de Imigração (ecl\_fild01 111,320)

   :   ![Garoto Sonhador002](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/07/Garoto_Sonhador002.png)

   - Você deverá aguardar 1 minuto para voltar a falar com este [NPC](../Sem%20categoria/NPC.md "NPC").
3. Retorne ao Guarda da Cidade. A partir daqui seu acesso a [Eclage](../Lugares%20em%20Ragnar%C3%B6k/Eclage.md "Eclage") já está liberado.
4. Aproxime-se da Fada Carpinteiro (eclage 102,32)

   :   ![Garoto Sonhador03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/15/Garoto_Sonhador03.png)

   - É necessário ter nível 120 para continuar.
   - Digite:
     - `002`
5. Pegue 2 [Pedaços da Ponte de Folhas](http://www.divine-pride.net/database/item/6553/) com Golie (ecl\_fild01 192,94)

   :   ![Garoto Sonhador04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/04/Garoto_Sonhador04.png)
6. Reencontre a Fada Carpinteiro.
7. Procure por Glaces (eclage 283,275)

   :   ![Garoto Sonhador05](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/27/Garoto_Sonhador05.png)
8. Fale com Yube (eclage 191,200)

   :   ![Garoto Sonhador05a](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/d/d8/Garoto_Sonhador05a.png)
9. Retorne a Glaces.
10. Encontre Murah (ecl\_in01 73,51)

    :   ![Garoto Sonhador07](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/29/Garoto_Sonhador07.png)
11. Converse com Shul (eclage 265,166)

    :   ![Garoto Sonhador06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/4/48/Garoto_Sonhador06.png)
12. Localize Tato (eclage 136,169)

    :   ![Garoto Sonhador07a](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3a/Garoto_Sonhador07a.png)
13. Traga 128 [Bolas de Poeira](http://www.divine-pride.net/database/item/6561/) a Yube.
    - Digite:
      - `128`
    - Você receberá 128 [Velas de Aniversário](http://www.divine-pride.net/database/item/6555/).
14. Retorne a Glaces.
15. Procure por Moreng (eclage 266,216)

    :   ![Garoto Sonhador08](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/9/9e/Garoto_Sonhador08.png)
16. Diga que não conhece a história para o Velho (ecl\_in01 60,71)

    :   ![Garoto Sonhador09](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/9/9f/Garoto_Sonhador09.png)
17. Dirija-se ao Viajante (ecl\_fild01 97,315)

    :   ![Garoto Sonhador010](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/7a/Garoto_Sonhador010.png)
18. Encontre o Mercador Sack (ecl\_in01 70,88)

    :   ![Garoto Sonhador010a](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/11/Garoto_Sonhador010a.png)
19. Volte a Moreng.
20. Avise Glaces.
21. Entre no portal do *Yai* e fale mais de uma vez com Yube (eclage 292,265)

    :   ![Garoto Sonhador011](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/ed/Garoto_Sonhador011.png)
22. Dê as sementes ao Entregador (ecl\_in01 44,53)

    :   ![Garoto Sonhador011a](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/8f/Garoto_Sonhador011a.png)
23. Retorne ao Yai de Yube.
24. Traga 1 [Pilha de Avelãs](http://www.divine-pride.net/database/item/6558/) para Oliver no mesmo Yai, seu personagem será expulso da casa.
25. Retorne ao Yai e converse com os aventureiros até receber o [Capuz do Lobo Oliver](http://www.divine-pride.net/database/item/19543/).

    :   ![Garoto Sonhador013](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/05/Garoto_Sonhador013.png)

    - Fale mais uma vez com todos os personagens da sala.
    - Caso tenha curiosidade, Oliver é o personagem da missão [Corvo do Destino](../Quests%20de%20experi%C3%AAncia/Corvo%20do%20Destino.md "Corvo do Destino").
