# Punho do Dragão

|  |  |
| --- | --- |
| **Punho do Dragão Punho do Dragão** | |
| Punho do Dragão | |
| **Dragon Combo / Combo de Dragón** | |
| **ID** | SR\_DRAGONCOMBO (2326) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | Nv. da habilidade + 2 |
| **Alvo** | Oponente |
| **Alcance** | 2 células |
| **Efeito negativo** | [Atordoamento](Atordoamento.md "Atordoamento") |
| **Pré-requisitos** | |
| - [Combo Triplo](Combo Triplo.md "Combo Triplo") [Combo Triplo](Combo Triplo.md "Combo Triplo") Nv. 5 | |

Atinge um alvo único com seu punho.

Causa dano físico corpo a corpo com chance de infligir [Atordoamento](Atordoamento.md "Atordoamento").

Também pode ser usada como combo, após [Combo Triplo](Combo Triplo.md "Combo Triplo") e antes de [Ruína](Ruína.md "Ruína").

| 1 | 180% | 2% | 3 |
| 2 | 260% | 3% | 4 |
| 3 | 340% | 4% | 5 |
| 4 | 500% | 5% | 6 |
| 5 | 580% | 6% | 7 |
| 6 | 340% | 7% | 8 |
| 7 | 660% | 8% | 9 |
| 8 | 740% | 9% | 10 |
| 9 | 820% | 10% | 11 |
| 10 | 900% | 11% | 12 |

### Fórmula

`ATQ = [Dano base × (Nv. de base ÷ 100)]%`
