# Vila Cautelosa

| NQ ico Vila Cautelosa | |
| --- | --- |
| Os nativos da ilha ainda estranham os estrangeiros na cidade. É melhor você causar uma boa primeira impressão! | |
| Requisitos mínimos | |
| **Nv. de base** | 100 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 523.png 6 [Água Benta](http://www.divine-pride.net/database/item/523/)  717.png 6 [Gema Azul](http://www.divine-pride.net/database/item/717/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 200.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 200.000 |
| **Recompensa** | Libera os [serviços](Serviços.md "Serviços") da cidade e a missão [Aventuras em Malaya](Aventuras em Malaya.md "Aventuras em Malaya") |

1. Fale com o Guarda Rondel (malaya 266,76)

   :   ![Vila Cautelosax](images/8/82/Vila_Cautelosax.png)
2. Encontre Phong, o Líder Mumbaki (malaya 185,358). Qualquer opção selecionada não altera a missão.

   :   ![Vila Cautelosa2](images/e/ee/Vila_Cautelosa2.png)
3. Converse com o Ancião Nardo (malaya 181,353)

   :   ![Malaya1](images/9/9f/Malaya1.png)
4. Fale com Ferreiro Pandoi (malaya 119,217)

   :   ![Malaya2](images/0/01/Malaya2.png)
5. Localize Romel (malaya 224,267)

   :   ![Malaya3](images/c/cd/Malaya3.png)
6. Dialogue com Talah (malaya 223,267)

   :   ![Malaya4](images/9/9f/Malaya4.png)
7. Apresente-se ao Colecionador Woeon (malaya 295,171)

   :   ![Malaya5](images/1/17/Malaya5.png)
8. Explique o que aconteceu a Phong (malaya 185,358). Em seguida, sugira [Água Benta](http://www.divine-pride.net/database/item/523/) e [Gema Azul](http://www.divine-pride.net/database/item/717/).
9. Encontre o Marinheiro Pedro (malaya 326,68)

   :   ![Malaya6](images/2/2c/Malaya6.png)
10. Colete os itens e entregue-os para cada [NPC](NPC.md "NPC") já mencionados acima:
    - Ancião Nardo (malaya 181,353)
    - Ferreiro Pandoi (malaya 119,217)
    - Romel (malaya 224,267)
    - Talah (malaya 223,267)
    - Colecionador Woeon (malaya 295,171)
    - Marinheiro Pedro (malaya 326,68)
11. Retorne a Phong (malaya 185,358)
    - Você recebe sua recompensa em [experiência](Experiência.md "Experiência") e tem acesso aos prédios da cidade, tais como a loja de ferramentas, a [estalagem](Estalagem.md "Estalagem") e usar o *jeepney* para se locomover na cidade.
