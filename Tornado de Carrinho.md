# Tornado de Carrinho

| --- | --- |

| **[SP](SP.md "SP")** | 30 |
| **Pós-conjuração** | 1 segundo |

| **Alvo** | Usuário |
| **Área** | 5x5 células |
| **Requisito** | [Carrinho](Usar Carrinho.md "Usar Carrinho") |
| **Pré-requisitos** | |
| - [Aprimorar Carrinho](Aprimorar Carrinho.md "Aprimorar Carrinho") [Aprimorar Carrinho](Aprimorar Carrinho.md "Aprimorar Carrinho") Nv. 1 | |

Rodopia rapidamente seu carrinho.

Causa dano físico corpo a corpo nos oponentes próximos.

Se [Cultivar Bárbaro](Cultivar Bárbaro.md "Cultivar Bárbaro") estiver ativo, o dano aumenta.

Quanto mais pesado seu carrinho estiver, maior o dano causado.

| 8 | 1.600% | 3.200% |
| 9 | 1.800% | 3.600% |
| 10 | 2.000% | 4.000% |

### Fórmula

`Dano = {Dano base + [Peso atual do Carrinho ÷ (150 - FOR base)] + (Nv. de Aprimorar Carrinho × 50)}%`
