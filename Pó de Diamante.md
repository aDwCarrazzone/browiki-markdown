# Pó de Diamante

|  |  |
| --- | --- |
| **Pó de Diamante Pó de Diamante** | |
| Pó de Diamante | |
| **Diamond Dust / Polvo Diamantino** | |
| **ID** | SO\_DIAMONDDUST (2447) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | 1 - (Nv. × 0,2) + 4,5 + (Nv. × 0,5) seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 5 segundos |
| **Alvo** | Chão |
| **Alcance** | 9 células |
| **Propriedade** | [Água](Monstros.md#Propriedade "Monstros") |
| **Efeito negativo** | [Cristalização](Cristalização.md "Cristalização") |
| **Pré-requisitos** | |
| - [Dilúvio](Dilúvio.md "Dilúvio") [Dilúvio](Dilúvio.md "Dilúvio") Nv. 3 | |

Pulveriza gelo na área selecionada.

Causa dano mágico de propriedade [Água](Água.md "Água").

Tem chance de infligir [Cristalização](Cristalização.md "Cristalização").

O dano é influenciado pelo nível de [Encantar com Geada](Encantar com Geada.md "Encantar com Geada").

| Nv. | Área | [Cristalização](Cristalização.md "Cristalização") | | [Conjuração](Conjuração.md "Conjuração") | | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- | --- |
| Chance | Duração | Fixa | Variável |
| 1 | 7x7 células | 10% | 12 segundos | 0,8 segundos | 5,0 segundos | 50 |
| 2 | 15% | 14 segundos | 0,6 segundos | 5,5 segundos | 56 |
| 3 | 20% | 16 segundos | 0,4 segundos | 6,0 segundos | 62 |
| 4 | 9x9 células | 25% | 18 segundos | 0,2 segundos | 6,5 segundos | 68 |
| 5 | 30% | 20 segundos | 0,0 segundos | 7,0 segundos | 74 |

### Fórmula

`Dano = {[(Nv. da hab. + 2) × INT] + (Nv. de Enc. com Geada × 300)} × (Nv. de base ÷ 100)`

`Cristalização = [Duração da habilidade - (VIT base do alvo ÷ 10)] segundos`
