# Aumentar Recuperação de HP

|  |  |
| --- | --- |
| **Aumentar Recuperação de HP Aumentar Recuperação de HP** | |
| Aumentar Recuperação de HP | |
| **Increase HP Recovery / Aumentar Recuperación de HP** | |
| **ID** | SM\_RECOVERY (4) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| - Nenhum | |

Enquanto imóvel, regenera o [HP](HP.md "HP") a cada 10 segundos.

A taxa de regeneração aumenta conforme o [HP](HP.md "HP") máximo do usuário.

Também melhora a cura recebida de habilidades e por [itens](Itens.md "Itens") de recuperação.

| Nv. | Regeneração de [HP](HP.md "HP") | | Cura recebida |
| --- | --- | --- | --- |
| Base | Bônus |
| 1 | 5 | +0,2% | 10% |
| 2 | 10 | +0,4% | 20% |
| 3 | 15 | +0,6% | 30% |
| 4 | 20 | +0,8% | 40% |
| 5 | 25 | +1,0% | 50% |
| 6 | 30 | +1,2% | 60% |
| 7 | 35 | +1,4% | 70% |
| 8 | 40 | +1,6% | 80% |
| 9 | 45 | +1,8% | 90% |
| 10 | 50 | +2,0% | 100% |

## Notas

- O efeito é cancelado se o usuário não puder regenerar o [HP](HP.md "HP") naturalmente, como por exemplo, 70% acima do [peso](Peso.md "Peso") permitido.
- A cura recebida de itens e habilidades é cumulativo com os bônus de [VIT](VIT.md "VIT").
