# Autodestruição

|  |  |
| --- | --- |
| **Autodestruição Autodestruição** | |

| **Suicidal Destruction / Destrucción Suicida** | |
| **ID** | NC\_SELFDESTRUCTION (2267) |
| **Tipo** | Ofensiva |
| **Níveis** | 3 |
| **[SP](SP.md "SP")** | 1 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,5 + 2,5 - (Nv. × 0,5) seg. |
| **Recarga** | 300 segundos |
| **Alvo** | Usuário |
| **Área** | 5x5 a 9x9 células |
| **Propriedade** | [Neutro](Monstros.md#Propriedade "Monstros") |
| **Item** | 6146.png 3 [Gasolinas](http://www.divine-pride.net/database/item/6146/) |
| **Requisito** | 2802.png 1 [Detonador](http://www.divine-pride.net/database/item/2802/) |
| **Pré-requisitos** | |
| - [Reforçar Estrutura](Reforçar Estrutura.md "Reforçar Estrutura") [Reforçar Estrutura](Reforçar Estrutura.md "Reforçar Estrutura") Nv. 2 | |

Ejeta-se do M.E.C.H.A. e o explode.

Consume todo o [SP](SP.md "SP") para causar dano físico corpo a corpo

A explosão ignora a [esquiva](Esquiva.md "Esquiva") do inimigos ao redor.

| Nv. | Fator | Área | [Conjuração variável](Conjuração variável.md "Conjuração variável") |
| --- | --- | --- | --- |
| 1 | 2 | 5x5 células | 2,0 segundos |
| 2 | 3 | 7x7 células | 1,5 segundos |
| 3 | 4 | 9x9 células | 1,0 segundo |

### Fórmula

`Dano = {[Fator × (Nv. de Reforçar Estrutura + 8)] × (SP restante + VIT) × (Nv. de base ÷ 100)} + HP restante`

## Notas

- A [conjuração](Conjuração.md "Conjuração") não pode ser interrompida.
- O dano ignora reduções de [raça](Raça.md "Raça"), [tamanho](Tamanho.md "Tamanho") e [propriedade](Propriedade.md "Propriedade").
- Pode ser reduzido pela soma da [DEF](DEF.md "DEF") do oponente, sendo subtraído no dano final.
- O usuário poderá receber dano por [Vínculo Sombrio](Vínculo Sombrio.md "Vínculo Sombrio").
- O [Detonador](http://www.divine-pride.net/database/item/2802/) não precisa estar equipado para usar esta habilidade.
