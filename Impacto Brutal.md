| **Arma** | Katar |
| **Pré-requisitos** | |
| - [Lâminas Retalhadoras](Lâminas Retalhadoras.md "Lâminas Retalhadoras") [Lâminas Retalhadoras](Lâminas Retalhadoras.md "Lâminas Retalhadoras") Nv. 5 - [Senso das Sombras](Senso das Sombras.md "Senso das Sombras") [Senso das Sombras](Senso das Sombras.md "Senso das Sombras") Nv. 3 | |

Corre até o alvo, atingindo-o 3 vezes.

Causa dano físico corpo a corpo na área de 3x3 células ao redor do oponente.

Se estiver em [Ocultação](Ocultação.md "Ocultação"), atinge 5 vezes e causa mais dano.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é a metade do [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") por golpe | | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| Normal | [Ocultação](Ocultação.md "Ocultação") |
| 1 | 60% | 100% | 28 |
| 2 | 120% | 200% | 31 |
| 3 | 180% | 300% | 34 |
| 4 | 240% | 400% | 37 |
| 5 | 300% | 500% | 40 |
| 6 | 360% | 600% | 43 |
| 7 | 420% | 700% | 46 |
| 8 | 480% | 800% | 49 |
| 9 | 540% | 900% | 52 |
| 10 | 600% | 1.000% | 55 |

### Fórmula

Normal (3 golpes):  
`ATQ = [Dano base + (POD x 5)] x (Nv. de base ÷ 100)]%`

Em Ocultação (5 golpes):  
`ATQ = [Dano base + (POD x 7)] x (Nv. de base ÷ 100)]%`
