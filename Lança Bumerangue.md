# Lança Bumerangue

|  |  |
| --- | --- |
| **Lança Bumerangue Lança Bumerangue** | |
| Lança Bumerangue | |
| **Spear Boomerang / Lanza Bumerán** | |
| **ID** | KN\_SPEARBOOMERANG (59) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 12 |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | 3 + (Nv. da habilidade ×2) células |
| **Arma** | Lança |
| **Pré-requisitos** | |
| - [Perfurar](Perfurar.md "Perfurar") [Perfurar](Perfurar.md "Perfurar") Nv. 3 | |

Arremessa a lança no alvo, causando dano físico corpo a corpo.

| Nv. | [ATQ](ATQ.md "ATQ") | Alcance |
| --- | --- | --- |
| 1 | 150% | 3 células |
| 2 | 200% | 5 células |
| 3 | 250% | 7 células |
| 4 | 300% | 9 células |
| 5 | 350% | 11 células |

## Notas

- A partir do nível 2, o dano passa a ser considerado de longa distância.
