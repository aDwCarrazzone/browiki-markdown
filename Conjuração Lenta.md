# Conjuração Lenta

|  |  |
| --- | --- |
| **NPC Skill Icon Conjuração Lenta** | |
| Conjuração Lenta | |
| **ID:** | NPC\_SLOWCAST (672) |
| **Níveis:** | 5 |
| **Exclusiva:** | Não |
| **Duração:** | 30 segundos |
| **[Ícone](Ícone.md "Ícone"):** | I Conjuração Lenta |

Aumenta a [conjuração variável](Conjuração variável.md "Conjuração variável") dos alvos afetados por 30 segundos.

| Nv. | [Conjuração variável](Conjuração variável.md "Conjuração variável") | Área |
| --- | --- | --- |
| 1 | +20% | 5x5 células |
| 2 | +40% | 11x11 células |
| 3 | +60% | 17x17 células |
| 4 | +80% | 23x23 células |
| 5 | +100% | 29x29 células |

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Conjuração Lenta](http://www.divine-pride.net/database/skill/672)*
- O efeito é acumulativo com outros equipamentos que aumentam a [conjuração variável](Conjuração variável.md "Conjuração variável").
- É possível ter imunidade ao efeito se o usuário conseguir alcançar 100% de redução da [conjuração variável](Conjuração variável.md "Conjuração variável").
- [Itens](Itens.md "Itens") que reduzem a [conjuração variável](Conjuração variável.md "Conjuração variável") de uma [habilidade](Habilidade.md "Habilidade") específica são afetados.
