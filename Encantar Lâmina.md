# Encantar Lâmina

|  |  |
| --- | --- |

| **Enchant Blade / Encantar Espada** | |

| **Tipo** | Suporte |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 30 + (Nv. da habilidade × 4) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 seg. |
| **Duração** | 300 segundos |
| **Alvo** | Usuário |
| **Ícone** | I Encantar Lâmina |

Concentra poderes mágicos na sua arma.

Por 5 minutos, encanta a arma com magia para acrescentar [ATQM](ATQM.md "ATQM") no dano dos ataques básicos.

| Nv. | Bônus | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | +120 | 34 |
| 2 | +140 | 38 |
| 3 | +160 | 42 |
| 4 | +180 | 46 |
| 5 | +200 | 50 |
| 6 | +220 | 54 |
| 7 | +240 | 58 |
| 8 | +260 | 62 |
| 9 | +280 | 66 |
| 10 | +300 | 70 |

### Fórmula

`ATQM = [(Nv. da habilidade × 20) + 100] x [(Nv. de base ÷ 100)] + INT`

## Notas

- Não afeta [habilidades](Habilidades.md "Habilidades"), apenas ataques básicos.
- Pode ser cancelada por [Desencantar](Desencantar.md "Desencantar") e [Gênese](Gênese.md "Gênese").
- O bônus ignora itens que melhoram a resistência física do oponente.
