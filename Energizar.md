# Energizar

|  |  |
| --- | --- |
| **NPC Skill Icon Energizar** | |
| Energizar | |
| **ID:** | NPC\_POWERUP (349) |
| **Níveis:** | 5 |
| **Exclusiva:** | Sim |
| **Duração:** | [5 + (Nv. da habilidade × 5)] segundos |

Temporariamente concede:

- [ATQ](ATQ.md "ATQ") +200%
- [Precisão](Precisão (Atributo).md "Precisão (Atributo)") +100%

| Nv. | Duração |
| --- | --- |
| 1 | 10 segundos |
| 2 | 15 segundos |
| 3 | 20 segundos |
| 4 | 25 segundos |
| 5 | 30 segundos |

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Energizar](http://www.divine-pride.net/database/skill/349)*
- O bônus de [ATQ](ATQ.md "ATQ") não afeta o dano de [Terremoto](Terremoto.md "Terremoto").
- Não pode ser removida por [Desencantar](Desencantar.md "Desencantar").
- A [precisão](Precisão (Atributo).md "Precisão (Atributo)") é aumentada de acordo com a [DES](DES.md "DES") do [monstro](Monstro.md "Monstro").
- Exemplo:
  - Imagine um [MVP](MVP.md "MVP") com [ATQ](ATQ.md "ATQ") entre 780 ~ 1428, com 155 de [precisão](Precisão (Atributo).md "Precisão (Atributo)").
  - Com o efeito da habilidade, esses atributos aumentariam para [ATQ](ATQ.md "ATQ") entre 2340 ~ 4284 e 310 de [precisão](Precisão (Atributo).md "Precisão (Atributo)").
  - Seria então, necessário 535 de [esquiva](Esquiva.md "Esquiva") para desviar dos ataques com 95% de chance.
