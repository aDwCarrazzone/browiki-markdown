# Festival de Pintados

| DQ ico Festival de Pintados | |
| --- | --- |
| O Festival de Pintados é um evento especial que dura 7 dias. Durante as festividades, será possível fabricar 3 tipos diferentes de tatuagem! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Repetível |

| Premiação | |
| **[Itens](Itens.md "Itens")** | Cria [acessórios](Acessório.md "Acessório") |

- Para dar início ao evento, jogadores deverão doar os seguintes itens:
  - [Amuletos Menores](http://www.divine-pride.net/database/item/6497/), conseguidos nas [quests](Quests.md "Quests") diárias de [Porto Malaya](Porto Malaya.md "Porto Malaya");
  - [Cruzes de Prata](http://www.divine-pride.net/database/item/6502/), forjadas pelo Ferreiro Pandoi (malaya 119,217)
  - [Tinturas](Tinturas.md "Tinturas"), podem ser de qualquer cor.
- Assim que o período do evento terminar, os personagens deverão juntar os itens novamente para recomeçar o festival.

1. Faça sua doação ao Ajudante do Festival (ma\_fild01 179,260)

   :   ![Pintados1](images/c/c8/Pintados1.png)

   - O festival começa quando a doação atingir o número de itens requeridos:
     - 1.000 [Amuletos Menores](http://www.divine-pride.net/database/item/6497/)
     - 1.000 [Cruzes de Prata](http://www.divine-pride.net/database/item/6502/)
     - 300 [Tinturas](Tinturas.md "Tinturas")
   - Ao fazer sua doação, o Ajudante só recolhe até o 50 [itens](Itens.md "Itens") por vez.
2. Quando o Festival de Pintados começar, fale com o Tatuador (ma\_fild01 172,223)

   :   ![Pintados2](images/4/4d/Pintados2.png)

   - Você precisa ficar sempre selecionando a opção "**...**", caso contrário, perderá os rancores.
   - O Tatuador fará 1 [acessório](Acessório.md "Acessório") de acordo com os itens que você trouxer:
