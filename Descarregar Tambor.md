# Descarregar Tambor

|  |  |
| --- | --- |
| **Descarregar Tambor Descarregar Tambor** | |
| Descarregar Tambor | |
| **Fire Dance / Baile de Fuego** | |
| **ID** | RL\_FIREDANCE (2561) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 3) |
| **Pós-conjuração** | 0,5 segundos |
| **Alvo** | Usuário |
| **Área** | 7x7 células |
| **Munição** | [Over Bullet](Munições.md "Munições") 5 Projéteis |
| **Arma** | Pistola |
| **Pré-requisitos** | |
| [Desperado](Desperado.md "Desperado") [Desperado](Desperado.md "Desperado") Nv. 1 | |

Gira o tambor do revólver para iniciar uma dança de disparos de fogo.

Causa dano físico a distância ao seu redor em uma área de 7x7 células.

Consome 5 projéteis e o dano aumenta de acordo com o nível de [Desperado](Desperado.md "Desperado").

| Nv. | [ATQ](ATQ.md "ATQ") | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 300% | 13 |
| 2 | 400% | 16 |
| 3 | 500% | 19 |
| 4 | 600% | 22 |
| 5 | 700% | 25 |
| 6 | 800% | 28 |
| 7 | 900% | 31 |
| 8 | 1.000% | 34 |
| 9 | 1.100% | 37 |
| 10 | 1.200% | 40 |

### Fórmula

`ATQ = {[(Nv. da hab. × 100) + 200] + (Nv. de Desperado × 50)} × (Nv. de base ÷ 100)`
