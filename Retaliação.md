# Retaliação

|  |  |
| --- | --- |
| **Retaliação Retaliação** | |

| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 2 + (Nv. da habilidade × 3) |
| **Pós-conjuração** | 2 a 1 segundos |
| **Alvo** | Usuário |
| **Área** | 3x3 a 5x5 células |
| **Requisito** | [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") |
| **Pré-requisitos** | |
| - [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") Nv. 1 | |

Reage à um ataque inimigo.

Só pode ser usada em estado de [Reflexo](Reflexo de Combate.md "Reflexo de Combate").

Causa dano físico corpo a corpo que ignora a [DEF](DEF.md "DEF") dos oponentes ao redor.

| Nv. | Dano base | Área | Pós-conjuração | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| 1 | 450% | 3x3 células | 2 segundos | 5 |
| 2 | 600% | 8 |
| 3 | 750% | 11 |
| 4 | 900% | 14 |
| 5 | 1.050% | 17 |
| 6 | 1.200% | 5x5 células | 1 segundo | 19 |
| 7 | 1.350% | 21 |
| 8 | 1.500% | 23 |
| 9 | 1.650% | 25 |
| 10 | 1.800% | 27 |

### Fórmula

`Dano = {[Dano base] × (Nv. de base ÷ 120)] + [(AGI × 2) + (Nv. de classe × 4)]}%`
