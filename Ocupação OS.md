# Ocupação OS

| NQ ico Ocupação OS | |
| --- | --- |
| Requisitos mínimos | |

| **Tipo** | Normal |
| **Caça** | [Miguel](http://www.divine-pride.net/database/monster/20347/) |
| **Quests anteriores** | [Ilusión](Ilusión.md "Ilusión") |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 1.200.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 1.080.000 |
| **[Itens](Itens.md "Itens")** | 25723.png 5 [Núcleo COR](http://www.divine-pride.net/database/item/25723/) 25668.png 1 [Arma Danificada](http://www.divine-pride.net/database/item/25668/) |
| **Recompensa** | Libera a instância [Missão OS](Missão OS.md "Missão OS") |

1. No Distrito COR, fale com Est (sp\_cor 164,56) para gerar a instância.

   :   ![Ilusion12](images/1/1e/Ilusion12.png)

   - Essa [NPC](NPC.md "NPC") é cinemática, ou seja, só aparece para seu personagem e só quando você se aproxima.
2. Dentro da instância, espere a introdução finalizar e os [NPCs](NPC.md "NPC") sumirem.
3. Como pedido pela comandante, fique no local, basta ir enfrentando alguns monstros.
4. Quando o anúncio pedir que a *Equipe Eco* se comunique, basta clicar no robô e selecionar qualquer opção.

   :   ![WalkieTalkie](images/d/d6/WalkieTalkie.png)
5. Após a ajuda do Esquadrão Anti-Bomba, siga pela [navegação](Navegação.md "Navegação").
   - Fale com o robô e continue seguindo a navegação e matando monstros no caminho.
   - É importante que você **AGUARDE** um pouco após derrotar os monstros, pois o [NPC](NPC.md "NPC") de navegação leva um tempo para aparecer.
   - Sempre que você eliminar os mutantes remanescentes, a ocupação estará concluída e uma luz azul aparecerá, aproxime-se dela para indicar o próximo caminho.
6. Enfrente Miguel, com [HP](HP.md "HP") menor que o original e depois, fale com Est.

   :   ![EstOS1](images/8/82/EstOS1.png)

   - Você receberá 200.000 de [EXP](EXP.md "EXP") de base e 180.000 de [EXP](EXP.md "EXP") de classe parcelado em 6 vezes.
   - Você ganha ![25723.png](http://static.divine-pride.net/images/items/item/25723.png) 5 [Núcleo COR](http://www.divine-pride.net/database/item/25723/) e ![25668.png](http://static.divine-pride.net/images/items/item/25668.png) 1 [Arma Danificada](http://www.divine-pride.net/database/item/25668/)
7. Fale com Est para finalizar a instância.
   - Você liberou a instância [Missão OS](Missão OS.md "Missão OS").
