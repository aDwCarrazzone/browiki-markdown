# Explosão de Moedas

|  |  |
| --- | --- |
| **Explosão de Moedas Explosão de Moedas** | |
| Explosão de Moedas | |
| **Rapid Throw / Lanzamiento Rápido** | |
| **ID** | KO\_MUCHANAGE (3008) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 seg. |
| **Recarga** | 10 segundos |
| **Alvo** | Chão |
| **Alcance** | 11 células |
| **Área** | 3x3~5x5 células |
| **Consome:** | [Zeny](Zeny.md "Zeny") |
| **Pré-requisitos** | |
| [Estrepes](Estrepes.md "Estrepes") [Estrepes](Estrepes.md "Estrepes") Nv. 3 | |

Atira suas moedas na área selecionada.

Desfere dano fixo aleatório que ignora a [DEF](DEF.md "DEF").

Esse ataque tem chances de sucesso e, em casos de falha, o [zeny](Zeny.md "Zeny") ainda será consumido.

| Nv. | Dano | | Área | [Zeny](Zeny.md "Zeny") |
| --- | --- | --- | --- | --- |
| Mínimo | Máximo |
| 1 | 5.000 | 10.000 | 3x3 células | 10.000 |
| 2 | 10.000 | 20.000 | 20.000 |
| 3 | 15.000 | 30.000 | 30.000 |
| 4 | 20.000 | 40.000 | 40.000 |
| 5 | 25.000 | 50.000 | 50.000 |
| 6 | 30.000 | 60.000 | 60.000 |
| 7 | 35.000 | 70.000 | 70.000 |
| 8 | 40.000 | 80.000 | 80.000 |
| 9 | 45.000 | 90.000 | 90.000 |
| 10 | 50.000 | 100.000 | 5x5 células | 100.000 |

### Fórmula

`Chance = {10 - [1 ÷ (DES + SOR)] × 500} × [(Nv. da habilidade ÷ 2) + 5]`

## Notas

- O dano será dividido entre os inimigos na área.
- É reduzida pela metade em monstros do tipo chefe e em personagens que não aprenderam [Perícia com Shuriken](Perícia com Shuriken.md "Perícia com Shuriken").
- As chances de sucesso em um personagem com 120 de [DES](DES.md "DES") e [SOR](SOR.md "SOR") serão de 80%.
