# Turbilhão de Kunais

|  |  |
| --- | --- |
| **Turbilhão de Kunais Turbilhão de Kunais** | |
| Turbilhão de Kunais | |
| **Kunai Splash / Salpicadura de Kunai** | |
| **ID** | KO\_HAPPOKUNAI (3007) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 2) |
| **Pós-conjuração** | 0,5 segundos |
| **Alvo** | Usuário |
| **Área** | 9x9 a 11x11 células |
| **Munição** | [Over Kunai](Munições.md "Munições") 2 Kunais |
| **Pré-requisitos** | |
| - [Kunai Explosiva](Kunai Explosiva.md "Kunai Explosiva") [Kunai Explosiva](Kunai Explosiva.md "Kunai Explosiva") Nv. 1 | |

Atira kunais em todas as direções enquanto gira.

Desfere dano físico a distância nos oponentes ao redor do usuário.

O nível de [Perícia com Shuriken](Perícia com Shuriken.md "Perícia com Shuriken") influencia no dano causado.

| 1 | 360% | 9x9 células | 12 |
| 2 | 420% | 14 |
| 3 | 480% | 16 |
| 4 | 540% | 18 |
