# Metamorfose

|  |  |
| --- | --- |
| **NPC Skill Icon Metamorfose** | |
| Metamorfose | |
| **ID:** | NPC\_METAMORPHOSIS (193) |
| **Níveis:** | 1 |
| **Exclusiva:** | Sim |

Transforma-se em um novo monstro de sua espécie.

### Escala evolutiva

- [Fabre](http://www.divine-pride.net/database/monster/1007/) → [Pupa](http://www.divine-pride.net/database/monster/1008/) → [Creamy](http://www.divine-pride.net/database/monster/1018/)
- [Ovo de Andre](http://www.divine-pride.net/database/monster/1097/) → [Andre](http://www.divine-pride.net/database/monster/1095/) ou [Piere](http://www.divine-pride.net/database/monster/1160/) ou [Deniro](http://www.divine-pride.net/database/monster/1105/)
- [Ovo de Peco Peco](http://www.divine-pride.net/database/monster/1047/) → [Picky](http://www.divine-pride.net/database/monster/1049/) ou [Picky com Casca](http://www.divine-pride.net/database/monster/1050/)
- [Ovo de Besouro-Ladrão](http://www.divine-pride.net/database/monster/1048/) → [Besouro-Ladrão](http://www.divine-pride.net/database/monster/1051/)
- [Ovo de Chifres Duplos](http://www.divine-pride.net/database/monster/2089/) → [Chifres Duplos](http://www.divine-pride.net/database/monster/2084/)
- [Ovo de Scaraba Chifre](http://www.divine-pride.net/database/monster/2088/) → [Scaraba Chifre](http://www.divine-pride.net/database/monster/2083/)
- [Ovo de Scaraba Ancinho](http://www.divine-pride.net/database/monster/2091/) → [Scaraba Ancinho](http://www.divine-pride.net/database/monster/2086/)
- [Ovo de Scaraba Galhado](http://www.divine-pride.net/database/monster/2090/) → [Scaraba Galhado](http://www.divine-pride.net/database/monster/2085/)
- [Ovo de Draco](http://www.divine-pride.net/database/monster/2014/) → [Draco](http://www.divine-pride.net/database/monster/2013/)
- [Ovo de Dragão](http://www.divine-pride.net/database/monster/1721/) → [Novus Dourado](http://www.divine-pride.net/database/monster/1718/)
- [Lude](http://www.divine-pride.net/database/monster/1509/) → [Quve](http://www.divine-pride.net/database/monster/1508/) (e vice versa)
- [Mascarado](http://www.divine-pride.net/database/monster/1506/) → [Lude](http://www.divine-pride.net/database/monster/1509/) e [Quve](http://www.divine-pride.net/database/monster/1508/)
- [Belzebu](http://www.divine-pride.net/database/monster/1873/) (mosca) → [Belzebu](http://www.divine-pride.net/database/monster/1874/) (demônio)
