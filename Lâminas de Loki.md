# Lâminas de Loki

| **Pré-requisitos** | |
| - [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") Nv. 10 | |

Gira o Katar para atingir oponentes ao redor.

Causa dano físico corpo a corpo na área.

O personagem receberá 1 Ponto de Rotação para cada rodopio, num máximo de 10 pontos.

Os pontos duram 10 segundos e podem ser usados em [Castigo de Loki](Castigo de Loki.md "Castigo de Loki").

| Nv. | [ATQ](ATQ.md "ATQ") | Área |
| --- | --- | --- |
| 1 | 130% | 3x3 células |
| 2 | 210% |
| 3 | 290% | 5x5 células |
| 4 | 370% |
| 5 | 450% | 7x7 células |

### Fórmula

`Dano = {[Dano base × (Nv. de base ÷ 100)]%`

## Notas

- Manter a tecla de atalho pressionada permite usar a habilidade continuadamente.
- Os Pontos de Rotação serão zerados quando a duração acabar ou se o usuário mudar de posição, como usar [Recuar](Recuar.md "Recuar") ou ser empurrado.
