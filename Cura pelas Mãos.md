# Cura pelas Mãos

|  |  |
| --- | --- |
| **Cura pelas Mãos Cura pelas Mãos** | |
| Cura pelas Mãos | |
| **Touch of Heal / Healing Hands** | |
| **ID** | HLIF\_HEAL (8001) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 3) |
| **Recarga** | 20 segundos |
| **Alvo** | Mestre |
| **Item** | 545.png 1 [Poção Vermelha Compacta](http://www.divine-pride.net/database/item/545/) |

Regenera o [HP](HP.md "HP") do mestre, consumindo uma [Poção Vermelha Compacta](http://www.divine-pride.net/database/item/545/) por uso.

A quantia de [HP](HP.md "HP") recuperada aumenta de acordo com a [INT](INT.md "INT") e o nível de base da homunculus.

| Nv. | [SP](SP.md "SP") |
| --- | --- |
| 1 | 13 |
| 2 | 16 |
| 3 | 19 |
| 4 | 22 |
| 5 | 25 |

### Fórmula

`Cura = [(Nv. do Homunculus + INT do Homunculus) ÷ 8] × [(Nv. da habilidade × 8) + 4]`
