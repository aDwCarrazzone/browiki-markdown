# Trindade

| - [Formação Real](Formação Real.md "Formação Real") [Formação Real](Formação Real.md "Formação Real") Nv. 3 - [Prestígio Divino](Prestígio Divino.md "Prestígio Divino") [Prestígio Divino](Prestígio Divino.md "Prestígio Divino") Nv. 3 | |

Causa dano físico corpo a corpo no alvo.

O dano é afetado pela [VIT](VIT.md "VIT") e nível de base do usuário.

Se [Consagração](Consagração.md "Consagração") estiver ativa, garante um bônus no dano causado.

| Nv. | [ATQ](ATQ.md "ATQ") | | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| Normal | Bônus |
| 1 | 300% | 450% | 37 |
| 2 | 600% | 900% | 44 |
| 3 | 900% | 1.350% | 51 |
| 4 | 1.200% | 1.800% | 58 |
| 5 | 1.500% | 2.250% | 65 |

### Fórmula

Normal:  
`Dano = [(Dano base + VIT) × (Nv. de base ÷ 100)]%`

Consagração:  
`Dano = {[(Dano base + (VIT × 2)] × (Nv. de base ÷ 100)}%`
