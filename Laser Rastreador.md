# Laser Rastreador

|  |  |
| --- | --- |
| **NPC Skill Icon Laser Rastreador** | |
| Laser Rastreador | |
| **ID:** | NPC\_LOCKON\_LASER (773) |
| **Níveis:** | 5 |
| **Exclusiva:** | Sim |
| **Alcance:** | 11 células |
| **Propriedade:** | [Neutro](Monstros.md#Propriedade "Monstros") |
| **Duração:** | Ver tabela |
| **[Ícone](Ícone.md "Ícone"):** | I Marcar em Brasa |

Trava uma mira no alvo, que dura por alguns segundos.

A cada 1 segundo, o alvo sofrerá dano fixo de 30.000 de [HP](HP.md "HP").

Oponentes ao redor de 7x7 células ao redor do alvo também sofrerão dano.

| Nv. | Duração |
| --- | --- |
| 1 | 4 segundos |
| 2 | 6 segundos |
| 3 | 8 segundos |
| 4 | 10 segundos |
| 5 | 12 segundos |

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Laser Rastreador](http://www.divine-pride.net/database/skill/773)*
- O ataque cai na célula em que o personagem se encontra. Caso o jogador se mova antes do ataque, ele não será atingido.
- A mira não pode ser removida por [Desencantar](Desencantar.md "Desencantar") ou [Gênese](Gênese.md "Gênese").
- O dano não pode ser bloqueado por [Imunidade Mágica](Imunidade Mágica.md "Imunidade Mágica").
- O dano é fixo, ignorando [DEF](DEF.md "DEF") e [DEFM](DEFM.md "DEFM"), mas pode ser reduzido por:
  - [![Corpo Fechado](images/c/c4/Corpo_Fechado.png)](Corpo Fechado.md "Corpo Fechado") [Corpo Fechado](Corpo Fechado.md "Corpo Fechado")
  - [![Proteção Arcana](images/d/d6/Proteção_Arcana.png)](Proteção Arcana.md "Proteção Arcana") [Proteção Arcana](Proteção Arcana.md "Proteção Arcana")
  - [![Kyrie Eleison](images/f/f4/Kyrie_Eleison.png)](Kyrie Eleison.md "Kyrie Eleison") [Kyrie Eleison](Kyrie Eleison.md "Kyrie Eleison")
- Pode ser esquivada por [Cambalhota](Cambalhota.md "Cambalhota").
- Pode bloqueado por [Escudo Mágico](Escudo Mágico.md "Escudo Mágico")
