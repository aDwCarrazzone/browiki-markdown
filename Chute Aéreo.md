# Chute Aéreo

|  |  |
| --- | --- |
| **Chute Aéreo Chute Aéreo** | |
| Chute Aéreo | |
| **Flying Kick / Patada Voladora** | |
| **ID** | TK\_JUMPKICK (421) |
| **Tipo** | Ofensiva |
| **Níveis** | 7 |
| **[SP](SP.md "SP")** | 80 - (Nv. da habilidade × 10) |
| **Alvo** | Personagens e [monstros](Monstros.md "Monstros") |
| **Alcance** | 10 células |
| **Pré-requisitos** | |
| - Nenhum | |

Pula instantaneamente no alvo, causando dano físico e cancelando alguns efeitos.

O dano pode ser aumentado se usado após um bloqueio de [Cambalhota](Cambalhota.md "Cambalhota") ou durante a [Corrida](Corrida.md "Corrida").

| Nv. | [ATQ](ATQ.md "ATQ") | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 40% | 70 |
| 2 | 50% | 60 |
| 3 | 60% | 50 |
| 4 | 70% | 40 |
| 5 | 80% | 30 |
| 6 | 90% | 20 |
| 7 | 100% | 10 |

### Efeitos cancelados

- [Kaahi](Kaahi.md "Kaahi")
- [Kaite](Kaite.md "Kaite")
- [Kaizel](Kaizel.md "Kaizel")
- [Kaupe](Kaupe.md "Kaupe")
- Espíritos de [Espiritualistas](Espiritualistas.md "Espiritualistas").
- [Poção da Fúria Selvagem](http://www.divine-pride.net/database/item/657/)

## Notas

- Não pode ser usada por [Espiritualistas](Espiritualistas.md "Espiritualistas").
- Segurar a tecla *Shift* permite que a habilidade seja usada em outros personagens, pulando instantaneamente até o alvo.
- [Desordeiros](Desordeiros.md "Desordeiros") e evoluções com [Preservar](Preservar.md "Preservar") ativo e [Espiritualistas](Espiritualistas.md "Espiritualistas") são imunes a remoção dos efeitos.
