# Perícia com a Mão Esquerda

|  |  |
| --- | --- |
| **Perícia com a Mão Esquerda Perícia com a Mão Esquerda** | |
| SemImagem | |
| **Left Hand Mastery / Maestría de Mano Izquierda** | |
| **ID** | KO\_LEFT (3003) |
| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Pré-requisitos** | |
| Nenhuma | |

Ao equipar duas armas, reduz a penalidade de dano ao equipar uma arma na mão esquerda.

| Nv. | Dano |
| --- | --- |
| 1 | 60% |
| 2 | 70% |
| 3 | 80% |
| 4 | 90% |
| 5 | 100% |

## Notas

- [Kagerou](Kagerou.md "Kagerou") e [Oboro](Oboro.md "Oboro") podem equipar duas armas, mas haverá penalidade no dano físico ao equipar uma segunda arma.
- A arma da mão esquerda causará apenas 50% do seu dano original.
- Por conta disso, cada nível aprendido desta habilidade, a penalidade de dano é corrigida.
- No nível 5, a penalidade é totalmente anulada, retornando ao dano original.
- Sobre Manejo Duplo:
  - Apenas [Mercenários](Mercenários.md "Mercenários"), [Kagerou](Kagerou.md "Kagerou"), [Oboro](Oboro.md "Oboro") e evoluções podem equipar armas nas duas mãos.
  - A mão esquerda pode equipar escudos ou armas.
  - No caso de habilidades mágicas, os bônus mágicos da mão esquerda são contabilizados.
  - No caso de habilidades físicas:
    - Apenas os bônus de equipamento são contados.
    - O ATQ natural da arma, o [refino](Refino.md "Refino") e a propriedade não são contabilizados.
    - [Cartas](Cartas.md "Cartas") e [Bônus Aleatórios](Bônus Aleatórios.md "Bônus Aleatórios") são contados.
  - Em ataques físicos básicos, todos os bônus da mão esquerda são contados normalmente (ATQ natural e propriedade).
