# Visita do Manuk

| EP ico Visita do Manuk | |
| --- | --- |
| Interessados nos conhecimentos dos humanos, os Manuks enviaram um ministro de Diel para permitir a entrada de aventureiros na capital do mundo de gelo. | |
| Requisitos mínimos | |
| **Nv. de base** | 70 |
| **Tipo** | Episódio |
| **Equipado** | 2782.png [Anel do Rei Sábio](http://www.divine-pride.net/database/item/2782/) |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 2844.png 1 [Luz de El Dicastes](http://www.divine-pride.net/database/item/2844/) |
| **Recompensa** | Acesso a [El Dicastes](El Dicastes.md "El Dicastes") e ao [Túnel Kamidal](http://www.divine-pride.net/database/map/dic_dun01/) |

1. Você não precisa ter completado a missão [Anel do Rei Sábio](Anel do Rei Sábio.md "Anel do Rei Sábio") caso já tenha o anel.
2. Converse com o Viajante Ben (mid\_camp 222,246)

   :   ![VdS01](images/3/3e/VdS01.png)
3. Fale com Guarda Aello (mid\_campin 93,114)

   :   ![VdS02](images/e/e0/VdS02.png)
4. Entre no quarto falando com o Soldado (mid\_campin 111,120)

   :   ![VdS03](images/3/34/VdS03.png)
5. Converse com o Embaixador de Manuk (mid\_campin 168,128)

   :   ![VdS04](images/e/e6/VdS04.png)

   - Selecione:
     - *Digo o mesmo.*
     - *El Dicastes?*
     - *Hound, eu ouvi direito?*
   - Fale com o Hound, ao lado. Converse mais de uma vez com esses [NPCs](NPCs.md "NPCs").
6. Dentro do prédio e subindo as escadas, procure pelo Comandante Avalanche (man\_in01 314,52)

   :   ![VdS04b](images/2/23/VdS04b.png)
7. Acesse o [Túnel Kamidal](http://www.divine-pride.net/database/map/dic_dun01/) falando com o Gerente da Entrada (manuk 320,181)

   :   ![VdS04a](images/d/d1/VdS04a.png)
8. Após atravessar o [Túnel Kamidal](http://www.divine-pride.net/database/map/dic_dun01/), entre em [El Dicastes](El Dicastes.md "El Dicastes") com o Guarda da Capital (dic\_fild01 147,280)

   :   ![VdS04c](images/8/82/VdS04c.png)
9. Dirija-se ao norte de [El Dicastes](El Dicastes.md "El Dicastes"), entre no castelo e registre-se com o Aventureiro Escrivão (dic\_in01 40,261).

   :   ![VdS05](images/7/77/VdS05.png)

   - Você receberá uma [Luz de El Dicastes](http://www.divine-pride.net/database/item/2844/).
10. Você pode [encantar](Encantamento.md "Encantamento") a [Luz de El Dicastes](http://www.divine-pride.net/database/item/2844/) com Kareka (dicastes01 96,257)

    :   ![VdS06](images/e/e0/VdS06.png)

    - O serviço de [encantamento](Encantamento.md "Encantamento") funciona da seguinte forma:
      - Encantar o 1º slot, custará 1 [Certificado do Explorador](http://www.divine-pride.net/database/item/6304/).
      - Encantar o 2º slot, custará 2 [Certificados do Explorador](http://www.divine-pride.net/database/item/6304/).
      - Encantar o 3º slot, custará 3 [Certificados do Explorador](http://www.divine-pride.net/database/item/6304/).
      - Não há opção de reset.
      - Os [Certificados do Explorador](http://www.divine-pride.net/database/item/6304/) são adquiridos nas missões do [Departamento de Exploradores](Departamento de Exploradores.md "Departamento de Exploradores") e [Relíquias Desconhecidas](Relíquias Desconhecidas.md "Relíquias Desconhecidas").
