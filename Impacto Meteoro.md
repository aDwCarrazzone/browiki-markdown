# Impacto Meteoro

|  |  |
| --- | --- |
| **Impacto Meteoro Impacto Meteoro** | |
| Impacto Meteoro | |
| **Meteor Assault / Asalto Meteórico** | |
| **ID** | ASC\_METEORASSAULT (406) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 3) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,25 + 0,25 seg. |
| **Recarga** | 0,5 segundos |
| **Alvo** | Usuário |
| **Área** | 5x5 células |
| **Efeito negativo** | [Cegueira](Cegueira.md "Cegueira"), [Atordoamento](Atordoamento.md "Atordoamento") e [Sangramento](Sangramento.md "Sangramento") |
| **Pré-requisitos** | |
| - [Perícia com Katar](Perícia com Katar.md "Perícia com Katar") [Perícia com Katar](Perícia com Katar.md "Perícia com Katar") Nv. 5 - [Destruidor de Almas](Destruidor de Almas.md "Destruidor de Almas") [Destruidor de Almas](Destruidor de Almas.md "Destruidor de Almas") Nv. 1 - [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") Nv. 5 - [Perícia com Mão Direita](Perícia com Mão Direita.md "Perícia com Mão Direita") [Perícia com Mão Direita](Perícia com Mão Direita.md "Perícia com Mão Direita") Nv. 3 | |

Desfere um poderoso golpe 5x5 células ao redor de si.

Causa dano físico corpo a corpo com chance de infligir [Cegueira](Cegueira.md "Cegueira"), [Atordoamento](Atordoamento.md "Atordoamento") ou [Sangramento](Sangramento.md "Sangramento").

| Nv. | [ATQ](ATQ.md "ATQ") | Chance | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 320% | 10% | 13 |
| 2 | 440% | 15% | 16 |
| 3 | 560% | 20% | 19 |
| 4 | 680% | 25% | 22 |
| 5 | 800% | 30% | 25 |
| 6 | 920% | 35% | 28 |
| 7 | 1.040% | 40% | 31 |
| 8 | 1.160% | 45% | 34 |
| 9 | 1.280% | 50% | 37 |
| 10 | 1.400% | 55% | 40 |

### Fórmula

`ATQ = [{[(Nv. da hab. × 120) + 200] + (FOR × 5)} × (Nv. de base ÷ 100)]%`

## Notas

- É afetada por [Investigar](Investigar.md "Investigar").
- Não é afetada por [Veneno Mortal](Encantar com Veneno Mortal.md "Encantar com Veneno Mortal").
- Não inflige os efeitos de destruir arma ou armadura.
