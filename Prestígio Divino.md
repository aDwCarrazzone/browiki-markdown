# Prestígio Divino

|  |  |
| --- | --- |
| **Prestígio Divino Prestígio Divino** | |
| Prestígio Divino | |
| **Prestige / Prestigio** | |
| **ID** | LG\_PRESTIGE (2318) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 70 + (Nv. da habilidade × 5) |
| **[Conjuração](Conjuração.md "Conjuração")** | 2 + 1 seg. |
| **Recarga** | 60 segundos |
| **Duração** | [15 + (Nv. da hab. ×15]) segundos |
| **Alvo** | Usuário |
| **Ícone** | I Prestígio Divino |
| **Pré-requisitos** | |
| - [Pisotear Armadilha](Pisotear Armadilha.md "Pisotear Armadilha") [Pisotear Armadilha](Pisotear Armadilha.md "Pisotear Armadilha") Nv. 3 | |

O usuário se concentra para oração.

Melhora a [DEF](DEF.md "DEF") dos [equipamentos](Equipamentos.md "Equipamentos") e há chance de bloquear ataques mágicos.

| Nv. | [DEF](DEF.md "DEF") | Chance | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| 1 | +15 | 1% | 30 segundos | 75 |
| 2 | +30 | 2% | 45 segundos | 80 |
| 3 | +45 | 3% | 60 segundos | 85 |
| 4 | +60 | 4% | 75 segundos | 90 |
| 5 | +75 | 5% | 90 segundos | 95 |

### Fórmula

`Chance = {[(INT + SOR) × (Nv. da hab. ÷ 20) × (Nv. de Base ÷ 200)] + Nv. da hab.}%`

`DEF = [(Nv. da hab. × 15) + (Nv. de Aura Sagrada × 10)] × (Nv. de base ÷ 100)`

## Notas

- Não pode ser usada simultaneamente com [Formação Real](Formação Real.md "Formação Real").
- Funciona com magias de alvo único ou com habilidades mágicas de área, como [Nevasca](Nevasca.md "Nevasca"), [Ira de Thor](Ira de Thor.md "Ira de Thor") e etc.
