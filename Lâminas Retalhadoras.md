# Lâminas Retalhadoras

|  |  |
| --- | --- |
| **Lâminas Retalhadoras Lâminas Retalhadoras** | |
| Lâminas Retalhadoras | |
| **Cross Impact / Impacto Cruzado** | |
| **ID** | GC\_CROSSIMPACT (2022) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 40 |
| **Pós-conjuração** | [3 - (Nv. da hab. × 0,5)] segundos |
| **Recarga** | 0,35 segundos |
| **Alvo** | Oponente |
| **Alcance** | 7 células |
| **Pré-requisitos** | |
| - [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") [Lâminas Destruidoras](Lâminas Destruidoras.md "Lâminas Destruidoras") Nv. 10 | |

Avança até o alvo para causar dano físico corpo a corpo.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual à metade do [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") | Pós-conjuração |
| --- | --- | --- |
| 1 | 1.550% | 2,5 segundos |
| 2 | 1.700% | 2,0 segundos |
| 3 | 1.850% | 1,5 segundos |
| 4 | 2.000% | 1,0 segundo |
| 5 | 2.150% | 0,5 segundos |

### Fórmula

`Dano = [Dano base × (Nv. de base ÷ 100)]%`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- Por ser uma habilidade de locomoção, pode resetar os [Pontos de Rotação](Lâminas de Loki.md "Lâminas de Loki"), mesmo que o personagem esteja na mesma célula do alvo.
