# Amplificar Poder

|  |  |
| --- | --- |
| **Amplificar Poder Amplificar Poder** | |
| Amplificar Poder | |
| **Power Maximize / Poder Maximizado** | |
| **ID** | BS\_MAXIMIZE (114) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 |
| **Duração** | Até o [SP](SP.md "SP") chegar a zero |
| **Alvo** | Usuário |
| **Ícone** | I Amplificar Poder |
| **Pré-requisitos** | |
| - [Força Violenta](Força Violenta.md "Força Violenta") [Força Violenta](Força Violenta.md "Força Violenta") Nv. 2 - [Manejo Perfeito](Manejo Perfeito.md "Manejo Perfeito") [Manejo Perfeito](Manejo Perfeito.md "Manejo Perfeito") Nv. 3 | |

Desativa a regeneração natural de [SP](SP.md "SP") para causar dano máximo ao atacar fisicamente.

Enquanto estiver ativa, drena 1 de [SP](SP.md "SP") de acordo com o nível da habilidade.

| Nível | Dreno de [SP](SP.md "SP") |
| --- | --- |
| 1 | A cada 1 segundo |
| 2 | A cada 2 segundos |
| 3 | A cada 3 segundos |
| 4 | A cada 4 segundos |
| 5 | A cada 5 segundos |

## Notas

- Enquanto estiver ativa, ataques críticos não serão causados.
- É cancelada se usada novamente ou o [SP](SP.md "SP") chegar a zero.
