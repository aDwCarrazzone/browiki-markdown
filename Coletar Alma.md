# Coletar Alma

|  |  |
| --- | --- |
| **Coletar Alma Coletar Alma** | |
| Coletar Alma | |
| **Soul Collection / Recolección de Almas** | |
| **ID** | SP\_SOULCOLLECT (2609) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 100 |
| **Alvo** | Usuário |
| **Ícone** | I Coletar Alma |
| **Pré-requisitos** | |
| Nenhum | |

Permite atrair as almas que estão vagando pelo mundo para perto de você.

Ganha 1 Alma a cada tempo determinado pelo nível da habilidade.

Cada Alma concede [ATQM](ATQM.md "ATQM") +3 e duram 10 minutos.

| Nv. | Coleta de almas |
| --- | --- |
| 1 | 1 Alma a cada 60 segundos |
| 2 | 1 Alma a cada 50 segundos |
| 3 | 1 Alma a cada 40 segundos |
| 4 | 1 Alma a cada 30 segundos |
| 5 | 1 Alma a cada 20 segundos |

## Notas

- A coleta termina quando o número de almas atinge o limite determinado por [Perícia com Almas](Perícia com Almas.md "Perícia com Almas") ou até 5 almas, se não tiver aprendido a habilidade.
- O status pode ser cancelado por [Gênese](Gênese.md "Gênese"), mas não remove as Almas já coletadas.
- As Almas não podem ser removidas por [Absorver Esferas Espirituais](Absorver Esferas Espirituais.md "Absorver Esferas Espirituais"), [Máscara da Fraqueza](Máscara da Fraqueza.md "Máscara da Fraqueza"), [Desencantar](Desencantar.md "Desencantar") e derivados.
- As Almas não serão perdidas se o usuário morrer ou mudar de zonas.
- O status pode ser ativado ou desativado.
