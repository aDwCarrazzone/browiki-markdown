# Comandos

Configure o jogo da melhor forma para uma jogabilidade mais personalizada.

Para acessar o menu de *Configurações do Jogo* e de *Teclas de Atalho*, basta apertar o botão `ESC`

![ESCbutau](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/0a/ESCbutau.png)

## Configurações

Na janela de configurações, você pode ajustar música, efeitos sonoros, efeitos gráficos, idioma e até alterar a *Skin* do jogo.

![Settingwnd](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/0/0a/Settingwnd.png)

## Atalhos

Os atalhos rápidos podem ser alterados de acordo com o seu gosto, não apenas para uso de [habilidades](../Habilidades%20especiais/Habilidades.md "Habilidades"), mas também para acesso a outras janelas.

![Hotkeyteclas](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/31/Hotkeyteclas.png)

## Teclas

É possível inserir comandos no campo de mensagens para alterar o sistema do jogo e exibir informações.

Lembre-se de que algumas funções, como exibir as coordenadas de posição de um número, só podem ser operadas por meio de comandos.

| Tecla | Descrição |
| --- | --- |
| ESC | Abre a janela de configurações do jogo. |
| F1 até F9 | Teclas de atalho rápido, podem ser personalizadas pelo usuário. |
| F10 | Aumenta ou diminui o tamanho da janela de conversa. |
| F11 | Esconde ou exibe as janelas da interface. |
| F12 | Mostra, expande ou esconde a barra de atalhos. |
| Print Screen | Tira uma foto do jogo e salva em formato *.jpg*. As imagens são armazenadas na pasta "*screenshot*", dentro da pasta "Ragnarok Online". |
| Tab | Alterna entre as abas na janela de conversa. |
| Insert | Faz o personagem sentar ou levantar. Precisa ter [Instintos Básicos](../Passiva/Instintos%20B%C3%A1sicos.md "Instintos Básicos") ou nível 3 de [Habilidades Básicas](../Passiva/Habilidades%20B%C3%A1sicas.md "Habilidades Básicas"). |
| Alt + Clique direito | Transfere [itens](../Itens/Itens.md "Itens") do inventário para o [armazém](Servi%C3%A7os.md "Serviços") ou [carrinho](../Passiva/Usar%20Carrinho.md "Usar Carrinho") e vice-versa. |
| Alt + Enter | Envia mensagens para o [clã](../PvP/Cl%C3%A3.md "Clã") do personagem. |
| Alt + End | Exibe ou esconde a barra de [HP](../Sem%20categoria/HP.md "HP")/[SP](../Sem%20categoria/SP.md "SP"). |
| Alt + F10 | Mostra ou esconde a janela de conversa. |
| Alt + Home | Trava ou destrava o cursor verde do solo. |
| Alt + Q | Abre a janela de [equipamentos](../Sem%20categoria/Equipamentos.md "Equipamentos"). |
| Alt + X | Abre a janela de [visuais](../Sem%20categoria/Visuais.md "Visuais"). |
| Alt + A | Abre a janela de [atributos](Atributos.md "Atributos"). |
| Alt + W | Abre a janela de itens do carrinho. Apenas [classes](Classes.md "Classes") que possuem um [carrinho](../Passiva/Usar%20Carrinho.md "Usar Carrinho") podem acessá-la. |
| Alt + E | Abre o seu Inventário. |
| Alt + R | Abre a janela de informações do seu [Homunculus](../Homunculus/Homunculus.md "Homunculus"). Apenas [Alquimistas](../Sem%20categoria/Alquimista.md "Alquimista") e suas evoluções podem acessá-la. |
| Alt + T | Alterna a A.I. do [Homunculus](../Homunculus/Homunculus.md "Homunculus") entre agressivo e passivo. |
| Alt + Y | Uma janela para vários comandos se abrirá. |
| Alt + U | Abre a janela de [quests](../Quest/Quests.md "Quests"). |
| Alt + S | Abre a árvore de [habilidades](../Habilidades%20especiais/Habilidades.md "Habilidades"). |
| Alt + G | Abre a janela de [clã](../PvP/Cl%C3%A3.md "Clã"). |
| Alt + H | Abre a janela de contatos. |
| Alt + J | Abre a janela de [mascotes](Mascotes.md "Mascotes"), caso você tenha um. |
| Alt + L | Abre a janela de emoticons. |
| Alt + Z | Abre a janela de [grupo](Grupo.md "Grupo"). |
| Alt + C | Abre uma janela onde você poderá criar uma sala de chat. |
| Alt + V | Abre a janela de informações do seu personagem. |
| Alt + B | Alterna a exibição da janela de [instância](../Sem%20categoria/Inst%C3%A2ncia.md "Instância") sem que ela feche em definitivo. |
| Alt + M | Abre uma janela de atalhos de comando. |
| Alt + ' | Abre o menu da interface de informações. |
| Ctrl + Tab | Alterna a aparência do mini-mapa entre desligado, semitransparente ou sem transparência. |
| Ctrl + Enter | Envia uma mensagem para o [grupo](Grupo.md "Grupo"). |
| Ctrl + End | Mostra/Esconde personagens (transparência). Funciona apenas em cidades. |
| Ctrl + Print Screen | Tira uma foto do jogo e salva em formato *.bmp*. As imagens são armazenadas na pasta "*screenshot*", dentro da pasta "Ragnarok Online". |
| Ctrl + + | Aumenta o zoom do mini-mapa. |
| Ctrl + - | Diminui o zoom do mini-mapa. |
| Ctrl + ' | Exibe o mapa-múndi. |
| Ctrl + \* | Ativa o modo de transparência do mapa. |
| Ctrl + B | Abre a janela do [Banco](../Sem%20categoria/Banco.md "Banco"). |
| Ctrl + G | Abre a janela de [Clãs Reais](Cl%C3%A3s%20Reais.md "Clãs Reais"). |
| Ctrl + Q | Abre a janela de soma dos bônus de [equipamentos](../Sem%20categoria/Equipamentos.md "Equipamentos"). |
| Ctrl + R | Abre a janela de [assistentes](../Itens/Assistentes.md "Assistentes"). |
| Ctrl + T | Alterna a A.I. do [assistente](../Sem%20categoria/Assistente.md "Assistente") entre agressivo e passivo. |
| Ctrl + Z | Abre a janela da [Agência de Aventuras](../Sem%20categoria/Ag%C3%AAncia%20de%20Aventuras.md "Agência de Aventuras"). |
| Ctrl + Clique direito | Identifica [itens](../Itens/Itens.md "Itens") não identificados no inventário. Consome 1 Lupa. |
| Ctrl + F12 | Alterna as abas do atalho rápido. |
| Shift + Clique esquerdo no item | Cria um *link* de descrição do item clicado para o chat de conversa. |
| Shift + Clique direito no item | Move um item para a aba de favoritos do inventário. |
| Shift + Insert | Cola textos da área de transferência. Funciona como um *Ctrl + V* no jogo. |
