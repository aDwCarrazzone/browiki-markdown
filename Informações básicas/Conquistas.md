# Conquistas

Diferente das [quests](../Quest/Quests.md "Quests"), o sistema de Conquistas traz tarefas para que um jogador possa explorar mais um pouco de [Ragnarök Online](Ragnar%C3%B6k%20Online.md "Ragnarök Online").

Essas conquistas podem coincidir com os objetivos principais do jogo, como alcançar o nível máximo.

Algumas conquistas podem recompensar o usuário com um [item](../Sem%20categoria/Item.md "Item"), Título ou um efeito especial como [Bênção](../Suporte/B%C3%AAn%C3%A7%C3%A3o.md "Bênção") e [Aumentar Agilidade](../Suporte/Aumentar%20Agilidade.md "Aumentar Agilidade").

As conquistas que dão [itens](../Itens/Itens.md "Itens") são enviadas ao personagem através do [RODEX](../Sem%20categoria/RODEX.md "RODEX").

Já os Títulos podem ser ativados através da Janela de Equipamentos ([ALT + Q](Comandos.md "Comandos")), clicando na aba *Título*.

### Aventura

- As missões de aventura consistem em encontrar um baú de tesouro em áreas escondidas no mapa.
- O Baú não aparece para outros personagens, somente para você.
- O Baú só aparece quando você se aproxima das coordenadas ao redor dele.
- O prêmio por desbloquear a conquista é uma [Bolsa de Zenys](http://www.divine-pride.net/database/item/22876/).
- Ao encontrar todos os baús de uma região, você receberá uma [Caixa de Presente](http://www.divine-pride.net/database/item/644/).
- Quando encontrar todos os baús de um continente, você receberá uma [Velha Caixa Roxa](http://www.divine-pride.net/database/item/617/).
- Clique nas imagens para ampliar os mapas.
