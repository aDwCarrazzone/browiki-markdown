# Esmagamento Sombrio

|  |  |
| --- | --- |
| **Esmagamento Sombrio Esmagamento Sombrio** | |

| **Shadow Trampling / Pisoteo de Sombra** | |
| **ID** | KG\_KAGEHUMI (3023) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 + (5× Nv. da habilidade) |
| **Duração** | (4× Nv. da habilidade) segundos |
| **Alvo** | Usuário |
| **Área** | 3x3~13x13 células |
| **Ícone** | I Esmagamento Sombrio |
| **Pré-requisitos** | |
| [Clone das Sombras](Clone das Sombras.md "Clone das Sombras") [Clone das Sombras](Clone das Sombras.md "Clone das Sombras") Nv. 1 | |

Revela alvos do [esconderijo](Esconderijo.md "Esconderijo") e os mantém imóveis, sendo impedidos de conjurar [habilidades](Habilidades.md "Habilidades").

| Nv. | Área | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 5x5 células | 5 segundos | 25 |
| 2 | 7x7 células | 6 segundos | 30 |
| 3 | 9x9 células | 7 segundos | 35 |
| 4 | 11x11 células | 8 segundos | 40 |
| 5 | 13x13 células | 9 segundos | 45 |

## Notas

- Só funciona em áreas de [PvP](PvP.md "PvP") e na [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
- Afeta qualquer habilidade de esconderijo, tais como [Furtividade](Furtividade.md "Furtividade"), [Ocultação](Ocultação.md "Ocultação"), [Vínculo Sombrio](Vínculo Sombrio.md "Vínculo Sombrio") e [Camuflagem](Camuflagem.md "Camuflagem").
- Pode cancelar a habilidade [Controle de Marionete](Controle de Marionete.md "Controle de Marionete").
