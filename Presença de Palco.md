# Presença de Palco

|  |  |
| --- | --- |
| **Presença de Palco Presença de Palco** | |
| SemImagem | |
| **Stage Etiquette / Presencia Escénica** | |
| **ID** | TR\_STAGE\_MANNER (5349) |
| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Arma** | Arco ou Chicote ou Instrumento musical |
| **Pré-requisitos** | |
| - Nenhum | |

Melhora sua postura em apresentações.

Ao usar [habilidades](Habilidades.md "Habilidades") ofensivas de regeneração de [AP](AP.md "AP"), regenera +1 de [AP](AP.md "AP").

Equipar armas do tipo Arco, Chicote ou Instrumento Musical aumenta o [P.ATQ](P.ATQ.md "P.ATQ") e [S.ATQM](S.ATQM.md "S.ATQM").

| Nv. | [P.ATQ](P.ATQ.md "P.ATQ") e [S.ATQM](S.ATQM.md "S.ATQM") |
| --- | --- |
| 1 | +3 |
| 2 | +6 |
| 3 | +9 |
| 4 | +12 |
| 5 | +15 |
