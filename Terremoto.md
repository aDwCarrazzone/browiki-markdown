# Terremoto

|  |  |
| --- | --- |
| **NPC Skill Icon Terremoto** | |
| Terremoto | |
| **ID:** | NPC\_EARTHQUAKE\_K (750) |
| **Níveis:** | 10 |
| **Exclusiva:** | Não |
| **Propriedade:** | [Neutro](Monstros.md#Propriedade "Monstros") |

Cria um tremor que provoca 3 ondas.

Causa dano mágico de propriedade [Neutro](Neutro.md "Neutro") ao redor.

O dano é baseado no [ATQ](ATQ.md "ATQ") do conjurador.

Cada onda atinge um alvo único, tendo seu dano dividido entre eles.

Quanto mais oponentes dentro da área, menor será o dano.

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Terremoto](https://www.divine-pride.net/database/skill/750)*.
- Cada onda pode ser refletida por [itens](Itens.md "Itens") como a [carta Maya](http://www.divine-pride.net/database/item/4146/) e a [Travessa de Orleans](http://www.divine-pride.net/database/item/2123/).
- [Kaupe](Kaupe.md "Kaupe") e [Cambalhota](Cambalhota.md "Cambalhota") só bloqueiam o primeiro dano.
- Os danos não podem ser bloqueados por [Escudo Mágico](Escudo Mágico.md "Escudo Mágico"), [Escudo Sagrado](Escudo Sagrado.md "Escudo Sagrado"), [Kyrie Eleison](Kyrie Eleison.md "Kyrie Eleison") e [Proteger Terreno](Proteger Terreno.md "Proteger Terreno").
- Até o nível 6, o dano não pode ser reduzido por resistências de [Raça](Raça.md "Raça"), [Propriedade](Propriedade.md "Propriedade"), [Tamanho](Tamanho.md "Tamanho") e de monstros Chefes, como a [Carta Alice](http://www.divine-pride.net/database/item/4253/).
- Independente do nível, o dano ainda pode ser reduzido por:
  - Habilidades como [Proteção Arcana](Proteção Arcana.md "Proteção Arcana") e [Analgésico](Analgésico.md "Analgésico").
  - Número de personagens, [assistentes](Assistentes.md "Assistentes") e [homunculus](Homunculus.md "Homunculus") na área do efeito.
- O dano é recalculado a cada uma das três ondas. Se os personagens morrerem durante a primeira ou segunda onda, o dano das ondas subsequentes não será mais dividido entre eles. Porém, se ainda houver um alvo, ele receberá todo o dano do terceiro golpe.
