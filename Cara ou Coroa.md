# Cara ou Coroa

|  |  |
| --- | --- |
| **Cara ou Coroa Cara ou Coroa** | |
| Cara ou Coroa | |
| **Coin Flip / Cara o Cruz** | |
| **ID** | GS\_GLITTERING (500) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 2 |
| **Alvo** | Usuário |
| **Consome:** | 1 [zeny](Zeny.md "Zeny") |
| **Pré-requisitos** | |
| Nenhum | |

Consome 1 [zeny](Zeny.md "Zeny") para dar a chance de conseguir 1 moeda, num máximo de 10.

Em casos de falha, o personagem perde 1 moeda.

Cada moeda fornece [ATQ](ATQ.md "ATQ") +3.

| Nv. | Chance |
| --- | --- |
| 1 | 30% |
| 2 | 40% |
| 3 | 50% |
| 4 | 60% |
| 5 | 70% |

## Notas

- As moedas ficam ativas no personagem por 10 minutos.
- Ao aprender [Farta Fortuna](Farta Fortuna.md "Farta Fortuna"), a habilidade não perderá 1 moeda em caso de falha.
