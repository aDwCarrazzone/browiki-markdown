| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 + (Nv. da habilidade × 20) |

| **Pós-conjuração** | 1 segundo |

| **Alcance** | 9 células |
| **Área** | 3x3 a 7x7 células |
| **Propriedade** | [Veneno](Monstros.md#Propriedade "Monstros") |
| **Pré-requisitos** | |
| - [Maldição de Jormungand](Maldição de Jormungand.md "Maldição de Jormungand") [Maldição de Jormungand](Maldição de Jormungand.md "Maldição de Jormungand") Nv. 2 | |

Controla o veneno que corre nas veias de um alvo, implodindo-o.

Causa dano mágico de propriedade [Veneno](Veneno.md "Veneno") no oponente e arredores dele.

Se oponentes estiverem sofrendo de [Infecção](Maldição de Jormungand.md "Maldição de Jormungand"), o dano será ainda maior.

`Dano = [(Dano base + INT) × (Nv. de base ÷ 100)]%`
