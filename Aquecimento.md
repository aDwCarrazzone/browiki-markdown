# Aquecimento

|  |  |
| --- | --- |
| **Aquecimento Aquecimento** | |
| Aquecimento | |
| **Amp / Sutileza** | |
| **ID** | BD\_ADAPTATION (304) |
| **Tipo** | Suporte |
| **Níveis** | 1 |
| **[SP](SP.md "SP")** | 10 |
| **Pós-conjuração** | 0,3 segundos |
| **Recarga** | 300 segundos |
| **Duração** | 180 segundos |
| **Alvo** | Usuário |
| **Ícone** | I Aquecimento |
| **Pré-requisitos** | |

Faz um aquecimento para adaptar-se às circunstâncias.

Por 3 minutos, reduz em 20% o consumo de [SP](SP.md "SP") das apresentações.

## Notas

- Acumula com [Bis](Bis.md "Bis").
- Não funciona com habilidades de [classes](Classes.md "Classes") 3.
- Em casos de Duetos, apenas quem está com Aquecimento ativo terá o [SP](SP.md "SP") reduzido.
- É cancelada ao morrer, mas não é removida por [Desencantar](Desencantar.md "Desencantar") e [Gênese](Gênese.md "Gênese").
