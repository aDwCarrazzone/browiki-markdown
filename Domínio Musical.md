# Domínio Musical

|  |  |
| --- | --- |
| **Domínio Musical Domínio Musical** | |
| Domínio Musical | |
| **Voice Lessons / Lecciones de Voz** | |
| **ID** | WM\_LESSON (2412) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| - Nenhum | |

Domina as técnicas de música, permitindo aumentar o [SP](SP.md "SP") máx.

Enquanto estiver imóvel, a cada 10 segundos, regenera o [SP](SP.md "SP") em uma quantia fixa.

| Nv. | [SP](SP.md "SP") máx. | Regeneração |
| --- | --- | --- |
| 1 | +30 | 6 |
| 2 | +60 | 9 |
| 3 | +90 | 12 |
| 4 | +120 | 15 |
| 5 | +150 | 18 |
| 6 | +180 | 21 |
| 7 | +210 | 24 |
| 8 | +240 | 27 |
| 9 | +270 | 30 |
| 10 | +300 | 33 |

## Notas

- O efeito é cancelado se o usuário não puder regenerar o [SP](SP.md "SP") naturalmente, por exemplo, acima do [peso](Peso.md "Peso") permitido.
- Diferente de [Aumentar Recuperação de SP](Aumentar Recuperação de SP.md "Aumentar Recuperação de SP"), a regeneração de [SP](SP.md "SP") não é aumentada proporcionalmente ao [SP](SP.md "SP") máximo.
