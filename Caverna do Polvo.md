# Caverna do Polvo

| SC ico Caverna do Polvo | |
| --- | --- |
| Perturbe o sono do Polvo mais insolente da ilha Malangdo. | |
| Requisitos mínimos | |
| **Nv. de base** | 90 |
| **Tipo** | Instância |
| **Duração** | 2 horas |
| Skilldelayinfo **Retorno** | 3 horas |
| **[Itens](Itens.md "Itens")** | 6442.png 1 [Arpão de Caçar Polvos](http://www.divine-pride.net/database/item/6442/) |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| Premiação | |
| **Chefe** | Monster PolvoGigante [Polvo Gigante](http://www.divine-pride.net/database/monster/2194/) |

1. Entre na fila da [instância](Instância.md "Instância") falando com a Estrela do Mar (mal\_dun01 151,235)

   :   ![Caverna do Polvo1](images/a/a8/Caverna_do_Polvo1.png)

   - Todos os membros do [grupo](Grupo.md "Grupo") deverão ter 1 [Arpão de Caçar Polvos](http://www.divine-pride.net/database/item/6442/) no inventário para participar.
2. Clique na Entrada Estranha para acessar.

   :   ![Caverna do Polvo2](images/d/d4/Caverna_do_Polvo2.png)
3. Sua tarefa é eliminar 1 [Tentáculo de Polvo](http://www.divine-pride.net/database/monster/2193/) no final de cada corredor. Como existem 4 portais, serão ao todo 4 Tentáculos.

   :   ![Caverna do Polvo3](images/1/1e/Caverna_do_Polvo3.png)
4. Entre no portal ao norte para enfrentar o [Polvo Gigante](http://www.divine-pride.net/database/monster/2194/).
5. Saia da [instância](Instância.md "Instância") pelo portal ao norte.

:   *Diferente de outras [instâncias](Instâncias.md "Instâncias"), o usuário poderá entrar novamente se sair do mapa, desde que ainda tenha o item necessário.*
