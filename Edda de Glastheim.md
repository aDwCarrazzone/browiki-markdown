# Edda de Glastheim

| SC ico Edda de Glastheim | |
| --- | --- |
| A lenda de que o último rei de Glastheim sucumbiu à loucura por gostar de magia das trevas pode ser falsa... acompanhe Oscar, o Dragão-do-Tempo, nessa viagem ao passado e descubra a verdade por trás dos fatos. | |
| Requisitos mínimos | |
| **Nv. de base** | 130 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa apenas |
| Premiação | |
| **Chefe** | Nenhum |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 200.000 |
| **Recompensa** | Libera o acesso à [Ruína de Glastheim](Ruína de Glastheim.md "Ruína de Glastheim") e [Queda de Glastheim](Queda de Glastheim.md "Queda de Glastheim") |

1. Aproxime-se para Oscar aparecer (glast\_01 241,290) e reserve a [instância](Instância.md "Instância").

   :   ![EddaGH01](images/a/ab/EddaGH01.png)
2. Basta seguir o caminho falando com os [NPCs](NPC.md "NPC").

   :   ![EddaGH02](images/d/d7/EddaGH02.png)
3. Mantenha sempre o [assistente](Assistente.md "Assistente") Rei Schmidt ativo para as [Khalitzburgs](http://www.divine-pride.net/database/monster/20388/) não atacarem você. Você deve se aproximar das *Sobreviventes* para ativar o diálogo.
4. No corredor final, selecione "*Vou seguir você.*" para ver o restante da história e ganhar [EXP](EXP.md "EXP").

   :   ![EddaGH03](images/6/6d/EddaGH03.png)
5. De volta à [Glastheim](Glastheim.md "Glastheim"), converse com Oscar para finalizar.
6. Essa instância só é feita uma única vez, e dá acesso à:
   - [Ruína de Glastheim](Ruína de Glastheim.md "Ruína de Glastheim") (Modo Normal)
   - [Queda de Glastheim](Queda de Glastheim.md "Queda de Glastheim") (Modo Difícil)
   - Máquinas **OSC**.
7. As máquinas aparecem ao se aproximar de Oscar (glast\_01 241,290)
