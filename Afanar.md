# Afanar

|  |  |
| --- | --- |
| **Afanar Afanar** | |

| **Steal Coin / Robar Moneda** | |
| **ID** | RG\_STEALCOIN (211) |
| **Tipo** | Ativa |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 15 |
| **Pós-conjuração** | 0,5 segundos |
| **Alvo** | [Monstros](Monstros.md "Monstros") |
| **Pré-requisitos** | |
| - [Mãos Leves](Mãos Leves.md "Mãos Leves") [Mãos Leves](Mãos Leves.md "Mãos Leves") Nv. 4 | |

Tenta extorquir um [monstro](Monstro.md "Monstro") normal uma única vez.

Em caso de sucesso, você receberá uma pequena quantia em [zenys](Zenys.md "Zenys").

| Nv. | Chance | Fator |
| --- | --- | --- |
| 1 | 1% | 8,1 a 10,1 |
| 2 | 2% | 8,2 a 10,2 |
| 3 | 3% | 8,3 a 10,3 |
| 4 | 4% | 8,4 a 10,4 |
| 5 | 5% | 8,5 a 10,5 |
| 6 | 6% | 8,6 a 10,6 |
| 7 | 7% | 8,7 a 10,7 |
| 8 | 8% | 8,8 a 10,8 |
| 9 | 9% | 8,9 a 10,9 |
| 10 | 10% | 9,0 a 11,0 |

### Fórmula

`Chance de sucesso = {[Nv. da hab. + (Nv. do alvo - Nv. do usuário) ×3] + [(DES ×2) + (SOR ×2)] ÷ 1.000)%`

`Zeny adquirido = (Nv. do alvo × Fator) zenys`

## Notas

- Não funciona em [monstros](Monstros.md "Monstros") do tipo Chefe.
- Usar a habilidade em [monstros](Monstros.md "Monstros") passivos, os tornarão agressivos.
