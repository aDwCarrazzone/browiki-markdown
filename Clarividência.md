# Clarividência

|  |  |
| --- | --- |
| **Clarividência Clarividência** | |

| **Clairvoyance** | |

| **Tipo** | Passiva |
| **Níveis** | 1 |
| **Pré-requisitos** | |
| Nenhum | |

Permite detectar oponentes escondidos.

## Notas

- Não funciona em [castelos TE](Guerra do Emperium.md "Guerra do Emperium").
- Mesmo detectados, o oponente continua imune a [habilidades](Habilidades.md "Habilidades") de alvo até serem revelados.
- Detecta as seguintes habilidades:
  - [Esconderijo](Esconderijo.md "Esconderijo")
  - [Furtividade](Furtividade.md "Furtividade")
  - [Corte da Névoa](Corte da Névoa.md "Corte da Névoa")
  - [Espreitar](Espreitar.md "Espreitar")
  - [Refúgio das Sombras](Refúgio das Sombras.md "Refúgio das Sombras")
  - [Ocultação](Ocultação.md "Ocultação")
- **Não** detecta:
  - [Camuflagem](Camuflagem.md "Camuflagem")
  - [Cópia Explosiva](Cópia Explosiva.md "Cópia Explosiva")
  - [Campo de Invisibilidade](Campo de Invisibilidade.md "Campo de Invisibilidade")
- Consumíveis relacionados:
  - [Caixa da Luz do Sol](http://www.divine-pride.net/database/item/12033/)
- Cartas relacionadas:
  - [Carta Maya Macho](http://www.divine-pride.net/database/item/4198/)
