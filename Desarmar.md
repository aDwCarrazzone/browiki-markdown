# Desarmar

|  |  |
| --- | --- |
| **Desarmar Desarmar** | |
| Desarmar | |
| **Disarm / Desarme** | |
| **ID** | GS\_DISARM (513) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade ×5) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,4 + 1,6 seg. |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | 9 células |
| **Munição** | [Over Bullet](Munições.md "Munições") 1 Projétil |
| **Arma** | Pistola ou Rifle |
| **Pré-requisitos** | |
| [Rastrear o Alvo](Rastrear o Alvo.md "Rastrear o Alvo") [Rastrear o Alvo](Rastrear o Alvo.md "Rastrear o Alvo") Nv. 7 | |

Atira nas mãos do alvo.

Causa 100% de dano físico a distância.

Tem chance de remover a arma do oponente por 30 segundos.

Se usada em [monstros](Monstros.md "Monstros"), o efeito será de reduzir o [ATQ](ATQ.md "ATQ") deles em 25%.

| Nv. | Chance | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 3% | 15 |
| 2 | 6% | 20 |
| 3 | 9% | 25 |
| 4 | 12% | 30 |
| 5 | 15% | 35 |

## Notas

- Funciona contra monstros do tipo Chefe.
- A chance de remover a arma é baseada na diferença de [DES](DES.md "DES") entre o alvo e o usuário.
