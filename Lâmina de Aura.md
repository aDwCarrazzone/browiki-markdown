# Lâmina de Aura

|  |  |
| --- | --- |

| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 8) |
| **Duração** | 20 + (Nv. da habilidade × 20) segundos |
| **Alvo** | Usuário |
| **Ícone** | I Instinto de Defesa |
| **Pré-requisitos** | |
| - [Impacto Explosivo](Impacto Explosivo.md "Impacto Explosivo") [Impacto Explosivo](Impacto Explosivo.md "Impacto Explosivo") Nv. 5 - [Perícia com Espada de Duas Mãos](Perícia com Espada de Duas Mãos.md "Perícia com Espada de Duas Mãos") [Perícia com Espada de Duas Mãos](Perícia com Espada de Duas Mãos.md "Perícia com Espada de Duas Mãos") Nv. 5 | |

Envolve a arma com uma aura.

Acrescenta um ataque fixo no dano final que o usuário causa.

| Nv. | Dano final | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | + (Nv. de base × 4) | 40 segundos | 18 |
| 2 | + (Nv. de base × 5) | 60 segundos | 26 |
| 3 | + (Nv. de base × 6) | 80 segundos | 34 |
| 4 | + (Nv. de base × 7) | 100 segundos | 42 |
| 5 | + (Nv. de base × 8) | 120 segundos | 50 |

## Notas

- Exemplo:

1. Um personagem nível 100 causa 1.600 de dano em um monstro.
2. Usar a habilidade em nível 5 garante +800 de bônus.
3. O dano final causado será de 2.400 de dano nesse monstro.

- É cancelada ao trocar de arma.
- Não se aplica a alvos que tomam 1 de dano.
- O bônus não possui [propriedade](Propriedade.md "Propriedade"), independente do elemento da arma.
