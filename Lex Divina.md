# Lex Divina

|  |  |
| --- | --- |
| **Lex Divina Lex Divina** | |

| **Tipo** | Ativa |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 20 a 10 |
| **Pós-conjuração** | 3 segundos |
| **Duração** | 30 a 60 segundos |
| **Alvo** | Oponente |
| **Efeito negativo** | [Silêncio](Silêncio.md "Silêncio") |
| **Pré-requisitos** | |

Evoca a lei divina, concedendo a chance de infligir [silêncio](Silêncio.md "Silêncio") no alvo.

| Nv. | Duração | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 30 segundos | 20 |
| 2 | 35 segundos |
| 3 | 40 segundos |
| 4 | 45 segundos |
| 5 | 50 segundos |
| 6 | 60 segundos | 18 |
| 7 | 16 |
| 8 | 14 |
| 9 | 12 |
| 10 | 10 |

### Fórmula

`Chance = {100 - [VIT do alvo + (SOR do alvo ÷ 8)] + (INT do usuário ÷ 15)}%`

## Notas

- Pode remover o [silêncio](Silêncio.md "Silêncio") em personagens do [grupo](Grupo.md "Grupo") ao conjurar a habilidade apertando a tecla SHIFT.
