# Explosão Protetora

|  |  |
| --- | --- |
| **Explosão Protetora Explosão Protetora** | |
| Explosão Protetora | |
| **Sight Blaster / Destello de Visión** | |
| **ID** | WZ\_SIGHTBLASTER (1006) |
| **Tipo** | Ofensiva |
| **Níveis** | 1 |
| **[SP](SP.md "SP")** | 80 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,32 + 1,28 seg. |
| **Duração** | 900 segundos |
| **Alvo** | Usuário |
| **Empurra** | 3 células |
| **Área** | 3x3 células |
| **Propriedade** | [Fogo](Monstros.md#Propriedade "Monstros") |
| **Ícone** | I Explosão Protetora |
| **Pré-requisitos** | |
| *[Completar Missão](Novas habilidades- Magos e evoluções.md#Bruxos "Novas habilidades: Magos e evoluções")* | |

Conjura uma bola de fogo para flutuar ao seu redor por 15 minutos.

Oponentes que se aproximarem 3x3 células da bola receberão 600% de dano mágico e serão empurrados para trás.

## Notas

- É cancelada quando a duração acabar ou se um alvo se aproximar da bola.
- Não empurra se errar o alvo.
