# Defesa de Duas Mãos

|  |  |
| --- | --- |
| **Defesa de Duas Mãos Defesa de Duas Mãos** | |
| SemImagem | |
| **Two-handed Defense / Defensa de Dos Manos** | |
| **ID** | DK\_TWOHANDDEF (5207) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Arma** | Espada de Duas Mãos, Lança de Duas Mãos ou Machado de Duas Mãos |
| **Pré-requisitos** | |

Melhora sua defesa com o manejo duplo.

Aumenta sua resistência à oponentes de todos os [tamanhos](Tamanhos.md "Tamanhos") ao usar armas dos seguintes tipos:

- Espada de Duas Mãos
- Lança de Duas Mãos
- Machado de Duas Mãos
