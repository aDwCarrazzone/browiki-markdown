# O Passado deu Errado

| NQ ico O Passado deu Errado | |
| --- | --- |
| Um pai saudoso ainda almeja por um encontro com sua filha perdida. | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 7311.png 4 [Bilhetes para Aeroplano](http://www.divine-pride.net/database/item/7311/) |
| **Recompensa** | Habilita o conserto de [armas](Armas.md "Armas") e [armaduras](Armaduras.md "Armaduras") quebradas em [Einbech](Einbech.md "Einbech") |

1. Encontre-se com Kain Himere (airplane 50,61)

   :   ![O passado deu errado00](images/0/0e/O_passado_deu_errado00.png)

   - Selecione:
     - *O que há de errado?*
     - *Juno*
     - *Einbroch*
2. Pergunte sobre Defra ao Funcionário do Hotel (ein\_in01 206,224)

   :   ![O passado deu errado01](images/4/45/O_passado_deu_errado01.png)
3. Entre no quarto (ein\_in01 264,246)

   :   ![O passado deu errado010](images/3/31/O_passado_deu_errado010.png)
4. Pergunte sobre a canção para Ellen Cherno (ein\_in01 261,241)

   :   ![O passado deu errado02](images/2/29/O_passado_deu_errado02.png)
5. Localize Tarsha Cherno (ein\_in01 125,99)

   :   ![O passado deu errado03](images/0/03/O_passado_deu_errado03.png)

   - Selecione:
     - *Sua filha pediu que eu o visitasse.*
6. Clique no Espelho (ein\_in01 121,99)

   :   ![O passado deu errado04](images/f/ff/O_passado_deu_errado04.png)
7. Fale com Tarsha novamente e pergunte sobre a cicatriz.
8. Volte a Kain no [aeroplano](Aeroplano.md "Aeroplano") e mencione a canção.
9. Leia o diário enterrado na árvore (é preciso tentar encontrar o diário clicando na árvore várias vezes). (einbech 47,107)

   :   ![O passado deu errado05](images/5/5e/O_passado_deu_errado05.png)
10. Retorne a Tarsha e verifique suas mãos.
11. Volte até Elle Cherno para receber uma [Mensagem Rabiscada](http://www.divine-pride.net/database/item/7276/).
12. Entregue o item para Kain e receba 4 [Bilhetes para Aeroplano](http://www.divine-pride.net/database/item/7311/) como recompensa.
13. Agora você pode reparar um equipamento quebrado clicando na Máquina Desconhecida (ein\_in01 117,80)

    :   ![O passado deu errado06](images/9/9b/O_passado_deu_errado06.png)
