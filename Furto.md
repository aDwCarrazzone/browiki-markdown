# Furto

|  |  |
| --- | --- |
| **Furto Furto** | |
| Furto | |
| **Steal / Robar** | |
| **ID** | TF\_STEAL (50) |
| **Tipo** | Ativa |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 10 |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | [Monstros](Monstros.md "Monstros") |
| **Alcance** | 1 célula |
| **Pré-requisitos** | |
| - Nenhum | |

Tentar roubar [itens](Itens.md "Itens") dos [monstros](Monstros.md "Monstros").

| Nv. | Chance |
| --- | --- |
| 1 | 8% |
| 2 | 14% |
| 3 | 20% |
| 4 | 26% |
| 5 | 32% |
| 6 | 38% |
| 7 | 44% |
| 8 | 50% |
| 9 | 56% |
| 10 | 62% |

### Fórmula

`Chance do furto = {Chance de Drop do item × [DES do personagem + (chance da hab.) − (DES do monstro + 10)]}%`

## Notas

- Não furta [cartas](Cartas.md "Cartas").
- Chances de sucesso exibidas na tabela são baseadas em que o alvo e o usuário possuem o mesmo valor de [DES](DES.md "DES").
- Não pode ser usada duas vezes no mesmo [monstro](Monstro.md "Monstro") e em [monstros](Monstros.md "Monstros") do tipo chefe.
- É possível furtar apenas a lista de itens comuns dos monstros. Se um [Poring](http://www.divine-pride.net/database/monster/1002/) pegar uma [Poção Vermelha](http://www.divine-pride.net/database/item/501/), por exemplo, você não poderá furtá-la.
- Furtar [monstros](Monstros.md "Monstros") atacados por outros personagens não altera as chances de queda dos [itens](Itens.md "Itens").
- Existe um efeito chamado "zero furto", quando as chances do seu personagem de furtar são baixas, todos os itens terão a mesma chance de serem furtados e isso independe da taxa de drop deles.
- A habilidade não é afetada pela a penalidade de drop por nível.
- Não é possível aumentar as chances de furto.
- Equipamentos que recebem [Bônus Aleatórios](Bônus Aleatórios.md "Bônus Aleatórios") de [monstros](Monstros.md "Monstros") não receberão o efeito se forem furtados.
