# Lanças de Fogo

|  |  |
| --- | --- |
| **Lanças de Fogo Lanças de Fogo** | |
| Lanças de Fogo | |
| **Fire Bolt / Lanzas de Fuego** | |
| **ID** | MG\_FIREBOLT (19) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 2) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,2 + (Nv. × 0,1) + 0,2 + (Nv. × 0,3) seg. |
| **Pós-conjuração** | 1,4 segundos |
| **Alvo** | Oponente |
| **Alcance** | 9 células |
| **Propriedade** | [Fogo](Monstros.md#Propriedade "Monstros") |
| **Pré-requisitos** | |
| - Nenhum | |

Ataca um oponente com lanças de fogo vindas do céu.

Causa de dano mágico da propriedade [Fogo](Fogo.md "Fogo") equivalente a 100% do [ATQM](ATQM.md "ATQM") por lança.

O número de lanças aumenta de acordo com o nível aprendido.

| Nv. | Lanças | [Conjuração](Conjuração.md "Conjuração") | | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| Fixa | Variável |
| 1 | 1 | 0,3 segundos | 0,5 segundos | 12 |
| 2 | 2 | 0,4 segundos | 0,8 segundos | 14 |
| 3 | 3 | 0,5 segundos | 1,1 segundos | 16 |
| 4 | 4 | 0,6 segundos | 1,4 segundos | 18 |
| 5 | 5 | 0,7 segundos | 1,7 segundos | 20 |
| 6 | 6 | 0,8 segundos | 2,0 segundos | 22 |
| 7 | 7 | 0,9 segundos | 2,3 segundos | 24 |
| 8 | 8 | 1,0 segundos | 2,6 segundos | 26 |
| 9 | 9 | 1,1 segundos | 2,9 segundos | 28 |
| 10 | 10 | 1,2 segundos | 3,2 segundos | 30 |
