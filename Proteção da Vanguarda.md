# Proteção da Vanguarda

|  |  |
| --- | --- |
| **Proteção da Vanguarda Proteção da Vanguarda** | |
| Proteção da Vanguarda | |
| **Vanguard Force / Fuerza de Vanguardia** | |
| **ID** | LG\_FORCEOFVANGUARD (2313) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 30 |
| **[Conjuração](Conjuração.md "Conjuração")** | 1 + 0 seg. |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Usuário |
| **Ícone** | I Proteção da Vanguarda |
| **Pré-requisitos** | |
| - Nenhum | |

Ativa o estado de Protetor Vanguarda.

Aumenta a [DEF](DEF.md "DEF") dos [equipamentos](Equipamentos.md "Equipamentos") e o [HP](HP.md "HP") máximo.

Drena [SP](SP.md "SP") a cada 10 segundos.

Há chance de ganhar 1 Ponto de Fúria sempre que receber dano.

Aumentar o nível da habilidade aumenta o limite de Pontos de Fúria adquiridos.

| Nv. | [HP](HP.md "HP") máx. | [DEF](DEF.md "DEF") | Pontos de Fúria | | Dreno de [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| Chance | Limite |
| 1 | +3% | +2% | 20% | 7 | 20 |
| 2 | +6% | +4% | 32% | 9 | 16 |
| 3 | +9% | +6% | 44% | 11 | 12 |
| 4 | +12% | +8% | 56% | 13 | 8 |
| 5 | +15% | +10% | 68% | 15 | 4 |

## Notas

- Pontos de Fúria são usados na habilidade [Retribuição da Vanguarda](Retribuição da Vanguarda.md "Retribuição da Vanguarda") e são semelhantes às [Esferas Espirituais](Zen.md "Zen") dos [Monges](Monges.md "Monges").
- O bônus de [DEF](DEF.md "DEF") não é exibido na janela de [equipamentos](Equipamentos.md "Equipamentos").
