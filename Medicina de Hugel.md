# Medicina de Hugel

| NQ ico Medicina de Hugel | |
| --- | --- |
| A difícil tarefa de um filho é estar ao lado de seu pai ao mesmo tempo em que sai em busca de um remédio, mas será preciso muita paciência para conseguir dar o remédio a ele. | |
| Requisitos mínimos | |
| **Nv. de base** | 50 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 504.png 1 [Poção Branca](http://www.divine-pride.net/database/item/504/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 5.000 |
| **[Itens](Itens.md "Itens")** | 603.png 1 [Velha Caixa Azul](http://www.divine-pride.net/database/item/603/) |

1. Converse com Allen Schuwell (hu\_in01 387,245)

   :   ![Humedci01](images/2/2b/Humedci01.png)
2. Encontrem Postel Schuwell (hu\_fild06 34,123)

   :   ![Humedci02](images/3/3d/Humedci02.png)
3. Reencontre Allen Schuwell e entregue a [Poção Branca](http://www.divine-pride.net/database/item/504/). Continue conversando com o [NPC](NPC.md "NPC").
4. Quando for pedido, aproxime-se da Gaveta de Cima (hu\_in 372,254)

   :   ![Humedci03](images/7/78/Humedci03.png)
5. Ande próximo a Allen Schuwell, ele perguntará sobre o remédio.
6. Retorne a caminhar nas coordenadas da Gaveta de Cima para ler algo sobre [Lighthalzen](Lighthalzen.md "Lighthalzen") ou sobre um livro que Allen guarda.
   - Você deve repetir esse procedimento (ir até a Gaveta e voltar ao Allen) até conseguir o remédio.
   - Tente ser ágil, se você demorar muito, verá um efeito parecido com a habilidade [Esconderijo](Esconderijo.md "Esconderijo") e isso significa que você não pegará o remédio.
   - Quando se falhar, suba na cama até que Allen fale com você e tente chegar na Gaveta novamente.
   - Este passo é aleatório, você deve fazer isso até Postell aparecer no quarto.
7. Em caso de sucesso, Postell Schuwell aparecerá no quarto.
8. Fale com Allen para receber sua premiação em [experiência](Experiência.md "Experiência").
9. Clique na Gaveta de Cima para receber a [Velha Caixa Azul](http://www.divine-pride.net/database/item/603/).
