# Ataque Duplo

|  |  |
| --- | --- |
| **Ataque Duplo Ataque Duplo** | |
| Ataque Duplo | |
| **Double Attack / Doble Ataque** | |
| **ID** | TF\_DOUBLE (48) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Arma** | Adaga |
| **Pré-requisitos** | |
| - Nenhum | |

Ao realizar ataques físicos com uma arma do tipo Adaga, tem uma chance de atingir o alvo mais uma vez.

Haverá uma correção da [Precisão](Precisão.md "Precisão") na hora do ataque.

Aumentar o nível da habilidade também aumenta o dano físico do Katar na mão esquerda.

| Nv. | Chance | [Precisão](Precisão.md "Precisão") | [ATQ](ATQ.md "ATQ") com Katar |
| --- | --- | --- | --- |
| 1 | 7% | +1% | +3% |
| 2 | 14% | +2% | +5% |
| 3 | 21% | +3% | +7% |
| 4 | 28% | +4% | +9% |
| 5 | 35% | +5% | +11% |
| 6 | 42% | +6% | +13% |
| 7 | 49% | +7% | +15% |
| 8 | 56% | +8% | +17% |
| 9 | 63% | +9% | +19% |
| 10 | 70% | +10% | +21% |

## Notas

- Se obtida através de itens, a habilidade funciona em qualquer arma, mas prevalecerá o maior nível aprendido.
- Itens que habilitam Ataque Duplo não acumulam, 2 [Cartas Sorrateiros](http://www.divine-pride.net/database/item/4117/) não darão a habilidade em nível 2.
- Ataques críticos podem ser duplicados, mas serão calculados separadamente e exibidos em um só dano.
