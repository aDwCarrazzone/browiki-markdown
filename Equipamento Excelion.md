# Equipamento Excelion

| DQ ico Equipamento Excelion | |
| --- | --- |
| Parece que uma antiga fábrica de robôs descartou 2 protótipos em Verus! Aparentemente eles ainda funcionam, mas são tão velhos que parecem uma pilha de sucatas! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Repetível |
| **[Itens](Itens.md "Itens")** | 6962.png 1 [Tanque Velho](http://www.divine-pride.net/database/item/6962/) |
| Premiação | |
| **Recompensa** | Libera os [Equipamentos](Equipamentos.md "Equipamentos") Excelion |

1. Com 1 [Tanque Velho](http://www.divine-pride.net/database/item/6962/) no inventário, fale com MARS\_01 (verus04 164,219)

   :   ![MARS](images/e/eb/MARS.png)
2. Agora você pode falar com PLUTO\_09 (verus04 165,217)

   :   ![PLUTO](images/a/a7/PLUTO.png)

   - Entregue 5 [Tanques Velhos](http://www.divine-pride.net/database/item/6962/) e 10 [Blocos de Sucata](http://www.divine-pride.net/database/item/6961/) para conseguir o equipamento Excelion de sua escolha:
     - ![15128.png](http://static.divine-pride.net/images/items/item/15128.png) [Colete Excelion](http://www.divine-pride.net/database/item/15128/)
     - ![20773.png](http://static.divine-pride.net/images/items/item/20773.png) [Motor Excelion](http://www.divine-pride.net/database/item/20773/)
   - Disponíveis por [JoyCoins](JoyCoins.md "JoyCoins"):
     - ![470373.png](http://static.divine-pride.net/images/items/item/470373.png) [Perna Excelion](http://www.divine-pride.net/database/item/470373/)
     - ![28941.png](http://static.divine-pride.net/images/items/item/28941.png) [Escudo Excelion](http://www.divine-pride.net/database/item/28941/)
   - Para conseguir os Diagramas necessários para o [encantamento](Encantamento.md "Encantamento"), traga 5 [Tanques Velhos](http://www.divine-pride.net/database/item/6962/).
   - Existe 30% de chance de sucesso de conseguir um Diagrama.
   - Em caso de falha, você receberá [Minério de Ferro](http://www.divine-pride.net/database/item/1002/), [Brigan](http://www.divine-pride.net/database/item/7054/), [Ferro](http://www.divine-pride.net/database/item/998/) ou [Aço](http://www.divine-pride.net/database/item/999/).
   - Cada diagrama tem o nome relacionado ao [encantamento](Encantamento.md "Encantamento") que você pode colocar no seu equipamento.
3. Para encantar seu equipamento, **equipe-o** e entregue os Diagramas para MARS\_01.
   - Não há chances de falha.
   - Não há possibilidade de reiniciar os [encantamentos](Encantamentos.md "Encantamentos").
   - O limite é de 3 encantos para os quatro equipamentos, mas existem restrições, descritas abaixo.

## Equipamentos Excelion

Sobre os encantos do [Colete Excelion](http://www.divine-pride.net/database/item/15128/):

- Apenas os diagramas da categoria **E-** possuem restrição de [slot](Slot.md "Slot").
- Esses diagramas devem ocupar sempre o 1º slot, ou seja, devem ser os primeiros a ser encantados na armadura.
- Categorias:
  - **I-** Imunidade
  - **E-** Elemental
  - **A-** Atributo
  - **R-** Resistência
  - **C-** Cura
