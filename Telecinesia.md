# Telecinesia

|  |  |
| --- | --- |
| **Telecinesia Telecinesia** | |
| Telecinesia | |
| **Intensification / Intense Telekinesis** | |
| **ID** | WL\_TELEKINESIS\_INTENSE (5012) |
| **Tipo** | Suporte |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 + (Nv. da habilidade × 50) |
| **Pós-conjuração** | 0,5 segundos |
| **Recarga** | 300 segundos |
| **Duração** | 60 segundos |
| **Alvo** | Usuário |
| **Ícone** | I Telecinesia |
| **Pré-requisitos** | |
| - [Impacto Espiritual](Impacto Espiritual.md "Impacto Espiritual") [Impacto Espiritual](Impacto Espiritual.md "Impacto Espiritual") Nv. 5 | |

Intensifica os poderes telecinéticos.

Aumenta o dano de magias [fantasmas](Fantasma.md "Fantasma") e reduz o custo de [SP](SP.md "SP") delas.

Também diminui a [conjuração variável](Conjuração variável.md "Conjuração variável") de todas as habilidades.

| Nv. | Magias Fantasmas | | [Conjuração variável](Conjuração variável.md "Conjuração variável") | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| Dano mágico | Custo de SP |
| 1 | ×1,4 | -10% | -10% | 100 |
| 2 | ×1,8 | -20% | -20% | 150 |
| 3 | ×2,2 | -30% | -30% | 200 |
| 4 | ×2,6 | -40% | -40% | 250 |
| 5 | ×3,0 | -50% | -50% | 300 |
