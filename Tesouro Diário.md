# Tesouro Diário

| DQ ico Tesouro Diário | |
| --- | --- |
| Um misterioso vidente tem algo a lhe dar... por uma pequena bagatela, é claro!! | |
| Requisitos mínimos | |

| **Tipo** | Repetível |
| **[Zeny](Zeny.md "Zeny")** | Aleatório |
| Skilldelayinfo **Espera:** | 4 da manhã. |
| Premiação | |
| **[Itens](Itens.md "Itens")** | Aleatório |

1. Aproxime-se do Vidente (geffen 110,104)

   :   ![TesouroDiarioNPC](images/0/08/TesouroDiarioNPC.png)
2. Esse [NPC](NPC.md "NPC") se comunica apenas por aproximação. Ele pedirá zeny duas vezes para você.
   - A quantidade de zeny pedida varia de acordo com a seguinte fórmula:
     - `x < [(Aleatório entre 10.000 e 12.500) × 100]`
   - Se seu zeny atual for menor que **x**, você não poderá doar.
3. Na segunda vez que pedir zeny, ele recompensará você com um *Prêmio Aleatório*.
4. Você pode repetir essa interação 3 vezes por dia.
