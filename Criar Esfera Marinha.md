# Criar Esfera Marinha

| **Tipo** | Ativa |
| **Níveis** | 5 |

| **[Conjuração](Conjuração.md "Conjuração")** | 0,4 + 1,6 seg. |
| **Pós-conjuração** | 0,5 segundos |
| **Duração** | 30 segundos |
| **Alvo** | Chão |
| **Alcance** | 1 célula |

| - [Preparar Poção](Preparar Poção.md "Preparar Poção") [Preparar Poção](Preparar Poção.md "Preparar Poção") Nv. 2 | |

Cria até 3 [Esferas Marinhas](http://www.divine-pride.net/database/monster/1142/) que vão usar [Explodir](Explodir.md "Explodir") após serem atacadas.

Qualquer alvo na área de 11x11 células da explosão receberá dano mágico de propriedade [fogo](Fogo.md "Fogo"), incluindo seu [homunculus](Homunculus.md "Homunculus") e [assistentes](Assistentes.md "Assistentes").

Aumentar o nível da habilidade melhora o [HP](HP.md "HP") da [Esfera Marinha](http://www.divine-pride.net/database/monster/1142/), maximizando o dano da explosão.

| Nv. | [HP](HP.md "HP") |
| --- | --- |
| 1 | 2.400 |
| 2 | 2.800 |
| 3 | 3.200 |
| 4 | 3.600 |
| 5 | 4.000 |

## Notas

- Em mapas de [PvP](PvP.md "PvP") a explosão afeta todos os personagens, incluindo aliados e o próprio usuário.
- [Esferas Marinhas](http://www.divine-pride.net/database/monster/1142/) invocadas andam pelo mapa após atingidas, num máximo de até 7 células antes de explodir.
- [Monstros](Monstros.md "Monstros") atingidos pela explosão atacarão o usuário.
