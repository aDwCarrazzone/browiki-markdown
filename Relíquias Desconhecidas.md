# Relíquias Desconhecidas

| DQ ico Relíquias Desconhecidas | |
| --- | --- |
| Existem peças estranhas espalhadas por Midgard que somente o mais curioso pesquisador poderá descobrir! | |
| Requisitos mínimos | |
| **Nv. de base** | 80 |
| **Tipo** | Repetível |
| **Equipado** | 2782.png [Anel do Rei Sábio](http://www.divine-pride.net/database/item/2782/) |
| **Quests anteriores** | [Visita do Manuk](Visita do Manuk.md "Visita do Manuk") |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 6304.png 1 [Certificado do Explorador](http://www.divine-pride.net/database/item/6304/) |

1. Em [Diel](El Dicastes.md "El Dicastes"), clique no Elevador e selecione "*Sala de Arquivo*".

   :   ![Relíquias Desconhecidas03](images/f/f5/Relíquias_Desconhecidas03.png)
2. Aceite ajudar Papyrus (dic\_in01 40,193)

   :   ![Relíquias Desconhecidas01](images/f/fe/Relíquias_Desconhecidas01.png)
3. Pegue as missões com os Arquivos de Documentos, na mesma sala.

   :   ![Relíquias Desconhecidas02](images/4/40/Relíquias_Desconhecidas02.png)

   - Você pode pegar quantas missões puder.
   - Há uma chance de impedir que você pegue novas missões por 6 horas.
   - As missões consistem em seguir até uma localização determinada pela missão e escavar.
   - Ao escavar, há uma chance de receber itens e uma [Relíquia Desconhecida](http://www.divine-pride.net/database/item/6304/).
4. Traga 3 [Relíquias Desconhecidas](http://www.divine-pride.net/database/item/6304/) a Papyrus para receber 1 [Certificado do Explorador](http://www.divine-pride.net/database/item/6304/).
   - Se trouxer 30 [Relíquias Desconhecidas](http://www.divine-pride.net/database/item/6304/), você recebe 11 [Certificados do Explorador](http://www.divine-pride.net/database/item/6304/).

Use *CTRL+F* para localizar
