# Mudança de Classe: Shuras

| JQ ico Mudança de Classe: Shuras | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 99 |
| **Nv. de classe** | 70 |
| **Tipo** | Classe |
| **Classe** | [Mestre](Mestre.md "Mestre") |
| **Caça** | 100 [Lobos do Deserto](http://www.divine-pride.net/database/monster/1106/) |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 2795.png 1 [Anel de Maçã Verde](http://www.divine-pride.net/database/item/2795/)  5754.png 1 [Disciplina de Espírito](http://www.divine-pride.net/database/item/5754/) |
| **Recompensa** | Mudança de classe para [Shura](Shura.md "Shura") |

1. No bar de [Veins](Veins.md "Veins"), fale com o Verme Asqueroso (ve\_in 241,128)

   :   ![MdCSu01](images/7/74/MdCSu01.png)
2. Saia do bar e dirija-se até a Lutadora, descendo as escadas (veins 127,238)

   :   ![MdCSu02](images/b/b7/MdCSu02.png)

   - Responda:
     - Mas eu ainda quero saber mais.
3. Elimine 100 [Lobos do Deserto](http://www.divine-pride.net/database/monster/1106/) e volte a falar com a Lutadora.
4. Inicie o teste prático com o Colega (yuno\_fild07 254,176)

   :   ![MdCSu03](images/3/3a/MdCSu03.png)

   - Seu personagem será teleportado para uma sala de espera, clique no chat e aguarde sua vez.
5. O teste prático consiste em ficar 5 minutos numa sala repleta de [monstros](Monstros.md "Monstros").
   - Os monstros são: [Injustiçados](http://www.divine-pride.net/database/monster/1257/), [Rybios](http://www.divine-pride.net/database/monster/1201/), [Esqueletos Prisioneiros](http://www.divine-pride.net/database/monster/1196/), [Zumbis Prisioneiros](http://www.divine-pride.net/database/monster/1197/) e [Ridewords](http://www.divine-pride.net/database/monster/1195/)
   - Você não precisa matar os [monstros](Monstros.md "Monstros"), basta esperar que o tempo acabe. Leve consigo [Amuleto de Ziegfried](http://www.divine-pride.net/database/item/7621/). Caso você morra, só clique em ressuscitar após o término dos 5 minutos.
   - Uma dica é se esconder atrás de um dos pilares da sala e matar somente os bichos que se aproximarem de você.
   - Passado no teste, fale com o Colega no centro da sala para teleportar.
6. Converse com o Mestre para mudar sua classe e receber 1 [Disciplina de Espírito](http://www.divine-pride.net/database/item/5754/) e 1 [Anel de Maçã Verde](http://www.divine-pride.net/database/item/2795/).

   :   ![MdCSu07](images/5/5a/MdCSu07.png)

- Ao chegar no nível 100, pegue sua *[Caixa Inicial](Conquista.md "Conquista")* para ajudar na sua nova jornada.
