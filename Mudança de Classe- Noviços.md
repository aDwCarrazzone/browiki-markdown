# Mudança de Classe: Noviços

| JQ ico Mudança de Classe: Noviços | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Nv. de classe** | 10 |
| **Tipo** | Classe |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 1545.png 1 [Maça de Iniciante](http://www.divine-pride.net/database/item/1545/) |
| **Recompensa** | Muda a classe para [Noviço](Noviço.md "Noviço") |

1. No 2º andar da [Academia Criatura](Academia Criatura.md "Academia Criatura"), fale com Alice (iz\_ac02 156,169)

   :   ![Aca Alcolyte01](images/3/3b/Aca_Alcolyte01.png)

   - Ao lado, a Religiosa lhe dará um [Manual do Noviço](http://www.divine-pride.net/database/item/2821/).
   - Esse passo é opcional, mas ajuda a chegar mais rápido na Guilda dos Noviços.
   - Você pode optar por mudar de classe diretamente na Academia, mas é recomendável mudar de classe na Guilda.
2. Na Catedral de [Prontera](Prontera.md "Prontera"), fale com o Padre Necrus (prt\_church 184,41)

   :   ![Aca Alcolyte02](images/c/c9/Aca_Alcolyte02.png)

   - Responda "*Sim*" para mudar de classe.
   - Mudando de Classe na Guilda, você recebe:
     - ![1545.png](http://static.divine-pride.net/images/items/item/1545.png) 1 [Maça de Iniciante](http://www.divine-pride.net/database/item/1545/)
     - Aprende [![Luz Divina](images/9/97/Luz_Divina.png)](Luz Divina.md "Luz Divina") [Luz Divina](Luz Divina.md "Luz Divina")
3. Após a mudança, volte para a [Academia Criatura](Academia Criatura.md "Academia Criatura") e fale com Sprakki (iz\_ac01 102,44)

   :   ![TutorialNewbie010a](images/f/fd/TutorialNewbie010a.png)

   - Lenore, do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden") aparecerá e lhe dará:
     - 1.000 de [EXP](EXP.md "EXP") de base e 200 de classe.
     - ![5583.png](http://static.divine-pride.net/images/items/item/5583.png) [Chapéu do Éden I](http://www.divine-pride.net/database/item/5583/)
     - ![2560.png](http://static.divine-pride.net/images/items/item/2560.png) [Capa do Éden I](http://www.divine-pride.net/database/item/2560/)
     - ![15009.png](http://static.divine-pride.net/images/items/item/15009.png) [Uniforme do Éden I](http://www.divine-pride.net/database/item/15009/)
     - ![2456.png](http://static.divine-pride.net/images/items/item/2456.png) [Botas do Éden I](http://www.divine-pride.net/database/item/2456/)
4. Fale com Sprakki novamente se deseja receber a missão [Explorando o Mundo](Explorando o Mundo.md "Explorando o Mundo").

## Treinamento

1. Fale com Alice (prt\_church 186,36)

   :   ![Tu Alcolyte01](images/d/df/Tu_Alcolyte01.png)

   - Selecione a opção "*Missões*".
   - As missões não podem ser repetidas.
   - Acima do nível 21, você não poderá mais pegar as [quests](Quests.md "Quests").
   - Seu personagem será enviado para o [modo pesadelo](Mapas.md#Modo_Pesadelo "Mapas") do [Navio Fantasma](http://www.divine-pride.net/database/map/treasure_n1/).
   - Para sair do navio, basta falar com a Ovelhinha Lapore (treasure\_n1 167,44)
