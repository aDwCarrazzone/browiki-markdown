# Corte da Névoa

| **Pré-requisitos** | |
| [Salto das Sombras](Salto das Sombras.md "Salto das Sombras") [Salto das Sombras](Salto das Sombras.md "Salto das Sombras") Nv. 1 | |

Ataca um alvo rapidamente.

Causa dano físico corpo a corpo no alvo.

Em seguida, o usuário entra em estado de [Esconderijo](Esconderijo.md "Esconderijo").

Enquanto estiver escondido, drena 1 de [SP](SP.md "SP") a cada 5 segundos.

Durante o estado, é possível usar [Salto das Sombras](Salto das Sombras.md "Salto das Sombras") ou [Corte das Sombras](Corte das Sombras.md "Corte das Sombras").

Quanto maior o nível da habilidade, maior a duração de [Esconderijo](Esconderijo.md "Esconderijo").

## Notas

- Se não atingir o alvo, não entra em [Esconderijo](Esconderijo.md "Esconderijo").
- Personagens escondidos podem ser detectados pela [Clarividência](Clarividência.md "Clarividência").
- O [esconderijo](Esconderijo.md "Esconderijo") é cancelado se o personagem usar [Corte das Sombras](Corte das Sombras.md "Corte das Sombras") ou [Salto das Sombras](Salto das Sombras.md "Salto das Sombras"), for revelado ou se o [SP](SP.md "SP") chegar a zero.
