# Comandos

Configure o jogo da melhor forma para uma jogabilidade mais personalizada.

Para acessar o menu de *Configurações do Jogo* e de *Teclas de Atalho*, basta apertar o botão `ESC`

![ESCbutau](images/0/0a/ESCbutau.png)

## Configurações

Na janela de configurações, você pode ajustar música, efeitos sonoros, efeitos gráficos, idioma e até alterar a *Skin* do jogo.

![Settingwnd](images/0/0a/Settingwnd.png)

## Atalhos

Os atalhos rápidos podem ser alterados de acordo com o seu gosto, não apenas para uso de [habilidades](Habilidades.md "Habilidades"), mas também para acesso a outras janelas.

![Hotkeyteclas](images/3/31/Hotkeyteclas.png)

## Teclas

É possível inserir comandos no campo de mensagens para alterar o sistema do jogo e exibir informações.

Lembre-se de que algumas funções, como exibir as coordenadas de posição de um número, só podem ser operadas por meio de comandos.

| Tecla | Descrição |
| --- | --- |
| ESC | Abre a janela de configurações do jogo. |
| F1 até F9 | Teclas de atalho rápido, podem ser personalizadas pelo usuário. |
| F10 | Aumenta ou diminui o tamanho da janela de conversa. |
| F11 | Esconde ou exibe as janelas da interface. |
| F12 | Mostra, expande ou esconde a barra de atalhos. |

| Tab | Alterna entre as abas na janela de conversa. |
| Insert | Faz o personagem sentar ou levantar. Precisa ter [Instintos Básicos](Instintos Básicos.md "Instintos Básicos") ou nível 3 de [Habilidades Básicas](Habilidades Básicas.md "Habilidades Básicas"). |
| Alt + Clique direito | Transfere [itens](Itens.md "Itens") do inventário para o [armazém](Serviços.md "Serviços") ou [carrinho](Usar Carrinho.md "Usar Carrinho") e vice-versa. |
| Alt + Enter | Envia mensagens para o [clã](Clã.md "Clã") do personagem. |
| Alt + End | Exibe ou esconde a barra de [HP](HP.md "HP")/[SP](SP.md "SP"). |

| Alt + Home | Trava ou destrava o cursor verde do solo. |

| Alt + X | Abre a janela de [visuais](Visuais.md "Visuais"). |
| Alt + A | Abre a janela de [atributos](Atributos.md "Atributos"). |

| Alt + E | Abre o seu Inventário. |
| Alt + R | Abre a janela de informações do seu [Homunculus](Homunculus.md "Homunculus"). Apenas [Alquimistas](Alquimista.md "Alquimista") e suas evoluções podem acessá-la. |

| Alt + Z | Abre a janela de [grupo](Grupo.md "Grupo"). |
| Alt + C | Abre uma janela onde você poderá criar uma sala de chat. |
| Alt + V | Abre a janela de informações do seu personagem. |
| Alt + B | Alterna a exibição da janela de [instância](Instância.md "Instância") sem que ela feche em definitivo. |
| Alt + M | Abre uma janela de atalhos de comando. |
| Alt + ' | Abre o menu da interface de informações. |
| Ctrl + Tab | Alterna a aparência do mini-mapa entre desligado, semitransparente ou sem transparência. |
| Ctrl + Enter | Envia uma mensagem para o [grupo](Grupo.md "Grupo"). |
| Ctrl + End | Mostra/Esconde personagens (transparência). Funciona apenas em cidades. |

| Shift + Clique esquerdo no item | Cria um *link* de descrição do item clicado para o chat de conversa. |
| Shift + Clique direito no item | Move um item para a aba de favoritos do inventário. |
| Shift + Insert | Cola textos da área de transferência. Funciona como um *Ctrl + V* no jogo. |
