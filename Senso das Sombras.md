# Senso das Sombras

|  |  |
| --- | --- |
| **Senso das Sombras Senso das Sombras** | |
| SemImagem | |
| **Shadow Sense / Sentido de la Sombra** | |
| **ID** | SHC\_SHADOW\_SENSE (5288) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| - Nenhum | |

Aprimora o seu sentido nas sombras, melhorando sua [Esquiva](Esquiva.md "Esquiva").

Ao usar armas do tipo Adaga ou Katar, aumenta o [CRIT](CRIT.md "CRIT").

| Nv. | [Esquiva](Esquiva.md "Esquiva") | [CRIT](CRIT.md "CRIT") | |
| --- | --- | --- | --- |
| Adaga | Katar |
| 1 | +10 | +14 | +7 |
| 2 | +20 | +18 | +9 |
| 3 | +30 | +22 | +11 |
| 4 | +40 | +26 | +13 |
| 5 | +50 | +30 | +15 |
| 6 | +60 | +34 | +17 |
| 7 | +70 | +38 | +19 |
| 8 | +80 | +42 | +21 |
| 9 | +90 | +46 | +23 |
| 10 | +100 | +50 | +25 |

## Notas

- A adaga precisa estar equipada na mão direita para o bônus ocorrer.
