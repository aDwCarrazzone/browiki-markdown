# Habilidades

![Skillwnd](../images/f/fc/Skillwnd.png)

Árvore de habilidades

Habilidades são ações aprendidas no seu personagem, acessável pelo comando `ALT + S`.

Sua janela de habilidades divide-se em algumas abas laterais, que vão aumentando conforme seu personagem muda de [classe](../Sem%20categoria/Classe.md "Classe").

As habilidades dividem-se entre passivas e ativas, onde as passivas não precisam de gasto de [SP](../Sem%20categoria/SP.md "SP").

As ativas dividem-se em ofensivas ou de suporte, podendo causar dano nos seus oponentes ou ajudar amigos!

As habilidades passivas não podem ser arrastadas para sua *Barra de Atalhos*, acessada pelo [comando](../Sem%20categoria/Comando.md "Comando") `F12`

### Atalhos

1. Invista pontos na habilidade desejada.
2. Algumas habilidades possuem níveis selecionáveis, verifique as setas azuis abaixo da habilidade.
3. Arraste o ícone de uma habilidade para a barra de atalhos.

   :   ![Skillhotkeywnd](../images/1/17/Skillhotkeywnd.png)
4. Para usar o atalho, não esqueça de configurar uma tecla em:

   :   Botão ESC > Teclas de Atalho
5. Existem duas barras de habilidades que podem ser alternadas.
6. A linha de uma barra pode ser alterada por `Ctrl + F12`

### Reset

- Você pode resetar suas habilidades pelas seguintes maneiras:

1. Hipnotizador (lasagna 222,123) - gratuitamente, quantas vezes quiser.
   - Somente para [invocadores](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Invocadores.md "Invocadores") até o nível base 49.
2. Hipnotizadora (izlude 127,175) - gratuitamente, quantas vezes quiser.
   - Somente para personagens até o nível base 45 (não inclui Transclasses e Expandidas).
3. [Neuralizador](http://www.divine-pride.net/database/item/12213/) - comprado em loja [JoyCoins](../Itens/JoyCoins.md "JoyCoins").
4. [Sino Prateado](http://www.divine-pride.net/database/item/22908/) - comprado em loja [JoyCoins](../Itens/JoyCoins.md "JoyCoins").

## Fases

|  |  |
| --- | --- |
| Skill1 | Início |
| - O nome da habilidade aparece em cima do personagem. - É nessa etapa que os requisitos da habilidade, como [HP](../Sem%20categoria/HP.md "HP"), [SP](../Sem%20categoria/SP.md "SP") ou [itens](../Itens/Itens.md "Itens") tipo Gema Azul, são consumidos. |
| Skill2 | [Conjuração](../Sem%20categoria/Conjura%C3%A7%C3%A3o.md "Conjuração") |
| - Uma espécie de "barra de *download*" aparece. - Se a habilidade não tem conjuração, ela é executada instantaneamente. |
| Skill3 | Execução |
| - O nome da habilidade aparece novamente em cima do personagem. - A habilidade é executada, aparecendo uma animação. |
| Skillcd | [Pós-conjuração](../Sem%20categoria/P%C3%B3s-conjura%C3%A7%C3%A3o.md "Pós-conjuração") |
| - É o término da conjuração, também chamada de "*delay*" ou "*global cooltime*". - Um cronômetro aparece em cima de todas as habilidades na [barra de atalhos](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Comandos.md "Comandos"). - Seu personagem estará impedido de usar todas as habilidades nesse tempo. |
| Skilldly | Recarga |
| - Vem do inglês "*cooldown*", onde a habilidade executada não pode ser conjurada novamente. - Será necessário esperar o tempo de recarga dela terminar. - Entretanto, outras habilidades estão liberadas para o uso. |

## Ícones

![StatusIcons](../images/0/08/StatusIcons.png)

Quando for alvo de alguma habilidade que altera o estado normal do personagem, algumas vezes um ícone é exibido na lateral direita da sua tela.

Esses ícones podem indicar [efeitos negativos](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Efeitos%20negativos.md "Efeitos negativos"), positivos, alterações de [propriedade](../Sem%20categoria/Propriedade.md "Propriedade") e até mesmo se o personagem estiver sentado.

Assim que um ícone surgir, você pode passar o mouse sobre ele para ler sua descrição.

Você pode mostrar/esconder a descrição dos ícones usando `/stateinfo`
