# Sopa Tailandesa

| NQ ico Sopa Tailandesa | |
| --- | --- |
| Se você deseja experimentar o prato mais famoso da cidade, você deverá ajudar uma simpática cozinheira com os ingredientes que estão faltando! | |
| Requisitos mínimos | |
| **Nv. de base** | 50 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | Nenhum |
| **[Zeny](Zeny.md "Zeny")** | 13.000 |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 566.png 10 [Sopas Tailandesas](http://www.divine-pride.net/database/item/566/) |

1. Fale com Einon (ayothaya 196,265)

   :   ![A Sopa Tailandesa01](images/3/3b/A_Sopa_Tailandesa01.png)
2. Experimente a sopa da Cozinheira (ayo\_in01 145,163)

   :   ![A Sopa Tailandesa02](images/f/f9/A_Sopa_Tailandesa02.png)
3. Compre 20 [camarões](http://www.divine-pride.net/database/item/567/) por 11.000 [zenys](Zenys.md "Zenys") com Thongpool (ayothaya 153,86)

   :   ![A Sopa Tailandesa03](images/c/ce/A_Sopa_Tailandesa03.png)
4. Pergunte para a Cozinheira onde é possível encontrar limões.
5. Aproxime-se do Sr. Jun (ayothaya 121,240)

   :   ![A Sopa Tailandesa04](images/3/3f/A_Sopa_Tailandesa04.png)

   - O [NPC](NPC.md "NPC") te convida a jogar pedra, papel ou tesoura. Você precisa vencê-lo 3 vezes para ganhar 10 [limões](http://www.divine-pride.net/database/item/568/).
     - Caso perca, basta se afastar e em seguida se aproximar do [NPC](NPC.md "NPC") para tentar denovo.
     - Você pode optar por não entregar os [limões](http://www.divine-pride.net/database/item/568/) e manter a missão incompleta. Esse método é utilizado para conseguir [limão](http://www.divine-pride.net/database/item/568/) de maneira mais rápida.
6. Volte para a Cozinheira.
7. Compre 30 [Pimentas Vermelhas](http://www.divine-pride.net/database/item/7286/) por 2.000 [zenys](Zenys.md "Zenys") com o Mercador (ayothaya 213,94)

   :   ![A Sopa Tailandesa05](images/c/c4/A_Sopa_Tailandesa05.png)
8. Retorne ao restaurante e entregue os itens para receber 10 [Sopas Tailandesas](http://www.divine-pride.net/database/item/566/), que podem ser vendidas em [NPC](NPC.md "NPC") por 5.000 [zenys](Zenys.md "Zenys") cada.
