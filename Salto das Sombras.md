# Salto das Sombras

|  |  |
| --- | --- |

| Salto das Sombras | |
| **Shadow Leap / Salto Sombrío** | |
| **ID** | NJ\_SHADOWJUMP (529) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 10 |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Chão |
| **Alcance** | (Nv. da habilidade +4) células |
| **Pré-requisitos** | |
| [Virar Tatami](Virar Tatami.md "Virar Tatami") [Virar Tatami](Virar Tatami.md "Virar Tatami") Nv. 1 | |

Só pode ser usada em estado de [Esconderijo](Esconderijo.md "Esconderijo").

Move-se instantaneamente para a célula selecionada.

Essa habilidade atravessando obstáculos e cancela o estado de [Esconderijo](Esconderijo.md "Esconderijo").

| Nv. | Alcance |
| --- | --- |
| 1 | 5 células |
| 2 | 6 células |
| 3 | 7 células |
| 4 | 8 células |
| 5 | 9 células |

## Notas

- Não move o personagem em castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
- Pode ser ativada pelo [esconderijo](Esconderijo.md "Esconderijo") da habilidade [Corte da Névoa](Corte da Névoa.md "Corte da Névoa") ou pela [carta Fumacento](http://www.divine-pride.net/database/item/4044/).
