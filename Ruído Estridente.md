# Ruído Estridente

|  |  |
| --- | --- |

| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 60 + (Nv. da habilidade × 2) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 + (Nv. × 0,3) seg. |

Emite dois ruídos agudos.

Causa dano mágico de propriedade [Neutro](Neutro.md "Neutro") no alvo.

Oponentes com [Sono](Sono.md "Sono") ou [Sono Profundo](Sono Profundo.md "Sono Profundo") sofrerão 150% a mais de dano.

Se o alvo estiver sofrendo de [Desarranjo Musical](Arranjo Musical.md "Arranjo Musical"), há um bônus no dano causado.

### Fórmula

`Dano = {[Dano base + (Nv. de Domínio Musical × 60)] × (Nv. de base ÷ 100)}%`

## Notas

- Sobre [Sono Profundo](Sono Profundo.md "Sono Profundo"):
  - Alvos sofrendo de [sono profundo](Sono profundo.md "Sono profundo") já recebem 150% de dano normalmente.
  - A habilidade acumula com esse efeito, totalizando 225% de dano.
  - Esse dano total também é acumulativo com [Lex Aeterna](Lex Aeterna.md "Lex Aeterna").
  - O dano com [Sono Profundo](Sono Profundo.md "Sono Profundo") e [Arranjo Musical](Arranjo Musical.md "Arranjo Musical") também se acumulam.
