# Saque Rápido

|  |  |
| --- | --- |

| **Níveis** | 1 |

| **Alvo** | Usuário |
| **Munição** | [Over Bullet](Munições.md "Munições") Projéteis |
| **Pré-requisitos** | |
| [Reação em Cadeia](Reação em Cadeia.md "Reação em Cadeia") [Reação em Cadeia](Reação em Cadeia.md "Reação em Cadeia") Nv. 1 | |

Saca rapidamente sua arma.

Só pode ser usada após [Reação em Cadeia](Reação em Cadeia.md "Reação em Cadeia").

Permite causar dano extra no alvo ao custo de consumir projéteis adicionais.

Os tiros variam de acordo com o nível de classe do usuário.

| Nv. de classe | Tiros |
| --- | --- |
| 1 a 19 | 1 |
| 20 a 39 | 2 |
| 40 a 59 | 3 |
| 60 a 70 | 4 |

### Fórmula

`Dano = 100% × Tiros adicionais`

## Notas

- Não é ativado se a [velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") for 193.
- Se o ataque no alvo for um [crítico](Crítico.md "Crítico") duplo, o dano extra não será crítico também.
- Se oponentes estiverem [marcados em brasa](Marcar em Brasa.md "Marcar em Brasa") na área de 21x21 células ao redor do usuário, eles também receberão os tiros.
