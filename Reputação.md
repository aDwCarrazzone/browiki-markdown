# Reputação

![Reputa](images/e/ea/Reputa.png)

Seus pontos de reputação

Durante suas aventuras em [Ragnarök Online](Ragnarök Online.md "Ragnarök Online") é normal que seus feitos sejam reconhecidos por diversas pessoas.

Para acessar a janela de reputação, clique no ícone das estrelinhas ![Bt repute](images/2/2c/Bt_repute.png)

A sua reputação em um povoado pode lhe render benefícios se você auxiliar os habitantes em suas tarefas.

Porém, algumas tribos rivais podem não ficar felizes se souberem que você está ajudando o inimigo!

- O nível de reputação aumenta 1 a cada 1.000 pontos.
- Pontos negativos são exibidos como uma barra vermelha e você pode perder níveis de reputação.
- A reputação é vinculada ao personagem, não à conta.
- Quanto maior o nível de reputação, mais benefícios você consegue.

## Vila dos Orcs

1. Converse com o Líder da Tribo dos Orcs (in\_orcs01 106,177)

   :   ![ReputaOrc](images/6/62/ReputaOrc.png)
2. Para aumentar sua reputação na vila, você pode fazer as seguintes tarefas:
   - Entregar 1x [Carta Goblin](http://www.divine-pride.net/database/item/4060/) ⮕ **+50** pontos para os Orcs
   - Completar [Goblins Ferozes](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Orcs / **-5** para os Goblins
   - Completar [Goblins Velozes](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Orcs / **-5** para os Goblins
   - Completar [Goblins Atrozes](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Orcs / **-5** para os Goblins
3. O Comerciante da Tribo (in\_orcs01 116,179) deve abrir opções de permuta de acordo com seu nível de reputação
   - [Desconto](Desconto.md "Desconto") não funciona.

## Vila dos Goblins

1. Converse com o Líder da Tribo dos Goblins (gef\_fild11 247,253)

   :   ![ReputaGoblin](images/8/82/ReputaGoblin.png)
2. Para aumentar sua reputação na vila, você pode fazer as seguintes tarefas:
   - Entregar 1x [Carta Guerreiro Orc](http://www.divine-pride.net/database/item/4066/) ⮕ **+50** pontos para os Goblins
   - Entregar 100x [Amulero Orc](http://www.divine-pride.net/database/item/931/) (1 vez por dia) ⮕ **+30** pontos para os Goblins
   - Completar [Orcs Infernais](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Goblins / **-5** para os Orcs
   - Completar [Ganancioso](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Goblins / **-5** para os Orcs
   - Completar [Família Orc](Grupo do Éden.md#Nível_40_a_69 "Grupo do Éden") ⮕ **+10** pontos para os Goblins / **-5** para os Orcs
3. O Comerciante da Tribo (gef\_fild11 249,246) deve abrir opções de permuta de acordo com seu nível de reputação
   - [Desconto](Desconto.md "Desconto") não funciona.
