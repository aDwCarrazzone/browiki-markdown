# Salvando Elyumina

| ![NQ ico](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/82/NQ_ico.png) Salvando Elyumina | |
| --- | --- |
| Elyumina, antes membra da Ilusión, ainda sofre preconceito das autômatas. Após ser empurrada em uma água de esgoto radioativo, ela precisa de um antídoto coletado das Papilias. Salve a vida de Elyumina enquanto ajuda as jardineiras a reestabelecer a conexão com outras robôs. | |
| Requisitos mínimos | |
| **Nv. de base** | 130 |
| **Tipo** | Normal |
| **Caça** | [Papilia Violeta](http://www.divine-pride.net/database/monster/20667/) |
| **Quests anteriores** | [Legado do Sábio](Legado%20do%20S%C3%A1bio.md "Legado do Sábio") |
| Premiação | |
| **[EXP base](../../Redirecionamentos/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 1.547.004 |
| **[Itens](../../Itens/Itens.md "Itens")** | ![1000103.png](https://static.divine-pride.net/images/items/item/1000103.png) 20 [Vale Varmida](http://www.divine-pride.net/database/item/1000103/) |
| **Recompensa** | Libera o acesso à [Ortus Aqua](../../Calabou%C3%A7os%20e%20Inst%C3%A2ncias/Ortus%20Aqua.md "Ortus Aqua") |

1. Fale com Renshiz (ba\_pw02 41,267).

   :   ![Salvaelyu1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/e4/Salvaelyu1.png)
2. Encontre Nid (ba\_maison 170,270).

   :   ![Salvaelyu2](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/ff/Salvaelyu2.png)
3. Converse com Seiyu (ba\_maison 239,47).

   :   ![Seiyu172](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/16/Seiyu172.png)
4. Você receberá a missão de eliminar uma [Papilia Violeta](http://www.divine-pride.net/database/monster/20667/) em [Ortus Aqua](../../Calabou%C3%A7os%20e%20Inst%C3%A2ncias/Ortus%20Aqua.md "Ortus Aqua").
5. Após completar a [instância](../../Redirecionamentos/Inst%C3%A2ncia.md "Instância"), retorne a Seiyu (ba\_maison 239,47).
6. Entregue o antídoto para Elyumina (ba\_maison 234,51).

   :   ![Salvaelyu4](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/a/ad/Salvaelyu4.png)

   - Você receberá sua recompensa.
   - Após um dia, você poderá iniciar [Ortus Aqua](../../Calabou%C3%A7os%20e%20Inst%C3%A2ncias/Ortus%20Aqua.md "Ortus Aqua").
