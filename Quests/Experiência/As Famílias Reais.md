1. No quarto dos Gaebolg, fale com o Guarda Costas da Rainha, Jared (prt\_cas\_q 85,127)

3. Converse com Lidion Gaebolg, próximo a Scarlet.

   :   ![FamiliaReal 3](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/b9/FamiliaReal_3.png)
4. Retorne a Scarlet Jaha, que pedirá pistas para encontrar o Príncipe Ernest Gaebolg.
5. Seu objetivo é caminhar nas coordenadas para ativar diálogos.

   :   ![FamiliaReal 4](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/7a/FamiliaReal_4.png)

   - Localizações:
     - (prt\_cas 77,258)
     - (prt\_cas 87,235)
     - (prt\_cas 109,240)
     - (prt\_cas 108,221)
     - (prt\_cas 90,221)
     - (prt\_cas 78,220)
     - (prt\_cas 90,229)
6. Volte a falar com Scarlet para receber sua recompensa.
   - Você pode usar as Marcas de Honra para [trocar por itens](../../Lugares/Prontera.md#Marcas_de_Honra "Prontera")

   :   ![FamiliaReal 8](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/b5/FamiliaReal_8.png)

   - Você receberá 1 [Carta com Selo](http://www.divine-pride.net/database/item/6929/).
6. Entregue o item a Isenhonor Walther para receber a sua recompensa.
   - Você pode usar as Marcas de Honra para [trocar por itens](../../Lugares/Prontera.md#Marcas_de_Honra "Prontera")
   - Para desbloquear a conquista [Assaltante de Geladeira](../../Redirecionamentos/Conquista.md "Conquista"), traga 1 [Vinho de Mastela](http://www.divine-pride.net/database/item/12050/) para o Pé Grande entre 1 e 2 horas da manhã.

| Requisitos mínimos | |
| **Nv. de base** | 140 |
| **Tipo** | Repetível |
| **Caça** | 50 Monstros |
| **Quests anteriores** | [Banquete Real](Banquete%20Real.md "Banquete Real") |
| ![Skilldelayinfo](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/84/Skilldelayinfo.png) **Espera:** | 4 da manhã |
| Premiação | |
| **[EXP base](../../Redirecionamentos/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 500.000 |
| **[EXP classe](../../Redirecionamentos/Experi%C3%AAncia%20de%20Classe.md "Experiência de Classe")** | 550.000 |
| **[Itens](../../Itens/Itens.md "Itens")** | ![6919.png](https://static.divine-pride.net/images/items/item/6919.png) 4 [Marca de Honra](http://www.divine-pride.net/database/item/6919/) |

1. No quarto dos Richards, fale com o Mordomo (prt\_cas\_q 80,80)

   :   ![Mordomo Richard](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/e4/Mordomo_Richard.png)

   - O [NPC](../../Redirecionamentos/NPC.md "NPC") disponibiliza uma lista para matar 50 monstros de uma determinada raça.
   - A contagem de monstros mortos é feita por raça, ou seja, não importa *qual o monstro* que você caçar, mas a sua *raça*.
   - Será contado apenas os monstros da raça selecionada e que tenham nível 140 ou maior.
   - Portanto tome cuidado, pois não existem monstros da raça dragão com esse nível.
   - As raças disponíveis são:
     - [Humanoide](../../Redirecionamentos/Humanoide.md "Humanoide"), [Bruto](../../Redirecionamentos/Bruto.md "Bruto"), [Anjo](../../Redirecionamentos/Anjo.md "Anjo"), [Inseto](../../Redirecionamentos/Inseto.md "Inseto"), [Morto-Vivo](../../Redirecionamentos/Morto-Vivo.md "Morto-Vivo"), [Demônio](../../Redirecionamentos/Dem%C3%B4nio.md "Demônio"), [Dragão](../../Redirecionamentos/Drag%C3%A3o.md "Dragão"), [Peixe](../../Redirecionamentos/Peixe.md "Peixe"), [Planta](../../Redirecionamentos/Planta.md "Planta") ou [Amorfo](../../Redirecionamentos/Amorfo.md "Amorfo").
2. Retorne ao Mordomo após a caçada para receber a sua recompensa.
   - Você pode usar as Marcas de Honra para [trocar por itens](../../Lugares/Prontera.md#Marcas_de_Honra "Prontera")
