# Instrutores do Éden

As missões dadas pelos Instrutores do Éden são voltadas para iniciantes e consistem em lições básicas do jogo.

   :   ![MestredoEden2](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1e/MestredoEden2.png)

   1. Mestre de Batalha
   2. Mestre de Grupo
   3. Mestre de Guerra
   4. Mestre de Instância
   5. Mestre de Mapeamento
3. Retorne à Instrutora para receber sua recompensa.

## Instrutor

| ![GQ ico](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/19/GQ_ico.png) Instrutores do Éden | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 15 |
| **Tipo** | Tutorial |
| **[Itens](../../Itens/Itens.md "Itens")** | 1 [Catálogo de Vendas Prata](http://www.divine-pride.net/database/item/12580/), 10 [Jellopys](http://www.divine-pride.net/database/item/909/), 1 Pergaminho de [assistente](../../Redirecionamentos/Assistente.md "Assistente") |
| **[Zeny](../../Redirecionamentos/Zeny.md "Zeny")** | 10.200 |
| Premiação | |
| **[EXP base](../../Redirecionamentos/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 5.000 |
| **[EXP classe](../../Redirecionamentos/Experi%C3%AAncia%20de%20Classe.md "Experiência de Classe")** | 2.500 |

1. Aceite o desafio do Instrutor, no segundo andar do [Grupo do Éden](../Gerais/Grupo%20do%20%C3%89den.md "Grupo do Éden").

   :   ![InstrutorEden1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/61/InstrutorEden1.png)
2. Compre um pergaminho de [assistente](../../Redirecionamentos/Assistente.md "Assistente"), a guilda mais próxima está em (prontera 41,337)
3. Use o pergaminho e retorne ao Instrutor, que pedirá que você confira sua janela de missões, sem seguida, digite:

   :   `Jellopy`
4. Compre um [Catálogo de Vendas Prata](http://www.divine-pride.net/database/item/12580/) com o o [Catalogador Mágico](../../Redirecionamentos/Cat%C3%A1logo%20de%20Vendas.md "Catálogo de Vendas").
5. Retorne ao Instrutor, que pedirá que você use o item.
   - Dica: basta jogar o catálogo no chão, clicar no [NPC](../../Redirecionamentos/NPC.md "NPC") e pegar novamente.
6. Traga 10 [Jellopys](http://www.divine-pride.net/database/item/909/) ao instrutor para receber 3 [Capotes do Tutorial](http://www.divine-pride.net/database/item/15033/).
7. Pela última vez, fale com o Instrutor para receber sua recompensa. Os Capotes serão removidos de seu inventário.
