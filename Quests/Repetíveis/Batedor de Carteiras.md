1. Seu objetivo é ser "vítima" de um assalto, perdendo de 100 a 200 [zenys](../../Redirecionamentos/Zenys.md "Zenys") durante a missão. O [NPC](../../Redirecionamentos/NPC.md "NPC") ficará fugindo de uma coordenada para outra se você não o capturar.
2. Aproxime-se do Cara Suspeito (lighthalzen 240,216)

   :   ![Carlhz01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1e/Carlhz01.png)
3. Se o [NPC](../../Redirecionamentos/NPC.md "NPC") fugir, continue caminhando em torno do Cara Suspeito (lighthalzen 220,169)

   :   ![Carlhz02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/a/ab/Carlhz02.png)
4. Nesta última coordenada, se capturá-lo, o Suspeito venderá 1, 2, ou 3 [Poções do Vento](http://www.divine-pride.net/database/item/12016/) por 15.000 [zenys](../../Redirecionamentos/Zenys.md "Zenys") cada garrafa.

   :   ![Carlhz03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/cd/Carlhz03.png)

   - (lighthalzen 164,127)
5. Caso não tenha capturado ou comprado as poções, ele volta para a primeira coordenada.
