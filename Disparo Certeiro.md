# Disparo Certeiro

|  |  |
| --- | --- |

| **ID** | RA\_AIMEDBOLT (2236) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 40 |
| **[Conjuração](Conjuração.md "Conjuração")** | 1 + 2 seg. |
| **Pós-conjuração** | 2 segundos |
| **Recarga** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | (Nv. de [Olhos de Águia](Olhos de Águia.md "Olhos de Águia") + 9) células |
| **Munição** | [Over Arrow](Munições.md "Munições") 3 Flechas |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - [Instalar Armadilha](Instalar Armadilha.md "Instalar Armadilha") [Instalar Armadilha](Instalar Armadilha.md "Instalar Armadilha") Nv. 5 | |

Atira uma flecha que causa dano físico a distância em um único alvo.

O dano aumenta se o usuário estiver em estado de [![Disparo Selvagem](images/6/6b/Disparo_Selvagem.png)](Disparo Selvagem.md "Disparo Selvagem") [Disparo Selvagem](Disparo Selvagem.md "Disparo Selvagem").

| Nv. | [ATQ](ATQ.md "ATQ") sem bônus | [ATQ](ATQ.md "ATQ") com bônus |
| --- | --- | --- |
| 1 | 520% | 835% |
| 2 | 540% | 870% |
| 3 | 560% | 905% |
| 4 | 580% | 940% |
| 5 | 600% | 975% |
| 6 | 620% | 1.010% |
| 7 | 640% | 1.045% |
| 8 | 660% | 1.080% |
| 9 | 680% | 1.115% |
| 10 | 700% | 1.150% |

### Fórmula

`ATQ = [Dano base × (Nv. de base ÷ 100)]%`
