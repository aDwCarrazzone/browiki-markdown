# Onda Psíquica

|  |  |
| --- | --- |
| **Onda Psíquica Onda Psíquica** | |
| Onda Psíquica | |
| **Psychic Wave** | |
| **ID** | SO\_PSYCHIC\_WAVE (2449) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 40 + (Nv. da habilidade × 8) |
| **[Conjuração](Conjuração.md "Conjuração")** | 1,1 - (Nv. × 0,1) + (Nv. + 7) seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 5 segundos |
| **Duração** | [1 + (Nv. da hab. × 0,5)] segundos |
| **Alvo** | Chão |
| **Alcance** | 9 células |
| **Área** | 7x7 a 11x11 células |
| **Propriedade** | [Neutro](Monstros.md#Propriedade "Monstros") |
| **Pré-requisitos** | |
| - [Desencantar](Desencantar.md "Desencantar") [Desencantar](Desencantar.md "Desencantar") Nv. 2 | |

Libera energia psíquica para criar ondas a cada 0,5 segundos.

Causa dano mágico de propriedade [Neutro](Neutro.md "Neutro") na área selecionada.

A [propriedade](Propriedade.md "Propriedade") da magia pode ser alterada de acordo com o [Elemental](Domínio Elemental.md "Domínio Elemental").

Se estiver usando Livro ou Cajado, dobra as ondas invocadas, como um *bônus*.

| Nv. | [ATQM](ATQM.md "ATQM") | Ondas | | Área | [Conjuração](Conjuração.md "Conjuração") | | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Normal | Bônus | Fixa | Variável |
| 1 | 70% | ×3 | ×6 | 7x7 células | 1,0 segundo | 8 segundos | 48 |
| 2 | 140% | ×4 | ×8 | 0,9 segundos | 9 segundos | 56 |
| 3 | 210% | ×5 | ×10 | 9x9 células | 0,8 segundos | 10 segundos | 64 |
| 4 | 280% | ×6 | ×12 | 0,7 segundos | 11 segundos | 70 |
| 5 | 350% | ×7 | ×14 | 11x11 células | 0,6 segundos | 12 segundos | 78 |

### Fórmula

`Dano = [Dano base + (INT × 3)] × (Nv. de base ÷ 100)`

## Notas

- O *bônus* funciona com qualquer Cajado, de uma ou duas mãos.
- [Proteger Terreno](Proteger Terreno.md "Proteger Terreno") cancela e previne a o efeito da habilidade.
- A [propriedade](Propriedade.md "Propriedade") do dano mágico pode ser alterada de acordo com o Elemental em nível 2:
  - [Chandra](Chandra.md "Chandra"): [Terra](Terra.md "Terra")
  - [Agni](Agni.md "Agni"): [Fogo](Fogo.md "Fogo")
  - [Varuna](Varuna.md "Varuna"): [Água](Água.md "Água")
  - [Vayu](Vayu.md "Vayu"): [Vento](Vento.md "Vento")
  - Serpens: [Veneno](Veneno.md "Veneno")
- Caso a habilidade seja autoconjurada, também leva em consideração a [propriedade](Propriedade.md "Propriedade") ativada pelo Elemental.
