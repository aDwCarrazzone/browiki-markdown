# Perfurar em Espiral

|  |  |
| --- | --- |
| **Perfurar em Espiral Perfurar em Espiral** | |

| **Clashing Spiral / Espiral Perforante** | |
| **ID** | LK\_SPIRALPIERCE (397) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 15 + (Nv. da habilidade × 3) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,3 + (Nv. × 0,05) seg. |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | 5 células |
| **Arma** | Lança ou Espada ou Espada de Duas Mãos |
| **Pré-requisitos** | |
| - [Perfurar](Perfurar.md "Perfurar") [Perfurar](Perfurar.md "Perfurar") Nv. 5 - [Estocada](Estocada.md "Estocada") [Estocada](Estocada.md "Estocada") Nv. 5 - [Perícia com Lança](Perícia com Lança.md "Perícia com Lança") [Perícia com Lança](Perícia com Lança.md "Perícia com Lança") Nv. 5 | |

Ataca usando a técnica espiral, golpeando 5 vezes um alvo.

Causa dano físico a distância e deixando o oponente imóvel por 1 segundo.

O dano é afetado pelo peso da arma e o tamanho do alvo.

| Nv. | [ATQ](ATQ.md "ATQ") base | [Conjuração variável](Conjuração variável.md "Conjuração variável") | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 200% | 0,05 segundos | 18 |
| 2 | 250% | 0,10 segundos | 21 |
| 3 | 300% | 0,15 segundos | 24 |
| 4 | 350% | 0,20 segundos | 27 |
| 5 | 400% | 0,25 segundos | 30 |

### Fórmula

`ATQ = [{[(ATQ × 0,7) + (Peso da arma × 0,7)] × Tamanho} + ATQ de Perícia] × [ATQ base x (Nv. de base ÷ 100)]`

- Tamanho:
  - Alvo pequeno: 1,30
  - Alvo médio: 1,15
  - Alvo grande: 1,0
