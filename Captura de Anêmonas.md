# Captura de Anêmonas

| DQ ico Captura de Anêmonas | |
| --- | --- |
| Um estranho crescimento da população de anêmonas está invadindo as praias da cidade! Tente controlar as criaturas com a ajuda da Bióloga. | |
| Requisitos mínimos | |
| **Nv. de base** | 40 |
| **Tipo** | Repetível |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 11502.png 5 [Poção Azul Leve](http://www.divine-pride.net/database/item/11502/) |
| **Recompensa** | [DES](DES.md "DES"), [VIT](VIT.md "VIT") e [SOR](SOR.md "SOR") +5 por 20 minutos |

1. Ajude a Bióloga Marinha (brasilis 193,133)

   :   ![Captura das Anêmonas01](images/c/c2/Captura_das_Anêmonas01.png)
2. Use a [Esfera de Captura](http://www.divine-pride.net/database/item/12408/) nas Anêmonas. O método é igual ao de capturar uma [mascote](Mascote.md "Mascote").

   :   ![Captura das Anêmonas02](images/1/1d/Captura_das_Anêmonas02.png)

   - Se desejar outra [Esfera de Captura](http://www.divine-pride.net/database/item/12408/), basta falar com a Bióloga.
3. Entregue a [Esfera Mística](http://www.divine-pride.net/database/item/6221/) para a Bióloga lhe agradecer com 5 [Poção Azul Leve](http://www.divine-pride.net/database/item/11502/) e um bônus de [DES](DES.md "DES"), [VIT](VIT.md "VIT") e [SOR](SOR.md "SOR") +5 por 20 minutos.
