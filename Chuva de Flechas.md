# Chuva de Flechas

|  |  |
| --- | --- |
| **Chuva de Flechas Chuva de Flechas** | |
| Chuva de Flechas | |
| **Arrow Shower / Lluvia de Flechas** | |
| **ID** | AC\_SHOWER (47) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 15 |
| **Pós-conjuração** | 0,1 segundo |
| **Alvo** | Chão |
| **Alcance** | (9 + Nv. de [Olhos de Águia](Olhos de Águia.md "Olhos de Águia")) células |
| **Empurra** | 2 células |
| **Área** | 3x3 a 5x5 células |
| **Munição** | [Over Arrow](Munições.md "Munições") 1 Flecha |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") Nv. 5 | |

Atira flechas em uma área selecionada.

Causa dano físico a distância e empurra oponentes atingidos.

| 1 | 160% | 3x3 células |
| 2 | 170% |
| 3 | 180% |
| 4 | 190% |
| 5 | 200% |
| 6 | 210% | 5x5 células |
| 7 | 220% |
| 8 | 230% |
| 9 | 240% |
| 10 | 250% |
