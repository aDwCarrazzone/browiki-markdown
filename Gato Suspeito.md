# Gato Suspeito

| DQ ico Gato Suspeito | |
| --- | --- |
| Para um gato comum, explorar as catacumbas de uma pirâmide cheia de cachorros pode ser um verdadeiro pesadelo! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Repetível |
| **Caça** | 20 [Verits do Pesadelo](http://www.divine-pride.net/database/monster/2355/) ou 20 [Múmias Anciãs do Pesadelo](http://www.divine-pride.net/database/monster/2360/) |
| Skilldelayinfo **Espera:** | 4 da manhã |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 300.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 100.000 |

1. No [Modo Pesadelo](Modo Pesadelo.md "Modo Pesadelo") da Pirâmide, converse com o Gato Suspeito (moc\_prydn1 94,98)

   :   ![GatoSuspeito2](images/7/72/GatoSuspeito2.png)
2. O [NPC](NPC.md "NPC") dará 2 missões diferentes, podendo ser escolhida separadamente:
   - Selecione *Eliminar Verit* para caçar 20 [Verits do Pesadelo](http://www.divine-pride.net/database/monster/2355/)
   - Selecione *Eliminar Múmias* para caçar 20 [Múmias Anciãs do Pesadelo](http://www.divine-pride.net/database/monster/2360/)
3. Retorne ao Gato Suspeito para receber sua recompensa:

:   :   | Missão | [EXP](EXP.md "EXP") Base | [EXP](EXP.md "EXP") Classe |
        | --- | --- | --- |
        | Cães Inimigos | 300.000 | 100.000 |
        | Múmias Entediadas | 300.000 | 100.000 |
