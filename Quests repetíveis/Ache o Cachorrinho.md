# Ache o Cachorrinho

| DQ ico Ache o Cachorrinho | |
| --- | --- |
| O dono de 3 cachorrinhos está a procura de aventureiros que possam ajudá-lo a encontrar seus cães perdidos na floresta! | |
| Requisitos mínimos | |
| **Nv. de base** | 40 |
| **Tipo** | Repetível |
| Premiação | |
| **[EXP base](../Sem%20categoria/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 50.000 |

1. Ajude Angelo (brasilis 297,306)

   :   ![Ache o cachorrinho01](../images/a/ad/Ache_o_cachorrinho01.png)
2. Encontre 3 cachorrinhos espalhados pela selva. Se um cachorrinho não estiver na localização marcada no mapa, procure outro ponto.

   :   ![Ache o cachorrinho02](../images/f/f4/Ache_o_cachorrinho02.png)

   - Localizações:
     - (bra\_fild01 98,96)
     - (bra\_fild01 234,101)
     - (bra\_fild01 81,163)
     - (bra\_fild01 80,210)
     - (bra\_fild01 62,142)
3. Retornando a Angelo para receber [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência") e recuperar seu [HP](../Sem%20categoria/HP.md "HP") e [SP](../Sem%20categoria/SP.md "SP") totalmente.
