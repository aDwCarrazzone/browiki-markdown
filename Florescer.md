Cria um jardim de flores explosivas.

A cada intervalo de tempo, uma flor explode para causar dano mágico.

Cada flor ocupa uma área de 7x7 células dentro do terreno que você criou.

As flores explodem de maneira aleatória dentro da área.

Quanto maior o nível, mais flores poderão explodir.

| Nv. | Flores Explosivas | | | Área | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| [ATQM](ATQM.md "ATQM") | Núm. | Intervalo |
| 1 | 100% | 4 | 1,50 segundos | 7x7 células | 80 |
| 2 | 200% | 8 | 0,75 segundos | 90 |
| 3 | 300% | 12 | 0,50 segundos | 9x9 células | 100 |
| 4 | 400% | 16 | 0,37 segundos | 110 |
| 5 | 500% | 20 | 0,30 segundos | 120 |

### Fórmula

`ATQM por flor = {[Dano base + (5 × FEI)] × (Nv. base ÷ 100)}%`

### Potencialização

- Essa habilidade pode atingir seu clímax a cada nível de [![Potencializar Magia](images/4/49/Potencializar_Magia.png)](Potencializar Magia.md "Potencializar Magia") [Potencializar Magia](Potencializar Magia.md "Potencializar Magia").
- Seu solo mágico muda de cor conforme o nível usado.

|  |  |
| --- | --- |
| Clímax | Nível 1 |
| - Reduz pela metade o intervalo entre as explosões.   - Redução por nível: 0,75 / 0,375 / 0,25 / 0,18 / 0,15 segundos. | |
| Clímax | Nível 2 |
| - Dano de Florescer -50%. - Dobra o número de flores explosivas.   - Aumento por nível: 8 / 16 / 24 / 32 / 40 | |
| Clímax | Nível 3 |
| - Dano de Florescer +100%.   - Esse valor é cumulativo com [equipamentos](Equipamentos.md "Equipamentos") de efeitos semelhantes. | |
| Clímax | Nível 4 |
| - Não causa mais dano. Inflige o estado de [*Pólen*] em quem estiver na área. - **Ícone:** I ClimaxPolen - **Duração:** 30 segundos - **Efeitos no oponente:**   - Resistência à propriedade [Fogo](Fogo.md "Fogo") -100%. | |
| Clímax | Nível 5 |
| - Quando todas as flores explodirem, você queima seu jardim de uma vez. - Causa 7.000% de dano mágico de propriedade [Fogo](Fogo.md "Fogo"). | |

## Notas

- A duração da habilidade acaba quando todas as flores explodirem.
- O estado de *Pólen* não pode ser removido por [Desencantar](Desencantar.md "Desencantar") ou [Gênese](Gênese.md "Gênese").
- Não causa dano, nem inflige efeito em oponentes dentro do [Proteger Terreno](Proteger Terreno.md "Proteger Terreno").
- *Pólen* e [Empalamento](Pilares de Pedra.md "Pilares de Pedra") podem estar atuantes simultaneamente.
