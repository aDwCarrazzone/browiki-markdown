# Arremesso de Machado

|  |  |
| --- | --- |
| **Arremesso de Machado Arremesso de Machado** | |
| Arremesso de Machado | |
| **Axe Boomerang / Hacha Bumerán** | |
| **ID** | NC\_AXEBOOMERANG (2278) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 18 + (Nv. da habilidade × 2) |
| **Recarga** | [5,5 - (Nv. da hab. × 0,5)] segundos |
| **Alvo** | Oponente |
| **Alcance** | (Nv. da habilidade + 4) células |
| **Empurra** | (Nv. da habilidade + 1) células |
| **Arma** | Machado |
| **Pré-requisitos** | |
| - [Maestria com Machados](Maestria com Machados.md "Maestria com Machados") [Maestria com Machados](Maestria com Machados.md "Maestria com Machados") Nv. 1 | |

Atira o machado como um bumerangue.

Causa dano físico a distância e empurra o alvo para trás.

| Nv. | Dano base | Alcance | Empurra | Recarga | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| 1 | 300% | 5 | 2 | 5,0 segundos | 20 |
| 2 | 350% | 6 | 3 | 4,5 segundos | 22 |
| 3 | 400% | 7 | 4 | 4,0 segundos | 24 |
| 4 | 450% | 8 | 5 | 3,5 segundos | 26 |
| 5 | 500% | 9 | 6 | 3,0 segundos | 28 |

### Fórmula

`Dano = {[Dano base] + Peso do machado} × (Nv. de base ÷ 100)`
