# Improviso

| --- | --- |

| **Improvised Song / Canción Improvisada** | |
| **ID** | WM\_RANDOMIZESPELL (2424) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 |
| **Pós-conjuração** | 0,3 segundos |
| **Recarga** | 5 segundos |
| **Alvo** | Personagens |
| **Arma** | Chicote ou Instrumento musical |
| **Pré-requisitos** | |
| - [Poema de Nifflheim](Poema de Nifflheim.md "Poema de Nifflheim") [Poema de Nifflheim](Poema de Nifflheim.md "Poema de Nifflheim") Nv. 1 | |

Improvisa uma apresentação.

Tem chance de retirar efeitos de Coral atuantes sobre o personagem selecionado.

| Nv. | Chance |
| --- | --- |
| 1 | 40% |
| 2 | 50% |
| 3 | 60% |
| 4 | 70% |
| 5 | 80% |

## Notas

- Não pode ser usada em si próprio.
- Pode ser usada em aliados para remover Corais negativos ou contra inimigos para remover Corais positivos.
- Para usar em aliados, é necessário segurar a tecla `Shift`
