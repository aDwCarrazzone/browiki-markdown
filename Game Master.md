# Game Master

Game Masters são funcionários contratados pela *GnJoy LATAM* para trabalhar em [Ragnarök Online](Ragnarök Online.md "Ragnarök Online"), ficando responsáveis pelas seguintes funções no jogo:

- Checar a funcionalidade dos [itens](Itens.md "Itens");
- Atender consumidores na Central de Suporte;
- Verificar anormalidades no jogo;
- Aplicar punições a infratores.

### Identificação

Por ser uma autoridade em [Ragnarök](Ragnarök.md "Ragnarök"), os Game Masters possuem roupa própria e a sigla **[GM]** antes do nome.

![GameMaster](images/d/d6/GameMaster.png)

- Todas as mensagens públicas e o próprio nome dos Game Masters são exibidas em amarelo claro.
- Um Game Master nunca pedirá [itens](Itens.md "Itens") ou [zenys](Zenys.md "Zenys") ou detalhes da sua conta, como e-mail, senha, nome de usuário.
- Game Masters não enviam mensagens particulares alertando uma suspensão da conta ou punição.
  - As mensagens particulares de um Game Master aparecem no topo da sua tela, como um anúncio de [broadcast](Broadcast.md "Broadcast").

### Comandos

Se você tiver dúvidas de alguém que se diz ser Game Master, você pode pedir que ele te teleporte para um mapa reservado.

- `/b <mensagem>` Envia uma mensagem com o nome para o servidor inteiro.
- `/nb <mensagem>` Envia uma mensagem sem nome para todos no servidor.
- `/bb <mensagem>` Envia uma mensagem de cor azul para todos do mapa.
- `/nlb <mensagem>` Envia uma mensagem somente para todos do mapa.
- `/mm <mapa>` Teleporta o Game Master para o mapa digitado.
- `/shift <nome>` Move o Game Master diretamente para próximo de um personagem.
- `/remove <nome>` Move o Game Master para um personagem de acordo com a conta digitada.
- `/recall <nome>` Teleporta um personagem pelo nome de usuário da conta.
- `/monster <nome>` Invoca um monstro no mapa.
- `/monster whereisboss` Exibe as coordenadas do [MVP](MVP.md "MVP") do mapa.
- `/hide` Torna-se invisível para todos, não podendo ser detectado por habilidades nem sofrer ataques.
- `/item refinitem dbname 10` Cria um item já refinado no +10.
- `/item packageitemtest itemid 10000 0` Cria uma pasta com as chances de um pacote loteria.
- `/item pet,relationship,número`
- `/item pet,getegg,dbname do monstro`
