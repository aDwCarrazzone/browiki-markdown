Permite criar poções a partir de itens definidos pelo manual de criação.

Aumentar o nível da habilidade melhora as chances de sucesso.

| Nv. | Chance |
| --- | --- |
| 1 | +3% |
| 2 | +6% |
| 3 | +9% |
| 4 | +12% |
| 5 | +15% |
| 6 | +18% |
| 7 | +21% |
| 8 | +24% |
| 9 | +27% |
| 10 | +30% |

### Fórmula

`Chance = [(Nv. de Pesquisa de Poções, Preparar Poção e Mudança de Planos somados) + (Nv. de classe × 0,2) + (DES × 0,1) + (SOR × 0,1) + (INT × 0,05) + (Dificuldade)]%`

| Dificuldade | Poções |
| --- | --- |
| +15% ~ +25% | Poção Vermelha, Poção Amarela, Poção Branca. |
| +5% ~ +15% | Álcool. |
| -5% ~ +5% | Frasco de Ácido, Frasco de Esfera Marinha, Frasco de Fogo Grego, Frasco de Planta Carnívora. |
| -5% | Poção Azul, Analgésico, Aloe Vera, Embrião, Poções Anti-Propriedade, Poção Compacta Vermelha. |
| -10% ~ -5% | Poção Compacta Amarela. |
| -15% ~ -5% | Poção Compacta Branca, Frasco de Revestimento. |

## Ranking

Os 10 maiores criadores de poção do servidor listados no Ranking terão bônus nas poções criadas: Poder de cura por quaisquer poções de cura produzidas +50%

O ranking pode ser visualizado pelo comando `/alchemist`

- Pontuação:
  - **1 ponto:** Criar 3 Poções Compactas seguidas, sem falha;
  - **3 pontos:** Criar 5 Poções Compactas seguidas, sem falha;
  - **10 pontos:** Criar 7 Poções Compactas seguidas, sem falha;
  - **50 pontos:** Criar 10 Poções Compactas seguidas, sem falha;
- Usar [Criação Espiritual de Poções II](Criação Espiritual de Poções II.md "Criação Espiritual de Poções II") conta no ranking.
