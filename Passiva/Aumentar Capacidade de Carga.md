# Aumentar Capacidade de Carga

|  |  |
| --- | --- |
| **Aumentar Capacidade de Carga Aumentar Capacidade de Carga** | |
| SemImagem | |
| **Enlarge Weight Limit / Aumentar Límite de Peso** | |
| **ID** | MC\_INCCARRY (36) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| - Nenhum | |

Aumenta o limite de [peso](../Sem%20categoria/Peso.md "Peso") máximo que pode ser carregado.

| Nv. | Limite |
| --- | --- |
| 1 | +200 |
| 2 | +400 |
| 3 | +600 |
| 4 | +800 |
| 5 | +1.000 |
| 6 | +1.200 |
| 7 | +1.400 |
| 8 | +1.600 |
| 9 | +1.800 |
| 10 | +2.000 |

## Nota

- Não influencia o limite de peso do [carrinho](Usar%20Carrinho.md "Usar Carrinho").
- Acumula com [Aumentar Capacidade de Carga Especial](Aumentar%20Capacidade%20de%20Carga%20Especial.md "Aumentar Capacidade de Carga Especial").
