# Perícia com Mão Direita

|  |  |

| **Right hand Mastery / Maestría de la Mano Derecha** | |
| **ID** | AS\_RIGHT (132) |
| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Pré-requisitos** | |
| - Nenhum | |

Ao equipar duas armas, reduz a penalidade de dano ao equipar uma arma na mão direita.

| Nv. | Dano |
| --- | --- |
| 1 | 60% |
| 2 | 70% |
| 3 | 80% |
| 4 | 90% |
| 5 | 100% |

## Notas

- Não funciona com armas do tipo Katar.
- [Mercenários](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Mercen%C3%A1rios.md "Mercenários") e evoluções podem equipar duas armas, mas haverá penalidade no dano físico ao equipar uma segunda arma.
- A arma da mão direita causará apenas 50% do seu dano original.
- Por conta disso, cada nível aprendido desta habilidade, a penalidade de dano é corrigida.
- No nível 5, a penalidade é totalmente anulada, retornando ao dano original.
- Sobre Manejo Duplo:
  - Apenas [Mercenários](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Mercen%C3%A1rios.md "Mercenários"), [Kagerou](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Kagerou.md "Kagerou"), [Oboro](../P%C3%A1ginas%20que%20usam%20a%20etiqueta%20Tabber%20do%20analisador%20sint%C3%A1tico/Oboro.md "Oboro") e evoluções podem equipar armas nas duas mãos.
  - A mão esquerda pode equipar escudos ou armas.
  - No caso de habilidades mágicas, os bônus mágicos da mão esquerda são contabilizados.
  - No caso de habilidades físicas:
    - Apenas os bônus de equipamento são contados.
    - O ATQ natural da arma, o [refino](../Sem%20categoria/Refino.md "Refino") e a propriedade não são contabilizados.
    - [Cartas](../Itens/Cartas.md "Cartas") e [Bônus Aleatórios](../Sem%20categoria/B%C3%B4nus%20Aleat%C3%B3rios.md "Bônus Aleatórios") são contados.
  - Em ataques físicos básicos, todos os bônus da mão esquerda são contados normalmente (ATQ natural e propriedade).
