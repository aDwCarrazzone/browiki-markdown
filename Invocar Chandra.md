Desperta o espírito da montanha, Chandra.

Concede bônus especiais para seu mestre.

Manter o elemental vivo drena [SP](SP.md "SP") a cada 10 segundos.

| [SP](SP.md "SP") | (SP do usuário ÷ 4) |
| [ATQ](ATQ.md "ATQ") (mín.) | [SP do usuário x (Nv. de Invocar ÷ 18)] + (1 ~ 100) |
| [ATQ](ATQ.md "ATQ") (máx.) | [SP do usuário x (Nv. de Invocar ÷ 18)] |

| [DEF](DEF.md "DEF") | {DEF do usuário + [Nv. de base ÷ (5 - Nv. de Invocar )]} |
| [DEFM](DEFM.md "DEFM") | {DEFM do usuário + [INT ÷ (5 - Nv. de Invocar )]} |
| [Esquiva](Esquiva.md "Esquiva") | {Esquiva do usuário + [Nv. de base ÷ (5 - Nv. de Invocar )]} |

## Notas

- Apenas 1 elemental pode estar ativo por vez.
- Os elementais não funcionam como os [homunculus](Homunculus.md "Homunculus"), não havendo necessidade de programar uma [AI](AI.md "AI") ou alimentá-los.
- As habilidades dos elementais são conjuradas de acordo com o modo definido por [Domínio Elemental](Domínio Elemental.md "Domínio Elemental").
