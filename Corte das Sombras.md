# Corte das Sombras

|  |  |
| --- | --- |
| **Corte das Sombras Corte das Sombras** | |
| Corte das Sombras | |
| **Shadow Slash / Golpe de Sombra** | |
| **ID** | NJ\_KIRIKAGE (530) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 9 + Nv. da habilidade |
| **Alvo** | Oponente |
| **Alcance** | 4 + Nv. de [Salto das Sombras](Salto das Sombras.md "Salto das Sombras") |
| **Pré-requisitos** | |
| [Corte da Névoa](Corte da Névoa.md "Corte da Névoa") [Corte da Névoa](Corte da Névoa.md "Corte da Névoa") Nv. 5 | |

Só pode ser usada se o usuário estiver em um estado de [esconderijo](Esconderijo.md "Esconderijo").

Surpreende o oponente com um dano físico corpo a corpo, aumentando o [CRIT](CRIT.md "CRIT") durante o ataque.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") | [CRIT](CRIT.md "CRIT") | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 200% | +30 | 10 |
| 2 | 350% | +35 | 11 |
| 3 | 500% | +40 | 12 |
| 4 | 650% | +45 | 13 |
| 5 | 800% | +50 | 14 |

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- Não pode ser usada nos castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
- O estado de [Esconderijo](Esconderijo.md "Esconderijo") pode ser ativada por [Corte da Névoa](Corte da Névoa.md "Corte da Névoa"), pela [carta Fumacento](http://www.divine-pride.net/database/item/4044/) e etc.
- Pode atingir oponentes a distância de acordo com o nível de [Salto das Sombras](Salto das Sombras.md "Salto das Sombras"), mas o dano é sempre considerado corpo a corpo.
