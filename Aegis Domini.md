# Aegis Domini

| **Tipo** | Suporte |

| **Alvo** | Usuário |

| **Pré-requisitos** | |
| - [Aegis Inferi](Aegis Inferi.md "Aegis Inferi") [Aegis Inferi](Aegis Inferi.md "Aegis Inferi") Nv. 2 - [Escudo Compressor](Escudo Compressor.md "Escudo Compressor") [Escudo Compressor](Escudo Compressor.md "Escudo Compressor") Nv. 3 | |

Concentra magia no seu escudo por 90 segundos.

Recebe um bônus que muda de acordo com o nível da habilidade.

O bônus não acumula com os bônus dos outros níveis.

| Nv. | Bônus |
| --- | --- |
| 1 | Cura 3% do [HP](HP.md "HP") máx. a cada 3 segundos. |
| 2 | Cura 5% do [SP](SP.md "SP") máx. a cada 5 segundos. |
| 3 | [ATQ](ATQ.md "ATQ") e [ATQM](ATQM.md "ATQM") +150. |

## Notas

- Retirar ou trocar de escudo após conjurar a habilidade não cancela a duração dos efeitos.
- Os efeitos são cancelados se o usuário receber [Desencantar](Desencantar.md "Desencantar"), [Canto de Batalha](Canto de Batalha.md "Canto de Batalha"), [Bastão de Hermod](Bastão de Hermod.md "Bastão de Hermod") e [Consagração](Consagração.md "Consagração").
