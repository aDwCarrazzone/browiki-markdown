# Renovatio

|  |  |
| --- | --- |

| **Tipo** | Suporte |
| **Níveis** | 4 |
| **[SP](SP.md "SP")** | 70 |
| **[Conjuração](Conjuração.md "Conjuração")** | 2 + 3 seg. |
| **Recarga** | 1 segundo |
| **Duração** | 90 segundos |
| **Alvo** | Usuário |
| **Área** | 31x31 células |
| **Ícone** | I Renovatio |
| **Pré-requisitos** | |
| - [Sopro Divino](Sopro Divino.md "Sopro Divino") [Sopro Divino](Sopro Divino.md "Sopro Divino") Nv. 3 | |

Invoca um brilho divino.

Regenera [HP](HP.md "HP") a cada 5 segundos de todos os membros do [grupo](Grupo.md "Grupo") na tela do usuário.

| Nv. | [HP](HP.md "HP") curado | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 5% | 90 segundos | 240 |
| 2 | 6% | 120 segundos | 280 |
| 3 | 7% | 150 segundos | 320 |
| 4 | 8% | 180 segundos | 360 |

## Notas

- Não é removida por [Desencantar](Desencantar.md "Desencantar") ou [Gênese](Gênese.md "Gênese").
- Membros do [grupo](Grupo.md "Grupo") com armadura de propriedade [Maldito](Maldito.md "Maldito") não receberão o efeito.
