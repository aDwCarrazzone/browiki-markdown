# Mudança de Classe: Mercenários

| JQ ico Mudança de Classe: Mercenários | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Nv. de classe** | 40 |
| **Tipo** | Classe |
| **Classe** | [Gatuno](Gatuno.md "Gatuno") |
| Premiação | |
| **[Itens](Itens.md "Itens")** | Ver tabela |
| **Recompensa** | Muda a classe para [Mercenário](Mercenário.md "Mercenário") |

1. Dirija-se para a Guilda dos Mercenários e fale com o Mercenário (in\_moc\_16 19,33)

   :   ![MdCsin01](images/8/85/MdCsin01.png)
2. Seu teste começa ao se aproximar do Funcionário da Guilda, que se moverá para um outro ponto. Ande até ele para se inscrever.

   :   ![MdCsin02](images/6/63/MdCsin02.png)
3. Na nova sala, siga em frente até um diálogo começar a fazer perguntas. Você deve acertar 9, no mínimo, para passar.

   :   ![MdCsin03](images/c/cd/MdCsin03.png)

:   :   |  |  |
        | --- | --- |
        | Possibilidade I | |

        | Qual desses monstros ignora você estar escondido furtivo? | Andre |

:   4. Após o questionário, siga em frente até ouvir uma conversa. Para iniciar o teste, basta clicar no chat e aguardar sua vez.

    :   ![MdCsin04](images/2/29/MdCsin04.png)

        - Seu objetivo será eliminar 6 [monstros](Monstros.md "Monstros") de nome *Vítima* em 3 minutos.
        - Os [monstros](Monstros.md "Monstros") são 3 Porings, 1 Lunático, 1 Poporing e 1 Drops
        - O teste falha se você:
          - Cair em um buraco.
          - Atacar um monstro que não tenha o nome *Vítima*.
:   5. Em caso de sucesso, a próxima etapa é chegar perto de Dante (in\_moc\_16 89,98) sem matar os [monstros](Monstros.md "Monstros") que surgem no caminho. O ideal é abusar da sua [esquiva](Esquiva.md "Esquiva") e usar [esconderijo](Esconderijo.md "Esconderijo") quando puder.

    :   ![MdCsin07](images/c/ca/MdCsin07.png)
:   6. A última etapa consiste em aproximar-se do Líder da Guilda (in\_moc\_16 149,80)

    :   ![MdCsin06](images/b/b8/MdCsin06.png)

        - Aproxime-se do Líder e responda às perguntas, suas escolhas não influenciam na missão. Você receberá 1 [Colar do Oblívio](http://www.divine-pride.net/database/item/1008/).
        - Se seu [nível de classe](Nível de classe.md "Nível de classe") estiver entre 40 a 48 quando começou a missão, você receberá 1 arma aleatória.
          - [Main Gauche](http://www.divine-pride.net/database/item/1207/) [3]
          - [Jur](http://www.divine-pride.net/database/item/1250/) [2]
          - [Stiletto](http://www.divine-pride.net/database/item/1216/) [2]
          - [Faca](http://www.divine-pride.net/database/item/1201/) [3]
          - [Katar](http://www.divine-pride.net/database/item/1252/) [1]
        - Se seu [nível de classe](Nível de classe.md "Nível de classe") for 49 ou 50 quando começou a missão, você escolherá a arma.
          - [Jur](http://www.divine-pride.net/database/item/1251/) [3]
          - [Katar](http://www.divine-pride.net/database/item/1253/) [2]
          - [Main Gauche](http://www.divine-pride.net/database/item/1208/) [4]
          - [Gladius](http://www.divine-pride.net/database/item/1220/) [3]
        - Se seu personagem for desconectado do servidor ou você cancelar a conversa com o Líder, será necessário caminhar ao redor dele novamente para continuar o diálogo.
:   7. Entregue o [Colar do Oblívio](http://www.divine-pride.net/database/item/1008/) ao Mercenário (in\_moc\_16 19,33) para mudar de classe.

    :   ![MdCsin01](images/8/85/MdCsin01.png)
