# Conhecendo a Morte

| NQ ico Conhecendo a Morte | |
| --- | --- |
| Nem sempre as pessoas conseguem se despedir de alguém na hora da morte. Ajude Aleuce a transmitir uma mensagem de afeto para sua querida esposa! | |
| Requisitos mínimos | |
| **Nv. de base** | 80 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 748.png 1 [Rosa Eterna](http://www.divine-pride.net/database/item/748/)   610.png 10 [Folhas de Yggdrasil](http://www.divine-pride.net/database/item/610/) |
| Premiação | |
| **[Itens](Itens.md "Itens")** | Variável |

1. Fale com Aleuce (prt\_in 161,20)

   :   ![Conhecendo a Morte01](images/a/a5/Conhecendo_a_Morte01.png)
2. Diga que está atrás de uma pessoa para Billy (umbala 215,181)

   :   ![Conhecendo a Morte02](images/a/af/Conhecendo_a_Morte02.png)
3. Com os itens no inventário, fale mais de uma vez com Fayelyn (niflheim 153,215)

   :   ![Conhecendo a Morte03](images/f/fa/Conhecendo_a_Morte03.png)

   - Selecione sempre a primeira opção. Não selecione a opção "*Desculpe*".
   - Fayelyn tentará realizar uma magia, a premiação varia se essa magia der certo ou errado.
   - Em caso de sucesso, você receberá 1 [Espólio](http://www.divine-pride.net/database/item/934/).
   - Em caso de falha, você perderá 50% do [HP](HP.md "HP").
4. Retorne a Aleuce em [Prontera](Prontera.md "Prontera") para receber sua premiação dependendo do resultado.
   - Em caso de falha, você receberá 10 [Folhas de Yggdrasil](http://www.divine-pride.net/database/item/610/).
   - Se bem sucedido, você receberá 10 [Folhas de Yggdrasil](http://www.divine-pride.net/database/item/610/), 2 [Sementes de Yggdrasil](http://www.divine-pride.net/database/item/608/) e 2 [Frutos de Yggdrasil](http://www.divine-pride.net/database/item/607/).
