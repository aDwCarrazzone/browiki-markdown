# Mudar de Elemento

|  |  |
| --- | --- |
| **NPC Skill Icon Mudar de Elemento** | |
| SemImagem | |
| **ID:** | NPC\_CHANGE (Vários) |
| **Níveis:** | 1 |
| **Exclusiva:** | Sim |

O [monstro](Monstro.md "Monstro") altera a [propriedade](Propriedade.md "Propriedade") elemental.

O efeito é permanente ou até usar outra habilidade de mudança elemental.

A habilidade possui vários nomes, conforme a [propriedade](Propriedade.md "Propriedade").

## Notas

- Não há indicação gráfica do uso da [habilidade](Habilidade.md "Habilidade") ou mudança visual no [monstro](Monstro.md "Monstro").
- O número elemental é alterado de maneira aleatória, se o monstro for [Água](Água.md "Água") 4 e mudar para [Fogo](Fogo.md "Fogo"), ele poderá ser [Fogo](Fogo.md "Fogo") 2, por exemplo.
