# Pesquisa de Poções

|  |  |
| --- | --- |
| **Pesquisa de Poções Pesquisa de Poções** | |
| SemImagem | |

| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| - Nenhum | |

Através do estudo, aumenta as chances de produzir poções.

Melhora a efetividade de cura das poções criadas e [arremessadas](Arremessar Poção.md "Arremessar Poção").

| Nv. | Chance | Efetividade |
| --- | --- | --- |
| 1 | +1% | +5% |
| 2 | +2% | +10% |
| 3 | +3% | +15% |
| 4 | +4% | +20% |
| 5 | +5% | +25% |
| 6 | +6% | +30% |
| 7 | +7% | +35% |
| 8 | +8% | +40% |
| 9 | +9% | +45% |
| 10 | +10% | +50% |
