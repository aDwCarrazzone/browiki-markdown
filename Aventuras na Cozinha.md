# Aventuras na Cozinha

## Emergência: Molhos

| DQ ico Aventuras na Cozinha | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | 100 |
| **Tipo** | Repetível |
| **Quests anteriores** | [Emergência Culinária](Emergência Culinária.md "Emergência Culinária") |
| Skilldelayinfo **Espera:** | 4 da manhã |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 6919.png 4 [Marca de Honra](http://www.divine-pride.net/database/item/6919/) |

1. Encontre o Mestre dos Molhos (prt\_cas 319,211)

   :   ![EmergeCu 03](images/3/3d/EmergeCu_03.png)

   - Seu objetivo é conseguir fazer o molho com as indicações dadas no diálogo.
   - Após o diálogo citar o ingrediente, basta seguir até o [NPC](NPC.md "NPC") relacionado.

     :   ![EmergeCu 03a](images/c/cd/EmergeCu_03a.png)

     - **Manteira** (losango): Sabor suave
     - **Gelatina de Poring** (triângulo): Sabor texturizado
     - **Suco de Muka** (círculo): Sabor refrescante
     - **Planta Brilhante** (quadrado): Sabor brilhante
2. Ao completar, fale novamente com o Mestre dos Molhos para receber sua recompensa.
