# Salto

|  |  |
| --- | --- |
| **Salto Salto** | |
| Salto | |
| **Leap / High Jump** | |
| **ID** | TK\_HIGHJUMP (426) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | (6 - Nv. da habilidade) segundos |
| **Alvo** | Chão |
| **Alcance** | (Nv. da habilidade × 2) células |
| **Pré-requisitos** | |
| - Nenhum | |

Pula em direção a uma célula válida do mapa.

Pode saltar de muros, montanhas e outros obstáculos atravessáveis.

| Nv. | Alcance | [Conjuração](Conjuração.md "Conjuração") | |
| --- | --- | --- | --- |
| Fixa | Variável |
| 1 | 2 células | 2,5 segundos | 2,5 segundos |
| 2 | 4 células | 2,0 segundos | 2,0 segundos |
| 3 | 6 células | 1,5 segundos | 1,5 segundos |
| 4 | 8 células | 1,0 segundo | 1,0 segundo |
| 5 | 10 células | 0,5 segundos | 0,5 segundos |

## Notas

- Não pode ser usada em mapas onde o [teleporte](Teleporte.md "Teleporte") é desabilitado, com exceção dos castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium") clássica.
- A habilidade falha em mapas de [PvP](PvP.md "PvP"), em células inválidas ou em castelos da segunda e terceira edição da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
- Enquanto líder de [clã](Clã.md "Clã"), aprender esta habilidade dobra o tempo de [conjuração](Conjuração.md "Conjuração") de [Chamado Urgente](Chamado Urgente.md "Chamado Urgente").
