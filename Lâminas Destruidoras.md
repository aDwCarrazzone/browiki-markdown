# Lâminas Destruidoras

| **Sonic Blow / Impacto Sónico** | |
| **ID** | AS\_SONICBLOW (136) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 14 + (Nv. da habilidade × 2) |
| **Recarga** | 1 segundo |
| **Alvo** | Oponente |
| **Alcance** | 1 célula |
| **Efeito negativo** | [Atordoamento](Atordoamento.md "Atordoamento") |
| **Arma** | Katar |
| **Pré-requisitos** | |
| - [Perícia com Katar](Perícia com Katar.md "Perícia com Katar") [Perícia com Katar](Perícia com Katar.md "Perícia com Katar") Nv. 4 | |

Usa o katar para desferir 8 ataques sucessivos contra um alvo.

Causa dano físico corpo a corpo com chance de infligir [atordoamento](Atordoamento.md "Atordoamento").

Se o oponente estiver com o [HP](HP.md "HP") menor que 50%, o dano aumenta 1,5 vezes.

- É melhorada por [Lâminas Aceleradas](Lâminas Aceleradas.md "Lâminas Aceleradas") e [Espírito do Mercenário](Espírito do Mercenário.md "Espírito do Mercenário")
- A animação da habilidade exibe o caractere **滅**, que significa destruir.
- Ao ser adquirida ou autoconjurada por itens, a habilidade funciona com qualquer arma.
