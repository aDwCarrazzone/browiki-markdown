# Onda de Choque

|  |  |
| --- | --- |
| **Onda de Choque Onda de Choque** | |
| Onda de Choque | |
| **Sonic Wave / Onda Sónica** | |
| **ID** | RK\_SONICWAVE (2002) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 30 + (Nv. da habilidade × 3) |
| **Pós-conjuração** | 0,5 segundo |
| **Recarga** | 1,75 segundos |
| **Alvo** | Oponente |
| **Alcance** | 7 a 11 células |
| **Pré-requisitos** | |
| - [Encantar Lâmina](Encantar Lâmina.md "Encantar Lâmina") [Encantar Lâmina](Encantar Lâmina.md "Encantar Lâmina") Nv. 3 | |

Maneja a arma rapidamente, provocando uma onda sonora.

Causa dano físico a distância.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

Aumentar o nível da habilidade melhora a [precisão](Precisão.md "Precisão") do ataque.

`ATQ = [Dano base × (Nv. de Base ÷ 100]%`
