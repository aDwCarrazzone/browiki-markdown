# Remover Escudo

|  |  |
| --- | --- |
| **Remover Escudo Remover Escudo** | |
| Remover Escudo | |
| **Strip Shield / Quitar Escudo** | |
| **ID** | RG\_STRIPSHIELD (216) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 15 + (Nv. da habilidade × 2) |
| **[Conjuração](Conjuração.md "Conjuração")** | [0,5 + (Nv. da hab. × 0,2)] segundos |
| **Pós-conjuração** | 1 segundo |
| **Duração** | [60 + (Nv. da hab. × 15)] segundos |
| **Alvo** | Oponente |
| **Alcance** | 2 células |
| **Ícone** | I Remover Escudo |
| **Pré-requisitos** | |
| - [Remover Capacete](Remover Capacete.md "Remover Capacete") [Remover Capacete](Remover Capacete.md "Remover Capacete") Nv. 5 | |

Tenta remover o escudo do oponente, impedindo-o de equipá-lo temporariamente.

Em [monstros](Monstros.md "Monstros"), reduz a [DEF](DEF.md "DEF") em 15%.

| Nv. | Remoção | | [Conjuração](Conjuração.md "Conjuração") | | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| Chance base | Duração base | Fixa | Variável |
| 1 | 10% | 75 segundos | 0,14 segundos | 0,56 segundos | 17 |
| 2 | 15% | 90 segundos | 0,18 segundos | 0,72 segundos | 19 |
| 3 | 20% | 105 segundos | 0,22 segundos | 0,88 segundos | 21 |
| 4 | 25% | 120 segundos | 0,26 segundos | 1,14 segundos | 23 |
| 5 | 30% | 135 segundos | 0,30 segundos | 1,20 segundos | 25 |

### Fórmula

`Chance = {Chance base + [(DES do usuário - DES do alvo) ÷5]}%`  
`Duração = {Duração base + [(DES do usuário - DES do alvo) ÷5]}%`

## Notas

- A chance de sucesso não pode ser menor que 5% e a duração não pode ser menor que 2 segundos.
- Não exibe animação em caso de falha.
- Não funciona contra [Mecânicos](Mecânicos.md "Mecânicos") pilotando um [MECHA](Licença de Pilotagem.md "Licença de Pilotagem").
- Funciona em [monstros](Monstros.md "Monstros") do tipo chefe e em alvos sofrendo de [congelamento](Congelamento.md "Congelamento") e [petrificação](Petrificação.md "Petrificação").
- O efeito de remoção pode ser prevenido por [Revestir Escudo](Revestir Escudo.md "Revestir Escudo") e cancelado por [Canto de Batalha](Canto de Batalha.md "Canto de Batalha") ou desconectar e conectar no jogo novamente.
