# Impacto Flamejante

|  |  |
| --- | --- |
| **Impacto Flamejante Impacto Flamejante** | |
| Impacto Flamejante | |
| **Ignition Break / Explosión Ígnea** | |
| **ID** | RK\_IGNITIONBREAK (2006) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 30 + (Nv. da habilidade × 5) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 seg. |
| **Recarga** | 2 segundos |
| **Alvo** | Usuário |
| **Área** | 9x9 células |
| **Pré-requisitos** | |
| - [Onda de Choque](Onda de Choque.md "Onda de Choque") [Onda de Choque](Onda de Choque.md "Onda de Choque") Nv. 2 - [Vento Cortante](Vento Cortante.md "Vento Cortante") [Vento Cortante](Vento Cortante.md "Vento Cortante") Nv. 3 - [Revidar Dano](Revidar Dano.md "Revidar Dano") [Revidar Dano](Revidar Dano.md "Revidar Dano") Nv. 5 | |

Crava a espada no chão, liberando uma explosão.

Causar dano físico corpo a corpo ao redor de 9x9 células do usuário.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 450% | 35 |
| 2 | 900% | 40 |
| 3 | 1.350% | 45 |
| 4 | 1.800% | 50 |
| 5 | 2.250% | 55 |

### Fórmula

`ATQ = [(Nv. da habilidade × 450) × (Nv. de base ÷ 100)]%`
