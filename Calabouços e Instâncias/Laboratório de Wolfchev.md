| Requisitos mínimos | |

| **Tipo** | Instância |
| **Duração** | 2 horas e 30 minutos |
| ![Skilldelayinfo](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/84/Skilldelayinfo.png) **Retorno** | 3 dias |
| **[Grupo](../Sistema%20e%20Mec%C3%A2nicas/Grupo.md "Grupo")** | 1 pessoa ou mais |

| Premiação | |
| **Chefe** | Aleatório |
| **Recompensa** | [Material Biológico](../Lugares/Lighthalzen.md#Encantamento "Lighthalzen") |

1. Entre na fila da [instância](../Redirecionamentos/Inst%C3%A2ncia.md "Instância") falando com o Pesquisador (lhz\_dun04 151,276)

   :   ![Investigando os Rumores07](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/6e/Investigando_os_Rumores07.png)
2. Acesse o mapa pela Entrada do Laboratório.

   :   ![Investigando os Rumores08](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/bc/Investigando_os_Rumores08.png)
3. **Laboratório 1**:
   1. Nesta fase, derrote todas as [Cobaias Famintas](http://www.divine-pride.net/database/monster/2243/).
   2. Leia as Folhas do Manual.

      :   ![LabWolfchev01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/d/d1/LabWolfchev01.png)
   3. Gire a Válvula conforme recomendação.

      :   ![LabWolfchev02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/29/LabWolfchev02.png)

      - As Folhas do Manual indicarão quais as direções que você deve girar as **2** Válvulas.
      - Se errar, é necessário ler as Folhas do Manual novamente para resetar as direções.
      - Cada direção é aleatória e muda sempre que você ler o manual.
      - Exemplo:
        - Girar a Válvula Esquerda
        - Uma vez na Anti-horária
        - Duas vezes na Horária
        - Uma vez na Anti-horária
   4. Em caso de sucesso, entre no portal para a próxima fase.
4. **Laboratório 2**:
   1. Encontre as Válvulas e libere a pressão.
      - Este passo parece estar bugado, basta eliminar os monstros para seguir ao próximo lugar.
      - Durante esse processo, uma barra de conjuração aparecerá.
      - O personagem não poderá ser atacado enquanto gira a válvula.
      - O [grupo](../Sistema%20e%20Mec%C3%A2nicas/Grupo.md "Grupo") tem 3 chances e fechar as válvulas. Em caso de falha, serão mandados para fora da [instância](../Redirecionamentos/Inst%C3%A2ncia.md "Instância").
      - A cada falha, mais [Animais Famintos](http://www.divine-pride.net/database/monster/2242/) surgirão.
   2. Em caso de sucesso, entre no portal para a próxima fase.
5. **Laboratório 3**:
   1. Você tem 3 minutos para derrotar todos os [Animais Famintos](http://www.divine-pride.net/database/monster/2242/) e [Cobaias Famintas](http://www.divine-pride.net/database/monster/2243/).
   2. Em caso de sucesso, entre no portal para a próxima fase.
6. **Laboratório 4**:
   1. Prepare-se para a batalha! Um dos 13 [MVPs](../Sistema%20e%20Mec%C3%A2nicas/MVP.md "MVP") fugirá dos tubos de ensaio e seu [grupo](../Sistema%20e%20Mec%C3%A2nicas/Grupo.md "Grupo") deve derrotá-lo.
      - O [MVP](../Sistema%20e%20Mec%C3%A2nicas/MVP.md "MVP") é escolhido aleatoriamente:
        - [Algoz Eremes](http://www.divine-pride.net/database/monster/1647/)
        - [Mestre-Ferreiro Howard](http://www.divine-pride.net/database/monster/1648/)
        - [Lorde Seyren](http://www.divine-pride.net/database/monster/1646/)
        - [Menestrel Alphoccio](http://www.divine-pride.net/database/monster/2240/)
        - [Criador Flamel](http://www.divine-pride.net/database/monster/2236/)
        - [Paladino Randel](http://www.divine-pride.net/database/monster/2235/)
        - [Mestre Chen](http://www.divine-pride.net/database/monster/2238/)
        - [Sumo Sacerdotisa Margaretha](http://www.divine-pride.net/database/monster/1649/)
        - [Atiradora de Elite Cecil](http://www.divine-pride.net/database/monster/1650/)
        - [Arquimaga Kathryne](http://www.divine-pride.net/database/monster/1651/)
        - [Cigana Trentini](http://www.divine-pride.net/database/monster/2241/)
        - [Professora Celia](http://www.divine-pride.net/database/monster/2237/)
        - [Desordeira Gertie](http://www.divine-pride.net/database/monster/2239/)
7. Fale com Wolfchev no centro do laboratório para encerrar a [instância](../Redirecionamentos/Inst%C3%A2ncia.md "Instância").
   - Você pode trocar seus itens adquiridos por [itens biológicos](../Lugares/Lighthalzen.md#Itens_Biol.C3.B3gicos "Lighthalzen").
