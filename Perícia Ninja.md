# Perícia Ninja

|  |  |
| --- | --- |
| **Perícia Ninja Perícia Ninja** | |
| SemImagem | |
| **Ninja Mastery / Maestría de Ninja** | |
| **ID** | NJ\_NINPOU (533) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| Nenhum | |

Enquanto imóvel, regenera o [SP](SP.md "SP") a cada 10 segundos.

A taxa de regeneração aumenta conforme o [SP](SP.md "SP") do usuário.

| Nv. | Regeneração de [SP](SP.md "SP") | |
| --- | --- | --- |
| Base | Bônus |
| 1 | 3 | +0,2% |
| 2 | 6 | +0,4% |
| 3 | 9 | +0,6% |
| 4 | 12 | +0,8% |
| 5 | 15 | +1% |
| 6 | 18 | +1,2% |
| 7 | 21 | +1,4% |
| 8 | 24 | +1,6% |
| 9 | 27 | +1,8% |
| 10 | 30 | +2% |

## Notas

- O efeito é cancelado se o usuário não puder regenerar o [SP](SP.md "SP") naturalmente, por exemplo, acima do [peso](Peso.md "Peso") permitido.
- O tempo de regeneração é reduzido para 5 segundos se estiver sentado.
