# Inspeção no Templo de Odin

| EP ico Inspeção no Templo de Odin | |
| --- | --- |
| Dois grupos de escavação estão procurando aventureiros para investigar as ruínas do Templo de Odin. Ao se deparar com uma organização secreta chamada "Vento Secreto", você pode descobrir que os mistérios não estão somente nas ruínas... | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Episódio |
| **[Itens](Itens.md "Itens")** | 7511.png 5 [Runas da Escuridão](http://www.divine-pride.net/database/item/7511/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 100.000 |
| **[Itens](Itens.md "Itens")** | 603.png 1 [Velha Caixa Azul](http://www.divine-pride.net/database/item/603/) |

1. Leia o Aviso (hugel 91,152)

   :   ![TdO00](images/b/b9/TdO00.png)
2. Dirija-se ao Escritório de Arqueologia (hu\_in01 14,11)

   :   ![TdO01](images/e/e6/TdO01.png) ![TdO02](images/0/00/TdO02.png)

   - Nesse mapa há duas salas com as arqueólogas Laura e Alex.
   - Você pode escolher se deseja ajudar Laura ou Alex.
   - Este guia optará pela Laura, mas tenha uma conversa com as duas primeiro.
   - Independente da escolha, você precisará pisar no mapa do Templo de Odin.
3. Para acessar o Templo, pague 800 [zenys](Zenys.md "Zenys") ao Barqueiro (hugel 209,109)

   :   ![TdO03a](images/7/7c/TdO03a.png)

   - Assim que acessar o Templo, basta retornar para [Hugel](Hugel.md "Hugel").
4. Converse com Laura e aceite ajudá-la.
   - Ela pedirá 5 [Runas da Escuridão](http://www.divine-pride.net/database/item/7511/).
   - Mesmo que tenha os itens no inventário, será necessário repetir o passo acima.
   - Então fale com o Barqueiro para acessar o Templo e retorne para Hugel.
5. Entregue as 5 [Runas da Escuridão](http://www.divine-pride.net/database/item/7511/) para Laura.
   - Você receberá 1 [Velha Caixa Azul](http://www.divine-pride.net/database/item/603/).
   - Continue conversando com Laura até ela pedir que você investigue Alex.
6. Aproxime-se da porta do quarto de Alex para ouvir a conversa.

   :   ![TdO03](images/0/02/TdO03.png)
7. Retorne ao quarto de Laura.
   - Selecione `Giantes` e depois, qualquer opção.
8. Dirija-se para o Templo de Odin novamente e procure pelas coordenadas abaixo.

   :   ![Escavação do Templo de Odin010](images/7/73/Escavação_do_Templo_de_Odin010.png)
   :   (odin\_tem03 288,271)
   :   (odin\_tem03 266,302)
   :   (odin\_tem03 321,263)
   :   (odin\_tem03 179,275)
   :   (odin\_tem03 283,241)

   - A coordenada correta teleportará você para as ruínas.
   - Caso não consiga ativar o teleporte, tente outra coordenada.
9. Nas ruínas, siga reto e ative um diálogo (que\_hugel 35,29)

   :   ![TdO04a](images/3/38/TdO04a.png)

   - Selecione:
     - "*Explorar*"
     - "*Pegar*"
   - Se a opção não aparecer, continue andando em volta.
   - Após selecionar pegar, você pode optar por sair das ruínas por onde entrou ou usar asa de borboleta.
10. Retorne ao escritório, fale com Laura.

    :   ![TdO08](images/6/60/TdO08.png)
11. Converse com Ashe.

    :   ![TdO05](images/0/08/TdO05.png)
12. Aproxime-se das coordenadas (moc\_fild12 161,367)

    :   ![TdO045](images/a/a7/TdO045.png)

    - Você receberá 100.000 de [experiência](Experiência.md "Experiência").
13. Ao retornar para o escritório, você encontrará o lugar vazio.
