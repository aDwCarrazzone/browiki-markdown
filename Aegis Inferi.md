# Aegis Inferi

|  |  |
| --- | --- |
| **Aegis Inferi Aegis Inferi** | |
| Aegis Inferi | |
| **Earth Drive / Golpe de Tierra** | |
| **ID** | LG\_EARTHDRIVE (2323) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 44 + (Nv. da habilidade × 8) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 seg. |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | (6,5 - Nv. da hab.) segundos |
| **Alvo** | Usuário |
| **Área** | 3x3 a 7x7 células |
| **Equipamento** | Escudo |
| **Pré-requisitos** | |
| - [Escudo Supressor](Escudo Supressor.md "Escudo Supressor") [Escudo Supressor](Escudo Supressor.md "Escudo Supressor") Nv. 3 | |

Bate o escudo no chão, destruindo o solo ao redor do usuário.

Causa dano físico corpo a corpo até em alvos em [Esconderijo](Esconderijo.md "Esconderijo"), revelando-os.

Magias de solo atuantes ao redor do impacto serão canceladas.

Se [Escudo Reforçado](Arremessar Escudo.md "Arremessar Escudo") estiver ativo, há um bônus no dano causado.

| Nv. | [ATQ](ATQ.md "ATQ") | | Área | Recarga | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| Normal | [Bônus](Arremessar Escudo.md "Arremessar Escudo") |
| 1 | 380% | 600% | 3x3 células | 6,5 segundos | 52 |
| 2 | 760% | 1.200% | 5,5 segundos | 60 |
| 3 | 1.140% | 1.800% | 5x5 células | 4,5 segundos | 68 |
| 4 | 1.520% | 2.400% | 3,5 segundos | 76 |
| 5 | 1.900% | 3.000% | 7x7 células | 2,5 segundos | 84 |

### Fórmula

Normal:  
`ATQ = [(Dano base + FOR + VIT) × (Nv. de base ÷ 100)]%`

Escudo Reforçado:  
`ATQ = {Nv. da habilidade × (600 + [(Nv. de Perícia com Escudo × 15) + (FOR + VIT)]} × (Nv. de base ÷ 100)}%`

## Notas

- A [propriedade](Propriedade.md "Propriedade") do ataque pode ser alterada pela arma.
- Cancela as mesmas magias removidas por [Ganbantein](Ganbantein.md "Ganbantein"), exceto [Tornado](Tornado.md "Tornado").
- O dano é afetado por [cartas](Cartas.md "Cartas"), [itens](Itens.md "Itens") e [equipamentos](Equipamentos.md "Equipamentos") que aumentam o [ATQ](ATQ.md "ATQ") em porcentagem.
