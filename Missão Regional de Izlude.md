# Missão Regional de Izlude

| NQ ico Missão Regional de Izlude | |
| --- | --- |
| Estudantes de [Moscóvia](Moscóvia.md "Moscóvia"), [Porto Malaya](Porto Malaya.md "Porto Malaya") e [Amatsu](Amatsu.md "Amatsu") estão fazendo intercâmbio por Rune-Midgard, mas acabaram deixando um amigo para trás...  ---  Modo história do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden"). | |
| Requisitos mínimos | |
| **Nv. de base** | 50 |
| **Tipo** | Normal |
| **[Itens](Itens.md "Itens")** | 1024.png 10 [Tintas de Polvo](http://www.divine-pride.net/database/item/1024/) |
| **Caça** | 10 [Marses](http://www.divine-pride.net/database/monster/1144/) |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 10.640 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 10.240 |
| **[Itens](Itens.md "Itens")** | 12056.png 1 [Sopa de Lula e Sapo](http://www.divine-pride.net/database/item/12056/) |

1. Fale com Ernesto (iz\_dun01 121,158)

   :   ![IzludeRegional01](images/5/54/IzludeRegional01.png)
2. Encontre o Líder Maximus (iz\_dun02 229,231)

   :   ![IzludeRegional02](images/1/19/IzludeRegional02.png)

   - Agora você pode iniciar as missões diárias. Essas [quests](Quests.md "Quests") também estão disponíveis nas placas do [Grupo do Éden](Grupo do Éden.md "Grupo do Éden").
3. Ao lado de Maximus, converse com Kysekili (iz\_dun02 228,229)
   - Elimine 10 [Marses](http://www.divine-pride.net/database/monster/1144/) e traga 10 [Tintas de Polvo](http://www.divine-pride.net/database/item/1024/).
   - Você receberá 10.640 de [EXP](EXP.md "EXP") base e 10.240 de [EXP](EXP.md "EXP") de classe.
   - Se esta for a sua primeira vez, você receberá o dobro de [EXP](EXP.md "EXP").
4. Kysekili dará 1 [Sopa de Lula e Sapo](http://www.divine-pride.net/database/item/12056/).
5. Volte ao Membro Ernesto (iz\_dun01 121,158) e entregue o item.
   - Você receberá 10.640 de [EXP](EXP.md "EXP") base e 10.240 de [EXP](EXP.md "EXP") de classe.
