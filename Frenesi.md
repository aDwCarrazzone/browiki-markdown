# Frenesi

|  |  |
| --- | --- |
| **Frenesi Frenesi** | |
| Frenesi | |
| **Frenzy / Berserk** | |
| **ID** | LK\_BERSERK (359) |
| **Tipo** | Ativa |
| **Níveis** | 1 |
| **[SP](SP.md "SP")** | 200 |
| **Duração** | Até o [HP](HP.md "HP") chegar a 100 |
| **Alvo** | Usuário |
| **Pré-requisitos** | |
| - [Nível de classe](Nível.md "Nível") 50 | |

Entra em um estado de delírio total.

Regenera todo o [HP](HP.md "HP") e torna-se vulnerável.

### Efeitos

- [ATQ](ATQ.md "ATQ") +200%.
- [HP](HP.md "HP") +300%.
- [Velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") +30%.
- [Esquiva](Esquiva.md "Esquiva") -50%.
- Ativa [Vigor](Vigor.md "Vigor").
- Aumenta a [velocidade de movimento](Velocidade de movimento.md "Velocidade de movimento").
- Reduz a [DEF](DEF.md "DEF") e a [DEFM](DEFM.md "DEFM") a zero.
- Reduz o [SP](SP.md "SP") a zero.
- Drena 5% de [HP](HP.md "HP") a cada 15 segundos.
- Desabilita a regeneração natural de [HP](HP.md "HP") e [SP](SP.md "SP") durante a habilidade e 5 minutos após o término.
- Impossibilita o uso de habilidades, trocar de [equipamentos](Equipamentos.md "Equipamentos"), usar consumíveis, conversar, e com algumas exceções, recuperar o [HP](HP.md "HP").

## Notas

- O efeito de não reagir ao receber [ataques físicos](Atributos.md#Ataque_Físico "Atributos") de [Vigor](Vigor.md "Vigor") não é interrompido se o usuário receber 7 golpes.
- É cancelada se o personagem desconectar, chegar a 100 de [HP](HP.md "HP") ou receber [Gênese](Gênese.md "Gênese") e [Desencantar](Desencantar.md "Desencantar").
- [Habilidades](Habilidades.md "Habilidades") como [Renovatio](Renovatio.md "Renovatio") e [Dádiva do Atum](Dádiva do Atum.md "Dádiva do Atum") não recuperam o [HP](HP.md "HP"). Porém [Beijo do Vampiro](Beijo do Vampiro.md "Beijo do Vampiro"), [Chakra da Cura](Chakra da Cura.md "Chakra da Cura"), [Criar Monstro Planta](Criar Monstro Planta.md "Criar Monstro Planta"), [Drenar Vida](Drenar Vida.md "Drenar Vida"), [Epiclesis](Epiclesis.md "Epiclesis"), [Aquecer Terreno](Aquecer Terreno.md "Aquecer Terreno") e [Ajuda Angelical](Ajuda Angelical.md "Ajuda Angelical") recuperam.
- Efeitos de [itens](Itens.md "Itens") de conversão, tais como a [Carta Mosca Caçadora](http://www.divine-pride.net/database/item/4115/), não funcionam. Porém itens como a [Carta Exterminador](http://www.divine-pride.net/database/item/4158/) funcionam.
- [Exalar Alma](Exalar Alma.md "Exalar Alma") pode regenerar o [SP](SP.md "SP") do personagem.
- A [velocidade de movimento](Velocidade de movimento.md "Velocidade de movimento") não acumula com outros efeitos parecidos, somente com a [Montaria](Montaria.md "Montaria").
- Não afeta o [ATQ](ATQ.md "ATQ") dado por [Lâmina de Aura](Lâmina de Aura.md "Lâmina de Aura") ou [Encantar Lâmina](Encantar Lâmina.md "Encantar Lâmina").
- Aumenta todos os danos em uma porcentagem do seu ataque.
  - Se você causa um ataque básico, ele é aumentado em 200%.
  - Se você usa uma habilidade de 700% do ataque, ela passa a causar 900%.
