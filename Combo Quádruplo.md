# Combo Quádruplo

|  |  |
| --- | --- |
| **Combo Quádruplo Combo Quádruplo** | |
| Combo Quádruplo | |
| **Raging Quadruple Blow / Golpe Furioso Cuádruple** | |
| **ID** | MO\_CHAINCOMBO (272) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | (Nv. da habilidade + 4) |
| **Pós-conjuração** | 0,3 segundos |
| **Alvo** | Oponente |
| **Alcance** | 1 célula |
| **Pré-requisitos** | |
| - [Combo Triplo](Combo Triplo.md "Combo Triplo") [Combo Triplo](Combo Triplo.md "Combo Triplo") Nv. 5 | |

Só pode ser usada após 0,3 segundos depois de ativar [Combo Triplo](Combo Triplo.md "Combo Triplo") ou em alvos capturados por [Dilema](Dilema.md "Dilema").

Golpeia o oponente quatro vezes, causando dano físico corpo a corpo.

Se o usuário equipar uma Soqueira, atinge o alvo 6 vezes, causando um dano ainda maior.

| Soqueira | Outros |
| 1 | 600% | 300% | 5 |
| 2 | 700% | 350% | 6 |
| 3 | 800% | 400% | 7 |
| 4 | 900% | 450% | 8 |
| 5 | 1.000% | 500% | 9 |
