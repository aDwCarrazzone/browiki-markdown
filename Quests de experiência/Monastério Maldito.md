# Monastério Maldito

| EP ico Monastério Maldito | |
| --- | --- |
| O desaparecimento do Rei Tristan Gaebolg III pode estar relacionado a sua última viagem para uma ilha sem nome. | |
| Requisitos mínimos | |
| **Nv. de base** | 80 |
| **Tipo** | Episódio |
| **Caça** | 1 [Massacre](http://www.divine-pride.net/database/monster/1864/) e 1 [Tristam III](http://www.divine-pride.net/database/monster/1875/) |
| **[Zeny](../Sem%20categoria/Zeny.md "Zeny")** | 3.000 |
| **Quests anteriores** | [Segredo da Família Real](Segredo%20da%20Fam%C3%ADlia%20Real.md "Segredo da Família Real") [Irmãos em Veins](Irm%C3%A3os%20em%20Veins.md "Irmãos em Veins") |
| Premiação | |
| **[EXP base](../Sem%20categoria/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 150.000 |
| **[Itens](../Itens/Itens.md "Itens")** | 12052.png 4 [Frutos do Mar Sortidos](http://www.divine-pride.net/database/item/12052/) |
| **Recompensa** | Acesso a [Ilha Esquecida](http://www.divine-pride.net/database/map/nameless_n/) |

1. Pegue o Aeroplano internacional via [Izlude](../Lugares%20em%20Ragnar%C3%B6k/Izlude.md "Izlude"), [Juno](../Lugares%20em%20Ragnar%C3%B6k/Juno.md "Juno") ou [Rachel](../Lugares%20em%20Ragnar%C3%B6k/Rachel.md "Rachel").
2. Ao entrar no convés, desça as escadas para ativar o Agente (airplane\_01 96,61)

   :   ![Larjes0](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/31/Larjes0.png)
3. Vá de encontro ao Padre Bamph (prt\_church 185,106)

   :   ![Gaebolgs04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/6e/Gaebolgs04.png)
4. Fale mais de uma vez com Larjes (cmd\_in02 174,90)

   :   ![Larjes1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/16/Larjes1.png)
5. Fique conversando várias vezes com o Padre Bamph (prt\_church 185,106)

   :   ![Gaebolgs04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/6e/Gaebolgs04.png)
6. Na estalagem de [Rachel](../Lugares%20em%20Ragnar%C3%B6k/Rachel.md "Rachel"), faça um pedido ao Garçom (ra\_in01 308,59)

   :   ![Garssom01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/61/Garssom01.png)
7. Você estará no restaurante, ouvindo uma conversa. Ande ao redor aos homens famintos para abrir um diálogo (ra\_in01 301,43)

   :   ![Garssom02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/b/bb/Garssom02.png)

   - Pague 3.000 [zenys](../Sem%20categoria/Zenys.md "Zenys") pelos [Frutos do Mar Sortidos](http://www.divine-pride.net/database/item/12052/).
8. Fale com o Magistrado Al Hamad (ve\_in 78,311)

   :   ![Magistrado1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/83/Magistrado1.png)
9. Converse com o Soldado ao lado do Magistrado.

   :   ![Magistrado2](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/e9/Magistrado2.png)
10. Retorne ao Magistrado e pergunte sobre os contrabandistas.

    :   ![Magistrado1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/83/Magistrado1.png)
11. Dentro do Santuário de [Rachel](../Lugares%20em%20Ragnar%C3%B6k/Rachel.md "Rachel"), converse com o Sumo Sacerdote Zhed (ra\_temin 276,159). O nome do cientista é Varmunt. Para prosseguir nesse passo, é necessário ter concluído a quest [Irmãos em Veins](Irm%C3%A3os%20em%20Veins.md "Irmãos em Veins")

    :   ![Zhed](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/a/ad/Zhed.png)
12. Procure pela Sumo Sacerdotisa Niren (ra\_temple 165,56)

    :   ![Niren](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/9/9a/Niren.png)
13. Reencontre o Magistrado Al Hamad. (ve\_in 78,311)

    :   ![Magistrado1](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/8/83/Magistrado1.png)
14. Dentro do Bar de [Veins](../Lugares%20em%20Ragnar%C3%B6k/Veins.md "Veins"), encontre o Homem Risonho (ve\_in 239,115)

    :   ![Niren02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/69/Niren02.png)
15. Converse com o Garoto (veins 326,185)

    :   ![IeV01](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/4/46/IeV01.png)
16. Saindo de [Veins](../Lugares%20em%20Ragnar%C3%B6k/Veins.md "Veins"), clique no Barco (ve\_fild07 128,130)

    :   ![MdI02](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/e4/MdI02.png)
17. Larjes, o Agente Secreto, aparecerá para viajar com você para a [Ilha Esquecida](../Lugares%20em%20Ragnar%C3%B6k/Ilha%20Esquecida.md "Ilha Esquecida").

    :   ![Lerjes](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/36/Lerjes.png)
18. Na ilha, investigue o Corvo Morto (nameless\_i 125,208)

    :   ![MdI03](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/19/MdI03.png)
19. Você acordará na casa de Larjes. Converse com o [NPC](../Sem%20categoria/NPC.md "NPC") até surgir um [Massacre](http://www.divine-pride.net/database/monster/1864/). Derrote-o para prosseguir.

    :   ![MdI04](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/6/69/MdI04.png)

    - Caso você morra na casa, basta voltar ao mapa da [Ilha Esquecida](../Lugares%20em%20Ragnar%C3%B6k/Ilha%20Esquecida.md "Ilha Esquecida") e entrar na casa de Larjes.
    - Você não perde [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência") se morrer dentro da casa.
    - Cuidado ao sair casa de Lajes, o mapa estará cheio de [monstros](../Informa%C3%A7%C3%B5es%20b%C3%A1sicas/Monstros.md "Monstros") agressivos.
20. Dirija-se ao monastério no centro da ilha (nameless\_n 157,184)

    :   ![MdI05a](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/9/93/MdI05a.png)
21. No segundo andar do Monastério, receba 1 [Bloco de Pesquisa](http://www.divine-pride.net/database/item/7755/) ao examinar o Livro (abbey02 223,71)

    :   ![MdI05](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/ee/MdI05.png)
22. No terceiro andar do Monastério, toque no Homem (abbey03 232,231)

    :   ![MdI06](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/27/MdI06.png)

    - Derrote o [Tristam III](http://www.divine-pride.net/database/monster/1875/) e toque no [NPC](../Sem%20categoria/NPC.md "NPC") novamente.
    - Você receberá uma [Recordação do Rei](http://www.divine-pride.net/database/item/7726/).
23. Saia do Monastério e entregue a [Recordação do Rei](http://www.divine-pride.net/database/item/7726/) para Larjes na ilha (nameless\_n 256,218)

    :   ![MdI07](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/c/c1/MdI07.png)

    - Você receberá 100.000 de [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência").
24. Reporte suas descobertas ao Padre Bamph em [Prontera](../Lugares%20em%20Ragnar%C3%B6k/Prontera.md "Prontera") (prt\_church 185,106)
25. Volte a falar com a Sumo Sacerdotisa Niren em [Rachel](../Lugares%20em%20Ragnar%C3%B6k/Rachel.md "Rachel") (ra\_temple 165,56)
    - Pergunte sobre os contrabandistas para receber 50.000 de [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência").
