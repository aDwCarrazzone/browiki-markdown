| Requisitos mínimos | |
| **Nv. de base** | 70 |
| **Tipo** | Episódio |
| **[Itens](../Itens/Itens.md "Itens")** | 909.png 300 [Jellopy](http://www.divine-pride.net/database/item/909/)  721.png 1 [Esmeralda](http://www.divine-pride.net/database/item/721/)  723.png 1 [Rubi](http://www.divine-pride.net/database/item/723/) |
| **Caça** | 50 [Soldados da Aliança](http://www.divine-pride.net/database/monster/1313/) |
| Premiação | |
| **[EXP base](../Sem%20categoria/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 132.000 |
| **[EXP classe](../Sem%20categoria/Experi%C3%AAncia%20de%20Classe.md "Experiência de Classe")** | 42.000 |
| **Recompensa** | Acesso a [Ash Vacuum](../Lugares%20em%20Ragnar%C3%B6k/Ash%20Vacuum.md "Ash Vacuum") |

1. Ofereça ajuda ao Recrutador (prt\_cas 207,166)

   :   ![OD1 n](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/34/OD1_n.png)
2. Entregue 300 [Jellopys](http://www.divine-pride.net/database/item/909/) para o Membro da Equipe de Promoções (aldebaran 127,138)

   :   ![OD2](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/e/eb/OD2.png)
3. Entregue 1 [Esmeralda](http://www.divine-pride.net/database/item/721/) e 1 [Rubi](http://www.divine-pride.net/database/item/723/) para o Membro da Equipe de Promoções (geffen 90,67).  
    Ambas podem ser compradas com as Joalheiras em Morroc (moc\_ruins 110,105) ou Lighthalzen (lhz\_in02 105,21)

   :   ![OD3](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/fa/OD3.png)
4. Fale com o Membro da Equipe de Promoções (izlude 98,136)

   :   ![OD3a nw](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/f/f7/OD3a_nw.png)
5. Retorne ao Recrutador em [Prontera](../Lugares%20em%20Ragnar%C3%B6k/Prontera.md "Prontera") e ganhe um pouco de [experiência](../Sem%20categoria/Experi%C3%AAncia.md "Experiência"). (prt\_cas 207,166)

   :   ![OD1 n](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/34/OD1_n.png)
6. Entre na sala de reuniões atravessando os Guardas (lhz\_in01 124,234)

   :   ![OD9](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/55/OD9.png)
7. Fale mais de uma vez com Sikaiz (lhz\_in01 132,258)

   :   ![OD10](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/54/OD10.png)

   - Selecione:
     - *Ouvir.*
     - *A Aliança.*
8. Converse com o Coord. da Aliança (prt\_cas 19,98)

   :   ![OD4 n](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/4/47/OD4_n.png)
9. Fale com Sikaiz até ele pedir que você vá para [Rachel](../Lugares%20em%20Ragnar%C3%B6k/Rachel.md "Rachel"). (lhz\_in01 132,258)

   :   ![OD10](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/54/OD10.png)
10. Localize a Coord. da Aliança (ra\_temple 119,112)

    :   ![OD6](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/5/5a/OD6.png)
11. Retorne a Sikaiz e digite o nome do seu personagem. (lhz\_in01 132,258)
12. Converse mais de uma vez com um dos Oficiais (lhz\_in01 32,22)

    :   ![OD7](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/7/75/OD7.png)
13. Tente entrar novamente na sala de reuniões para ser impedido pelos guardas. (lhz\_in01 124,234)
14. Siga até o final do corredor à direita do salão (lhz\_in01 148,259) e fale com Munkenro para ouvir o último dialogo entre ele e Sikaiz.
15. Teleporte para a [Ruptura Dimensional](http://www.divine-pride.net/database/map/moc_fild20/) através do Guarda (lhz\_in01 130,231)

    :   ![OD8](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/1/1c/OD8.png)
16. Teleporte para a [Fenda Dimensional](http://www.divine-pride.net/database/map/moc_fild22b/) através do Guarda da Fenda (moc\_fild20 348,179)

    :   ![OD11](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/3/3d/OD11.png)
17. Aceite o desafio de Munkenro (moc\_fild22b 230,197)

    :   ![OD12](https://cdn.jsdelivr.net/gh/aDwCarrazzone/browiki-images@main/2/2e/OD12.png)

    - Sua missão é derrotar de 30 a 50 [Soldados da Aliança](http://www.divine-pride.net/database/monster/1313/) espalhados no mapa.
    - Você terá 15 minutos para completar esta tarefa.
    - Se houver mais de um personagem no mapa, você deve esperar o teste terminar dele para iniciar o seu.
18. Após derrotar os monstros, retorne a Munkenro para ter acesso a [Ash Vacuum](../Lugares%20em%20Ragnar%C3%B6k/Ash%20Vacuum.md "Ash Vacuum").
