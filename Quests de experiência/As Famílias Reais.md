## Os Richard

| DQ ico As Famílias Reais | |
| --- | --- |
| Richard | |
| Requisitos mínimos | |
| **Nv. de base** | 140 |
| **Tipo** | Repetível |
| **Caça** | 50 Monstros |
| **Quests anteriores** | [Banquete Real](Banquete%20Real.md "Banquete Real") |
| Skilldelayinfo **Espera:** | 4 da manhã |
| Premiação | |
| **[EXP base](../Sem%20categoria/Experi%C3%AAncia%20de%20Base.md "Experiência de Base")** | 500.000 |
| **[EXP classe](../Sem%20categoria/Experi%C3%AAncia%20de%20Classe.md "Experiência de Classe")** | 550.000 |
| **[Itens](../Itens/Itens.md "Itens")** | 6919.png 4 [Marca de Honra](http://www.divine-pride.net/database/item/6919/) |

1. No quarto dos Richards, fale com o Mordomo (prt\_cas\_q 80,80)

   :   ![Mordomo Richard](../images/e/e4/Mordomo_Richard.png)

   - O [NPC](../Sem%20categoria/NPC.md "NPC") disponibiliza uma lista para matar 50 monstros de uma determinada raça.
   - A contagem de monstros mortos é feita por raça, ou seja, não importa *qual o monstro* que você caçar, mas a sua *raça*.
   - Será contado apenas os monstros da raça selecionada e que tenham nível 140 ou maior.
   - Portanto tome cuidado, pois não existem monstros da raça dragão com esse nível.
   - As raças disponíveis são:
     - [Humanoide](../Sem%20categoria/Humanoide.md "Humanoide"), [Bruto](../Sem%20categoria/Bruto.md "Bruto"), [Anjo](../Sem%20categoria/Anjo.md "Anjo"), [Inseto](../Sem%20categoria/Inseto.md "Inseto"), [Morto-Vivo](../Sem%20categoria/Morto-Vivo.md "Morto-Vivo"), [Demônio](../Sem%20categoria/Dem%C3%B4nio.md "Demônio"), [Dragão](../Sem%20categoria/Drag%C3%A3o.md "Dragão"), [Peixe](../Sem%20categoria/Peixe.md "Peixe"), [Planta](../Sem%20categoria/Planta.md "Planta") ou [Amorfo](../Sem%20categoria/Amorfo.md "Amorfo").
2. Retorne ao Mordomo após a caçada para receber a sua recompensa.
   - Você pode usar as Marcas de Honra para [trocar por itens](../Lugares%20em%20Ragnar%C3%B6k/Prontera.md#Marcas_de_Honra "Prontera")
