# Cripta

| SC ico Cripta | |
| --- | --- |
| Instância criada para você evoluir com seus amigos em um mapa reservado só para você e seu grupo por uma horinha! Sem interferência de outros jogadores. | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Instância |
| **Duração** | 1 hora |
| Skilldelayinfo **Retorno** | Sábado e Domingo |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| Premiação | |
| **Chefe** | Nenhum |
| **[Itens](Itens.md "Itens")** | 23142.png [Poção Menor de EXP](http://www.divine-pride.net/database/item/23142/) 23143.png [Poção Menor de Classe](http://www.divine-pride.net/database/item/23143/) |

1. Fale com Mariaju (pay\_arche 44,124) para gerar a [instância](Instância.md "Instância")

   :   ![Cripta01](images/9/94/Cripta01.png)
2. Dentro da Cripta, fale novamente com a [NPC](NPC.md "NPC") e escolha o nível de dificuldade entre 60 a 160.
3. O mapa invocará [monstros](Monstros.md "Monstros") para você caçar de acordo com o nível de dificuldade escolhido, independente do seu nível de base.
4. Após um certo número de caçadas, aparece um Baú de Tesouro que derruba *Poções Menores*.

   :   ![Cripta021](images/8/83/Cripta021.png)

   1. (1@md\_pay 99,172)
   2. (1@md\_pay 235,60)
   3. (1@md\_pay 53,267)
   4. (1@md\_pay 238,252)
5. O único propósito da [instância](Instância.md "Instância") é ser um mapa de evolução sem interferência de outros jogadores, após 1 hora, ela termina.
6. As Poções Menores podem ser convertidas em [Embalagem de Poções](http://www.divine-pride.net/database/item/23144/) negociáveis falando com a Colecionadora (pay\_arche 44,121).
