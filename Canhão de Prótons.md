# Canhão de Prótons

|  |  |
| --- | --- |
| **Canhão de Prótons Canhão de Prótons** | |
| Canhão de Prótons | |
| **Cart Cannon / Cañón de Carreta** | |
| **ID** | GN\_CARTCANNON (2477) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 40 a 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + [0,5 + (Nv. da hab. × 0,5)] seg. |
| **Pós-conjuração** | 0,5 segundos |
| **Alvo** | Oponente |
| **Alcance** | (Nv. da habilidade + 7) células |
| **Área** | 3x3 a 7x7 células |
| **Munição** | [Over Mado](Munições.md "Munições") 1 Bala de Canhão |
| **Requisito** | [Carrinho](Usar Carrinho.md "Usar Carrinho") |
| **Pré-requisitos** | |
| - [Aprimorar Carrinho](Aprimorar Carrinho.md "Aprimorar Carrinho") [Aprimorar Carrinho](Aprimorar Carrinho.md "Aprimorar Carrinho") Nv. 2 | |

Dispara uma bala de um canhão equipado no carrinho.

Causa dano físico a distância que ignora [DEF](DEF.md "DEF") do alvo e dos oponentes próximos.

Se [Cultivar Bárbaro](Cultivar Bárbaro.md "Cultivar Bárbaro") estiver ativo, adiciona os seguintes efeitos:

- Reduz a animação do ataque.
- Passa a infligir 2 ataques, ao invés de 1.

| Nv. | [ATQ](ATQ.md "ATQ") | Alcance | [Conjuração variável](Conjuração variável.md "Conjuração variável") | Área | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- | --- |
| 1 | 250% | 8 células | 1,0 segundo | 3x3 células | 40 |
| 2 | 500% | 9 células | 1,5 segundos | 42 |
| 3 | 750% | 10 células | 2,0 segundos | 5x5 células | 46 |
| 4 | 1.000% | 11 células | 2,5 segundos | 48 |
| 5 | 1.250% | 12 células | 3,0 segundos | 7x7 células | 50 |

### Fórmula

`Dano = {[Dano base + (Nv. de Aprimorar Carrinho × 20)] + (INT x 2)} x (Nv. de base ÷ 100)%`

## Notas

- Não é afetado por [Investigar](Investigar.md "Investigar").
- A [propriedade](Propriedade.md "Propriedade") do dano é de acordo com a bala de canhão equipada e não pode ser alterada.
- A habilidade possui uma animação que dura 0,6 segundos, tornando difícil usá-la rapidamente e repetidas vezes mesmo com toda a conjuração variável reduzida.
