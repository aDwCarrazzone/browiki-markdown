# Fortalecer Guardiões

|  |  |
| --- | --- |
| **Fortalecer Guardiões Fortalecer Guardiões** | |
| SemImagem | |
| **Strengthen Guardians** | |
| **ID** | GD\_GUARDUP (10003) |
| **Tipo** | Passiva |
| **Níveis** | 3 |
| **Pré-requisitos** | |
| Nenhum | |

Aumenta o [ATQ](ATQ.md "ATQ") e a [velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") dos guardiões do seu castelo.

| Nv. | [Velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") | [ATQ](ATQ.md "ATQ") |
| --- | --- | --- |
| 1 | +5% | +150% |
| 2 | +10% | +200% |
| 3 | +15% | +250% |

## Notas

- Não funciona na segunda edição da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
