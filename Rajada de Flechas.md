# Rajada de Flechas

|  |  |
| --- | --- |

| **Double Strafe / Disparo Doble** | |
| **ID** | AC\_DOUBLE (46) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 12 |
| **Pós-conjuração** | 0,1 segundo |
| **Alvo** | Oponente |
| **Alcance** | (9 + Nv. de [Olhos de Águia](Olhos de Águia.md "Olhos de Águia")) células |
| **Munição** | [Over Arrow](Munições.md "Munições") 1 Flecha |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - Arqueiros Nenhum - Arruaceiros [Olhos de Águia](Olhos de Águia.md "Olhos de Águia") [Olhos de Águia](Olhos de Águia.md "Olhos de Águia") Nv. 10 | |

Atira duas flechas em um único ataque.

Causa dano físico a distância e consumindo apenas 1 flecha.
