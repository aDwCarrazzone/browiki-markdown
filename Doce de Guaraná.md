# Doce de Guaraná

| NQ ico Doce de Guaraná | |
| --- | --- |
| Uma estranha fruta com formato de olho pode ser o ingrediente necessário para uma Doceira fazer uma das melhores guloseimas da cidade! | |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Tipo** | Normal |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 70.000 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 10.000 |
| **[Itens](Itens.md "Itens")** | 12414.png 1 [Bala de Guaraná](http://www.divine-pride.net/database/item/12414/) |

1. Fale com a Doceira (brasilis 187,162)

   :   ![Doce de Guaraná01](images/4/42/Doce_de_Guaraná01.png)

   - Selecione:
     - "*Gostaria de Provar este Doce*"
     - "*Como encontrar o Guaraná?*"
2. Pergunte sobre o guaraná ao Paulão (bra\_in01 95,179)

   :   ![Doce de Guaraná02](images/1/1e/Doce_de_Guaraná02.png)
3. Converse com o Garoto Estranho (brasilis 205,65)

   :   ![Doce de Guaraná03](images/4/41/Doce_de_Guaraná03.png)

   - Selecione:
     - *Tentar conversar*
   - Após a escolha, selecione todas as opções.
     - *Parar de Falar*.
4. Retorne ao Paulão no Museu.
5. Procure pelo Pajé Ubirajá (brasilis 57,224)

   :   ![Doce de Guaraná04](images/9/99/Doce_de_Guaraná04.png)
6. Aproxime-se do Tucano (bra\_fild01 75,83)

   :   ![Doce de Guaraná05](images/8/89/Doce_de_Guaraná05.png)
7. Caminhe ao redor da Onça (bra\_fild01 34,184)

   :   ![Doce de Guaraná06](images/4/48/Doce_de_Guaraná06.png)
8. Fale com o Macaco (bra\_fild01 245,54)

   :   ![Doce de Guaraná07](images/5/57/Doce_de_Guaraná07.png)
9. Reencontre o Garoto Estranho (brasilis 205,65) para ganhar um [Guaraná](http://www.divine-pride.net/database/item/6237/).
10. Fale com a Doceira (brasilis 187,162) que o agradecerá com uma [Bala de Guaraná](http://www.divine-pride.net/database/item/12414/).
    - Agora você pode comprar 1 [Bala de Guaraná](http://www.divine-pride.net/database/item/12414/) por 4.000 [zenys](Zenys.md "Zenys") cada.
