# Estocada Precisa

|  |  |
| --- | --- |
| **Estocada Precisa Estocada Precisa** | |
| Estocada Precisa | |
| **Pinpoint Attack / Ataque Certero** | |
| **ID** | LG\_PINPOINTATTACK (2312) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 50 |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 5 segundos |
| **Alvo** | Oponente |
| **Alcance** | 5 células |
| **Efeito negativo** | [Sangramento](Sangramento.md "Sangramento") |
| **Arma** | Lança |
| **Pré-requisitos** | |
| - [Toque do Oblívio](Toque do Oblívio.md "Toque do Oblívio") [Toque do Oblívio](Toque do Oblívio.md "Toque do Oblívio") Nv. 5 | |

Pula instantaneamente no alvo, causando dano físico a distância.

O alvo sofrerá efeitos diferentes de acordo com o nível da habilidade.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

| Nv. | [ATQ](ATQ.md "ATQ") | Efeito | Chance |
| --- | --- | --- | --- |
| 1 | 100% | [Sangramento](Sangramento.md "Sangramento") | 35% |
| 2 | 200% | Elmo destruído | 40% |
| 3 | 300% | Escudo destruído | 45% |
| 4 | 400% | Armadura destruída | 50% |
| 5 | 500% | Arma destruída | 55% |

### Fórmula

`ATQ = {[(Nv. da habilidade × 100) + (AGI × 5)] × (Nv. de base ÷ 120)}%`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas").**
- Não pula em castelos da [Guerra do Emperium](Guerra do Emperium.md "Guerra do Emperium").
- Pode ser bloqueada por [Escudo Sagrado](Escudo Sagrado.md "Escudo Sagrado").
