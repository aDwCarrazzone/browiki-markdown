# Laboratório Werner

| SC ico Laboratório Werner | |
| --- | --- |
| Conheça a história de Werner e como ele continuou com os experimentos de Wolfchev para a replicação de cobaias e criação dos Homúnculos. | |
| Requisitos mínimos | |

| **Tipo** | Instância |

| Skilldelayinfo **Retorno** | Reseta 4 horas da manhã |
| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |
| **Quests anteriores** | [Terra Gloria](Terra Gloria.md "Terra Gloria") |
| Premiação | |
| **Chefe** | Monster Atroce [Fofinho](http://www.divine-pride.net/database/monster/3621/) Monster LordeSeyren [Cobaia YSF01](http://www.divine-pride.net/database/monster/3826/) |

1. Nos arredores de [Einbroch](Einbroch.md "Einbroch"), para entrar na [Base Militar](http://www.divine-pride.net/database/map/que_swat/) fale com Lukie (ein\_fild04 284,332)

   :   ![LukieLucky1](images/9/9a/LukieLucky1.png)
2. Na Base Militar, Lukie (que\_swat 150,58) tem algumas missões diárias para você completar. Você só terá acesso à instância se tiver uma missão de Lukie para fazer.

   :   ![LukiSwat](images/3/3a/LukiSwat.png)
3. Entre no [Laboratório Clandestino](http://www.divine-pride.net/database/map/slabw01/) e reserve a [instância](Instância.md "Instância").

   :   ![LuckyWerner](images/6/6b/LuckyWerner.png)
4. Assim que a entrada for liberada, clique no portal que aparece no centro do círculo.
5. Dentro da instância, siga ao norte, derrotando [Caça-Corações](http://www.divine-pride.net/database/monster/3622/) no caminho.

   :   ![CaçaCorações](images/9/94/CaçaCorações.png)
6. Ir para as próximas áreas requer ativar os Aparatos de Segurança.

   :   ![Aparato de Segurança](images/5/57/Aparato_de_Segurança.png)

   - Ao clicar nos Aparatos, ele exibe um nome que foi dado para uma das espécimes do Eisen Werner.
   - Você deve selecionar a cor correta das duas opções que aparecem.
   - Se acertar, o Aparato é ativado, ativando os 2 Aparatos, um portal se abre (ponto claro na imagem acima).
   - Se errar, os dois aparatos reiniciarão e você deverá começar tudo de novo.
7. Para selecionar a cor correta, atente-se ao nome indicado:
   - Aurora = Azul
   - Sonho = Azul
   - Eternidade = Azul
   - Pureza = Azul
   - Contradição = Vermelho
   - Brilho = Vermelho
   - Alegria = Vermelho
   - Solidão = Vermelho
   - Rosa = Vermelho
   - Mar = Vermelho
   - Crepúsculo = Vermelho
   - Retorno = Vermelho
   - Observação:
     - Jogadores daltônicos, a primeira opção é sempre de cor vermelha e a segunda é azul.
8. Assim que passar por todas as áreas, dirija-se ao portal que liga ao salão central.
   - Duas opções de escolha serão dadas para você:
     - Modo História (Onde você batalha contra a [Cobaia YSF01](http://www.divine-pride.net/database/monster/3826/)).
     - Modo Batalha (Onde você enfrenta [Fofinho](http://www.divine-pride.net/database/monster/3621/)).
9. Selecionando qualquer opção, aproxime-se do centro do mapa para iniciar a luta.

   :   ![Fofinho](images/6/64/Fofinho.png)
10. Ao derrotar o monstro, saia pelo portal ao sul e fale com Luki para finalizar a instância com sucesso (ein\_fild04 284,332).
