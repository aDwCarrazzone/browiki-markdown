# Revidar Dano

|  |  |
| --- | --- |
| **Revidar Dano Revidar Dano** | |

| **Death Bound / Reflejar Daño** | |
| **ID** | RK\_DEATHBOUND (2003) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 25 + (Nv. da habilidade ×3) |
| **Pós-conjuração** | 2 segundos |
| **Recarga** | 3 segundos |
| **Duração** | 3 segundos |
| **Alvo** | Usuário |
| **Empurra** | 3 células |
| **Pré-requisitos** | |
| - [Contra-Ataque](Contra-Ataque.md "Contra-Ataque") [Contra-Ataque](Contra-Ataque.md "Contra-Ataque") Nv. 1 - [Encantar Lâmina](Encantar Lâmina.md "Encantar Lâmina") [Encantar Lâmina](Encantar Lâmina.md "Encantar Lâmina") Nv. 2 | |

Prepara-se para um ataque.

Quando receber um dano físico corpo a corpo, reflete 70% do dano de volta ao atacante.

O atacante será empurrado para trás.

Aumentar o nível da habilidade aumenta o dano refletido.

| Nv. | Dano refletido | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 600% | 28 |
| 2 | 700% | 31 |
| 3 | 800% | 34 |
| 4 | 900% | 37 |
| 5 | 1.000% | 40 |
| 6 | 1.100% | 43 |
| 7 | 1.200% | 46 |
| 8 | 1.300% | 49 |
| 9 | 1.400% | 52 |
| 10 | 1.500% | 55 |

## Notas

- Não funciona com [monstros](Monstros.md "Monstros") do tipo chefe.
- Ao mirar outra [habilidade](Habilidade.md "Habilidade") no alvo durante o tempo de recarga de Revidar Dano, fará com que a [habilidade](Habilidade.md "Habilidade") usada seja autoconjurada assim que o usuário receber dano.
