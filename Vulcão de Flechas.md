# Vulcão de Flechas

|  |  |
| --- | --- |
| **Vulcão de Flechas Vulcão de Flechas** | |
| Vulcão de Flechas | |
| **Arrow Vulcan / Volcán Flecha** | |
| **ID** | CG\_ARROWVULCAN (394) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 10 + (Nv. da habilidade × 2) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,5 + 1,5 seg. |
| **Pós-conjuração** | 0,5 segundos |
| **Recarga** | 1,5 segundos |
| **Alvo** | Oponente |
| **Alcance** | (9 + Nv. de [Olhos de Águia](Olhos de Águia.md "Olhos de Águia")) células |
| **Munição** | [Over Arrow](Munições.md "Munições") 1 Flecha |
| **Arma** | Chicote ou Instrumento musical |
| **Pré-requisitos** | |
| - Menestréis [Flecha Melódica](Flecha Melódica.md "Flecha Melódica") [Flecha Melódica](Flecha Melódica.md "Flecha Melódica") Nv. 1 [Chuva de Flechas](Chuva de Flechas.md "Chuva de Flechas") [Chuva de Flechas](Chuva de Flechas.md "Chuva de Flechas") Nv. 5 - Ciganas [Estilingue](Estilingue.md "Estilingue") [Estilingue](Estilingue.md "Estilingue") Nv. 1 [Chuva de Flechas](Chuva de Flechas.md "Chuva de Flechas") [Chuva de Flechas](Chuva de Flechas.md "Chuva de Flechas") Nv. 5 | |

Dispara uma metralhadora de flechas em um único alvo.

Causa dano físico a distância de acordo com a [propriedade](Propriedade.md "Propriedade") da munição equipada.

| Nv. | [ATQ](ATQ.md "ATQ") | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 600% | 12 |
| 2 | 700% | 14 |
| 3 | 800% | 16 |
| 4 | 900% | 18 |
| 5 | 1.000% | 20 |
| 6 | 1.100% | 22 |
| 7 | 1.200% | 24 |
| 8 | 1.300% | 26 |
| 9 | 1.400% | 28 |
| 10 | 1.500% | 30 |

### Fórmula

`Dano = {[Dano base] × (Nv. de base ÷ 100)}%`
