# Campo Amaldiçoado

|  |  |
| --- | --- |
| **Campo Amaldiçoado Campo Amaldiçoado** | |
| Campo Amaldiçoado | |
| **Cursed Circle / Círculo Maldito** | |
| **ID** | SR\_CURSEDCIRCLE (2334) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 20) |
| **[HP](HP.md "HP")** | (Nv. da habilidade)% |
| **Pós-conjuração** | 1 segundo |
| **Recarga** | 10 segundos |
| **Duração** | (Nv. da hab. + 2) segundos |
| **Alvo** | Usuário |
| **Área** | 3x3 a 7x7 células |
| **Consome** | 1 Esfera espiritual por alvo |
| **Ícone** | I Punho Supremo de Asura |
| **Pré-requisitos** | |
| - [Dilema](Dilema.md "Dilema") [Dilema](Dilema.md "Dilema") Nv. 2 | |

Energiza um circulo maldito ao seu redor.

Imobiliza o usuário e oponentes, impedindo-os de andar, atacar e usar habilidades.

| Nv. | Área | Duração | [HP](HP.md "HP") | [SP](SP.md "SP") |
| --- | --- | --- | --- | --- |
| 1 | 3x3 células | 3 segundos | 1% | 40 |
| 2 | 4 segundos | 2% | 60 |
| 3 | 5x5 células | 5 segundos | 3% | 80 |
| 4 | 6 segundos | 4% | 100 |
| 5 | 7x7 células | 7 segundos | 5% | 120 |

## Notas

- Se houver mais alvos ao redor do que o número de esferas espirituais disponíveis, alguns não serão afetados.
- O usuário pode usar habilidades, mas isso libertará os alvos imobilizados.
- Não interrompe as conjurações em andamento dos alvos afetados.
- Não funciona contra [monstros](Monstros.md "Monstros") chefes.
