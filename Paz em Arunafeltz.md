# Paz em Arunafeltz

| EP ico Paz em Arunafeltz | |
| --- | --- |
| Um conflito interno no santuário está prestes a eclodir se a Papisa não for notificada a tempo de fazer um pronunciamento à população. | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Episódio |
| **[Itens](Itens.md "Itens")** | 725.png 1 [Peridoto](http://www.divine-pride.net/database/item/725/) |
| **Quests anteriores** | [Base no Vulcão](Base no Vulcão.md "Base no Vulcão") |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 250.000 |

1. Converse mais de uma vez com o Mercador Estrangeiro. (moc\_ruins 77,167)

   :   ![PeA01](images/5/56/PeA01.png)
2. Dirija-se ao oásis no centro do mapa. Para ativar o Mercador Estrangeiro, você precisa ficar caminhando ao redor da água. (moc\_ruins 101,133)

   :   ![PeA02](images/0/0f/PeA02.png)
3. Retorne ao Mercador Estrangeiro do primeiro passo. (moc\_ruins 77,167)
4. Faça o Sumo Sacerdote Zhed contar a verdade. (ra\_temin 277,159)

   :   ![Zhed](images/a/ad/Zhed.png)
5. Localize a Discípula (ra\_temin 103,151)

   :   ![PeA04](images/c/c1/PeA04.png)
6. Pergunte sobre Zhed à Sumo Sacerdotisa Niren (ra\_temple 165,57)

   :   ![Niren](images/9/9a/Niren.png)
7. Relate o ocorrido a Zhed. (ra\_temin 277,159)
8. Volte a Niren. Insista em conversar sobre Zhed até que ela peça para você se encontrar com a Papisa. (ra\_temple 165,57)
9. Vá até a Papisa (ra\_temin 134,131)

   :   ![Papisa](images/f/f7/Papisa.png)
10. Retorne a Niren. (ra\_temple 165,57)
11. Traga um [Peridoto](http://www.divine-pride.net/database/item/725/) a Ishmael (ra\_fild03 139,355)

    :   ![Paz em Arunafeltz000](images/9/9f/Paz_em_Arunafeltz000.png)
12. Reencontre Niren. (ra\_temple 165,57)
13. Fale com Zhed. (ra\_temin 277,159)
14. Ouça a história da Papisa, que te mandará para [Rachel](Rachel.md "Rachel"). (ra\_temin 134,131)
15. Na cidade, todos do mapa ouvirão o discurso da Papisa.
16. Fale com Niren para receber [experiência](Experiência.md "Experiência"). (ra\_temple 165,57)
17. Converse com Zhed pela última vez. (ra\_temin 277,159)
