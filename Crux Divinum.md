# Crux Divinum

|  |  |
| --- | --- |

| **ID** | CR\_HOLYCROSS (253) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | (Nv. da habilidade + 10) |
| **Alvo** | Oponente |
| **Alcance** | 3 células |
| **Propriedade** | [Sagrado](Monstros.md#Propriedade "Monstros") |
| **Efeito negativo** | [Cegueira](Cegueira.md "Cegueira") |

| - [Fé](Fé.md "Fé") [Fé](Fé.md "Fé") Nv. 7 | |

Corta o alvo com uma luz cruzada.

Causa dano físico de propriedade [sagrado](Sagrado.md "Sagrado") com chance de infligir [cegueira](Cegueira.md "Cegueira") no oponente.

O dano é dobrado ao equipar uma lança de duas mãos.
