# Dor Máxima

|  |  |
| --- | --- |
| **NPC Skill Icon Dor Máxima** | |
| Dor Máxima | |
| **ID:** | NPC\_MAXPAIN (716) |
| **Níveis:** | 10 |
| **Exclusiva:** | Sim |
| **Duração:** | 10 segundos |

Reveste o corpo com uma cera laranja.

Reflete todo o tipo de dano recebido para a área de 29x29 células.

O dano refletido causa dano físico nos oponentes próximos.

| Nv. | Dano refletido |
| --- | --- |
| 1 | 10% |
| 2 | 20% |
| 3 | 30% |
| 4 | 40% |
| 5 | 50% |
| 6 | 60% |
| 7 | 70% |
| 8 | 80% |
| 9 | 90% |
| 10 | 100% |

## Notas

- *Confira a lista de [monstros](Monstros.md "Monstros") que usam [Dor Máxima](http://www.divine-pride.net/database/skill/716)*
- Reflete danos mágicos e físicos, sejam eles à distância ou corpo a corpo.
- Independente do tipo de dano que você causa, o dano refletido pelo monstro será sempre físico, sendo portanto, bloqueado por [Escudo Mágico](Escudo Mágico.md "Escudo Mágico"), [Kyrie Eleison](Kyrie Eleison.md "Kyrie Eleison") e [Troca de Pele](Troca de Pele.md "Troca de Pele").
- Pode ser cancelado por [Garra Sombria](Garra Sombria.md "Garra Sombria").
