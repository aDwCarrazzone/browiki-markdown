# Invocar Esfera de Água

|  |  |
| --- | --- |
| **Invocar Esfera de Água Invocar Esfera de Água** | |
| Invocar Esfera de Água | |
| **Summon Water Ball / Invocar Esfera de Agua** | |
| **ID** | WL\_SUMMONWB (2224) |
| **Tipo** | Ativa |
| **Níveis** | 2 |
| **[SP](SP.md "SP")** | 10 a 50 |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 2 seg. |
| **Duração** | 280 segundos |
| **Alvo** | Usuário |
| **Pré-requisitos** | |
| - [Nevasca](Nevasca.md "Nevasca") [Nevasca](Nevasca.md "Nevasca") Nv. 1 | |

Invoca até 5 Esferas Elementais que flutuarão ao redor do usuário.

Elas podem causar dano mágico de propriedade [Água](Água.md "Água") se usadas por [Lançar Magia](Lançar Magia.md "Lançar Magia").

Em [Tetra Vortex](Tetra Vortex.md "Tetra Vortex"), elas podem infligir [Hipotermia](Hipotermia.md "Hipotermia").

| Nv. | Efeito | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | Invoca 1 Esfera | 10 |
| 2 | Invoca 5 Esferas | 50 |

## Notas

- Cada esfera ativa drena 1 de [SP](SP.md "SP") por segundo.
- Cada esfera possui um tempo de duração individual.
- Se usar a habilidade em nível 2 e existirem Esferas de outro elemento invocadas, elas serão substituídas.
- Invocar 1 esfera é útil para usar [Tetra Vortex](Tetra Vortex.md "Tetra Vortex") com os 4 elementos.
