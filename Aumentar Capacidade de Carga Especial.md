# Aumentar Capacidade de Carga Especial

|  |  |
| --- | --- |
| **Aumentar Capacidade de Carga Especial Aumentar Capacidade de Carga Especial** | |
| SemImagem | |

| **ID** | ALL\_INCCARRY (681) |
| **Tipo** | Passiva |
| **Níveis** | 10 |
| **Pré-requisitos** | |
| Nenhum | |

Aumenta o limite de [peso](Peso.md "Peso") máximo que pode ser carregado.

| Nv. | Peso |
| --- | --- |
| 1 | +200 |
| 2 | +400 |
| 3 | +600 |
| 4 | +800 |
| 5 | +1.000 |
| 6 | +1.200 |
| 7 | +1.400 |
| 8 | +1.600 |
| 9 | +1.800 |
| 10 | +2.000 |

## Nota

- Acumula com [Aumentar Capacidade de Carga](Aumentar Capacidade de Carga.md "Aumentar Capacidade de Carga").
- Para adquirir a habilidade, leve o [Passe Antigravitacional](http://www.divine-pride.net/database/item/7776/) ao Mister Peso dentro do [Shopping](JoyCoins.md "JoyCoins") (itemmall 35,75).
- Cada passe apresentado ao [NPC](NPC.md "NPC") aumenta 1 nível da habilidade, necessitando de 10 [Passes Antigravitacionais](http://www.divine-pride.net/database/item/7776/) para ter nível 10.
- Se você resetar suas [habilidades](Habilidades.md "Habilidades") e perder a habilidade, retorne ao *Mister Peso* e selecione a segunda opção para restituir sua habilidade.
