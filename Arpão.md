# Arpão

|  |  |
| --- | --- |
| **Arpão Arpão** | |
| Arpão | |
| **Phantom Thrust / Propulsión Espectral** | |
| **ID** | RK\_PHANTOMTHRUST (2020) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 12 + (3 × Nv. da habilidade) |
| **Alvo** | Oponentes |
| **Alcance** | (Nv. da hab. + 4) células |
| **Arma** | Lança |
| **Pré-requisitos** | |
| - [Brandir Lança](Brandir Lança.md "Brandir Lança") [Brandir Lança](Brandir Lança.md "Brandir Lança") Nv. 2 | |

Usa a lança com um gancho.

Causa dano físico a distância e puxa alvos para perto do usuário.

| Nv. | Dano base | Alcance | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 50% | 5 células | 15 |
| 2 | 100% | 6 células | 18 |
| 3 | 150% | 7 células | 21 |
| 4 | 200% | 8 células | 24 |
| 5 | 250% | 9 células | 27 |

### Fórmula

`ATQ = {[(Dano base + Nv. de Perícia com Lança × 10) ÷ Nv. de base] ÷ 150}%`

## Notas

- Pode ser usada em membros do [grupo](Grupo.md "Grupo") sem causar dano.
- Não funciona contra [monstros](Monstros.md "Monstros") Chefes e em personagens inimigos no [PvP](PvP.md "PvP").
