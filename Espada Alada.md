# Espada Alada

|  |  |
| --- | --- |
| **Espada Alada Espada Alada** | |
| Espada Alada | |
| **Servant Weapon / Espada Alada** | |
| **ID** | DK\_SERVANTWEAPON (5201) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 + (Nv. da habilidade × 10) |
| **Regen. de [AP](AP.md "AP")** | **(Nv. da habilidade × 6)** |
| **Recarga** | (Nv. da hab. × 30) segundos |
| **Duração** | (Nv. da hab. × 30) segundos |
| **Alvo** | Usuário |
| **Área** | 9x9 células |
| **Ícone** | I Espada Alada |
| **Pré-requisitos** | |
| - Nenhum | |

Invoca 5 Espadas Aladas para o combate.

Ao realizar ataques básicos, tem chance de consumir 1 Espada.

Ao consumir a espada, atinge todos na área de 9x9 células ao redor.

Causa dano físico corpo a corpo.

Enquanto estiver ativa, uma nova Espada pode nascer em um intervalo de tempo.

Tem chance do ataque ser crítico.

A chance é igual ao [CRIT](CRIT.md "CRIT") do usuário.

### Fórmula

`Dano = [(Dano base + (5 x POD)] x (Nv. de base ÷ 100)}%`

## Notas

- Esta é uma **[Habilidade Crítica](Categoria-Habilidades críticas.md "Categoria:Habilidades críticas")**.
- Não é possível ativar uma Espada Alada com ataques por [habilidades](Habilidades.md "Habilidades").
- É cancelada por [Gênese](Gênese.md "Gênese") ou [Desencantar](Desencantar.md "Desencantar").
- A regeneração de [AP](AP.md "AP") acontece no momento do uso da [habilidade](Habilidade.md "Habilidade"), não do ataque das Espadas.
