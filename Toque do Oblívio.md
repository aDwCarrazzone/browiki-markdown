# Toque do Oblívio

|  |  |
| --- | --- |
| **Toque do Oblívio Toque do Oblívio** | |
| Toque do Oblívio | |
| **Vanishing Point / Punto de Fuga** | |
| **ID** | LG\_BANISHINGPOINT (2308) |
| **Tipo** | Ofensiva |
| **Níveis** | 10 |
| **[SP](SP.md "SP")** | 18 + (Nv. da habilidade × 2) |
| **Alvo** | Oponente |
| **Alcance** | 7 células |
| **Arma** | Lança |
| **Pré-requisitos** | |
| - [Perícia com Lança](Perícia com Lança.md "Perícia com Lança") [Perícia com Lança](Perícia com Lança.md "Perícia com Lança") Nv. 1 | |

Toca a lança no alvo, causando dano físico a distância.

O dano é afetado pelo nível de [Golpe Fulminante](Golpe Fulminante.md "Golpe Fulminante").

Aumentar o nível da habilidade melhora a [precisão](Precisão.md "Precisão") para acertar o ataque.

Se [Cicatriz da Justiça](Lança da Justiça.md "Lança da Justiça") estiver ativo, o dano é maior.

| --- | --- | --- | --- | --- |
| Normal | [Cicatriz](Lança da Justiça.md "Lança da Justiça") |
| 1 | 100% | 180% | +5 | 20 |
| 2 | 200% | 360% | +10 | 22 |
| 3 | 300% | 540% | +15 | 24 |
| 4 | 400% | 720% | +20 | 26 |
| 5 | 500% | 900% | +25 | 28 |
| 6 | 600% | 100% | +30 | 30 |
| 7 | 700% | 1.080% | +35 | 32 |
| 8 | 800% | 1.260% | +40 | 34 |
| 9 | 900% | 1.440% | +45 | 36 |
| 10 | 1.000% | 1.620% | +50 | 38 |

### Fórmula

`ATQ = {[Dano base + (Nv. de Golpe Fulminante × 70)] × (Nv. de base ÷ 100)}%`
