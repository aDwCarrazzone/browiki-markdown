# Segredo da Família Real

| EP ico Segredo da Família Real | |
| --- | --- |
| Uma misteriosa maldição assombra os membros da família real de Rune-Midgard. Caberá a você investigar se a história é verdadeira ou se tudo não passa de um antigo poema. | |
| Requisitos mínimos | |
| **Nv. de base** | 60 |
| **Tipo** | Episódio |
| **[Itens](Itens.md "Itens")** | 506.png 1 [Poção Verde](http://www.divine-pride.net/database/item/506/) 715.png 1 [Gema Amarela](http://www.divine-pride.net/database/item/715/) |
| **[Zeny](Zeny.md "Zeny")** | 1.000 |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 90.000 |

1. Aproxime-se do Garoto Ocupado (prontera 248,212)

   :   ![Gaebolgs01](images/e/e8/Gaebolgs01.png)

   - Aceite ajudá-lo para receber uma [Pilha de Livros](http://www.divine-pride.net/database/item/7431/)
   - Se você perder a [Pilha de Livros](http://www.divine-pride.net/database/item/7431/), pode comprá-la por 700 [zenys](Zenys.md "Zenys") na biblioteca de [Prontera](Prontera.md "Prontera").
2. Leve a [Pilha de Livros](http://www.divine-pride.net/database/item/7431/) para o Historiador (yuno 311,195)

   :   ![Gaebolgs02](images/d/d1/Gaebolgs02.png)

   - Continue conversando com o Historiador e você receberá uma [Pasta de Arquivos](http://www.divine-pride.net/database/item/7342/)
3. Encontre a Historiadora Rodafrian (morocc\_in 45,126)

   :   ![Gaebolgs03](images/3/3b/Gaebolgs03.png)

   - Diga que o poema é diferente.
   - Digite qualquer coisa, não interfere na missão.
4. Diga que você conhece outra versão do poema para o Historiador Mondo (mjolnir\_01 135,168)

   :   ![Gaebolgs07](images/6/68/Gaebolgs07.png)

   - Digite linha por linha:
     - `A grande serpente engoliu o mar.`
     - `A águia do arco-íris engoliu a serpente.`
     - `A águia então construiu seu ninho.`
     - `Um ninho sob o mar encoberto.`
5. Em direção ao Mirante, fale com a Mulher com Aparência ao lado (mjolnir\_01 316,268)

   :   ![Gaebolgs08](images/d/d8/Gaebolgs08.png)

   - Você receberá uma [Bolsa de Couro](http://www.divine-pride.net/database/item/7432/).
6. Entregue a [Bolsa de Couro](http://www.divine-pride.net/database/item/7432/) ao Padre Bamph (prt\_church 185,106)

   :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
7. Reencontre a Mulher com Aparência. (mjolnir\_01 316,268)

   :   ![Gaebolgs08](images/d/d8/Gaebolgs08.png)
8. Volte ao Padre Bamph (prt\_church 185,106) e aceite examinar os cadáveres.

   :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
9. Clique no interruptor (prt\_church 177,113)

   :   ![Gaebolgs05](images/8/89/Gaebolgs05.png)
10. Você estará no necrotério. Fale com Bamph para iniciar os exames.

    :   ![Gaebolgs09](images/d/d4/Gaebolgs09.png)

    - Examine primeiro o corpo da esquerda e fale com o Padre novamente. (prt\_church 185,106)

    :   ![Gaebolgs corpo esquerda](images/c/ca/Gaebolgs_corpo_esquerda.png)

    - Em seguida, examine os outros dois príncipes. Ao ser perguntado, diga *Veneno!*.

    :   ![Gaebolgs corpos outros](images/1/1d/Gaebolgs_corpos_outros.png)
11. Saia da sala através do Padre Biscuss (prt\_church 23,89)

    :   ![Gaebolgs Biscuss](images/c/cc/Gaebolgs_Biscuss.png)
12. Fale novamente com o Padre Bamph, na catedral. (prt\_church 185,106)

    :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
13. Mencione o veneno para o Mercenário da Guilda (moc\_fild16 201,295)

    :   ![Gaebolgs065](images/a/aa/Gaebolgs065.png)
14. Para ter acesso a sala da Marjana, pague 1.000 [zenys](Zenys.md "Zenys") ao Cavaleiro (que\_job01 81,79)

    :   ![Gaebolgs010](images/d/d5/Gaebolgs010.png)
15. No corredor, entre no portal de cima, à esquerda. Você terá 4 minutos para falar com Marjana.

    :   ![Gabelo11](images/1/1b/Gabelo11.png)
16. Volte para a entrada da taverna **caminhando**, um diálogo deverá surgir no momento em que você pisa em [Morroc](Morroc.md "Morroc").

    :   ![Gaebolgs011](images/9/93/Gaebolgs011.png)

    - Selecione a opção "*falar com o Padre Bamph*".
    - Se tiver selecionado "*falar com Rodafrian*", você deverá conversar com a Historiadora, mas recomendamos não seguir essa escolha.
17. Traga 1 [Poção Verde](http://www.divine-pride.net/database/item/506/) e 1 [Gema Amarela](http://www.divine-pride.net/database/item/715/) e dirija-se ao Padre Bamph. (prt\_church 185,106)

    :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
18. Clique no interruptor para seguir ao necrotério.

    :   ![Gaebolgs05](images/8/89/Gaebolgs05.png)
19. Fale com Bamph e em seguida, verifique o terceiro corpo.

    :   ![Gaebolgs corpo direita](images/8/8e/Gaebolgs_corpo_direita.png)
20. Reporte ao Padre (prt\_church 185,106) sua descoberta.

    :   ![Gaebolgs09](images/d/d4/Gaebolgs09.png)
21. Saia do necrotério

    :   ![Gaebolgs Biscuss](images/c/cc/Gaebolgs_Biscuss.png)
22. Fale com Bamph na catedral e receba a [experiência](Experiência.md "Experiência"). (prt\_church 185,106)

    :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
23. Reporte o caso para a Mulher com Aparência. (mjolnir\_01 316,268)

    :   ![Gaebolgs08](images/d/d8/Gaebolgs08.png)
24. Volte até a Historiadora Rodafrian, você pode optar em contar o segredo ou não revelar a verdade. (morocc\_in 45,126)

    :   ![Gaebolgs03](images/3/3b/Gaebolgs03.png)
25. Encerre a missão falando com Bamph novamente. (prt\_church 185,106)

    :   ![Gaebolgs04](images/6/6e/Gaebolgs04.png)
