# Remover Acessório

|  |  |
| --- | --- |
| **Remover Acessório Remover Acessório** | |
| Remover Acessório | |
| **Strip Accessory / Descartar Accesorio** | |
| **ID** | SC\_STRIPACCESSARY (2298) |
| **Tipo** | Ativa |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 12 + (Nv. da habilidade × 3) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0 + 1 seg. |
| **Pós-conjuração** | 1 segundo |
| **Duração** | [50 + (Nv. da hab. × 10)] segundos |
| **Alvo** | Oponente |
| **Alcance** | 3 células |
| **Ícone** | I Remover Acessório |
| **Pré-requisitos** | |
| - [Remover Arma](Remover Arma.md "Remover Arma") [Remover Arma](Remover Arma.md "Remover Arma") Nv. 1 | |

Tenta remover o acessório do oponente, impedindo-o de equipá-lo temporariamente.

Em monstros, reduz a [INT](INT.md "INT"), [SOR](SOR.md "SOR") e [DES](DES.md "DES") em 20%.

| Nv. | Chance | Duração | [SP](SP.md "SP") |
| --- | --- | --- | --- |
| 1 | 14% | 60 segundos | 15 |
| 2 | 16% | 70 segundos | 18 |
| 3 | 18% | 80 segundos | 21 |
| 4 | 20% | 90 segundos | 24 |
| 5 | 22% | 100 segundos | 27 |

## Notas

- Funciona em alvos imobilizados por [Pintar Armadilha](Pintar Armadilha.md "Pintar Armadilha").
- Não funciona contra [Mecânicos](Mecânicos.md "Mecânicos") pilotando um [MECHA](Licença de Pilotagem.md "Licença de Pilotagem").
- Após aprender a habilidade, [Remoção Total](Remoção Total.md "Remoção Total") não inclui os acessórios.
- Não há métodos para evitar a remoção de um acessório.
