# História Perdida

| NQ ico História Perdida | |
| --- | --- |
| Tudo o que acontece no navio Conchiolina deve ser registrado em um relatório por um Arquivista. O problema é que algumas histórias não estão batendo... | |
| Requisitos mínimos | |
| **Nv. de base** | 1 |
| **Tipo** | Normal |
| **Quests anteriores** | [Clube dos Farfelle](Clube dos Farfelle.md "Clube dos Farfelle") ou [Buongiorno](Buongiorno.md "Buongiorno") |
| Premiação | |
| **[EXP base](Experiência de Base.md "Experiência de Base")** | 700 |
| **[EXP classe](Experiência de Classe.md "Experiência de Classe")** | 700 |

1. Dentro do Conchiolina, aceite ajudar o Arquivista.

   :   ![Arquivisty](images/4/43/Arquivisty.png)
2. Procure o Capitão Gamberi.

   :   ![Gamberi](images/8/83/Gamberi.png)
3. Retorne ao Arquivista, ele indicará um prédio chamado Clube dos Aventureiros.
4. Dirija-se até uma casa em [Lasagna](Lasagna.md "Lasagna") e converse com o Chefe Peperocino (lasagna 56,186)

   :   ![ClubeAventureiro](images/5/55/ClubeAventureiro.png)
5. Volte a falar com o Arquivista.
6. Encontre Mill Enair (lasagna 263,156)

   :   ![Millenair](images/5/5c/Millenair.png)
7. De volta ao Conchiolina, reporte a situação ao Arquivista.
   - Você receberá 700 de [EXP](EXP.md "EXP") de base e classe.
8. Agora você pode ler os Diários do Conchiolina.

   :   ![ConchilinaDi](images/0/07/ConchilinaDi.png)
