# Adestrar Dragão

|  |  |
| --- | --- |
| **Adestrar Dragão Adestrar Dragão** | |
| Adestrar Dragão | |
| **Dragon Training / Adiestramiento de Dragones** | |
| **ID** | RK\_DRAGONTRAINING (2007) |
| **Tipo** | Passiva |
| **Níveis** | 5 |
| **Pré-requisitos** | |
| - [Perícia em Montaria](Perícia em Montaria.md "Perícia em Montaria") [Perícia em Montaria](Perícia em Montaria.md "Perícia em Montaria") Nv. 1 | |

Permite usar um Ferus como montaria.

Remove a penalidade de [tamanho](Tamanho.md "Tamanho") ao equipar lanças.

Melhora a [velocidade de movimento](Velocidade de movimento.md "Velocidade de movimento") e o [peso](Peso.md "Peso") máximo.

Aumentar o nível da habilidade recupera a [velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") perdida enquanto monta.

Também aumenta o dano do dragão nas seguintes habilidades:

- [![Sopro do Dragão](images/2/27/Sopro_do_Dragão.png)](Sopro do Dragão.md "Sopro do Dragão") [Sopro do Dragão](Sopro do Dragão.md "Sopro do Dragão")
- [![Bafo do Dragão](images/5/5d/Bafo_do_Dragão.png)](Bafo do Dragão.md "Bafo do Dragão") [Bafo do Dragão](Bafo do Dragão.md "Bafo do Dragão")

| Nv. | [Peso](Peso.md "Peso") | [Velocidade de ataque](Velocidade de ataque.md "Velocidade de ataque") | Dano do Dragão |
| --- | --- | --- | --- |
| 1 | +700 | -20% | 0% |
| 2 | +900 | -15% | +10% |
| 3 | +1.100 | -10% | +20% |
| 4 | +1.300 | -5% | +30% |
| 5 | +1.500 | 0% | +40% |

## Notas

- O bônus de [Perícia com Lança](Perícia com Lança.md "Perícia com Lança") funciona com o Ferus.
- É possível alugar a montaria no Domador de Feras (prontera 130,213)
