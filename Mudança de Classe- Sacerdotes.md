# Mudança de Classe: Sacerdotes

| JQ ico Mudança de Classe: Sacerdotes | |
| --- | --- |
| Requisitos mínimos | |
| **Nv. de base** | Nenhum |
| **Nv. de classe** | 40 |
| **Tipo** | Classe |
| **Classe** | [Noviço](Noviço.md "Noviço") |
| Premiação | |
| **[Itens](Itens.md "Itens")** | 1550.png 1 [Livro](http://www.divine-pride.net/database/item/1550/) ou  1551.png 1 [Bíblia](http://www.divine-pride.net/database/item/1551/) |
| **Recompensa** | Muda a classe para [Sacerdote](Sacerdote.md "Sacerdote") |

1. Inicie a jornada com o Bispo Tomas (prt\_church 16,41)

   :   ![MdCscr01](images/d/d4/MdCscr01.png)
2. Aceitando fazer a peregrinação, o jogador deve seguir para os [NPCs](NPC.md "NPC") abaixo e conversar. Caso tenha o nível de classe 50, não há necessidade de fazer o passo **3**.
3. Fique atento a ordem da peregrinação no seu quest log (alt+u). Se for em um NPC antes do outro, terá que refazer o caminho certo.
   1. Encontre a Irmã Matilda (moc\_fild07 41,355)

      :   ![MdCscr02](images/6/6b/MdCscr02.png)
   2. Localize o Irmão Bartolomeu (prt\_fild00 208,218)

      :   ![MdCscr03](images/a/ad/MdCscr03.png)
   3. Procure o Irmão Simão (prt\_fild03 365,255)

      :   ![MdCscr04](images/a/a6/MdCscr04.png)
4. Fale com Bispo Tomas para ser teleportado para uma sala de testes.
5. Nesta sala, fale com o Irmão Pedro para iniciar seu teste de exorcismo, ele é dividido em 3 partes.

   :   ![MdCscr05](images/3/36/MdCscr05.png)
6. No primeiro quarto, seu objetivo é eliminar todos os [Zumbis](http://www.divine-pride.net/database/monster/1015/) e seguir para o portal ao norte.
7. Em seguida, haverá um corredor com 4 [NPCs](NPC.md "NPC") onde deve se aproximar deles para ativar o diálogo. Cada um fará ofertas diferentes a fim de que você desista do teste, se aceitá-las você fracassa e verá que são propostas mentirosas, mas se recusá-las pode passar para a etapa seguinte.

   :   ![MdCscr07](images/e/ec/MdCscr07.png)
8. A última sala terá mais alguns [monstros](Monstros.md "Monstros"), você não precisa matá-los, basta correr, mas caso não queira arriscar, elimine-os.
9. Concluído o teste, reencontre o Bispo Tomas.
10. Faça um juramento com a Irmã Cecilia, basta responder sim ou não, são perguntas bem simples e fáceis (prt\_church 27,24)

    :   ![MdCscr06](images/8/8b/MdCscr06.png)

| Conjunto de Perguntas | Respostas |
| --- | --- |
| Você vai devotar sua vida a Deus? | Sim |
| Você usará seus poderes apenas para si mesmo? | Não |
| Você vai usar os seus poderes para ajudar os outros? | Sim |
| Você se sacrificaria pela vida de outra pessoa? | Sim |
| Você perturbará pessoas nos campos e nas cidades, repetindo as mesmas palavras várias vezes? | Não |
| Você trará grandes quantidades de monstros para que seus amigos possam matá-los? | Não |
| Você viverá e morrerá pela glória divina | Sim |
| Eu o ouvi, irmão. Por fim, jure por Deus que você respondeu com sinceridade as minhas perguntas. | Eu juro |

:   11. Retorne ao Bispo Tomas para alterar sua classe e receber um livro de acordo com seu nível de classe.

    - Se seu [nível de classe](Nível de classe.md "Nível de classe") estiver entre 40 a 49 quando começou a quest, você receberá 1 [Livro](http://www.divine-pride.net/database/item/1550/).
    - Se seu [nível de classe](Nível de classe.md "Nível de classe") for 50 quando começou a quest, você receberá 1 [Bíblia](http://www.divine-pride.net/database/item/1551/).
