# Tiro Preciso

|  |  |
| --- | --- |
| **Tiro Preciso Tiro Preciso** | |
| Tiro Preciso | |
| **Focused Arrow Strike / Tiro Certero** | |
| **ID** | SN\_SHARPSHOOTING (382) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 14 + (Nv. da habilidade × 2) |
| **[Conjuração](Conjuração.md "Conjuração")** | 0,5 + 0,5 seg. |
| **Pós-conjuração** | 0,5 segundos |
| **Recarga** | 0,15 segundos |
| **Alvo** | Oponente |
| **Alcance** | 11 células |
| **Área** | 5x5 células |
| **Munição** | [Over Arrow](Munições.md "Munições") 1 Flecha |
| **Arma** | Arco |
| **Pré-requisitos** | |
| - [Concentrar](Concentrar.md "Concentrar") [Concentrar](Concentrar.md "Concentrar") Nv. 10 - [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") [Rajada de Flechas](Rajada de Flechas.md "Rajada de Flechas") Nv. 5 | |

Foca a mira no alvo, atirando uma flecha afiada no alvo e em oponentes ao redor.

É possível causar dano crítico com a habilidade.

A chance do ataque crítico é igual ao [CRIT](CRIT.md "CRIT") do usuário.

No momento do ataque, garante [CRIT](CRIT.md "CRIT") +50.

| Nv. | [ATQ](ATQ.md "ATQ") | [SP](SP.md "SP") |
| --- | --- | --- |
| 1 | 600% | 16 |
| 2 | 900% | 18 |
| 3 | 1.200% | 20 |
| 4 | 1.500% | 22 |
| 5 | 1.800% | 24 |

### Fórmula

`Dano = [Dano base × (Nv. de base ÷ 100)]%`
