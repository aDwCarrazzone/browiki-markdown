# Caverna de Buwaya

| SC ico Caverna de Buwaya | |
| --- | --- |
| A lenda de Buwaya conta que qualquer pessoa que for pega pelo monstro ficará preso eternamente dentro de seu baú. | |

| **Tipo** | Instância |

| **[Grupo](Grupo.md "Grupo")** | 1 pessoa ou mais |

| Premiação | |

1. Converse com o Chefe da Guarda (malaya 290,340)

   :   ![Malaya07](images/4/4c/Malaya07.png)
2. Entre na fila da [instância](Instância.md "Instância") falando com o Guarda (ma\_fild02 312,317)

   :   ![Cavernadebuwaya01](images/a/af/Cavernadebuwaya01.png)
3. Entre no portal para acessar o mapa.

   :   ![Cavernadebuwaya02](images/d/d8/Cavernadebuwaya02.png)
4. Seu objetivo é encontrar e eliminar o verdadeiro [Buwaya](http://www.divine-pride.net/database/monster/2319/).
   - Não será uma tarefa fácil, existem [Buwayas](http://www.divine-pride.net/database/monster/2332/) falsos que morrem fácil.
   - O lugar é cheio de [Escravos](http://www.divine-pride.net/database/monster/2330/) e [ovos](http://www.divine-pride.net/database/monster/2329/) que servem para dificultar.
   - As [Algas de Malaya](http://www.divine-pride.net/database/monster/2331/) conjuram [Diminuir Agilidade](Diminuir Agilidade.md "Diminuir Agilidade") no seu [grupo](Grupo.md "Grupo").
   - A cada 60 segundos, o [grupo](Grupo.md "Grupo") todo será enviado para o Baú de Buwaya.
5. No Baú, fale com a Pessoa Sequestrada para invocar dois [Pontos Fracos](http://www.divine-pride.net/database/monster/2333/).

   :   ![Cavernadebuwaya03](images/a/a9/Cavernadebuwaya03.png)

   - Existem dois [NPCs](NPC.md "NPC") no baú, uma senhora que ativa os [Pontos Fracos](http://www.divine-pride.net/database/monster/2333/) e um senhor que fornece um bônus de [ATQ](ATQ.md "ATQ") e [ATQM](ATQM.md "ATQM") por alguns segundos.
   - Dentro do baú, os personagens sofrerão de [envenenamento](Envenenamento.md "Envenenamento") e o [HP](HP.md "HP") será drenado rapidamente.
6. Ao derrotar [Buwaya](http://www.divine-pride.net/database/monster/2319/), retorne ao Chefe da Guarda para receber 3 [Amuletos Menores](http://www.divine-pride.net/database/item/6497/).
   - Você pode trocar seus itens adquiridos por  [Tatuagens](Festival de Pintados.md "Festival de Pintados").
