# Confinamento

|  |  |
| --- | --- |

| **ID** | RG\_CLOSECONFINE (1005) |
| **Tipo** | Ativa |
| **Níveis** | 1 |
| **[SP](SP.md "SP")** | 35 |
| **Pós-conjuração** | 1 segundo |
| **Duração** | 15 segundos |
| **Alvo** | Oponente |
| **Alcance** | 3 células |
| **Ícone** | I Confinamento |
| **Pré-requisitos** | |
| *[Completar Missão](Novas habilidades- Gatunos e evoluções.md#Arruaceiros "Novas habilidades: Gatunos e evoluções")* | |

Algema o oponente junto com você, tornando ambos imóveis por 10 segundos.

Após imobilizados, você ganha [Esquiva](Esquiva.md "Esquiva") +50.

É possível usar [habilidades](Habilidades.md "Habilidades"), [itens](Itens.md "Itens") e atacar durante esse estado.

Apenas um alvo pode estar confinado por vez.

## Notas

- É cancelada se o alvo se distanciar 3 células do usuário, entrar em [Esconderijo](Esconderijo.md "Esconderijo") ou se o usuário usar [Teleporte](Teleporte.md "Teleporte").
- Não é cancelada por se o alvo usar [Teleporte](Teleporte.md "Teleporte") ou se o usuário morrer.
- Não funciona contra [monstros](Monstros.md "Monstros") Chefes.
