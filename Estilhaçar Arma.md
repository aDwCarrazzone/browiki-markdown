# Estilhaçar Arma

|  |  |
| --- | --- |
| **Estilhaçar Arma Estilhaçar Arma** | |
| Estilhaçar Arma | |
| **Weapon Crush / Golpe Desarmador** | |
| **ID** | GC\_WEAPONCRUSH (2030) |
| **Tipo** | Ofensiva |
| **Níveis** | 5 |
| **[SP](SP.md "SP")** | 20 |
| **Pós-conjuração** | 1 segundo |
| **Alvo** | Usuário |
| **Requisito** | [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") |
| **Ícone** | I Remover Arma |
| **Pré-requisitos** | |
| - [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") [Reflexo de Combate](Reflexo de Combate.md "Reflexo de Combate") Nv. 1 | |

Reage à um ataque rapidamente.

Só pode ser usada em estado de [Reflexo](Reflexo de Combate.md "Reflexo de Combate").

Causa dano físico corpo a corpo equivalente a 100% do [ATQ](ATQ.md "ATQ").

Tem chance de remover a arma do atacante e impedi-lo de equipá-la temporariamente.

| --- | --- | --- | --- |
| Personagens | Monstros |
| 1 | 7% | 75 segundos | 90 segundos |
| 2 | 9% | 90 segundos | 120 segundos |
| 3 | 11% | 105 segundos | 150 segundos |
| 4 | 13% | 130 segundos | 170 segundos |
| 5 | 15% | 145 segundos | 200 segundos |

## Notas

- A duração do efeito é baseada na diferença de [DES](DES.md "DES") do alvo e do usuário.
- O efeito é semelhante à [Remover Arma](Remover Arma.md "Remover Arma").
